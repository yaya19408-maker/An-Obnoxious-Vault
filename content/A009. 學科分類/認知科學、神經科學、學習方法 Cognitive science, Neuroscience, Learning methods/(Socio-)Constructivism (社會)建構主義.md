# Socio-constructivism, Constructivism (社會)建構主義
![[(Socio-)Constructivism (社會)建構主義-20250319153237355.webp]]
- 建構主義認識論（constructivist epistemology）
	- 探討見解、結構和建構主義層面的知識本質。
- 建構主義在數學（證明）、建築（俄羅斯的建構主義運動）、心理學（療法）、語言學（建構主義觀點探討語言習得）皆有應用。
- 二十世紀早期，行為主義（behaviorism）和刺激－反應理論被推為學習主流。
	- **帕甫洛夫（Pavlov）帕甫洛夫的狗**
	- **史金納（Burrhus Skinner）獎勵機制**
	- **華生（John Watson）行為主義研究法：觀察法、條件反射、口頭報告、測驗法**
	- （行為主義）有些老師鼓勵快問快答，促進學生自發回答
- （建構主義）老師運用鷹架之類的技巧慢慢建構個人知識理解
- 皮亞傑（Jean Piaget）認知發展/發生認識論
	- 影響學習的兩個社會情境
		- 個人體會或身處的文化（如語言、邏輯、算術系統）
		- 和**博學多聞的他人（more knowledgeable other, MKO）**在社區的互動（尤其是內化語言的涵義）
	- **認知發展階段（Cognitive development stages, Maturation cycle）**
		- 感知動作期sensorimotor/sensory-motor stage (0~2y)
			- 物體恆存性
			- 伸手去摸（reaching, touching）
		-　前運思期pre-operations/preoperational stage (2~7y)
			- 語言發展（linguistic development）
			- 能直覺理解一些簡單流程（intuitive understanding of some simple processes）
		- 具體運思期concrete operations stage (7~11y)
			- 思考抽象物或概念(abstract conceptualization)
			- 形式邏輯(formal logic)
	- **發生認識論（genetic epistemology）**
		- 以認識論為目標和起點，通過生物學的方法論的類比，採用發生學的方法，建構於主客體相互作用活動之上的兒童發展心理，分為同化（assimilation）、調適（accommodation）、平衡（equilibration, equilibrium）三階段
		- 主張認知結構代表身心理的行動，也反映出孩童的智慧和發展階段
		- ![[(Socio-)Constructivism (社會)建構主義-20250319155317405.webp]]
		- **Scheme, schema 基模：**　
			- 基模會分類出主題，例如小孩可能會知道車子能帶他們旅行、外觀是紅色、有獨特的氣味、後座有嬰兒座椅，或在之後知道車子有喇叭、有不一樣的顏色、煞車會有尖銳聲等等新資訊，就會讓車子的基模快速成長，但也有新資訊與現有知識不相容的情況。因此整體分成「在現有基模上加入新資訊（同化）」及「改變現有基模，讓接收到的新證據加進來（調適與平衡）」兩種傾向
			- **基模不只能解釋，還能預測**
				- 如果有人要求你解釋某個東西（what is it），那麼你很可能在之前就已經知道該對象的涵義了。相對而言，遇到不符合（或找不到／不熟悉）的基模，其資訊就不太能理解，甚至誤解。
					- 這也代表有些學生在閱讀時可能不太懂自己不熟的文章，即便是看得懂每個字，也還是看不懂整篇文章想表達甚麼。
			- 學生（通常無意識地）選擇資訊，轉換資訊，開始假設，最後根據參考資料與依賴內部認知結構（即基模網絡）做決定，讓學生的基模網絡能在已知範圍延伸探索
		- **Assimilation 同化：**　收集、分類新基模，並確保不會和已有基模相悖
		- **Accommodation 調適**：調整、變換已有基模，好讓新的基模加進來
		- **Equilibration 平衡**: 沒有相悖的基模，有三種後續情況
			- 第一， 我們自己很滿意現在思路的平衡狀態
			- 第二， 萬一思路有相悖或不完整，就會陷入不平衡狀態（認知衝突）
			- 第三， 去除相悖，重新達到平衡
- 維高斯基（Lev Vygotsky）**近側發展區間ZPD**、鷹架（scaffolding）
	- **Zone of Proximal Development (ZPD) 近側發展區間；可能發展區**
		- 從完全不懂，到與專家學習，再學著自己獨力完成
		- 老師是學習夥伴，或者說「閱歷豐富的他者」（more knowledgeable other, MKO），負責帶孩子參與學習挑戰
		- 學生會在數位學習遊戲中嘗試解決問題和進行策略及推理的任務，而軟體給出的提示、老師或同儕會給予幫助，讓學生成為ZPD的中心焦點
		- 劃分潛力發展，大人手把手教你解決問題，或是和更有能力的同儕共同合作；而在心理發展方面，在別人幫助下的事也會比自己做來得更有幫助、更正確。
		- **Scaffolding 鷹架**
			- 老師往往都是提供鷹架的人（scaffolder），但 scaffolder 並不都是老師或大人
			- 專家幫助學生學習
				- 學生角度來看，活動可以當作完成學業的漸進式教學步驟
					- 例如數位教學能依優先次序劃分階級並構建起來，從學習目標的具體經驗逐步帶入抽象概念
				- 老師角度來看，鷹架就是學習經驗設計的隱喻，再配上適當的教學介入，好讓學生隨時都能應對課程
					- 老師旨在為學生現況提供適合他們的任務（下階段的墊腳石）
					- 學生就能運用充足資源和活動建構學習，精進自身成就、技能、理解力
			- Vygotsky其實沒有提到鷹架，而是Jerome Bruner與David Wood、Gail Ross在1976年提出「鷹架」概念
	- **Social Learning 社會學習**
		- 維高斯基認為社會互動是認知和智慧成功發展的基本要素
			- 個體之間的心理過程（人與人之間，**interpsychological**: between people, between-persons）
			- 個體內的心理過程（一個人內心中，**intrapsychological:** inside the child, within-persons）
			- 兩者皆可提升自主注意力、邏輯記憶和形成概念。
	- **Vygotskian Learning 維高斯基式學習**
		- 無論是在教室、公園、家裡、任何地方，只要與他人或「知識淵博的他者」（more knowledgeable other）互動就很有可能是維高斯基式學習
- 社會建構主義思想三大層面（現實、知識、學習）
	- **現實：** 社會建構主義者視現實為個體創造出來的實體。我們看見的現實很相似，但某些方面會因經驗和互動而不同
	- **知識：** 藉由社會與文化構建。有時和現實一樣會有所差異，也有可能是個人先前知識或經驗互動下的誤構（misconstruction）
	- **學習：** 在社會建構主義者眼裡是一種社會過程，且不單單只關乎於個人，也不僅只是處於被動。個體參與社會活動及重複性感官輸入，才能有效且永續的學習。
- **范葛拉士費德(von Glasersfeld) 的激進建構主義（radical constructivism）**
	- 知識由具有認知的主體（即學習者）構建，而不是被動接收（知識自己蓋，不是等著別人給）
		- 游家政（1998）建構主義取向課程設計的評析
			- von Glasersfeld (1990) 的第一原則: **知識並非被動接受,而是主動建構**的。
	- 認知功能會適應環境並組織經驗世界，而不是發現本體論現實（認知是用來適應環境，讓自己體驗人生，打造專屬自己定義的世界，不是為了挖掘不可能認知到的絕對客觀現實）
		- 游家政（1998）建構主義取向課程設計的評析
			- von Glasersfeld (1990) 的第二原則: **認知功能具有調適性以促成平衡與發展**
			- von Glasersfeld (1990) 的第三原則: **認知為主體提供組織經驗世界的服務**，而非發現客觀的本質性實體
	- 范葛拉士費德的極端建構主義（radical constructivism）有時被視為社會建構主義（socio-constructivism）的對立面，但極端派和社會派都認可個體用感官輸入構建主觀現實，且不能直接感知客觀現實
		- 極端派：個體自己創建世界觀
		- 社會派：個體參與社會互動創建世界觀
- 布魯納（Jerome Bruner）：主張學習是在現有知識基礎上主動構建新想法和概念的過程
	- **Spiral curriculum 螺旋式課程**
		- 根據布魯納(J.S. Bruner)的教育理論而設計的〔人的研究〕(MACOS)最具代表性，合乎課程組織的繼續性(continuity，跳脫螺旋)和順序性(sequence，由淺入深)等規準。
		- 概念在螺旋式課程組織下，難度逐漸加深，幫助學生在循序漸進中逐漸熟悉這些概念的意義，且可以透過學習活動，幫助學生作模擬思考、推理假設與重新思考以發現概念之意義，提升學生之認知發展能力，並理解人性特質的基本問題。
		- 特徵
			- 合乎學科結構之邏輯順序，結構嚴謹
			- 合乎學生之認知結構與認知發展過程
			- 合乎課程組織的繼續性與順序性之規準
			- 提供明確的概念架構做為教師「探究教學」之依據，並配合詳細的教師教學指引，可以做精密的教學設計，使教學順序流暢
			- 提供具體的實物或教育玩具，配合學生認知發展階段，設計吸引學生學習興趣的活動，滿足學生好奇探究之學習欲望，合乎學生學習之需求
- 班度拉（Bandura）的社會認知理論（social cognitive theory）
	- 集體能動性（collective agency）
		- 是「人類能動性」一詞的延伸，主張人是集體生物，常以共同信仰和期望共事並提升生活品質
	- 班度拉認為人就是在日復一日的實體觀察中學到東西的
- 情境式學習 （situated learning）
	- 屬於實際學習（authentic learning）
	- 認知學徒制（cognitive apprenticeship, apprenticeship model）
		- 學習都發生在高度社會化的情境下
	- 社會（social）
		- 兩位或多位一起以一種或多種方式互動，並鼓勵互動的參與者一起學習
	- 學徒制（apprenticeship）
		- 在師傅（等同於知識淵博的他者/老師/大人）身旁觀察他的示範、指導，耳濡目染（emulate），並學習新技能
	- 社會互動並不是以學習者為中心，學習者只是經認可的旁觀者（legitimate peripheral participant）
