# 1. 普通查釋義 (辭典)
先直接用Cambridge Dictionary中英解釋，如果沒有結果就用Merriam Webster Dictionary英英解釋。
- [Cambridge Dictionary](https://dictionary.cambridge.org/zht/)
- [Merriam Webster](https://www.merriam-webster.com/)
- [Oxford Dictionary](https://www.oxfordlearnersdictionaries.com/)
- [Britannica Dictionary](https://www.britannica.com/dictionary)
通常這兩者就很夠用了。Cambridge還有中查英功能，好用。
# 2. 查找同義詞 (索引典)
現在Cambridge在慢慢增加同義詞功能了，Merriam Webster也有內建Thesaurus
- [Thesaurus](https://www.thesaurus.com/)
- [Collins](https://www.collinsdictionary.com/#google_vignette)
- 在Google搜尋: (想找的字) synonym
# 3. 查找同源詞 (語源學)
想找同個來源或同字根的詞就用:
- [Etymonline](https://www.etymonline.com/)
- [The American Heritage Dictionary](https://www.ahdictionary.com/word/indoeurop.html)
- [Proto-Indo-European Lexicon](http://pielexicon.hum.helsinki.fi/)
# 4. 查找字根 (Affixes)
想找同字根就用
- [Eng.tw (似乎停止更新了)(注意中文解釋需要再次搜索查證，很多都是機翻)](https://root.eng.tw/)
- [Word Maker](https://wordmaker.info/result.php)
- [字根文件](https://drive.google.com/file/d/0B9RAVul6SWKEN2FkYzM1MTAtYzQ2Yy00NjY3LWI5MGYtODg0NTE0NzIyZjg4/view?hl=zh_TW&resourcekey=0-RcItYIZ5XYnLsAINbHfTHA)
# 5. 查找生僻字
想找不主動找的話這輩子可能都看不到的字就用
- [Wiktionary (搜索選項記得要把ZH換成EN)](https://www.wiktionary.org/)
	- 也可以找字根，更可以進階找原始印歐語詞根 (PIE-root，語源)
	- 由於是生僻字，如果注釋不好懂的話可能也只能用翻譯硬著頭皮解讀，除非找得到文獻
- 找文獻的方法可以嘗試在Google搜索欄打上
	- =="關鍵字"(你覺得它中文翻譯可能會有的字)==
- 不能說是字根，但是是"明顯的造字規律"的話，可以用:
	- 繼續Word Maker找找看
	- [The Free Dictionary](https://www.thefreedictionary.com/)
	- Google搜尋: "==words start with X end with X=="
- 用Merriam Webster找，沒有就是只有Wiktionary有了
- Oxford通常也有，但要付費
- [The Dictionary of Obscure Sorrows](https://www.dictionaryofobscuresorrows.com/)
	- 通常是用不太到的美詞
