## 調節技能、五項主要技能及其子技能
![[Web psy-20241201155148211.webp]]
<center>圖2 資訊問題解決技能的拆解示意圖</center>

1. **define the information problem 定義問題: 查找資料的首要步驟**
	1. read task 閱讀任務
	2. concretize problem 將問題變具體
		1. 制定所需資料
	3. activate prior knowledge 啟用先備知識
		1. 鎖定必須查找的資料為何
	4. clarify task requirements 闡明任務需求
		1. 說清楚任務需要的條件
2. **search information 搜尋資訊: 選擇重要或有趣的資料來源並簡明概述**
	1. Internet skills 具備一定的網路操作技能
	2. derive search terms 生出搜索詞
	3. judge search result 評比搜尋結果: 根據
		1. quality 資料的品質
		2. relevance 是否契合相關
		3. reliability 是否可以信任
3. **scan information 掃視資料: 掃視、評估並決定該資料要不要和現有問題做連結**
	1. Internet skills 具備一定的網路操作技能
	2. scan sites 掃視網頁
	3. elaborate on content 深入探討內容
	4. judge scanned information 評比掃視過後的資料 (是否能跟先備知識或其他筆資料相結合)
4. **process information 處理資料:**
	1. read text 詳讀內文
	2. elaborate on content 深入探討內容
		1. analyzing 分析資料
		2. selecting 選取資料
		3. structuring 建構資料
	3. judge processed information 評估處理後的資料
		1. usefulness 有沒有幫助/好不好用
		2. quality 資料品質
5. **organize and present information 組織並呈現資料: 此研究是讓學生寫一段論述**
	1. formulate problem 制定問題
		1. 要根據產品/作品的要求清楚制定問題
	2. outline the product 概述產品/作品
		1. 要決定好主要樣式或配置
	3. structure the product 建構產品/作品
	4. formulate text 撰寫內文
	5. elaborate on content 深化呈現資料的內容
6. regulation 調節: 上述技能都執行過後，調節技能就會接續執行
	1. orient on the task 專注於任務的導向
	2. monitor and steer their performance 監督和引導自身表現
	3. testing (evaluate product and process) 測試 (評估產物和進程)

## 在不同技能上的時間投入
![[Web psy-20241201174211561.webp]]
<center>表2 專家與新手在主技能上投入的時間對比</center>

1. 專家完成任務所花費的平均時間為91.7分鐘 (SD=6.46)，意味著專家會盡可能利用時間檢視任務成果是否完善且深沉
2. 新手完成任務所花費的平均時間為71.6分鐘 (SD=20.06)
3. *p* < 0.10
4. 兩者差距尤其體現在最一開始的定義問題(define problem)方面 (專家: M=3.32，新手: M=0.62，*p* < 0.05)，也間接揭示了任務結果在深度上的差異

## 執行主要技能和子技能的頻率
![[Web psy-20241201175449998.webp]]
- 表3以平均數和標準差呈現專家與新手使用的主要技能或子技能的次數
- 數據分析方式為單因子變異數分析 (one-way ANOVAs)
1. **define problem 定義問題: 新手和專家都僅使用一次**
	1. read task 閱讀任務: 比起新手(M=1.60)，專家會更常檢視任務 (M=3.00，*p* < 0.05)
	2. activate prior knowledge 使用先備知識: 比起新手 (M=0.00)，專家會更常運用先備知識 (M=0.60，*p* < 0.05)
2. **search information 搜尋資料: 沒有明顯差異**
	1. 鑒於「使用網路的技能 (Internet skills)」在搜尋資料和掃視資料兩項主要技能中都很重要，那使用:
		1. 搜尋引擎、
		2. URL輸入欄、
		3. 選擇網頁、
		4. 運用連結、
		5. 網頁內建搜索和
		6. 切換視窗等動作的頻率就是決定專家與新手搜尋資料差異的關鍵要素了。
3. **scan information 掃視資料: 兩項子技能有明顯差異**
	1. 專家 (M= 5.00) 比新手 (M= 2.00) 更常專注於深入鑽研 (elaborate) 內容，*p* < 0.05
	2. 專家 (M= 5.00) 比新手 (M= 2.00) 更常專注於評估 (judge) 資料，評估標準包含資料品質、相關程度和可信度，而三項標準在此研究並沒有進行質性分析，*p* < 0.10
4. **process information 處理資料: 專家比新手更常處理資料**，*p* < 0.05
	1. 銜接先前的表2，專家(12%)會比新手(2.68%)用更多時間在此主要技能。
		1. <font color="#ff0000">(因為原句描述太長，可以考慮口述或不講)</font> 此研究也推論有些專家會把整個處理資料的時間分段 (more short moments)，反覆咀嚼處理過後的資訊。而新手則反之。
	2. 其他三個子技能的差異並沒有那麼明顯
5. **organize and present information 組織並呈現資料: 專家更常制定問題**
	1. formulate problem 制定問題: 專家 (M= 1.00) 比新手 (M= 0.00) 更常組織問題，*p* < 0.05
