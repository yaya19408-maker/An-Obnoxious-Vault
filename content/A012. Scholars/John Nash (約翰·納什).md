# John Nash (約翰·納什)

#scholars

## Game Theory (賽局理論) (c. 1950)

- **Nash Equilibrium (納什均衡):** 在非合作賽局中，一種策略組合，其中沒有任何參與者能透過單方面改變策略而獲得更好的結果。
- **Non-cooperative Game (非合作賽局):** 參與者無法達成具約束力的協議，必須獨立做出決策的賽局模型。
- **Strategy (策略):** 參與者在賽局中根據規則與預期對手行為所制定的完整行動計畫。
- **Payoff (支付):** 賽局結束時，參與者因其選擇的策略組合而獲得的效用或回報。
- **Rationality (理性):** 賽局理論中的基本假設，指參與者會選擇能最大化自身利益的策略。
- **Zero-sum Game (零和賽局):** 一方參與者的收益必然來自另一方參與者的損失，所有參與者總收益為零。
- **Non-zero-sum Game (非零和賽局):** 參與者的總收益不固定為零，存在合作共贏或兩敗俱傷的可能性。
- **Prisoner's Dilemma (囚徒困境):** 一種經典的非零和賽局，說明為何即使合作對整體有利，個體理性仍可能導致不合作的結果。

## Bargaining and Other Concepts (議價與其他概念) (c. 1950s)

- **Bargaining Problem (議價問題):** 尋求兩個或多個合作方如何分配合作所產生利益的數學模型。
- **Nash Bargaining Solution (納什議價解):** 為議價問題提供的一個獨特解，滿足特定公理，如對稱性與帕雷托效率。
- **Cooperative Game (合作賽局):** 參與者可以形成具約束力的聯盟或協議，以協調策略並共同行動的賽局。
- **Hex (六貫棋):** 納什發明的一種抽象策略棋盤遊戲，與拓撲學的布勞威爾不動點定理相關。

## Pure Mathematics (純粹數學) (c. 1950s-1960s)

- **Nash Embedding Theorems (納什嵌入定理):** 證明任何黎曼流形都可以等距嵌入到某個歐幾里得空間中。
- **Riemannian Manifold (黎曼流形):** 一個在每點的切空間上都定義了內積的光滑流形，可用於測量距離與角度。
- **Partial Differential Equations (偏微分方程):** 納什在研究非線性拋物線型與橢圓型偏微分方程的解的存在性與正則性方面做出貢獻。
- **De Giorgi–Nash Theorem (德喬治-納什定理):** 關於某類橢圓型與拋物線型偏微分方程解的霍爾德連續性的結果。
