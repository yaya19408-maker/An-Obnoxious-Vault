# Maurice Halbwachs (莫里斯·哈布瓦赫)

#scholars

## Les cadres sociaux de la mémoire (記憶的社會框架) (1925)

- **Collective Memory (集體記憶)**： 一種由特定社會群體共享、傳承與建構的過去再現，其存在超越個人，並為群體成員提供認同感。
- **Social Frameworks of Memory (記憶的社會框架)**： 指語言、時間觀念、空間定位等由社會提供的結構，個人必須透過這些框架才能組織、定位與回憶其記憶。
- **Memory Reconstruction (記憶的重構)**： 記憶並非對過去的精確複製，而是在當下根據群體的需求與關懷，不斷被重新組織與詮釋的過程。
- **Groups as Bearers of Memory (作為記憶載體的群體)**： 家庭、宗教社群、社會階級等社會團體是集體記憶的主要承載者與傳遞者，個人的記憶依附於這些群體而存在。
- **Individual Memory (個人記憶)**： 表面上屬於個人的記憶，其實質內容與形式皆受到其所屬社會群體的框架所塑造，無法脫離社會脈絡獨立存在。

## La Topographie légendaire des Évangiles en Terre Sainte (福音故事在聖地的傳說地形學) (1941)

- **Memory and Space (記憶與空間)**： 集體記憶需要附著於具體的物質空間才能穩定與存續，空間成為記憶的外在支撐與觸發點。
- **Localization of Memory (記憶的定位)**： 將抽象的記憶與特定的地理位置連結起來的過程，使空間具有象徵意義，並成為群體回憶的憑藉。
- **Commemorative Space (紀念性空間)**： 透過儀式或標記，將特定地點轉化為承載集體記憶的場所，用以鞏固群體的共同過去與認同。
- **Legendary Topography (傳說地形學)**： 指涉群體記憶如何投射到地理空間上，形成一套與歷史事實有所出入，但對群體深具意義的空間敘事。
- **Evocation of Memory (記憶的喚起)**： 物理空間的在場，例如朝聖地的景物，能有效地喚起與之相關的集體記憶，強化信徒的共同感受。

## La mémoire collective (論集體記憶) (1950)

- **History and Memory (歷史與記憶)**： 歷史是學者建構的、統一且客觀的記錄；記憶是活在群體中的、多元且主觀的經驗。歷史的起點是記憶的終點。
- **Autobiographical Memory (自傳式記憶)**： 個人對自身經歷的回憶，但此種回憶的敘述方式與重點選擇，仍受到社會框架的影響，以便與他人溝通。
- **Internal Memory/Dream (內在記憶/夢)**： 純粹屬於個體、無法或難以與他人分享的心理狀態，因其缺乏共通的社會框架，故不屬於集體記憶的範疇。
- **Forgetting (遺忘)**： 社會性的遺忘源於承載特定記憶的社會群體消失，或其賴以維生的社會框架發生了根本性變動。
- **Stream of Collective Thought (集體思想之流)**： 指一個社會中持續流動的觀念與表象，集體記憶是其中的一部分，並與社會當下的思想保持互動。
