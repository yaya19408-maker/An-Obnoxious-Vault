# Ludwig von Bertalanffy (路德維希·馮·貝塔郎非)

#scholars

## Problems of Life (生命的問題) (1952)

- **Organismic Biology (有機體生物學)**：一種將生物視為整合的有機整體的觀點，強調生命現象的組織性與動態性，反對將生命簡化為物理化學反應的機械論。
- **Holism (整體論)**：主張系統應被視為一個不可分割的整體來理解。整體的特性無法單純由其組成部分的特性總和來解釋。
- **Teleology (目的論)**：探討自然現象中的目的或目標導向行為。在系統理論中，這體現為系統朝向特定狀態發展的內在趨勢，例如成長或穩定。
- **Anabolism (合成代謝)**：生物體內將較簡單的物質合成為更複雜物質的過程，此過程需要消耗能量。
- **Catabolism (分解代謝)**：生物體內將複雜的有機分子分解為較簡單分子的過程，此過程會釋放能量。

## General System Theory (一般系統論) (1968)

- **General System Theory (一般系統論)**：一個跨學科的理論框架，旨在尋找適用於所有類型系統的普遍原則與模型，無論其本質是物理、生物或社會性的。
- **System (系統)**：由一群相互關聯、相互作用的組件構成，形成一個具有特定功能的整合性整體。
- **Open System (開放系統)**：與環境持續交換物質、能量和資訊的系統。生物體與社會組織是典型的開放系統。
- **Closed System (封閉系統)**：與外界環境完全隔離，不存在物質、能量或資訊交換的理論模型系統。
- **Isomorphism (同構性)**：指在不同學科領域的系統中，存在著結構、原理或運作模式上的相似性。一般系統論的目標之一就是發現這些同構現象。
- **Equifinality (等結果性)**：開放系統的特性，指系統可以從不同的初始狀態及透過不同的途徑，達到相同的最終狀態。
- **Multifinality (多結果性)**：與等結果性相對的概念，指相似的初始條件可能導致系統發展出截然不同的最終狀態。
- **Subsystem (子系統)**：一個較大系統中的組成部分，本身也具備系統的特性，並為母系統的功能服務。
- **Suprasystem (超系統)**：包含特定系統的更大系統或環境，它為系統提供運作的脈絡並與之互動。
- **Boundary (邊界)**：區分系統與其外部環境的界線，控制著系統與環境之間的交換活動。
- **Input (輸入)**：系統從其環境中獲取的物質、能量或資訊。
- **Output (輸出)**：系統處理後，釋放到其環境中的物質、能量或資訊。
- **Throughput (流轉)**：系統內部將輸入轉換為輸出的過程與活動。
- **Feedback (回饋)**：系統的輸出訊息被重新導回作為輸入，從而影響系統後續行為的機制。
- **Homeostasis (恆定性)**：開放系統透過內部調節機制，維持內部環境相對穩定的狀態，以應對外部環境的變動。
- **Differentiation (分化)**：系統從較簡單、同質的狀態，朝向更複雜、各部分功能專門化的方向發展的趨勢。
- **Hierarchy (層級)**：系統由不同層次的子系統所組成，形成一個由下至上或由簡至繁的結構排列。
