# John Sweller (約翰斯威勒)

- **Cognitive Load Theory (認知負荷理論)：** 探討人類工作記憶處理資訊的方式與限制，指出學習效率取決於如何分配與管理有限的認知資源。教學設計的品質取決於是否將工作記憶的極限納入考量。
- **Intrinsic Cognitive Load (內在認知負荷)：** 學習材料本身固有的困難度與複雜性。這項指標取決於學習內容的性質以及學習者先備知識的程度，教師無法透過改變教學方式來消除。
- **Extraneous Cognitive Load (外外認知負荷)：** 由不良的教學設計、多餘資訊或不當呈現方式所引發的額外認知負擔。它會分散學習者的注意力，消耗可用於處理重要訊息的工作記憶空間。
- **Germane Cognitive Load (增生認知負荷)：** 學習者為了理解新知、建立基模並將資訊轉移至長期記憶所投入的有效心智努力。在減少外在負荷後，釋放出的認知資源可轉用於此，以提升學習成效。
- **Working Memory (工作記憶)：** 人類大腦用以暫時儲存與處理新資訊的機制。其容量極度有限，通常只能同時處理數個資訊區塊，過多資訊會導致學習中斷與理解困難。
- **Long-term Memory (長期記憶)：** 儲存基模與龐大知識庫的系統。人類能力的增長建立在將新資訊有效從工作記憶轉移並鞏固至長期記憶之中。
- **Schema (基模)：** 存儲在長期記憶中，用以組織與理解訊息的知識結構。學習的本質即為建構與完善基模，使我們在面對未來情境時能迅速提取與運用。
- **Human Cognitive Architecture (人類認知架構)：** 斯威勒理論的基礎，指人類大腦處理、儲存與提取資訊的生理機制，包含工作記憶與長期記憶的互動關係。
- **Worked Example Effect (已解範例效應)：** 學習者研讀附有完整解題步驟的範例，比直接進行問題解決能獲得更好的學習成果。此方法能降低初學者的外在認知負荷，讓他們專注於理解解題邏輯。
- **Expertise Reversal Effect (專業反轉效應)：** 適合初學者的教學方法應用在具備高先備知識的專家身上時，會造成干擾並降低學習效能。隨著專業度提升，過多的指引會成為一種冗餘負荷。
- **Guidance Fading Effect (引導漸隱效應)：** 隨著學習者的知識與技能逐漸成熟，教學者應逐步減少明確的步驟指導，過渡至讓學生獨立解決問題，以配合其改變的認知需求。
- **Redundancy Effect (冗餘效應)：** 同時以多種格式呈現相同的資訊會迫使學習者處理重複訊息，從而消耗額外的工作記憶資源並阻礙學習。
- **Split Attention Effect (分散注意力效應)：** 當學習者必須在多個分開的資訊來源之間來回切換以拼湊意義時，會產生額外的認知負擔。將文字整合進圖表中有助於提升學習效率。
- **Coherency Effect (連貫效應)：** 移除教材中與學習目標無關的背景音樂、裝飾性圖片或多餘文字，能避免分散學習者的有限注意力，進而提升對核心概念的理解與記憶。
- **Means Ends Analysis (手段目的分析)：** 初學者常使用的一種解題策略，透過不斷比對目前狀態與目標狀態的差距來尋找解法。這種策略會佔用大量工作記憶，導致學習者無暇觀察並記住解題規則。
- **Goal Free Effect (無目標效應)：** 在解題訓練中，不提供單一明確的最終目標，要求學習者盡可能計算出所有未知數。這能減少手段目的分析帶來的負荷，促使學習者專注於問題的結構與運算規則。
- **Biologically Primary Knowledge (生物性初階知識)：** 人類在演化過程中自然習得的技能，例如母語對話、辨識人臉或社會互動。這些知識的獲取通常不需要自覺的努力或正式教學。
- **Biologically Secondary Knowledge (生物性次階知識)：** 社會與文化發展中產生的專業技能，例如閱讀、寫作或高階數學。人類缺乏自然習得這些知識的本能，必須仰賴外部的明確教學才能建立相關基模。
