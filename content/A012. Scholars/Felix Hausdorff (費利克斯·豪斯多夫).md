# Felix Hausdorff (費利克斯·豪斯多夫)

#scholars

## Grundzüge der Mengenlehre 集合論基礎 (1914)

- Topological Space (拓撲空間)：一個集合與其上定義的一組開集結構，此結構決定了點的鄰近性，並允許定義連續性與收斂性等概念。
- Hausdorff Space (豪斯多夫空間)：一種拓撲空間，其中任意兩個相異的點都存在互不相交的鄰域。此性質確保點的分離性，是分析學中常見的空間類型。
- Neighborhood (鄰域)：拓撲空間中某個點的鄰域是一個包含該點的集合，且該點位於集合的內部，不處於邊界上。此為拓撲學的基本構成單元。
- Open Set (開集)：一個集合，其內部的每一個點都有一個完全包含在該集合內的鄰域。拓撲結構即由所有開集的集合所定義。
- Closed Set (閉集)：一個集合，其補集為開集。等價地說，一個包含其所有極限點的集合就是閉集。
- Limit Point (極限點)：若一個點的每個鄰域都含有集合中異於該點本身的至少一個點，則稱此點為該集合的極限點。
- Closure (閉包)：一個集合與其所有極限點的聯集，構成包含原集合的最小閉集。

## Measure Theory and Dimension 測度理論與維度 (c. 1918)

- Metric Space (度量空間)：一個定義了點與點之間距離函數的集合，此距離函數需滿足非負性、對稱性與三角不等式。
- Hausdorff Measure (豪斯多夫測度)：一種將長度、面積、體積等概念推廣到任意度量空間子集的方法，可用於測量極不規則的集合。
- Hausdorff Dimension (豪斯多夫維度)：利用豪斯多夫測度定義的一種維度，其值可以是任意非負實數。它能精確刻畫碎形等複雜幾何對象的維度。
- Fractal (碎形)：一種具有自相似性質的幾何結構，其豪斯多夫維度通常大於其拓撲維度，呈現出分數維度的特徵。
- Outer Measure (外測度)：在定義正式的測度之前，對一個空間的所有子集賦予一個值的函數，是構建豪斯多夫測度的第一步。

## Advanced Set Theory and Analysis 高等集合論與分析 (c. 1900s–1930s)

- Partially Ordered Set (偏序集)：一個集合及其上定義的一個二元關係，該關係滿足自反性、反對稱性與遞移性。
- Hausdorff Maximal Principle (豪斯多夫極大原理)：在任何偏序集中，每個全序子集都被包含在一個極大全序子集之中。此原理等價於選擇公理。
- Totally Ordered Set (全序集)：一種偏序集，其中任意兩個元素都是可以比較大小的。
- Baker-Campbell-Hausdorff Formula (BCH公式)：在李代數理論中，用於表達兩個矩陣指數乘積的對數的公式，豪斯多夫對其級數表示有重要貢獻。
- Hausdorff-Young Inequality (豪斯多夫-楊氏不等式)：傅立葉分析中的一個重要不等式，建立了函數在Lp空間上的範數與其傅立葉變換在Lq空間上範數之間的關係。
