# A. R. Radcliffe-Brown (阿弗烈·芮克里夫-布朗)

#scholars

## General Theoretical Framework (一般理論框架) (c. 1920s-1950s)

- **Structural Functionalism (結構功能論)**： 將社會視為一個有機系統，其中各個社會制度與實踐的功能是為了維持整體社會結構的穩定與存續。
- **Social Structure (社會結構)**： 個體之間所有社會關係的總和，構成一個由制度化角色與地位組成的複雜網絡。
- **Social Function (社會功能)**： 任何經常性的社會活動對於維持社會結構的持續性所做的貢獻。
- **Social Physiology (社會生理學)**： 研究社會生活如何運作的學科，關注法律、宗教、經濟等制度如何維持社會系統的生命過程。
- **Social Morphology (社會形態學)**： 對社會進行分類、比較與描述其結構類型，如同生物學的形態分類。
- **Synchronic Analysis (共時性分析)**： 在特定時間點上研究社會系統，分析其各組成部分的相互關係，忽略歷史變遷。
- **Comparative Method (比較方法)**： 透過比較不同社會的結構與制度，發現人類社會運作的一般性法則。

## *Structure and Function in Primitive Society* (原始社會的結構與功能) (1952)

- **Joking Relationship (詼諧關係)**： 兩個體或群體間一種制度化的友好戲謔與嘲弄關係，用以調節潛在的社會敵意或緊張。
- **Avoidance Relationship (迴避關係)**： 兩個體或群體間必須保持社交距離與尊敬的關係，藉由減少接觸來預防潛在的衝突。
- **Kinship System (親屬系統)**： 一套界定個人權利、義務與行為模式的社會關係網絡，是許多社會的結構基礎。
- **Social Value (社會價值)**： 某事物對社會群體所具有的重要性，此重要性透過集體的興趣與情感表現出來。
- **Sentiment (情感)**： 在社會結構中被組織與制度化的情感，例如尊敬或親密，是維持社會關係的重要元素。
- **Eunomia (社會和諧)**： 指一個社會系統處於健康、整合且運作良好的狀態，各部分協調一致。
- **Dysnomia (社會失調)**： 指社會系統中存在衝突與不一致，導致結構性矛盾的狀態。

## *The Andaman Islanders* (安達曼島民) (1922)

- **Ritual (儀式)**： 具有象徵意義的集體活動，其功能在於表達與強化維繫社會團結所必需的集體情感。
- **Social Integration (社會整合)**： 社會各部分凝聚成一個穩定整體的過程或狀態，通常透過共同價值與制度達成。
- **Ceremonial Custom (儀式習俗)**： 透過集體儀式活動來表達和強化群體情感，維持社會團結的習俗。
- **Social Process (社會過程)**： 構成社會生活的無數行動與互動的總和，這些過程共同維繫著社會結構。
