# Gregory Bateson (葛瑞格里·貝特森)
#scholars #cybernetics #psychology

# Steps to an Ecology of Mind 邁向心智生態學 (1972)
- **Nested Learning (嵌套式學習)：** 貝特森提出的階層式學習理論。他認為學習並非單一層次的資訊獲取，而是由一組相互包含的邏輯層次所構成，每一層級的變革都會重新定義其下方層級的運作邏輯。

- **Learning 0 (零級學習)：** 個體對特定的刺激產生固定的、重複的反應。這是一種不具備修正能力的資訊接收，反應方式在過程中不發生任何改變。
- **Learning I (一級學習)：** 在特定的選擇集合中，透過試錯來修正錯誤並選擇適當的反應。這是最常見的學習形式，即在既定的規則框架內改進行為。
- **Learning II (二級學習 / 學習如何學習)：** 針對「一級學習」過程的變革。個體學會了如何學習，掌握了特定情境的底層規律與類別，並能將其遷移至相似的情境中。這涉及對脈絡（Context）的識別。
- **Learning III (三級學習)：** 針對「二級學習」過程的變革。個體意識到自己過往所有的學習慣性與性格基調（Ethos），並能對其進行徹底的重塑與否定。這通常與深層的範式轉移、宗教體驗或精神崩潰後的重生有關。
- **Learning IV (四級學習)：** 超越單一個體的層次，涉及物種演化、大規模歷史變遷或生態系統整體的基因/文化轉變。
- **Deuterolearning (次級學習)：** 貝特森早期用於描述「學習如何學習」的術語，強調學習者在習得具體知識的同時，也無意識地習得了一套關於世界如何運作的抽象邏輯。
- **Double Bind (雙重束縛)：** 一種具破壞性的溝通情境。主體同時接收到兩個相互矛盾的指令，且無法逃離該情境，亦無法對此矛盾進行後設溝通。這是導致二級學習扭曲與精神分裂症發生的關鍵機制。

- **Metacommunication (後設溝通)：** 關於溝通的溝通。主體透過語言或非語言訊號來界定當下對話的性質（例如：這是在開玩笑）。這對於理解嵌套學習中的「脈絡」至關重要。
- **Schismogenesis (裂變產生)：** 社會群體中互動模式的持續分化過程。包含「對稱性裂變」（競爭加劇）與「互補性裂變」（主從關係極化），若缺乏反饋調節將導致系統崩潰。
- **Cybernetic Explanation (控制論式解釋)：** 貝特森的核心方法論。他主張解釋現象不應尋找「正向驅力」，而應尋找系統中的「限制因素」與「反饋迴路」，即探究為何某些狀態沒有發生。

# Mind and Nature 心智與自然 (1979)
- **Pattern that Connects (連結的模式)：** 存在於所有生物與心智現象中的抽象元模式。貝特森認為，理解自然界最深層的結構在於識別不同系統間共享的組織模式，而非研究單一的物質組件。
- **Stochastic Process (隨機歷程)：** 發展與演化的基本機制，結合了隨機的變異（產生新可能）與非隨機的選擇（過濾無效可能）。
- **Abductive Reasoning (溯因推理 / 類比思考)：** 在不同領域之間尋找形式上的相似性，藉此洞察深層系統規律的思考方式。這是貝特森用以連結生物學、人類學與心理學的關鍵工具。
- **Logical Types (邏輯類型)：** 借用羅素的數學概念，指出概念的「類別」與該類別下的「成員」屬於不同的邏輯層次，混淆兩者會導致知覺錯誤與心理病理。
- **Information as a Difference (資訊即差異)：** 貝特森對資訊的著名定義：「一種能造成後續差異的差異 (A difference which makes a difference)」。
- **Flexibility (靈活性)：** 系統中尚未被承諾、可用於應對未來環境變動的自由度。過度優化或資源飽和會降低系統的靈活性，導致其在面臨突發挑戰時斷裂。
