# George Gabriel Stokes (喬治·加布里埃爾·斯托克斯)
#scholars #fluid_mechanics #physics
- **Stokes' Law (斯托克斯定律)：** 描述球體在黏性流體中以極低雷諾數運動時所受到的摩擦阻力方程式。阻力與流體黏滯係數、球體半徑及運動速度成正比。
- **Navier-Stokes Equations (納維-斯托克斯方程式)：** 描述黏性流體運動的非線性偏微分方程式。斯托克斯推導出此方程式，正確識別出內摩擦力的物理本質。
- **Stokes' Theorem (斯托克斯定理)：** 向量微積分中的基本定理，將向量場在曲面上的旋度面積分轉化為沿著該曲面邊界曲線的線積分。
- **Stokes Shift (斯托克斯位移)：** 光譜學中，物質吸收光子後發射出的螢光或拉曼散射光，其波長大於吸收光波長的現象。這反映分子在激發態與基態之間的能量耗散。
- **Stokes Drift (斯托克斯漂移)：** 流體力學中，非線性波浪傳播時導致流體質點產生沿著波浪傳播方向的淨位移現象。質點的運動軌跡呈現未完全封閉的橢圓。
- **Stokes Flow (斯托克斯流)：** 流體運動狀態，特徵是極低的雷諾數。在這種流動中，黏滯力遠大於慣性力，慣性項在納維-斯托克斯方程式中可以被忽略。
- **Stokes Parameters (斯托克斯參數)：** 一組四個數值，用於在光學中完整描述電磁波的極化狀態，包含總光強、線極化程度與圓極化程度。
- **Stokes Boundary Layer (斯托克斯邊界層)：** 流體平行於固體邊界作週期性振盪運動時，在邊界附近形成的一層薄流體層。流體速度在此層內從零急遽變化至自由流速度。
- **Stokes Stream Function (斯托克斯流函數)：** 在軸對稱的三維不可壓縮流體中，用於描述流體運動軌跡與速度場的純量函數。其等值線代表流線。
- **Stokes Phenomenon (斯托克斯現象)：** 漸近分析中的數學特徵。當一個解析函數的漸近展開式在複數平面上跨越某些特定射線時，其係數會發生不連續跳躍。
- **Stokes Number (斯托克斯數)：** 無因次量，用於描述懸浮在流體中的微粒運動跟隨流體流動的能力。它定義為微粒的弛豫時間與流體的特徵時間之比。
- **Stokes Radius (斯托克斯半徑)：** 溶質分子在溶液中擴散時，被視為一個等效的硬球體所具有的半徑。它受溶劑化效應影響，與分子的實際物理尺寸有差異。
- **Stokes' Hypothesis (斯托克斯假設)：** 在流體力學中，假設單原子氣體的體積黏度為零。流體的熱力學壓力等於三個法向應力的平均值。
- **Stokes Wave (斯托克斯波)：** 描述水面非線性重力波的數學模型。波峰較尖銳而波谷較平緩，比線性波理論更能精確描述大振幅的海浪。
- **Anti-Stokes Line (反斯托克斯線)：** 在拉曼光譜學中，散射光頻率高於入射光頻率的譜線。這發生在物質分子從較高的振動能階躍遷至較低的振動能階時。
- **Stokes Line (斯托克斯線)：** 在拉曼光譜學中，散射光頻率低於入射光頻率的譜線。相對於反斯托克斯線，這在室溫下更為常見與強烈。
