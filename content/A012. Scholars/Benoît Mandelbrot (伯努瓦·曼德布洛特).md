# Benoît Mandelbrot (伯努瓦·曼德布洛特)

#scholars

## Foundational Concepts & Early Work (c. 1960s-1970s)

- Roughness (粗糙性)：自然界物體普遍存在的非平滑、不規則表面特徵，曼德布洛特主張此為常態而非例外，無法以傳統歐幾里得幾何描述。
- The Coastline Paradox (海岸線悖論)：指出海岸線的測量長度會隨著測量單位縮小而趨近於無限長，揭示自然形狀的尺度依賴性。
- Self-Similarity (自相似性)：物體的局部是整體縮小後的樣貌，此特徵在不同尺度上重複出現，是碎形結構的核心屬性。
- Noah Effect (諾亞效應)：指系統中突然發生、幅度極大的不連續變化，用以描述金融市場或自然現象中由長尾分佈引起的劇烈事件。
- Joseph Effect (約瑟夫效應)：指系統中存在長期持續的趨勢或狀態，即使出現波動，仍會持續一段時間，反映了系統的記憶性與持久性。

## The Fractal Geometry of Nature 大自然的碎形幾何 (1982)

- Fractal (碎形)：一個幾何物件，其特徵是具有在所有尺度下皆呈現複雜細節的自相似結構，此術語由曼德布洛特所創。
- Fractional Dimension (碎形維度)：一種非整數維度，用以量化碎形佔據空間的程度與其複雜性，數值越高代表圖案越曲折、越能填滿空間。
- Euclidean Geometry (歐幾里得幾何)：處理點、線、圓等理想化平滑形狀的傳統幾何學，曼德布洛特認為其無法有效模擬自然界的複雜形態。
- Scaling (尺度縮放)：描述一個系統的統計特性在不同觀察尺度下保持不變或以可預測方式變化的概念，是碎形分析的基礎。
- Natural Patterns (自然圖案)：曼德布洛特運用碎形幾何來模擬與解釋雲朵、山脈、星系團、血管網絡等自然界中不規則但具有結構的形態。

## Complex Dynamics & The Mandelbrot Set 複變動態學與曼德布洛特集 (c. 1980)

- Mandelbrot Set (曼德布洛特集)：在複數平面上，由所有使得迭代函數 z → z² + c 的結果保持有界的起始點 c 所構成的集合，其邊界形成無限複雜的碎形。
- Complex Plane (複數平面)：由一條實數軸與一條虛數軸所構成的二維座標系統，為曼德布洛特集與朱利亞集的運算與視覺化空間。
- Iteration (迭代)：將一個函數的輸出結果作為其下一次計算的輸入，並重複此過程的數學操作，是生成碎形圖案的根本機制。
- Julia Set (朱利亞集)：與曼德布洛特集相關的碎形集合。對於一個固定的複數 c，所有在迭代函數 z → z² + c 作用下保持有界的起始點 z 所形成的集合。
- Escape-time Algorithm (逃逸時間演算法)：一種用於繪製曼德布洛特集與朱利亞集的電腦程式方法，透過計算複數點需要迭代多少次才會超出特定邊界來為其著色。
