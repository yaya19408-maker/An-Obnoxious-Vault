# Gottlob Frege (弗雷格)

#scholars #philosophy #logic #math

## Begriffsschrift (概念文字) (1879)

- **Begriffsschrift (概念文字)**： 弗雷格所創造的一種形式化的邏輯語言，其全稱為純粹思維的形式語言，模仿算術語言而建構。此系統旨在以無歧義的符號精確地表達命題的邏輯結構與推論關係，從而擺脫自然語言的模糊性與不精確性。概念文字採用二維的空間佈局，不同於傳統線性排列的語言。一個判斷的表達是透過垂直與水平線條的組合來呈現其內在的邏輯關聯，例如條件句與量化關係。此系統的目標是提供一個工具，使得科學證明的有效性可以被嚴格地檢驗，確保結論是純粹基於邏輯法則從前提中推導出來，無需求助於直覺或經驗內容。它是現代謂詞邏輯的直接前身，引入了函項、變元、量詞等核心元素。

- **Function and Argument (函項與論元)**： 弗雷格將數學中的函項與論元概念應用於邏輯分析，以此取代了亞里斯多德以來的主詞-謂詞邏輯。在他看來，一個命題可以被分解為一個飽和的部分（論元，即對象）與一個不飽和的部分（函項，即概念）。例如，在語句「凱撒征服了高盧」中，「凱撒」是一個專名，指稱一個對象，作為論元。而「( ) 征服了高盧」則是一個函項，它本身是不完整的，需要一個論元來填充其空位，從而形成一個完整的思想（命題）。這種分析方式使得邏輯能夠處理具有多個論元的關係（例如，「羅馬位於巴黎與維也納之間」），這是傳統主謂邏輯難以處理的。函項的「不飽和性」成為弗雷格哲學中的一個核心隱喻，用以區分概念與對象。

- **Quantification (量化)**： 弗雷格革命性地發展了量化理論，為現代邏輯奠定了基礎。他將量詞（如「所有」、「有些」）理解為二階概念，即關於概念的概念。例如，「所有哺乳動物都是溫血的」這個命題，其邏輯形式並非將「所有」附加於主詞「哺乳動物」，而是表達了兩個概念之間的關係：凡是具有「哺乳動物」這個概念的對象，也都具有「溫血的」這個概念。他在概念文字中設計了專門的符號來表示全稱量化。透過一個帶有凹槽的橫線與變元，他可以精確地表達量詞的作用域。此方法成功地解決了涉及多重量詞的命題的歧義問題，例如「每個男孩都喜歡一個女孩」，這個系統可以清楚地區分是同一個女孩還是不同的女孩。

- **Judgment Stroke (判斷豎筆)**： 在概念文字的符號系統中，判斷豎筆是位於整個公式最左邊的一條垂直短線。它的功能是表示對其後續內容的斷定或斷言。弗雷格嚴格區分一個命題的內容（一個可被判斷為真或假的思想）與對該內容所做的判斷行為。例如，一個命題可以僅僅被設想、提問，或者作為一個更複雜命題的一部分出現，而在這些情況下它都未被斷定。只有當判斷豎筆出現時，才表示作者主張其後的命題內容為真。這個符號將邏輯內容與言說行為中的斷定力量分開，體現了弗雷格對語用學與語義學的早期區分。

- **Content Stroke (內容橫筆)**： 內容橫筆是緊接在判斷豎筆之後（如果有的話）的一條水平線。它的作用是指示其後的符號表達了一個可以被判斷的內容，即一個思想或命題。當內容橫筆單獨使用時，它將其後的命題轉化為一個指稱其真值的名稱。例如，若 `P` 代表一個命題，則 `─ P` 就表示「`P` 的真值」。弗雷格認為，所有真命題都指稱同一個對象，即「真」（The True），而所有假命題都指稱另一個對象，即「假」（The False）。內容橫筆是其符號系統中將命題語義與真值連結起來的關鍵部分。

## Die Grundlagen der Arithmetik (算術的基礎) (1884)

- **Logicism (邏輯主義)**： 邏輯主義是弗雷格在數學哲學中倡導的核心綱領，其核心論題是：算術是邏輯的一個分支，所有算術真理都可以僅僅透過邏輯公理與定義來證明。他試圖表明，算術概念（如數、加法）可以被還原為純粹的邏輯概念，而算術定理（如 2+2=4）則是邏輯定律的推論。這個計畫旨在為算術提供一個絕對確定的基礎，使其獨立於任何經驗直觀或心理活動。弗雷格的邏輯主義直接挑戰了康德的觀點，康德認為算術命題是綜合先驗的，需要依賴於時間的純粹直觀。弗雷格則力圖證明它們是分析的，其真確性完全蘊含在概念的定義之中。

- **Frege's Definition of Number (弗雷格的數的定義)**： 為了執行邏輯主義計畫，弗雷格提出了一個關於自然數的純邏輯定義。他首先定義了概念之間的「等數」關係（equinumerosity），即如果兩個概念下的對象可以建立一一對應，那麼這兩個概念就是等數的。接著，他定義「屬於概念 F 的數」為「所有與概念 F 等數的概念的外延」。基於這個框架，他定義了具體的數。數字 0 被定義為「與『不等於自身』這個概念等數」的概念的外延（因為沒有任何東西不等於自身，所以這個概念下有零個對象）。數字 1 則被定義為「與『等於 0』這個概念等數」的概念的外延（因為只有數字 0 這個對象等於 0）。以此類推，每一個自然數 n+1 都可以透過其前驅數 n 來定義，從而將整個自然數序列建立在純粹的邏輯基礎之上。

- **The Context Principle (脈絡原則)**： 弗雷格在《算術的基礎》中提出了三個基本原則，脈絡原則是其中之一。它主張：「一個詞的意義必須在命題的脈絡中去探求，而非在孤立的狀態下」。這個原則有幾個重要的哲學功能。首先，它反對心理主義，即認為詞的意義是其在心中喚起的觀念。弗雷格堅持意義的客觀性，而命題的真值條件提供了這種客觀性的基礎。其次，它使得處理抽象對象（如數）的指稱問題成為可能。我們或許無法直觀地把握「數」這個對象，但只要包含數詞的句子具有確定的真值條件，我們就可以承認數詞具有客觀的指稱。這個原則將分析的焦點從孤立的詞彙轉向完整的命題，是分析哲學的一個根本方法論轉變。

- **Analytic/Synthetic Distinction (分析/綜合區分)**： 弗雷格重新定義了康德的分析與綜合判斷的區分。在康德那裡，這個區分是基於概念內容的：一個分析判斷是謂詞概念已包含在主詞概念之中。弗雷格則將其轉化為一個關於證明基礎的區分。一個命題是分析的，如果它的證明僅僅依賴於普遍的邏輯定律和定義。一個命題是綜合的，如果它的證明需要援引特定科學領域的真理（非邏輯、非定義的真理）。此外，他還區分了先驗/後驗，這是關於判斷的知識論證成來源的區分。弗雷格的目標是證明算術命題是分析先驗的，以此反駁康德認為它們是綜合先驗的觀點。

## Über Sinn und Bedeutung (論意義與意謂) (1892)

- **Sense and Reference (Sinn und Bedeutung) (意義與意謂)**： 這是弗雷格在語言哲學中最著名的貢獻。他主張一個符號（如專名、詞語或整個句子）具有兩個層面的語義內容：意謂（Bedeutung，通常譯為 reference 或 denotation）和意義（Sinn，通常譯為 sense）。意謂是該符號在世界上所指稱的對象或實體。意義則是呈現該意謂的方式，是把握意謂的途徑。一個意謂可以有多種不同的意義來呈現它。這個區分解決了許多語義學上的難題。

- **Morning Star/Evening Star (晨星/暮星)**： 這是弗雷格用來說明意義與意謂區分的經典例子。「晨星」和「暮星」這兩個詞語具有相同的意謂，它們都指稱行星金星。然而，它們的意義是不同的。「晨星」的意義是「在早晨的天空中出現的明亮星體」，而「暮星」的意義是「在傍晚的天空中出現的明亮星體」。由於意義不同，等式「晨星等於暮星」就具有認知價值。這個發現是天文學的一項成就，它告訴我們兩個不同的呈現方式實際上指向同一個對象。如果詞語的語義內容只有意謂，那麼這個等式就等同於「a=a」，變成一個無資訊量的同義反覆。

- **Cognitive Significance (認知價值)**： 認知價值問題是弗雷格提出意義與意謂區分的動機之一。問題是：為什麼形式為「a=b」的等式會有時具有重大的知識價值，而形式為「a=a」的等式卻總是平凡的？弗雷格的解答是，認知價值來自於意義層面。一個「a=a」的陳述是平凡的，因為詞語「a」的意義與其自身相同。而一個「a=b」的陳述可能具有認知價值，因為「a」和「b」雖然具有相同的意謂，但它們可能具有不同的意義。認識到這兩個不同的意義（呈現方式）導向同一個意謂，就構成了新的知識。因此，認知價值根植於我們把握到了一個對象的不同呈現方式之間的同一性。

- **Indirect Reference (間接意謂)**： 弗雷格觀察到，在某些語言脈絡中，詞語的意謂會發生變化。這些脈絡通常是引語或表達命題態度的句子，例如「哥白尼相信地球繞著太陽轉」。在這樣的間接言談中，嵌入的子句「地球繞著太陽轉」的意謂不再是其通常的真值（真或假）。如果其意謂仍是真值，那麼根據替換原則，我們可以將任何具有相同真值的子句替換進去而保持整個句子的真值不變，但這顯然是錯的。弗雷格的解決方案是，在這種間接脈絡中，詞語的意謂是其通常的意義。因此，在上述例子中，「地球繞著太陽轉」這個子句的意謂是它所表達的思想（其通常的意義），而不是它的真值。這種現象揭示了語言的層次性與複雜性。

## Grundgesetze der Arithmetik (算術的基本法則) (1893/1903)

- **Basic Law V (基本法則 V)**： 這是弗雷格在其邏輯主義計畫的集大成之作《算術的基本法則》中引入的一條核心公理。該法則旨在形式化地連接概念與對象。它聲稱：兩個函項的「值域」（course-of-values）是同一的，若且唯若這兩個函項對於每一個相同的論元都產生相同的值。更簡潔地說，它等同於一個更普遍的原則，即「概念 F 的外延等於概念 G 的外延，若且唯若對於所有對象 x，Fx 與 Gx 的真值相同」。這個法則至關重要，因為它允許弗雷格從關於概念的陳述過渡到關於對象（即概念的外延，如集合或數）的陳述。數被定義為概念的外延，因此基本法則 V 提供了將數作為邏輯對象來處理的基礎。

- **Russell's Paradox (羅素悖論)**： 1902年，伯特蘭·羅素在審查弗雷格的《算術的基本法則》時發現了一個致命的悖論，並將其告知弗雷格。這個悖論直接源於弗雷格的基本法則 V。羅素構想了一個概念：「是一個不屬於自身的集合的集合」（或在弗雷格的術語中，「是一個不落在其自身下的概念的外延」）。讓 R 代表這個集合。現在提問：R 是否屬於它自身？情況一：如果 R 屬於 R，那麼根據 R 的定義，R 必須滿足「不屬於自身的集合」這個屬性，所以 R 不屬於 R。這是一個矛盾。情況二：如果 R 不屬於 R，那麼 R 就滿足了「不屬於自身的集合」這個屬性，根據 R 的定義，R 就應該屬於 R。這又是一個矛盾。由於無論哪種情況都會導致矛盾，這證明了弗雷格的邏輯系統，特別是基本法則 V，是不一致的。這個悖論宣告了弗雷格最初的邏輯主義計畫的失敗。

## Core Philosophical Distinctions (核心哲學區分)

- **Concept and Object (概念與對象)**： 這是弗雷格本體論中的一個根本區分。對象是飽和的、獨立的實體，可以由專名指稱。例如，「月亮」、「數字2」都是對象。概念則是不飽和的、具有預測性的實體，其本質就像一個有空位的函項。例如，「( ) 是行星」就代表一個概念。概念不能被專名直接指稱；當我們試圖這樣做時（例如，說「『是行星』是一個概念」），我們實際上是在談論一個代理對象，而非概念本身。概念的本性是在判斷中被應用於對象之上。一個對象可以「落在」一個概念之下，例如，月亮這個對象落在「是地球的衛星」這個概念之下。這個區分是絕對的，一個東西要嘛是概念，要嘛是對象，不可能兩者都是。

- **Third Realm (第三領域)**： 弗雷格為了確保思想和真理的客觀性，提出了一個本體論上的「第三領域」。第一領域是物理世界，由具體的、可感知的對象構成。第二領域是內在的、主觀的心理世界，由個人的觀念、感覺和心理狀態構成。第三領域則是一個客觀、非物質、非心理的領域，其中居住著思想（命題的客觀內容）、意義以及數這樣的抽象實體。第三領域中的存在物是永恆不變的，它們不為任何人所創造，而是被人類心靈所「把握」或「領會」。例如，畢達哥拉斯定理這個思想，在畢達哥拉斯發現它之前就已存在於第三領域，並且對於任何能夠理解它的人來說，它都是同一個思想。這個學說旨在對抗心理主義，確保邏輯和數學的客觀性與普遍性。
