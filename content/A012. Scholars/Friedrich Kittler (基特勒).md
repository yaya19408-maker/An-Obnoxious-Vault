- **Aufschreibesysteme / Discourse Networks (寫入系統 / 論述網絡)**：指涉一個特定文化中，用於選擇、儲存、處理與傳輸相關數據的技術與體制網絡。這些網絡決定了什麼是可以被談論的、以及知識如何被生產。
	- **Discourse Network 1800 (1800年的論述網絡)**：以「母親的口（Mother's Mouth）」與詮釋學為核心。這是一個意義具備連續性的時代，強調精神、理解與作者的聲音。字母被視為靈魂的載體，閱讀是為了理解意義。
    - **Discourse Network 1900 (1900年的論述網絡)**：以心理物理學（Psychophysics）與數據的離散性為核心。意義的連續性破裂，數據流被技術拆分。不再追求理解，而是關注生理刺激、測量與數據處理。
- **Media Archaeology (媒介考古學)**：反對將媒介視為意義的載體，主張深入研究硬體、線路與技術標準的物質層面。媒介決定了我們的處境。
- **Media Determinism (媒介決定論)**：基特勒的名言「媒介決定了我們的處境（Media determine our situation）」。媒介並非人類使用的工具，而是媒介設定了標準與框架，人類的主體性（Subjectivity）是由這些技術標準所建構的副產品。
	- **The So-called Man (所謂的人)**：人類並非歷史的主宰。「人」只是一種在特定技術條件下產生的生理-心理裝置。隨著媒介改變，「人」的概念也隨之解體。
    - **Post-Hermeneutics (後詮釋學)**：反對探討文本背後的「深層意義」或「精神」。主張只需分析文本表面的物質性、信號傳輸的物理路徑與技術標準。
- **Gramophone, Film, Typewriter (留聲機、電影、打字機)**：1900年代技術對人類感官數據的「分流（Differentiation）」。統一的書寫文字體系崩解，分化為三種處理不同拉岡式（Lacanian）心理寄存器的技術。
	- **Gramophone / The Real (留聲機 / 真實界)**：留聲機記錄聲波的物理痕跡，包含噪音、錯誤與無意識的聲音。它捕捉了不被象徵符號過濾的生理「真實」，證明了混沌與隨機性的存在。
    - **Film / The Imaginary (電影 / 想像界)**：電影通過視覺暫留技術創造了連續運動的幻覺。它對應鏡像階段，處理幻影、替身（Doppelgänger）與視覺的自戀機制，將破碎的身體重新統整為虛構的完整影像。
    - **Typewriter / The Symbolic (打字機 / 象徵界)**：打字機將寫作從手寫的連續流動（與身體相連）轉變為離散的、標準化的符號敲擊。它剝離了作者的個性，將語言還原為純粹的符號操作，並引發了性別分工（女性打字員的出現）。
- **War and Media (戰爭與媒介)**：媒介歷史本質上是軍事歷史。通訊技術的發展動力來自戰爭中對速度、加密與攔截的需求，娛樂只是軍事技術的副產品（濫用）。
	- **Optical Media (光學媒介)**：光學技術（如探照燈、雷達、攝影機）的歷史就是追蹤、瞄準與摧毀的歷史。電影攝影機與機槍共享相同的機械邏輯（連續進片/連續射擊）與術語（Shoot）。
	- **Blitzkrieg (閃電戰)**：二戰中的閃電戰是摩托化與無線電通訊結合的產物，它依賴於實時的數據處理與指令傳輸，而非傳統的戰略地圖。
- **Media Materiality (媒介物質性)**：反對將媒介視為內容的載體（Container），強調媒介本身的物理屬性（電壓、矽晶片、光纖）才是本體論上的現實。
	- **There is No Software (沒有軟體)**：軟體只是一種為了讓人類能夠操作機器而設計的邏輯抽象層與混淆機制。在最底層，只有電壓差異與硬體的可編程性。軟體掩蓋了機器的物理真相。
     - **Protected Mode (保護模式)**：現代處理器的保護模式是一種權力結構，它禁止使用者（甚至程式設計師）直接存取硬體核心，從而建立了一種技術上的階級與保密制度。
- **Time Axis Manipulation (時間軸操作)**：技術媒介（如錄音帶、膠卷）允許人類首次在物理上操縱時間軸（倒轉、暫停、加速）。這改變了人類對記憶與歷史的感知，使得過去可以被技術性地重現與編輯。
	- **Frequency vs. Amplitude (頻率與振幅)**：技術媒介將感官數據轉化為可測量的物理量（頻率與振幅），使得感官可以被數學化地處理與合成。
- **Signal-to-Noise Ratio (訊噪比)**：通訊理論的核心。任何傳輸都伴隨著噪音。媒介的歷史就是不斷對抗噪音、提升訊噪比的歷史，但噪音（真實界）永遠無法被完全消除。
- **Musik und Mathematik (音樂與數學)**：基特勒晚期轉向古希臘研究，認為音樂與數學同源。音樂是人類最早的數位系統，因為它基於整數比（畢達哥拉斯音律）。
	- **The Alphabet and Vowels (字母表與元音)**：希臘字母表的發明是一次技術革命，因為它首次引入了元音符號，使得語言的聲音、音樂的音符與數學的數字可以在同一個符號系統中被記錄。
    - **Sirens (賽壬)**：神話中的賽壬代表了前文字時代、未被編碼的純粹聲音誘惑，奧德修斯將自己綁在桅杆上是為了在聽見聲音的同時保持象徵秩序（理性/文字）的控制。
- **Recursive Loops (遞歸迴路)**：在電腦時代，媒介可以自我指涉、自我生成。電腦是通用圖靈機，可以模擬所有其他媒介，使得媒介之間的界線消失，統一於二進位代碼中。
- **Dracula as Data Processing (德古拉作為數據處理)**：基特勒對《德古拉》小說的經典解讀。他認為吸血鬼並非恐怖怪物，而是代表舊式口傳文化的「數據流」。擊敗德古拉的並非大蒜，而是現代化的辦公室技術（打字機、留聲機、速記法）所建立的數據標準化與官僚系統。
- **The Schreber Case (舒瑞伯案例)**：重新詮釋佛洛伊德著名的舒瑞伯法官案例。舒瑞伯的幻覺（如「神經語言」、「神蹟射線」）並非精神錯亂，而是對當時新興媒介（電話、無線電）運作邏輯的精確描述。精神病是人體神經系統遭受新媒介技術入侵與重寫時的反應。
- **Rock Music and Military Equipment (搖滾樂與軍事設備)**：流行音樂（特別是搖滾樂）的聲音特質來自於對二戰軍事聲學設備的「誤用（Misuse）」。電吉他的回授（Feedback）與擴大機技術，源自於無線電通訊與信號放大技術。搖滾樂將軍事上的「噪音」轉化為藝術上的「信號」。
    - **Brain Damage (腦損傷)**：基特勒深入分析 Pink Floyd 的歌曲《Brain Damage》，指出其錄音技術如何利用立體聲道模擬精神分裂症聽到的聲音，證明現代音樂如何重塑聽覺生理學。
- **Silicon Nature (矽的自然性)**：反對將「自然」與「科技」對立。矽晶片僅僅是純化的沙子（二氧化矽），因此電腦運算本質上是礦物界的一種物理過程。科技並非反自然，而是自然界（礦物）通過自我揭示而展現的一種形式。
- **The City as Medium (城市作為媒介)**：城市規劃本身是一種信息傳輸系統。街道與建築的布局決定了人流與資訊流的軌跡，如同電路板上的線路。古代城市的城牆是防禦技術，現代城市的網絡則是通訊技術。
- **Love as Media Effect (作為媒介效應的愛情)**：浪漫愛（Romantic Love）並非人類本能，而是「1800年論述網絡」的產物。當母親教導孩子將字母視為精神載體時，創造了對「靈魂伴侶」與內在性的渴望。愛情的表達完全依賴於當時的情書、文學技術與郵政系統標準。
- **Universities as Media Systems (大學作為媒介系統)**：大學並非純粹的知識殿堂，而是一種規範知識生產與傳輸的媒介體制。講座（Lecture）是口傳媒介，論文是書寫媒介，考核制度則是標準化的數據過濾器。
- **Code is Law (代碼即法律)**：雖然此概念常被歸於 Lessig，但基特勒從更底層指出，電腦代碼（Code）取代了自然法與人類法律，成為控制行為的絕對規則。在數位環境中，不可能做出代碼邏輯不允許的動作，技術限制成為了絕對的法律限制。
- **God as Switch (上帝作為開關)**：基特勒晚期的神學技術觀。他認為神聖性存在於硬體的邏輯門與開關之中。存在即是開關的閉合（ON/OFF），二進位的數位邏輯反映了宇宙本體論的結構，上帝是最早的系統工程師。
- **Zentralperspektive / Linear Perspective (單點透視 / 線性透視)**：透視法不僅是藝術技巧，更是現代「光學媒介」的前身。它是一種早期的軍事與測繪技術，將世界數學化為幾何坐標。
    - **Camera Obscura (暗箱)**：暗箱是攝影機與電腦圖學的技術祖先。它透過針孔（幾何光學）將三維世界壓平為二維平面，訓練了人類的眼睛去適應「機器視覺」，為後來影像的機械複製鋪平了道路。
    - **Aiming (瞄準)**：透視法的視覺中心（Vanishing Point）在軍事上對應準心。畫家的視線與槍手的視線在幾何結構上是同構的，都是為了控制與捕獲遠處的物體。
- **Urheberrecht / Authorship (著作權 / 作者身分)**：作者並非天才的創造者，而是印刷技術下的功能性產物。
    - **Protector of Style (風格的保護者)**：在1800年論述網絡中，為了區分大規模印刷的書籍，「作者」的概念被發明出來以保護「風格」（精神），而印刷商則擁有「物質」（紙張與墨水）。版權法是為了處理物質載體過剩而設計的法律虛構。
    - **Functional Equivalent (功能等價物)**：當文字處理器（Word Processor）出現後，作者的功能被軟體指令取代，作者已死，變成了操作鍵盤的輸入終端。
- **Computer Graphics / Ray Tracing (電腦圖學 / 光線追蹤)**：基特勒晚期對數位影像的分析。他認為光學鏡頭（Lens）時代已經結束，現在進入了數學計算光線的時代。
    - **Physics Simulation (物理模擬)**：現代電腦影像不再是對現實的「紀錄」（如照片），而是基於物理方程式（菲涅耳方程、光照模型）的「計算」。影像脫離了與外部世界的索引（Indexical）關係，成為純粹的數學運算結果。
    - **Eye of God (上帝之眼)**：光線追蹤技術模擬了上帝創造光線的過程，視點不再受限於攝影機的物理位置，可以任意穿梭，這是視覺的絕對權力化。
- **The Imitation Game / Turing Test (模仿遊戲 / 圖靈測試)**：基特勒對艾倫·圖靈（Alan Turing）的激進解讀。
    - **Gender Abolition (性別的廢除)**：圖靈測試最初是關於區分男人與女人（性別模仿），後來才變成區分人與機器。電腦的出現意味著性別差異在資訊處理層面上變得無意義，只有數據處理能力的差異。
    - **Intelligence as Simulation (智慧即模擬)**：智慧不再是內在的靈魂特質，而是「能夠成功欺騙觀察者」的技術能力。如果機器能完美模擬人類的輸出，它就是人（或者說人就是機器）。
- **World War II & V2 Rocket (二戰與V2火箭)**：受到托馬斯·品欽（Thomas Pynchon）《雷虹》的影響，基特勒將V2火箭視為現代技術的轉捩點。
    - **Ballistics to Guidance (從彈道學到導航)**：傳統火砲依賴發射前的計算（彈道學），V2火箭引入了飛行中的自動修正（回饋迴路）。這是控制論（Cybernetics）的實體化，武器變成了有「感知」與「修正」能力的自動機。
    - **Integration (積分)**：火箭的導航路徑本質上是對重力與空氣阻力的實時積分運算。戰爭推動了微積分從紙面理論轉向實時運算硬體。
- **Standardization / DIN (標準化 / 德國工業標準)**：現代世界的基礎不是政治契約，而是工業標準（如DIN A4紙張、電壓標準）。
    - **Interoperability (互操作性)**：媒介要形成網絡，必須先標準化。A4紙張的大小決定了官僚機構的檔案櫃尺寸、信封大小與打字機的滾筒寬度。標準是隱形的暴政，規範了所有物質交流的格式。
- **Cryptography (密碼學)**：書寫的歷史實質上是加密與解密的歷史。
    - **Secret Writing (秘密書寫)**：通訊技術總是優先服務於軍事加密需求。網際網路（Internet）源於軍方對抗核戰的分布式網絡（ARPANET）。公開的通訊只是加密通訊技術「解密」後的民用殘餘。
- **Pynchon’s Paranoia (品欽式的偏執)**：基特勒極度推崇美國作家品欽，認為他的小說準確地描繪了技術網絡如何構成一種巨大的、人類無法理解的陰謀（Paranoia）。
    - **Circuit Diagram (電路圖)**：文學作品在電子時代不應被視為敘事，而應被解讀為社會技術權力的電路圖。角色只是電流在電路中流動的節點。
- **The Trace vs. The Sign (痕跡 vs. 符號)**：區分模擬媒介（Analog）與數字媒介（Digital）的關鍵本體論差異。
    - **Trace (痕跡)**：留聲機與照片紀錄的是物理的「痕跡」（如唱片溝槽對應聲波振動），它們與現實有物理接觸（連續性）。
    - **Sign (符號)**：電腦與打字機處理的是「符號」（0與1、字母），它們與現實沒有物理接觸，只是任意的編碼（離散性）。數位化意味著我們失去了與真實界（The Real）的物理連結，只剩下數學模型。
