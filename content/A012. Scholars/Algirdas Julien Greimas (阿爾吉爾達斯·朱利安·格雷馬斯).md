# Algirdas Julien Greimas (阿爾吉爾達斯·朱利安·格雷馬斯)

#scholars #semiotics #linguistics

## Sémantique structurale: Recherche de méthode 結構語義學：方法論探討 (1966)

- **Elementary Structure of Signification (意義的基本結構)**： 此為格雷馬斯理論的哲學與邏輯起點，主張意義並非源於事物本身，而是源於關係與差異。任何可被感知的意義單元，其存在都預設了對立面的存在。例如，理解「高」的概念，必然同時潛在地理解「低」的概念。意義的產生來自於對兩個對立項之間差異的辨識。此結構是所有意義組織的最底層邏輯，它不處理具體的詞彙或符號，只處理純粹的關係性差異。這個二元對立的思維，深受索緒爾與雅各布森的結構語言學影響，但格雷馬斯將其從語言學層面提升至普遍的意義生成層面，為後續的符號方陣奠定基礎。

- **Semiotic Square (符號方陣 / 矩陣)**： 這是意義基本結構的視覺化與邏輯化模型，用以闡明一個給定概念所能展開的完整語義宇宙。它從一對基礎的對立項（S1與S2）出發，例如「生命」與「死亡」。接著，透過邏輯上的矛盾操作，生成兩個次對立項（non-S2與non-S1），即「非死亡」與「非生命」。此方陣包含三種核心關係：S1與S2之間的「對立關係」（contrary）；S1與non-S1之間，以及S2與non-S2之間的「矛盾關係」（contradiction）；S1與non-S2之間，以及S2與non-S1之間的「蘊含或互補關係」（implication/complementarity）。這個看似簡單的方陣，能夠系統化地展現一個文化或文本中，圍繞某一核心價值所建構的複雜意義網絡，並預測可能出現的敘事路徑與意識形態立場。例如，一個故事的發展可以在「生命」、「死亡」、「非死亡」（如昏迷、瀕死狀態）與「非生命」（如機器人、鬼魂）這四個語義位置之間移動。

- **Isotopy (同位素 / 等位性)**： 此概念借用自化學，用以描述文本中語義連貫性的基礎。Isotopy 指的是在一個話語段落中，反覆出現的某些語義特徵（semes），這些特徵像一條看不見的線，將不同的詞彙或句子串連起來，形成一個統一的解讀層面。例如，在一個關於金融的段落中，「牛市」、「熊市」、「流動性」、「泡沫」等詞彙共享一個或多個共同的語義特徵（如「經濟」、「市場」、「價值」），從而建立了一個金融的同位素層面，讓讀者能夠順暢地理解。一個複雜的文本可以同時包含多個同位素層面，例如一首詩可能同時存在愛情、戰爭與自然的同位素，當讀者在不同層面之間切換時，便產生了文本的豐富性與多義性。Isotopy是分析文本如何維持其內部一致性，以及如何產生歧義的關鍵工具。

## Du sens: Essais sémiotiques 意義論：符號學文集 (1970)

- **Actantial Model (行動元模型)**： 格雷馬斯對俄國形式主義學者弗拉基米爾·普羅普的童話角色功能分析進行了高度抽象化與普遍化，提出了行動元模型。此模型將敘事中無數具體的角色，歸納為六個功能性的「行動元」（Actant）。行動元是敘事中的語法位置，而非具體的角色人物。這六個行動元成三組對立關係：
    1.  **Subject ↔ Object (主體 ↔ 客體)**：主體是欲求或行動的發動者，客體是主體所欲求的目標（可能是一個人、一件物品、一個抽象價值如「正義」或「幸福」）。這條軸線構成了敘事的「慾望軸」。
    2.  **Sender ↔ Receiver (發送者 ↔ 接收者)**：發送者是賦予任務、設定價值觀的權威來源，它將客體指派給主體，並最終評判主體的行動。接收者則是主體成功後，分享或獲得客體價值的受益者。這條軸線構成了敘事的「溝通軸」或「價值傳遞軸」。
    3.  **Helper ↔ Opponent (輔助者 ↔ 反對者)**：輔助者是幫助主體克服困難、達成目標的力量或角色。反對者則是阻礙主體行動的力量或角色。這條軸線構成了敘事的「權力軸」或「衝突軸」。
    此模型的強大之處在於其分析的普遍性，可用於分析神話、小說、廣告、政治演說等任何形式的敘事結構。

- **Narrative Schema (敘事基模)**： 亦稱為「標準敘事圖式」，此模型描繪了一個完整敘事行動的四個邏輯階段，展現了行動元如何依序展開其功能。這四個階段是：
    1.  **Manipulation (操縱 / 契約)**：此為敘事的起點。發送者透過說服、誘惑、威脅或挑釁等方式，促使主體接受一個任務或追求一個目標。此階段建立了主體與發送者之間的契約關係，並賦予主體行動的動機（devoir，必須做；vouloir，想要做）。
    2.  **Competence (賦能 / 具備資格)**：主體在接受任務後，需要獲得執行任務所需的能力與知識。此階段涉及主體獲取必要的模態價值，例如「能夠做」（pouvoir）與「知道如何做」（savoir）。這可能表現為英雄獲得一把神劍，或偵探收集到關鍵線索。
    3.  **Performance (執行 / 實踐)**：此為敘事的核心行動階段，主體運用其具備的能力，去克服反對者的阻礙，完成主要的任務。這通常是故事中最具戲劇性的部分，如戰勝惡龍、破解謎題或贏得比賽。
    4.  **Sanction (認可 / 制裁)**：敘事的結尾階段。發送者或社會對主體的行動結果進行評判。如果主體成功，他會得到獎勵與認可（positive sanction），其英雄地位被確立。如果失敗，則會受到懲罰（negative sanction）。此階段確認了客體價值的轉移，並為整個敘事賦予最終的意義。

- **Modalities (模態)**： 模態是描述行動元狀態與行動傾向的關鍵概念，它們是敘事動力的微觀引擎。格雷馬斯將模態系統化，主要圍繞四個核心模態展開，這四個模態決定了主體的「存在狀態」與「行動狀態」。
    -   **Vouloir (意願)**：主體「想要」做某事或獲得某物。這是慾望與動機的根源。
    -   **Devoir (義務)**：主體「必須」或「應該」做某事。這源於社會規範、命令或契約。
    -   **Pouvoir (能力)**：主體「能夠」做某事。這涉及物理、心智或社會層面的能力。
    -   **Savoir (知曉)**：主體「知道」如何做或擁有相關資訊。這是策略與智識的基礎。
    這些模態可以相互組合，形成複雜的敘事狀態。例如，一個「想要」但「不能夠」的主體處於挫折狀態；一個「必須做」且「能夠做」的主體則準備好執行任務。敘事基模的「賦能」階段，正是主體從缺乏模態（如不能、不知）轉變為具備模態（能夠、知曉）的過程。

## Sémiotique et sciences sociales 符號學與社會科學 (1976)

- **Generative Trajectory (生成路徑)**： 這是格雷馬斯符號學理論的總綱，一個宏大的理論框架，旨在描繪意義從最深層、最抽象的結構，如何一步步生成為表層、具體的文本或話語。此路徑由上而下分為三個主要層次：
    1.  **Deep Structures (深層結構)**：此為最抽象的層次，處理意義生成的基本邏輯條件。其核心工具是「符號方陣」，它不涉及任何具體內容，只關心純粹的關係與對立。
    2.  **Surface Narrative Structures (表層敘事結構)**：此層次將深層結構的抽象關係「轉化」為可被敘述的形式。深層的對立關係在此被人格化、行動化，形成了「行動元模型」與「敘事基模」。抽象的價值對立（如善 vs. 惡）在此轉化為主體追求客體、克服反對者的故事。
    3.  **Discursive Structures (話語結構)**：此為最表層、最接近最終文本的層次。敘事結構在此被賦予具體的語言、形象與風格。此層次涉及將抽象的行動元轉化為具體的「角色」（actors），並將其置於特定的時間（temporalization）與空間（spatialization）之中。同時，透過「主題化」（thematization）與「形象化」（figurativization），為故事披上具體的外衣。
    生成路徑的目標是提供一個統一的分析框架，能夠解釋任何符號現象（文本、圖像、儀式等）的意義是如何被系統地建構出來的。

- **Disengagement / Engagement (離合 / 脫離與掛合)**： 這是話語結構層面的關鍵機制，用以解釋話語如何創造出一個看似獨立於說話者之外的世界。
    -   **Disengagement (Débrayage / 離合)**：指的是將話語內容從其「言說情境」（enunciation，即「我-這裡-現在」）中「脫離」出去的過程。它創造出一個客觀化的敘事世界。這包括：
        -   **Actorial Disengagement (行動者離合)**：創造出第三人稱（他/她/他們），使故事看起來是在講述他人的事。
        -   **Temporal Disengagement (時間離合)**：創造出過去時態或未來時態（那時），建立一個獨立於「現在」的敘事時間。
        -   **Spatial Disengagement (空間離合)**：創造出一個不同於「這裡」的敘事地點（那裡）。
    -   **Engagement (Embrayage / 掛合)**：是與離合相反的操作，意指將脫離出去的元素重新拉回到言說情境中。例如，在一段歷史敘述中突然出現「我們可以看到...」或「如今...」，就是將歷史事件與當下的評論者聯繫起來，這是一種掛合操作。
    離合與掛合的交替使用，是話語建構其真實效果與主觀立場的根本手段。

- **Actorialization, Temporalization, Spatialization (角色化、時間化、空間化)**： 這些是話語結構層次中，將表層敘事結構轉化為具體話語的實踐過程，是「離合」機制的具體體現。
    -   **Actorialization (角色化)**：將抽象的「行動元」轉化為具體的「角色」（Actor）。一個行動元（如「反對者」）可以由多個角色（惡龍、巫師、暴風雨）來扮演；反之，一個角色（如一個複雜的反派）也可以同時承擔多個行動元的功能（既是「反對者」，又是另一個故事線的「主體」）。
    -   **Temporalization (時間化)**：在話語中建構出時間感。這不僅指使用過去式、現在式等語法時態，更包括建立敘事節奏、順序（順敘、倒敘、插敘）等，創造出獨屬於該文本的時間流。
    -   **Spatialization (空間化)**：在話語中建構出空間感。這涉及對地點、場景的描寫，以及角色在空間中的移動。空間不僅是故事發生的背景，其本身也常帶有符號意義（如城堡代表權力，森林代表未知與考驗）。

- **Thematic Roles (主題角色)**： 在角色化的基礎上，賦予角色更為具體的社會或文化身份。如果說「行動元」是語法角色，「角色」是敘事載體，那麼「主題角色」就是帶有特定文化內涵的身份標籤。例如，一個角色在行動元層面是「主體」，在角色層面是「亞瑟王」，在主題角色層面，他可以是「國王」、「騎士」、「丈夫」等。這些主題角色攜帶著預設的行為腳本與價值觀（如騎士應有的「忠誠」、「勇敢」），豐富了角色的內涵，並將其與更廣泛的文化知識體系聯繫起來。

- **Figurativization (形象化)**： 這是話語層次最外層的「裝飾」過程，但對意義的傳達至關重要。它指的是用具體的、感官可知的「形象」（figures）來表現抽象的主題。例如，「自由」是一個抽象主題，可以透過「飛翔的鳥」、「掙脫的鎖鏈」等形象來使其具體化。形象化讓話語更加生動、易於理解，並能引發讀者的情感共鳴。與之相對的是「主題化」（Thematization），後者處理的是更為抽象的概念，如「正義對抗邪惡」。形象化與主題化共同構成了話語的內容層面，一個文本的風格往往取決於其形象化是傾向於寫實還是抽象。
