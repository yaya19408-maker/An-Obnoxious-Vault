# Tim Berners-Lee (蒂姆·柏納-李)

#scholars

## The Invention of the World Wide Web (全球資訊網的發明時期) (1989-1994)

- **Information Management: A Proposal (資訊管理：一項提案)**： 1989年向歐洲核子研究組織 (CERN) 提交的文件，勾勒出一個利用超文字連結分散式資訊系統的願景。
- **World Wide Web (WWW) (全球資訊網)**： 一個透過網際網路存取的超媒體資訊系統，允許使用者瀏覽和分享連結的文件。
- **Hypertext Transfer Protocol (HTTP) (超文本傳輸協定)**： 用於在網路上傳輸超媒體文件的應用層協定，是網頁伺服器與瀏覽器之間的溝通基礎。
- **Hypertext Markup Language (HTML) (超文本標記語言)**： 用於創建網頁的標準標記語言，定義了網頁的結構與內容。
- **Uniform Resource Locator (URL) (統一資源定位符)**： 為網路上每個資源提供獨一無二位址的字串，俗稱網址。
- **WorldWideWeb (全球資訊網瀏覽器)**： 他開發的第一個網頁瀏覽器與編輯器，後為避免與全球資訊網本身混淆而改名為Nexus。
- **CERN httpd (CERN httpd伺服器)**： 他開發的第一個網頁伺服器軟體，用於儲存與傳遞網頁文件。

## Standardization and Advocacy (標準化與倡議時期) (1994-Present)

- **World Wide Web Consortium (W3C) (全球資訊網協會)**： 於1994年創立的國際標準組織，負責制定全球資訊網的技術標準與指南，以確保其長期發展。
- **Royalty-Free Principle (免權利金原則)**： 主張全球資訊網的核心技術與標準應開放給所有人免費使用，無需支付專利授權費用，以促進其普及。
- **Net Neutrality (網路中立性)**： 主張網際網路服務供應商應平等對待所有網路數據，不得基於內容、來源或應用程式進行歧視性收費或限制速度。
- **World Wide Web Foundation (全球資訊網基金會)**： 於2009年成立的非營利組織，旨在推動開放網路的普及，確保網路能為全人類帶來益處。

## The Future of the Web (網路的未來願景) (2001-Present)

- **Semantic Web (語意網)**： 一個擴展現有網路的願景，旨在讓網路上的資料具備機器可理解的語意，使電腦能更智慧地處理與整合資訊。
- **Linked Data (連結資料)**： 一組在網路上發布、分享與連結結構化資料的最佳實踐方法，是實現語意網的技術基礎。
- **Giant Global Graph (GGG) (巨大全球圖譜)**： 描述全球資訊網的社會性連結，以及語意網中資料與概念間的連結，超越了單純的文件連結。
- **Solid (Social Linked Data) (社會性連結資料)**： 一項去中心化的網路應用程式開發計畫，旨在讓使用者能掌控自己的個人資料，並自由選擇儲存位置與授權應用程式存取。
- **Decentralization (去中心化)**： 倡導網路架構應避免依賴少數大型中央伺服器或平台，將權力與控制權交還給個別使用者。
