# The Hero with a Thousand Faces (千面英雄, 1949)
- **Monomyth (單一神話)**：借用自喬伊斯（James Joyce）的術語，指存在於世界各地神話中的通用結構核心。無論文化細節如何，英雄故事都遵循「啟程—啟蒙—回歸」的基本敘事語法。
- **Separation / Departure (分離/啟程)**：英雄旅程的第一階段，包含「冒險的召喚」、「拒絕召喚」與「跨越第一道門檻」。英雄必須離開熟悉的日常世界，進入未知的超自然領域。
- **The Belly of the Whale (鯨魚之腹)**：象徵英雄完全進入未知領域的時刻，通常表現為被怪獸吞噬或進入洞穴。這是一個像子宮般的黑暗空間，代表自我的死亡與重生的潛伏期。
- **Initiation (啟蒙/試煉)**：旅程的核心階段，英雄在此經歷「試煉之路」，遭遇女神（與靈魂結合）或誘惑者，並最終達成「與父贖罪」（Atonement with the Father），解決權威衝突。
- **Apotheosis (神格化)**：英雄超越了二元對立（生/死、男/女、善/惡），獲得了擴展的意識與神聖的知識。這是個體自我消解，與宇宙法則合一的頓悟時刻。
- **The Ultimate Boon (終極恩賜)**：英雄達成冒險的目標，獲得了能拯救世界的靈藥、寶藏或智慧。這不僅是物質獎勵，更是生命能量的源泉。
- **Master of Two Worlds (兩個世界的主宰)**：英雄回歸後，能夠在物質世界與精神世界之間自由穿梭。他不再鄙視凡俗生活，而是將神聖的洞見融入日常之中。

# The Masks of God (神的面具, 1959-1968)
- **Metaphysical Function (形而上學功能)**：神話的第一功能，旨在喚起人們對「存在之奧祕」的敬畏感（Awe）。它幫助個體感受到宇宙背後不可言說的源頭與神聖性。
- **Cosmological Function (宇宙論功能)**：神話提供了一個與當時科學知識相容的宇宙形象。它解釋了世界的形狀與運作方式，將宇宙呈現為一個有序的、可理解的整體。
- **Sociological Function (社會學功能)**：神話負責維護與驗證特定的社會秩序、道德規範與階級制度。它將人為的法律與習俗神聖化，使其看起來如同自然法則般不可動搖。
- **Pedagogical Function (教育/心理學功能)**：神話引導個體度過生命的不同階段（出生、成年、結婚、死亡）。它提供了心理轉換的榜樣與儀式，幫助個人在人生轉折點實現和諧的心理適應。
- **Mythogenic Zone (神話生成帶)**：指歷史上特定時期與地理區域，產生了具有深遠影響力的核心神話體系（如蘇美爾、埃及、印度）。這些區域是人類精神文明的輻射中心。

# The Power of Myth (神話的力量, 1988)
- **Follow Your Bliss (追隨你的至福)**：坎伯最著名的格言，指個體應當識別並追求自己內在最深層的熱情與喜悅。這不僅是享樂，而是透過與內在真實本性（Atman）的對齊，踏上屬於自己的英雄旅程。
- **The Hero's Journey as Inner Work (英雄旅程作為內在工程)**：在現代社會，外在的冒險（屠龍）已轉化為內在的心理整合過程。英雄面對的怪物是自身的恐懼、創傷與社會制約，寶藏則是完整的自我（Self）。
- **Ritual as Enactment (儀式作為演繹)**：儀式是「付諸行動的神話」。透過參與儀式，現代人能夠暫時脫離社會角色，重新體驗神話所指向的深層生命節奏與集體連結。
- **Indra's Net (因陀羅之網)**：借用佛教概念，描述宇宙萬物相互映射、重重無盡的關聯性。神話幫助我們看見這一點，即每個人都是宇宙整體的一個獨特反映，彼此並無根本的分離。
- **Eternity is Now (永恆即當下)**：永恆不是死後的無限時間，而是超越時間維度的當下時刻。神話的體驗在於幫助人們在世俗生活的瞬間中，瞥見並體驗這種超越時間的維度。
