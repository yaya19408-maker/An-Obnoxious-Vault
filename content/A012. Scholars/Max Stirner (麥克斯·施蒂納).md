# Max Stirner (麥克斯·施蒂納)

#scholars #philosophy #existentialism

## The Ego and Its Own (唯一者及其所有物) (1844)

- **The Unique One / The Ego (Der Einzige / 唯一者)**： 此為施蒂納哲學的核心，指涉具體、肉身存在、獨一無二的個體本身。唯一者並非抽象的「人」或「自我」概念，也非某種預設的本質或精神實體。它就是此時此地、活生生的你或我。施蒂納強調，唯一者是所有思想、概念與價值觀的創造者與中心。任何外在於唯一者的權威，無論是上帝、國家、道德或人類，都是由唯一者自身所創造出來的概念。因此，唯一者是自身世界的絕對主宰，其存在先於一切本質與定義。它是一個空的、無內容的起點，一個「創造性的虛無」，所有意義與價值都從這個點出發，並為其所用。這個概念徹底反對任何形式的普遍主義與本質主義，將個體的獨特性與具體性置於最高位置。

- **Ownness (Eigenheit / 獨我性)**： 意指個體完全擁有並掌控自身，包括其思想、意志與身體。獨我性並非外在的自由，後者僅僅是擺脫了某種限制，但可能陷入新的束縛。獨我性是一種內在的狀態，是個體成為自身主人的自覺。一個擁有獨我性的人，其行動的唯一依據是自身的意志與慾望，而非任何外在的道德律令、社會規範或神聖戒律。施蒂納認為，追求「自由」是一個無盡的過程，因為總有新的權威出現；而實現獨我性，則是將自我確立為一切事物的尺度，從而真正地擁有自我。獨我性是唯一者對抗所有異化力量，收回自身主權的過程與狀態，是其哲學的終極目標。

- **Property (Eigentum / 所有物)**： 在施蒂納的語境中，所有物不僅指物質財產，更涵蓋一切被唯一者憑藉其力量（Macht）所佔有、支配與享用的人、事、物與觀念。所有物的正當性來源於唯一者的力量，而非法律、權利或道德的許可。凡是個體有能力獲取並持有的，便是其所有物。這個概念挑戰了傳統的財產權觀念，後者通常建立在國家或社會契約的基礎上。施蒂納認為，這種基於「權利」的財產是脆弱的，因為權利可被授予，亦可被剝奪。唯有基於自身力量的佔有，才是真實的所有物。因此，世界及其中的一切，對於唯一者而言，都是其潛在的所有物，等待其以力量去征服與享用。

- **Spook / Phantasm (Spuk / 幽靈)**： 指代那些被神聖化、固定化並反過來支配個體的抽象概念或觀念。這些幽靈包括上帝、國家、民族、家庭、道德、法律、真理、人道主義等。施蒂納認為，這些概念本身並無實體，是人類心智的創造物，但它們被賦予了超然的權威，要求個體為其服務、犧牲與奉獻，從而使個體異化，喪失了獨我性。人們被教導要敬畏這些幽靈，將其視為比自身更高的存在。唯一者的任務就是識破這些幽靈的虛幻本質，將它們從神聖的祭壇上拉下來，使其從支配主人的角色還原為可供個體利用的工具。

- **Fixed Idea (Fixe Idee / 固執觀念)**： 與「幽靈」高度相關，特指那些深植於個人心中，使其著魔並失去自主判斷能力的觀念。當一個想法，例如「正義」、「責任」或「愛國」，變得神聖不可侵犯，以至於個體願意為了它而犧牲自己的利益甚至生命時，這個想法就成為了一個固執觀念。它像一個精神上的寄生蟲，控制著宿主的思想與行為。一個被固執觀念所困的人，即使在追求自己的目標，也誤以為是在為某個崇高的事業奮鬥。施蒂納的意識利己主義者，就是要擺脫所有固執觀念的糾纏，讓所有思想都服務於自身的獨我性，而非反過來被思想所奴役。

- **The Sacred (Das Heilige / 神聖之物)**： 任何被置於個體之上，要求個體無條件服從、尊敬與犧牲的對象、觀念或制度。神聖之物是「幽靈」得以維繫其權威的基礎。無論是宗教中的上帝、政治中的國家，還是倫理學中的道德律，只要它被認為具有凌駕於個人意志之上的絕對價值，它就是神聖的。施蒂納的哲學旨在進行一場徹底的「瀆聖」（desacralization）運動，即剝奪一切事物的神聖光環。對於唯一者而言，沒有任何事物是內在地神聖的。唯一神聖的只有唯一者自身。當個體不再敬畏任何神聖之物，他才能自由地利用或拋棄它們，從而實現完全的獨我性。

- **Union of Egoists (Verein von Egoisten / 利己者聯盟)**： 這是施蒂納提出的，用以取代國家、社會等強制性集體的一種人際關係模式。利己者聯盟是個體之間基於共同利益而建立的、自願的、暫時性的聯合。聯盟的每個成員都保持其獨我性與獨立性，他們加入聯盟是為了更有效地實現個人目標。聯盟的存在完全依賴於個體成員的意願，任何成員在任何時候都有權脫離。它與國家的根本區別在於：國家將自身置於個體之上，要求個體效忠；而聯盟則純粹是個體實現自身目的的工具，個體永遠高於聯盟。這是一種流動的、非制度化的合作關係。

- **Involuntary Egoist (Unfreiwilliger Egoist / 非自願的利己者)**： 指那些口頭上宣稱自己為某個崇高理想（如上帝、國家、人類）服務，但實際上其行為背後仍是利己動機的人。施蒂ナー認為，所有人本質上都是利己的，區別僅在於是否意識到這一點。非自願的利己者被固執觀念所蒙蔽，他們將自己的慾望和利益投射到一個外部的、神聖化的對象上，並相信自己是在無私奉獻。他們是自我欺騙的利己者，其利己主義是扭曲和壓抑的。例如，一個虔誠的教徒為上帝服務，其深層動機可能是為了獲得內心的平靜或死後的救贖，這仍然是為了滿足自身的需求。

- **Voluntary Egoist (Freiwilliger Egoist / 自願的利己者)**： 指那些清醒地認識到自己是唯一者，並公開承認自己所有行動的唯一目的就是滿足自身利益的人。他們擺脫了所有「幽靈」和固執觀念的束縛，不再進行自我欺騙。自願的利己者將世界視為自己的所有物，並以自身的獨我性作為衡量一切的標準。他們的利己主義是坦誠、自覺且徹底的。他們與他人交往，例如建立利己者聯盟，也是出於計算後的自身利益，而非出於道德義務或情感束縛。施蒂納本人即是這種自願利己主義的倡導者。

- **Power / Might (Macht / 力量)**： 這是唯一者實現其意志、獲取其所有物的唯一憑藉。在施蒂納看來，「權利」（Recht）是一個虛幻的「幽靈」，是國家或社會為了維護其秩序而授予個人的許可。這種權利隨時可能被收回。真正可靠的只有個體自身的力量。我的力量所及之處，就是我的權利範圍。因此，「強力即公理」（Might is Right）。這並非鼓吹盲目的暴力，而是強調個體實現自身價值的根本能力，這可以包括體力、智力、財富、魅力等多種形式。力量是獨我性的外在展現，也是將世界轉化為自身所有物的手段。

- **Insurrection / Rebellion (Empörung / 起義)**： 指個體為了掙脫壓迫、實現獨我性而發起的自我解放行動。施蒂納將其與「革命」（Revolution）嚴格區分開來。革命旨在推翻一個舊的政治或社會秩序，並建立一個新的秩序，這往往只是用一個「幽靈」取代另一個「幽靈」。起義則不同，它的目標並非建立新的制度，而是個體自身的崛起。起義者不關心外部世界的變革，只關心自己是否能夠擺脫束縛，成為自己的主人。這是一種內在的、持續的姿態，是個體對任何試圖支配自己的權威所做的反抗。即使外部環境沒有改變，個體也可以透過起義在精神上獲得解放。

- **Revolution (Revolution / 革命)**： 施蒂納對革命持批判態度。他認為，歷史上的革命，如法國大革命，其結果並非個體的解放，而是統治形式的變更。革命者推翻了君主制，卻又建立了以「民族」、「國家」或「人民」為名的新權威。個體從效忠國王變為效忠國家，依然處於被奴役的狀態。革命追求的是一種新的社會安排，它關心的是集體的、政治的目標。這在施蒂納看來，是本末倒置。真正的解放應始於個體，而非集體。因此，他提倡個人的「起義」，而非社會的「革命」。

- **Self-Enjoyment / Self-Consumption (Selbstgenuss / 自我享用)**： 描述唯一者與世界互動的終極方式。在擺脫了所有道德和義務的束縛後，唯一者將整個世界，包括他人、物質財富甚至自己的思想和身體，都視為可供自己消費和享用的對象。這並非單純的感官享樂，而是一種將萬物化為己有、為我所用的存在狀態。個體在消費世界的過程中確認並實現自身的獨我性。施蒂納寫道：「我把世界當作我的所有物來對待，並按照我的意願來使用它。」自我享用是獨我性與所有物概念的動態結合，是利己主義哲學在實踐層面的最高表現。

- **The Creative Nothing (Das schöpferische Nichts / 創造性的虛無)**： 這是對唯一者本質的深刻描述。施蒂納指出，唯一者在成為具體個體之前，沒有任何預設的本質、屬性或定義。它是一個純粹的「無」，一個潛能的源頭。我之所以是我，並非因為我具備「人性」、「理性」或任何普遍特質，而是因為我從這個「無」中不斷地創造我自己。我的思想、我的身份、我的世界，都是我這個創造性虛無的產物。這個概念預示了存在主義「存在先於本質」的思想。唯一者不是一個被給定的實體，而是一個持續不斷的自我創造過程，其根基是徹底的、無規定性的虛無。

- **Humanity (Die Menschheit / 人類)**： 在施蒂納的哲學中，「人類」或「人道」是其中一個最具欺騙性和壓迫性的「幽靈」。費爾巴哈等青年黑格爾派將「人」的本質神聖化，要求個體為實現這個抽象的「人類」概念而活。施蒂納對此尖銳批判，認為「人類」這個概念同樣是一個外在於個體的權威，它要求個體壓抑自己的獨特性，去符合某種理想化的「人之為人」的標準。為「人類」服務，與為上帝或國家服務一樣，都是一種自我異化。唯一者不屬於「人類」這個範疇，他就是他自己，一個獨一無二的、無法被歸類的具體存在。

- **Crime (Verbrechen / 罪行)**： 施蒂納對「罪行」的定義是顛覆性的。他認為，所謂罪行，就是個體對「神聖之物」（如法律、道德）的侵犯。國家將其不允許的行為定義為罪行。然而，對於覺醒的唯一者來說，沒有什麼是內在地「罪惡」的。他的行為只取決於他的力量和意願。當一個利己者的行為與國家的法律發生衝突時，國家稱之為「罪行」，而利己者則將其視為貫徹自身意志的行動。因此，「罪行」是一個相對的概念，取決於誰的意志佔了上風。利己者通過實施「罪行」，打破了神聖秩序的束縛，從而肯定了自身的獨我性與力量。
