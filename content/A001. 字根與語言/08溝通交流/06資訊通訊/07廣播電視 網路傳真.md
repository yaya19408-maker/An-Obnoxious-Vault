☀ <span class="category">**廣播電視 網路傳真：**</span>
<span class="vocabulary">**station**</span> ['steɪʃn] 
<span class="definition">n. [C] 常構成複合詞，指廣播公司或電視公司，以及其播出的節目，即廣播電臺、電視臺：</span>radio, television, TV station 廣播電臺；電視臺 / cable, satellite station 有線臺；衛星臺 / commercial station 商業臺 / news station 新聞臺
           
<span class="vocabulary">**broadcaster**</span> [ˈbrɔ:dkɑ:stə(r); 美 -kæst-]
<span class="definition">n. [C] 電視臺；廣播公司：</span>commercial broadcaster 商業廣播公司 / independent broadcaster 獨立的廣播電合 / public, public-service broadcaster 公共廣播公司 / The country has a national public-service broadcaster. 這個國家有一個全國公共廣播公司。/ local broadcaster 地方廣播電臺 / radio, television broadcaster 無線電廣播公司；電視廣播公司 / cable, satellite broadcaster 有線電視廣播公司；衛星廣播公司 / digital, Web broadcaster 數字廣播公司；網路廣播公司 / religious broadcaster（尤美）宗教廣播電臺

<span class="vocabulary">**radio**</span> ['reɪdɪəʊ] 
<span class="definition">n. 1 [U, sing.] 利用無線電進行節目廣播的活動，即無線電廣播：</span>radio network, radio station 廣播網；無線電臺 / radio ad, radio interview, radio news 電臺廣告；電臺採訪；電臺新聞 / radio announcer 電臺播音員 / radio programme 廣播節目 / radio drama, radio play 廣播劇 <span class="definition">2 [U] 利用無線電來傳送或接收資訊的過程，即無線電通訊，或無線電傳輸；也可指無線電通訊系統：</span>to keep in radio contact 保持無線電聯絡 / radio signals, radio waves 無線電訊號；無線電波 / army, police radio 軍方無線電通訊；警方無線電通訊 / radio silence 無線電靜默 <span class="definition">3 [C]（如船上或飛機上的）無線電收發裝置：</span>to hear a gale warning on the ship’s radio 在船上的無線電收發機中接到大風警報

<span class="vocabulary">**television**</span> ['telɪ͵vɪʒn] 
<span class="definition">n. [U] 播放電視節目的系統、過程或行業，即電視系統；電視學；電視廣播業：</span>cable, digital, satellite television 有線電視；數字電視；衛星電視 / television interview, television report 電視採訪；電視報道 / television drama, television news 電視劇；電視新聞 / television ad 電視廣告 / television audience 電視觀眾 / television channel, television station 電影片道；電視臺 / She works in television. 她從事電視行業。/ We were watching the news on television. 我們正在看電視新聞。
           
<span class="vocabulary">**cable**</span> [ˈkeɪbl]
<span class="definition">n. [U] 有線電視系統。相當於cable television：</span>We can receive up to 500 cable channels. 我們可以接收多達500個有線電影片道。/ They ran commercials on cable systems across the country. 他們在全國的有線電視網上投放商業廣告。/ The channel is only available on cable. 這個頻道只有有線電視才能收看。/ We just had cable installed. 我們剛裝了有線電視。/ cable television, cable TV 有線電視 / cable channel, cable network 有線頻道；有線網路 / cable company, cable operator, cable provider 有線電視公司；有線電視運營商；有線電視供應商 / cable subscriber 有線電視使用者 / cable subscription 有線電視使用者費 / cable bill有線電視費賬單 / cable connection 有線電視的連線 / cable box 有線電視機頂盒 / cable modem 電纜調變解調器 <span class="definition">vi. [常用被動式] 給一個國家、城市、某人的家等安裝有線電視：</span>In France, 27 major cities are soon to be cabled. 法國有27座大城市不久將安裝有線電視。/ In the UK, 254,000 homes are cabled. 在英國，有254,000戶住戶安裝了有線電視。

<span class="vocabulary">**network**</span> ['netwə:k] 
<span class="definition">n. [C] 連線起來的一系列廣播電臺或電視臺（這些電臺或電視臺同時播放相同的節目），即廣播網；電視網：</span>broadcast, news, radio, television network 廣播網；新聞網；電臺網；電視網 / cable, digital, satellite network 有線網路；數字網路；衛星網路 / network news, network television 網路新聞；網路電視

<span class="vocabulary">**link**</span> [lɪŋk] 
<span class="definition">n. [C] 兩人或兩地之間的通訊手段：</span>audio, video link 音訊連線；影片連線 / satellite, telephone link 衛星連線；電話連線 / Internet, network link 網際網路連線；網路連線 / wireless link 無線連線 / data, digital, email link 資料連線；數字連線；電子郵件連線 / The driver has a radio link to base. 駕駛員透過無線電鏈路與總部聯絡。<span class="definition">vt. link A to/with B / link A and B (together) 常用被動式，在兩個機器、地點等之間建立物理聯絡或電子聯絡，相當於connect：</span>The video cameras are linked to a powerful computer. 這些攝像機是與一臺功能強大的計算機相連線的。/ The computers are linked into a network. 這些計算機是聯網的。 

<span class="vocabulary">**feed**</span> [fi:d] 
<span class="definition">n. [U] [美] 在一個電視網路中，由一箇中心電視臺向其他各電視臺提供電視節目的一個系統：</span>瞭解即可

<span class="vocabulary">**fax**</span> [fæks]（正式為facsimile）
<span class="definition">n. 1 [C] 也可用fax machine，傳真機：</span>Do you have a fax? 你有傳真機嗎？<span class="definition">2 [U] 用傳真機來傳送檔案的一個系統，即傳真系統：</span>fax number 傳真號碼 / fax line 傳真線路 / fax message 傳真電文 / fax service 傳真服務 / by fax 透過傳真

相關詞義延伸：
· radio還可指收音機；
· television還可指電視機。

周邊詞彙補充：
· channel可指人們用於獲取資訊或進行溝通交流的方法或系統，即途徑、渠道；
· speaker可指（收音機、電腦或音樂裝置上）聲音出來的地方，即揚聲器，喇叭。

· 表示“廣播節目”、“電視節目”的詞群見[[06節目]]
