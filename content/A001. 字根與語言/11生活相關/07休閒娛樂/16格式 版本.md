☀ <span class="category">**格式 版本：**</span>
<span class="vocabulary">**format**</span> ['fɔ:mæt] 
<span class="definition">n. [C] 電影、音樂錄製品等的格式：</span>His latest album is available on all formats. 他的最新專輯有各種格式可以買到。/ digital format 數碼格式 / DVD format DVD格式

<span class="vocabulary">**adaptation**</span> [͵ædæp'teɪʃn] 
<span class="definition">n. [C] 一部作品的改編本（如由文學作品改編成的電影、圖書或戲劇）。其詞義已為version所涵蓋。因此在很多情況下，adaptation可以換成version。（但是version並不是所有情況下都可換成adaptation，因為version可表示各種事物的變體，而非僅限於文學作品的變體）：</span>cinematic, stage, TV adaptation 改編的電影；改編的戲劇；改編的電視節目 / a screen adaptation of Shakespeare’s ‘Henry the Fifth’ 莎士比亞悲劇《亨利五世》的螢幕改編本 / faithful adaptation 忠實於原作的改編

<span class="vocabulary">**version**</span> ['və:ʃn] 
<span class="definition">n. [C] 基於某個作品所改編而成的電影、戲劇或音樂等形式。通常是形式、風格或語言上的“改編版”（而內容基本保持不變），如小說的電影版。此時與edition的“圖書出版的形式”之義有交集，如abridged version和abridged edition都可翻譯為“刪節本”，但abridged edition僅表示圖書的“刪節本”，而abridged version則除此之外還可指影視、音樂等作品的“刪節版”。且涉及到不同語言之間的轉換時，也要用version，如English version（英文版）。總之，在此義項下，edition一般僅用於表示圖書的版本，且僅表示版本本身，而不如version那樣強調各種事物不同版本之間的變化：</span>audio version 音訊版 / screen, stage, television version 銀幕版；舞臺版；電視版 / cover version 翻唱版 / the film version of ‘War and Peace’ 根據《戰爭與和平》改編的電影
