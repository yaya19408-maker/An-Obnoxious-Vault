☀ <span class="category">**寵物：**</span>
<span class="vocabulary">**domestic**</span> [dəˈmestɪk]
<span class="definition">adj.（動物）馴養的，作為寵物來飼養的、非野生的：</span>a domestic cat 家貓

<span class="vocabulary">**pet**</span> [pet] 
<span class="definition">n. [C] 作為寵物來養在家裡的動物、禽鳥等：</span>house pet 家養寵物 / virtual pet 虛擬寵物 / pet cat 寵物貓 / pet owner 寵物主人

<span class="vocabulary">**cat**</span> [kæt] 
<span class="definition">n. [C] 貓：</span>family, pet cat 家貓；寵物貓 / wild cat 野貓 / alley, stray cat（街上的）流浪貓 / cat hair 貓毛 / cat owner 貓主人

<span class="vocabulary">**dog**</span> [dɒɡ] 
<span class="definition">n. [C] 狗，犬：</span>family, pet dog 家犬；寵物狗 / stray dog 流浪狗 / wild dog 野狗 / guide dog 導盲犬 / police dog 警犬 / guard dog 看門狗 / rescue dog 救生犬

<span class="vocabulary">**toy**</span> [tɒɪ] 
<span class="definition">adj. 只用於名詞前，（狗）品種的體型很小的：</span>a toy poodle 玩具貴賓犬

<span class="vocabulary">**birdcage**</span> ['bə:dkeɪdӡ] 
<span class="definition">n. [C]（尤指屋內的）鳥籠：</span>瞭解即可

<span class="vocabulary">**home**</span> [həʊm] 
<span class="definition">n. [C] 寵物相關收養所（專門照看無家可歸的寵物）：</span>dogs’ home 狗之家

<span class="vocabulary">**pound**</span> [paʊnd] 
<span class="definition">n. [C] 流浪貓狗的收容所。不太常用：</span>cages at the local pound 本地流浪寵物收容所的籠子

<span class="vocabulary">**litter**</span> ['lɪtə] 
<span class="definition">n. [U] 放在一個淺盒子裡的一種乾燥物質，供寵物（尤指貓）用作室內廁所：</span>litter tray（英）貓砂盤 / litter box（美）鋪了便溺墊物的箱子 / cat litter 貓砂

<span class="vocabulary">**collar**</span> ['kɒlə] 
<span class="definition">n. [C] 動物（尤指狗）的皮革項圈或塑膠項圈：</span>dog collar 狗項圈 / The dog slipped its collar and ran off. 那條狗掙脫了項圈，跑掉了。
           
<span class="vocabulary">**muzzle**</span> [ˈmʌzl]
<span class="definition">n. [C] 為防止動物（尤指狗）咬人而給其戴的口鼻套：</span>dogs like pit bulls which have to wear a muzzle 像位元犬這種得戴嘴套的狗 <span class="definition">vt. 給狗的口鼻戴套子（以防止其咬人）：</span>He was convicted of failing to muzzle a pit bull. 他因沒能給位元犬戴嘴套而被判有罪。

<span class="vocabulary">**sit**</span> [sɪt] 
<span class="definition">vi.（狗）蹲坐（即臀部著地，前腿直立）：</span>Rover! Sit! 羅弗！蹲下！

<span class="vocabulary">**beg**</span> [beɡ] 
<span class="definition">vi. 狗蹲坐在後腿上將前爪抬起以等待被給予某東西：</span>瞭解即可

<span class="vocabulary">**wag**</span> [wæɡ] 
<span class="definition">vt.＆vi.（狗）（從一側到另一側一遍又一遍地）搖尾巴，擺尾巴；（狗尾巴）擺動：</span>The dog raced ahead, its tail wagging furiously. 狗向前衝去，尾巴猛烈地搖擺著。

周邊詞彙補充：
· parrot可指鸚鵡；
· rabbit可指兔子；
· stray可指走失的寵物；無主的寵物；
· cage可指（用於關動物或禽鳥的）金屬籠子。也可指把動物關在籠子裡；
· mess可指動物（通常是一隻狗或一隻貓）的糞便。還可指（動物）隨地便溺。
