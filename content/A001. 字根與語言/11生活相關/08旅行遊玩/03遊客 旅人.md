☀ <span class="category">**遊客 旅人：**</span>
<span class="vocabulary">**tourist**</span> ['tʊərɪst] 
<span class="definition">n. [C]（為愉悅身心之目的而進行旅遊的）旅遊者，遊客。是此義的常規用詞，最為常用：</span>foreign tourist 外國遊客 / influx of tourists 遊客湧入 / tourist traffic 遊客量 / tourist destination 旅遊目的地 / tourist attraction 旅遊景點 / tourist route 遊覽路線 / tourist industry 旅遊業 / tourist guide 旅遊指南；導遊 / tourist map 旅遊地圖 / tourist season 旅遊季節 / tourist visa 旅遊簽證 / tourist bus 觀光巴士

<span class="vocabulary">**traveler**</span> ['trævlə]（尤美；英尤用traveller）
<span class="definition">n. [C] 正在旅行（或經常旅行）的人。其旅行的目的並不限於愉悅身心，也可能肩負著某種使命等（在愉悅身心的目的上，與tourist沒有本質區別，只是tourist更常用。但在總體概念上，tourist是一種traveler）：</span>leisure traveler休閒遊客 / holiday traveler（尤美）假期旅行者 / frequent traveler 經常旅行的人 / foreign traveler 外國遊客 / independent traveler（尤英）自助旅行者 / fellow traveler 旅伴 / space, time traveler 太空旅行者；穿越時空的人 / armchair traveler 足不出戶的神遊旅行者

<span class="vocabulary">**hiker**</span> [ˈhaɪkə(r)]
<span class="definition">n. [C]（在鄉間進行長途徒步旅行的）遠足者、徒步旅行者：</span>瞭解即可

<span class="vocabulary">**visitor**</span> ['vɪzɪtə] 
<span class="definition">n. [C] visitor (to ...)（某地的）到訪者，來訪者，參觀者，即遊客：</span>international visitor 國際遊客 / holiday visitor（英）度假遊客 / museum, park visitor 博物館參觀者；公園遊客
           
<span class="vocabulary">**cosmopolitan**</span> [ˌkɒzməˈpɒlɪtən; 美 ˌkɑ:zməˈpɑ:l-]
<span class="definition">n. [C] 周遊世界的人：</span>She's a real cosmopolitan. 她是個真正的四海為家的人。
