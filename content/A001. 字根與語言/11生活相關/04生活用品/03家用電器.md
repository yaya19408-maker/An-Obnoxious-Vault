☀ <span class="category">**家用電器：**</span>
<span class="vocabulary">**television**</span> ['telɪ͵vɪʒn] 
<span class="definition">n. [C] [英式非正式為telly]（也可用television set）電視機：</span>black-and-white, colour/color television 黑白電視；彩色電視

<span class="vocabulary">**TV = television**</span> ['telɪ͵vɪʒn] 
<span class="definition">n. [C, U] 電視機。相當於television：</span>What’s on TV tonight? 今晚電視有什麼節目？/ to bue a TV 買一臺電視

<span class="vocabulary">**box**</span> [bɒks] 
<span class="definition">n. [sing.] [尤英，非正式] the box 可指電視：</span>What’s on the box tonight? 今晚有什麼電視節目？

<span class="vocabulary">**tube**</span> [tju:b] 
<span class="definition">n. [sing.] [美，非正式] the tube可指電視，電視機：</span>瞭解即可

<span class="vocabulary">**digital**</span> ['dɪdʒɪtl] 
<span class="definition">n. [U] 數字電視：</span>How long have you had digital? 你們的數字電視買了多長時間了？

<span class="vocabulary">**portable**</span> ['pɔ:təbl] 
<span class="definition">n. [C] 便於攜帶的一種小型機器（尤指行動式電腦或電視機）：</span>We bought a colour portable for the bedroom. 我們買了一臺行動式小彩電放在臥室裡。

<span class="vocabulary">**set**</span> [set] 
<span class="definition">n. [C] 接收電視或廣播訊號的裝置，如電視機，或收音機：</span>in front of the television set 在電視機前（常指看電視）

<span class="vocabulary">**radio**</span> ['reɪdɪəʊ] 
<span class="definition">n. [C] 收音機：</span>a car radio 汽車收音機 / to turn on the radio 開啟收音機

<span class="vocabulary">**flashlight**</span> ['flæʃlaɪt] 
<span class="definition">n. [C] [尤美] 手電筒，相當於英式的torch：</span>pocket flashlight 可放在口袋裡的手電筒 / flashlight beam 手電筒發出的光柱 

<span class="vocabulary">**fan**</span> [fæn] 
<span class="definition">n. [C] 風扇：</span>electric fan 電（風）扇 / extractor fan 排氣扇 / ceiling, overhead fan 吊扇 / fan heater（尤英）熱風器

<span class="vocabulary">**remote**</span> [rɪ'məʊt] 
<span class="definition">n. [C] [非正式] 遙控器，相當於remote control：</span>He flipped through the channels with the remote. 他用遙控器快速瀏覽了各個頻道。

周邊詞彙補充：
· appliance可指家用電器；
· speaker可指（收音機、電腦或音樂裝置上）聲音出來的地方，即揚聲器，喇叭；
· iron可指熨斗；
· fax可指傳真機。
