☀ <span class="category">**休閒用品 娛樂玩具：**</span>
<span class="vocabulary">**recorder**</span> [rɪ'kɔ:də] 
<span class="definition">n. [C] 構成複合詞，錄音機；錄影機：</span>tape, cassette, video recorder 磁帶錄音機；盒式錄音機；錄影機

<span class="vocabulary">**CD = compact disk**</span> 
<span class="definition">n. [C]（全稱為compact disc；也可用disc）（記錄聲音或其他資訊的）光碟，光碟：</span>accompanying CD 附帶的光碟 / live CD 現場演出的光碟 / multimedia, music, photo, software CD 多媒體光碟；音樂光碟；照片光碟；軟體光碟 / CD drive 光碟驅動器

<span class="vocabulary">**DVD = digital versatile disc**</span> 
<span class="definition">n. [C]（全稱為digital videodisc或digital versatile disc）儲存了大量資訊（尤其是影象和影片資訊）的數字影碟（可在電腦上或DVD播放器上進行播放）：</span>DVD format, DVD version 數字影碟格式；數字影碟版 / DVD technology 數字影碟技術

<span class="vocabulary">**cassette**</span> [kæ'set] 
<span class="definition">n. 1 [C]（用於播放或錄製音樂、聲音或影片的）盒式磁帶，或稱卡式磁帶（由一個扁平的小塑膠盒和其中的磁帶條組成）：</span>audio, video cassette 盒式錄音帶；盒式錄影帶 / tape cassette 盒式磁帶 / blank cassette 空白磁帶 / music cassette 音樂磁帶 / accompanying cassette 配書磁帶 / cassette recorder 盒式磁帶錄音機 <span class="definition">2 [C] 膠片盒（內含膠片，且可放入攝像機）：</span>瞭解即可

<span class="vocabulary">**tape**</span> [teɪp] 
<span class="definition">n. 1 [U] 用於記錄聲音、影象等資訊的磁性材料，為一條狹長的帶子：</span>His albums are available on tape and CD. 他的歌曲專輯有磁帶和光碟出售。/ magnetic tape 磁帶 <span class="definition">2 [C] 錄上了資訊（如聲音、影象等）的一個盒式磁帶：</span>cassette tape 盒式磁帶 / music tape 音樂盒帶 / video tape（通常用videotape）錄影盒帶 / demo, master, original tape 樣帶；母帶；原帶 / blank tape 空白盒帶
           
<span class="vocabulary">**videotape**</span> [ˈvɪdiəʊteɪp; 美 -oʊ-]
<span class="definition">n. [U, C] 錄影帶（做不可數名詞時，指錄影用的帶狀的磁性材料；做可數名詞時，指盒式錄影帶）：</span>Film can also be copied onto videotape. 電影同樣可以複製到錄影帶上。/ Police are studying a videotape of the scene. 警方正在研究現場的錄影。/ The videotape of the attack caused outrage when it was screened on the news. 襲擊事件的錄影帶在新聞中播出，激起了人們的義憤。/ a videotape showing some of the highlights of her career 表現她職業生涯中某些最重要時期的錄影帶 / a videotape of the demonstration 示威遊行的錄影帶

<span class="vocabulary">**master**</span> ['mɑːstə] 
<span class="definition">adj. 常用作形容詞，指錄音、磁帶、電影等的母帶（可以此進行復制）：</span>Keep one as a master copy. 留下一盤作為母帶。
           
<span class="vocabulary">**loop**</span> [lu:p]
<span class="definition">n. [C] 迴圈音像磁帶：</span>The audio tape runs in a continuous loop lasting thirty minutes. 這盤磁帶可以一圈一圈地連續轉三十分鐘。
           
<span class="vocabulary">**soundtrack**</span> [ˈsaʊndtræk]
<span class="definition">n. [C] 電影音樂磁帶（或唱片）；電影對白磁帶（或唱片）：</span>I've just bought the soundtrack of the latest Tarantino movie. 我剛買了塔倫蒂諾最新影片的音樂磁帶。/ film（尤英）, movie（尤美）soundtrack 電影原聲帶 / original soundtrack 原聲帶 / The original Japanese soundtrack is included with English subtitles. 日文的原聲帶配有英文字幕。/ musical soundtrack 原聲音樂 / a musical soundtrack conceived by Quincy Jones 由昆西·瓊斯構思設計的原聲音樂 / Abba provided the soundtrack to my teenage years. 我少年時代聽阿巴樂隊的音樂帶。/ The movie has a soundtrack featuring music by Strauss. 該影片的配樂主旋律是施特勞斯的樂曲。/ She composed the soundtracks to several hit movies. 她曾為幾部熱門影片配樂。

周邊詞彙補充：
· video可指錄影帶（此時指用於儲存資料的磁性材料），也可指裝錄影帶的盒子（此時叫盒式錄影帶）。

· 表示“唱片”、“音樂專輯”的詞群見[[07音樂 聲樂]]
· 表示“電視機”的詞群見[[03家用電器]]

<span class="vocabulary">**album**</span> ['ælbəm] 
<span class="definition">n. [C]（儲存照片、郵票等的）冊子，即影集，相簿；集郵冊等：</span>photo album 相簿；影集 / family, wedding album 家庭相簿；結婚紀念相簿 / I keep the photographs in an album. 我把照片放在相簿裡。/ stamp album 集郵簿

<span class="vocabulary">**mobile**</span> ['məʊbaɪl] 
<span class="definition">n. [C] 線繩懸掛著一些小物件所構成的一個小裝飾品，懸掛在天花板上，可隨風擺動。如風鈴：</span>瞭解即可

<span class="vocabulary">**fan**</span> [fæn] 
<span class="definition">n. [C] 扇子：</span>瞭解即可

<span class="vocabulary">**toy**</span> [tɒɪ] 
<span class="definition">n. 1 [C] 兒童玩的玩具：</span>electronic toy 電子玩具 / plastic, wooden toy 塑膠玩具；木質玩具 / construction, educational, learning toy 搭建類玩具；益智玩具；學習翫具 / child’s, children’s toy 兒童玩具 <span class="definition">2 [C] 用來玩樂的一個物品，即玩物，玩意兒：</span>executive toys（尤英）行政人員的玩意兒 / His latest toy is the electric drill he bought last week. 他的新玩意兒是他上星期買的電鑽。<span class="definition">adj. 只用於名詞前，被做成某事物的一個仿製品而用來玩的：</span>toy car, toy gun, toy piano 玩具車；玩具槍；玩具鋼琴

<span class="vocabulary">**brick**</span> [brɪk] 
<span class="definition">n. [C]（塑膠或木質的）積木（是一種玩具）：</span>瞭解即可

<span class="vocabulary">**block**</span> [blɒk] 
<span class="definition">n. [pl.] Blocks可表示木質或塑膠的立方體，例如兒童玩的積木：</span>瞭解即可

<span class="vocabulary">**cap**</span> [kæp] 
<span class="definition">n. [C]（尤指在玩具槍裡）裝火藥的一個小紙容器：</span>瞭解即可

<span class="vocabulary">**doll**</span> [dɒl] 
<span class="definition">n. [C] 兒童的人形（尤指孩童狀的）玩具，即玩偶：</span>china, rag, wooden doll 瓷娃娃；布娃娃；木頭玩偶 / mechanical doll 機器娃娃 / Russian doll 俄羅斯套娃 / voodoo doll（常為幽默用法）巫毒娃娃
           
<span class="vocabulary">**puppet**</span> [ˈpʌpɪt]
<span class="definition">n. [C] 木偶：</span>finger, glove（英）, hand, shadow, sock puppet 手指木偶；布袋木偶；牽線木偶；皮影木偶 / wooden puppet 木偶 / How do you work these puppets? 你是如何操縱這些木偶的? / puppet play, puppet show 木偶表演 / puppet theatre/theater 木偶劇院 / puppet master木偶操縱者

<span class="vocabulary">**balloon**</span> [bə'lu:n] 
<span class="definition">n. [C]（用作玩具或裝飾等的）小氣球（為橡膠製品）：</span>helium balloons for the children’s party 兒童聚會用的氦氣球 / water balloon 水氣球 / party balloon 聚會用氣球

<span class="vocabulary">**top**</span> [tɒp] 
<span class="definition">n. [C] 兒童玩的陀螺玩具（用手或繩子將其旋轉起來後只有一個尖端接觸一個表面進行高速旋轉）：</span>She was so confused--her mind was spinning like a top. 她如墮五里霧中，被搞得暈頭轉向。

<span class="vocabulary">**marble**</span> ['mɑːbl] 
<span class="definition">n. [C]（小孩子在地上玩兒的）玻璃彈子：</span>a glass marble 一個玻璃彈子
           
<span class="vocabulary">**buggy**</span> [ˈbʌgi]
<span class="definition">n. [C]（也可用Baby Buggy[商標]）[均英]（美式英語用stroller）（有輕型摺疊椅的）嬰兒車（可以推著走）：</span>baby buggy 嬰兒車 / parents pushing buggies 推著嬰兒車的父母
           
<span class="vocabulary">**pushchair**</span> [ˈpʊʃtʃeə(r); 美 -tʃer]
<span class="definition">n. [C] 嬰兒手推車（其座椅可摺疊）：</span>collapsible, folding pushchair 摺疊式嬰兒推車

<span class="vocabulary">**pram**</span> [præm]
<span class="definition">n. [C] 推著走的四輪嬰兒車：</span>She was pushing her baby along in a pram. 她把寶寶放在嬰兒車裡推著走。

<span class="vocabulary">**stroller**</span> [ˈstrəʊlə(r); 美 ˈstroʊ-]
<span class="definition">n. [C] [美]（可以推著走的）嬰兒車、童車。相當於buggy和pushchair：</span>瞭解即可     
           
<span class="vocabulary">**baby carriage**</span>
<span class="definition">n. [C] [美]（可以躺在裡面的）四輪手推嬰兒車。相當於pram：</span>He saw a woman pushing a baby carriage. 他看見一個女人推著個嬰兒車。/ There's an old photo of me in a baby carriage. 有一張我躺在嬰兒車裡的老照片。

相關詞義延伸：
· balloon還可指大型的熱氣球（常帶有一個籃子，可運送乘客）。

周邊詞彙補充：
· souvenir可指為使自己想起（或保持記憶）一個地方、一個場合或假期而買下來的一個東西，即紀念品。也可指度假時買回來送人的東西；
· novelty可指（售賣的）廉價的小玩具或小飾品。
