<span class="vocabulary">**menu**</span> ['menju:] 
<span class="definition">n. [C]（餐館等的）選單：</span>special menu 特色選單 / set menu 套餐選單 / lunch menu 午餐選單 / dessert menu 甜點單 / drinks menu 酒水單 / May we have the menu? 我們可以看一下選單嗎？/ What’s on the menu this evening? 今晚有些什麼菜？ 

<span class="vocabulary">**recipe**</span> ['resɪpɪ] 
<span class="definition">n. [C] recipe (for sth)對烹飪過程（或所需的食材及配料）所進行的一系列指導性說明，即烹飪法，食譜：</span>a recipe for chicken soup 雞湯的做法 / vegetarian recipes 素菜食譜 / a recipe book 烹飪書 / healthy recipe 健康食譜 / easy recipe 簡易食譜
