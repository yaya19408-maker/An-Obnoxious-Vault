☀ <span class="category">**準備吃飯：**</span>
<span class="vocabulary">**set**</span> [set] 
<span class="definition">vt. set a/the table 將餐具擺放到餐桌上（以準備吃飯）：</span>The table was set for eight guests. 桌子上擺放了八位客人的餐具。<span class="definition">adj. 只作前置定語，指餐館的飯菜價格固定且選擇範圍有限的，即套（餐）的：</span>a set meal 一份套餐 / a set lunch 午餐套餐

<span class="vocabulary">**lay**</span> [leɪ] 
<span class="definition">vt. [英；舊時] 在桌子上擺放刀叉、盤子等餐具（準備用餐），相當於set：</span>to lay the table 擺好餐具準備用餐

<span class="vocabulary">**follow**</span> ['fɒləʊ] 
<span class="definition">vt.＆vi. 在飲食中可指接下來的食物是…（如下一道菜是…）：</span>The main course was followed by fresh fruit. 主菜以後是新鮮水果。

<span class="vocabulary">**spoon**</span> [spu:n] 
<span class="definition">vt. 用勺子掀起或移動食物，如舀：</span>She spooned the sauce over the chicken pieces. 她用勺把沙司澆到雞塊上。

<span class="vocabulary">**pour**</span> [pɔ:] 
<span class="definition">vt.＆vi. pour (sth) (out) 將飲料從容器中倒出（如倒進杯子裡）：</span>Will you pour the coffee? 你來倒咖啡好嗎？<span class="definition">vt. 使液體或其他物質從容器中以一股的形式持續地流出（尤指透過使該容器處於一定角度），即傾倒：</span>Pour the sauce over the pasta. 把醬汁澆在麵食上。/ Helen poured out two stiff drinks. 海倫倒了兩杯烈酒。

<span class="vocabulary">**refresh**</span> [rɪ'freʃ] 
<span class="definition">vt. [尤美，非正式] 給某人的杯子重新倒滿或斟滿：</span>Let me refresh your glass. 我給你再斟一杯吧。

<span class="vocabulary">**charge**</span> [tʃɑːdӡ] 
<span class="definition">vt. 英國正式用法中可指注滿玻璃杯：</span>Please charge your glasses and drink a toast to the bride and groom! 請各位斟滿酒杯向新娘、新郎敬酒！

周邊詞彙補充：
· serve可指（例如在餐館或用餐期間）給某人提供或端上食物或飲料；
· dip可指蘸。
