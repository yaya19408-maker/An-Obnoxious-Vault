☀ <span class="category">**蔬菜 豆類：**</span>
<span class="vocabulary">**vegetable**</span> ['vedӡɪtəbl] 
<span class="definition">n. [C] 蔬菜：</span>green vegetables 綠色蔬菜 / organic vegetables 有機蔬菜 / seasonal vegetables 時令蔬菜 / vegetable garden 菜園 / vegetable market 菜市場 / vegetable juice 蔬菜汁

<span class="vocabulary">**green**</span> [ɡri:n] 
<span class="definition">n. [pl.] [尤英] 綠色蔬菜、青菜：</span>I like eating greens. 我喜歡吃青菜。

<span class="vocabulary">**tomato**</span> [tə'mɑ:təu, tə'meitəu] 
<span class="definition">n. [C, U] 西紅柿，番茄：</span>green, red tomato 青西紅柿；紅西紅柿 / tomato soup 番茄湯

<span class="vocabulary">**jacket**</span> ['dӡækɪt] 
<span class="definition">n. [C] [英] 烤過的土豆的皮：</span>potatoes baked in their jackets 帶皮烤的土豆

<span class="vocabulary">**cabbage**</span> ['kæbɪdӡ] 
<span class="definition">n. [U, C] 甘藍，捲心菜：</span>red cabbage 紫甘藍 / Chinese cabbage 大白菜 / cabbage leaf 捲心菜葉

<span class="vocabulary">**heart**</span> [hɑːt] 
<span class="definition">n. [C] 捲心菜等的菜心：</span>瞭解即可

<span class="vocabulary">**onion**</span> ['ʌnjən] 
<span class="definition">n. [C, U] 洋蔥：</span>raw onion 生洋蔥 / slice onion 切洋蔥片 / onion ring 洋蔥圈
           
<span class="vocabulary">**lettuce**</span> [ˈletɪs]
<span class="definition">n. [U, C] 萵苣；生菜：</span>crisp, crunchy lettuce 爽脆的萵苣 / limp lettuce 發蔫的萵營 / chopped, shredded lettuce 切碎的萵苣 / cos（英）, iceberg, romaine（美）lettuce 直立萵苣；捲心萵苣；長葉萵苣 / You need one whole head of lettuce for this salad. 做這道色拉你需要一整棵生菜。/ lettuce leaf萵苣葉 / on a bed of lettuce 以生菜墊底 / a bacon, lettuce and tomato sandwich 鹹肉、生菜加番茄三明治 / Buy a lettuce and some tomatoes. 買一個生菜和一些番茄。

<span class="vocabulary">**mushroom**</span> ['mʌʃrʊm] 
<span class="definition">n. [C] 蘑菇，傘菌：</span>dried mushroom 幹蘑菇 / poisonous mushroom 毒蘑菇 / to pick wild mushrooms 採野蘑菇 / mushroom cap 蘑菇蓋 / mushroom soup 蘑菇湯

<span class="vocabulary">**eggplant**</span> ['eɡplɑːnt] 
<span class="definition">n. [C, U] [美] 茄子，相當於aubergine（aubergine為英式用法）：</span>瞭解即可
           
<span class="vocabulary">**pepper**</span> ['pepə]
<span class="definition">n. [C, U] [英]（英美還可用sweet pepper，美還用bell pepper）（紅色、綠色或黃色的）中空的辣椒，即甜椒，柿子椒，燈籠椒：</span>sweet pepper 甜椒 / hot pepper 辣椒 / green, red, yellow pepper 青椒；紅椒；黃椒 / pepper sauce 辣椒醬
           
<span class="vocabulary">**chilli**</span> [ˈtʃɪli]
<span class="definition">n. [C, U]（美也用chili pepper）辣椒：</span>fresh chilli 新鮮辣椒 / dried chilli 幹辣椒 / hot chilli 辣的辣椒 / mild chilli 不太辣的辣椒 / green, red chilli 青辣椒；紅辣椒 / chilli pepper（美）辣椒 / chilli oil, chilli powder, chilli sauce 辣椒油；辣椒麵；辣椒醬 / chilli beans 辣豆

<span class="vocabulary">**carrot**</span> ['kærət] 
<span class="definition">n. [U, C] 胡蘿蔔：</span>raw carrot 生胡蘿蔔 / cooked carrot 煮熟的胡蘿蔔 / baby carrot 小胡蘿蔔 / carrot juice 胡蘿蔔汁

<span class="vocabulary">**pea**</span> [pi:] 
<span class="definition">n. [C] 豌豆，豌豆粒：</span>pea soup 豌豆湯 / pea pod 豌豆莢 
           
<span class="vocabulary">**celery**</span> [ˈseləri]
<span class="definition">n. [U] 芹菜：</span>a stick of celery（尤英）, a stalk of celery（美）一根芹菜 / Chop the celery and add it to the salad. 把芹菜切碎放到色拉里。/ celery leaf, celery salt, celery stalk, celery stick 芹菜葉；香芹鹽；芹菜莖；芹菜梗
           
<span class="vocabulary">**parsley**</span> [ˈpɑ:sli; 美 ˈpɑ:rsli]
<span class="definition">n. [U] 歐芹；荷蘭芹：</span>fish with parsley sauce 歐芹沙司魚 / curly, flat-leaf parsley 卷葉歐芹；平葉歐芹 / a sprig of parsley 一小枝歐芹 / The eggs are sprinkled with coarsely chopped parsley. 雞蛋上撒有粗切的歐芹段。
           
<span class="vocabulary">**rape**</span> [reɪp]
<span class="definition">n. [U]（也可用oilseed rape）油菜：</span>瞭解即可

<span class="vocabulary">**rocket**</span> ['rɒkɪt] 
<span class="definition">n. [U] [英] [美用arugula] 大蒜芥；芝麻菜；紫花南芥（葉子較長，味道較衝，可生吃）：</span>a small salad of rocket and chopped tomato 一小盤芝麻菜和西紅柿碎片色拉
           
<span class="vocabulary">**squash**</span> [skwɒʃ; 美 skwɑ:ʃ; skwɔ:ʃ]
<span class="definition">n. [C, U]（複數為squash，英式也用squashes）番瓜類蔬菜（winter squash外皮堅硬，內裡為橙色；summer squash外皮柔軟，為黃色或綠色，內裡為白色，可稱為西葫蘆）：</span>瞭解即可

<span class="vocabulary">**baby**</span> ['beɪbɪ] 
<span class="definition">adj.（蔬菜品種）體型小的；非常小的時候就採摘下來的：</span>baby carrots 小胡蘿蔔
