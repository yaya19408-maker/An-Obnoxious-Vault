☀ <span class="category">**工具、儀器的名詞：**</span>
<span class="vocabulary">**tool**</span> [tu:l]  
<span class="definition">n. 1 [C]（握在手裡用於製作或修理…的）工具（如錘子、螺絲刀、鋸等）：</span>cutting, drawing, measuring tool 刀具；繪圖工具；量具 / farm, gardening, woodworking tool 農具；園藝工具；木工工具 / sophisticated, special tool 精密工具；特殊工具 <span class="definition">2 [C] 能幫助你做工作（或完成某事）的一個東西，可稱為工具，器具：</span>The computer is now an invaluable tool for me. 計算機現在對我來說是一種非常有用的工具。/ learning, teaching tool 學習工具；教學工具 / design, development, presentation tool 設計工具；開發工具；演示工具 / programming, software tool 程式設計工具；軟體工具

<span class="vocabulary">**instrument**</span> ['ɪnstrəmənt]  
<span class="definition">n. [C] 用於執行某項任務的工具或裝置，尤指精細的或科學工作所用的儀器、器械：</span>drawing, measuring, writing instrument 繪圖儀器；測量儀器；書寫工具 / astronomical, mathematical, medical, scientific instrument 天文儀器；數學儀器；醫療器械；科學儀器 / instruments of torture 刑具 / precision, sensitive instrument 精密儀器；靈敏儀器 / The autopsy revealed that the deceased had been hit with a blunt instrument. 屍檢表明死者曾遭受過鈍器的重擊。/ Even though it was a somewhat blunt instrument, our questionnaire provided us with some interesting ideas. 我們的調查問卷雖然有點兒不夠精確，但仍然為我們提供了一些有趣的觀點。

<span class="vocabulary">**apparatus**</span> [ˌæpəˈreɪtəs; 美 -ˈrætəs]
<span class="definition">n. [U] 某個活動或任務所需要的工具或其它裝置，如儀器、器械、裝置：</span>electrical, experimental, scientific apparatus 電氣裝置；實驗儀器；科學儀器 / a piece of laboratory apparatus 一件實驗室儀器 / Firefighters needed breathing apparatus to enter the burning house. 消防隊員需要呼吸器以便進入燃燒的大樓。
   
<span class="vocabulary">**appliance**</span> [əˈplaɪəns]
<span class="definition">n. [C] 家用電器：</span>electric, electrical, electronic appliances 電器 / gas appliance 燃氣用具 / cooking, heating appliance 炊具；加熱用具 / combustion appliance（美）燃燒裝置 / kitchen 廚房電器 / household, domestic appliances 家用電器 / surgical appliance（英）手術器械

<span class="vocabulary">**utensil**</span> [ju:ˈtensl]
<span class="definition">n. [C] 家用的（如做飯、幹其它活兒用的）工具、器具：</span>cooking, kitchen utensils 炊具；廚房用具 / utensils such as bowls, steamers and frying pans 碗、蒸鍋、煎鍋等用具 / carving utensil 雕刻工具

<span class="vocabulary">**gadget**</span> [ˈgædʒɪt]
<span class="definition">n. [C] 做某事所用的小器具或小裝置：</span>must-have gadget 必備的小工具 / high-tech gadget 高科技小裝置 / digital, electrical, electronic gadget 小型數碼裝置；小型電動工具；小型電子裝置 / wireless gadget 小型無繩裝置 / household gadget（尤英）家用小器具 / kitchen gadgets including toasters and kettles 烤麵包機、水壺等廚用小家電 / gadget lover 小器具愛好者 / the latest gadget for the technology obsessed: pocket-sized computers that you write on with a pen 面向科技發燒友的最新玩意兒：可在上面用筆書寫的袖珍電腦
           
<span class="vocabulary">**widget**</span> [ˈwɪdʒɪt]
<span class="definition">n. [C] [非正式]（不知道名字或不知其原理的）小裝置、小器物、小玩意兒：</span>The secret is a little widget in the can. 奧秘在於易拉罐中的一個小玩意兒。
           
<span class="vocabulary">**gear**</span> [gɪə(r); 美 gɪr]
<span class="definition">n. [U]（針對某種活動的）裝置，用具，衣服：</span>camping, climbing, fishing, hiking, riding, running, sports, swimming gear 露營裝備；登山用具；漁具；徒步旅行用具；騎馬裝備；跑步裝備；運動裝備；游泳裝備 / breathing gear（尤英）呼吸裝置 / firemen in breathing gear 戴著呼吸器的消防隊員
           
<span class="vocabulary">**kit**</span> [kɪt]
<span class="definition">n. 1 [C, U] 做某事所需要的成套工具、成套裝置：</span>emergency, first-aid, make-up, medical, mess, repair, sewing, shaving, survival, test, tool kit 應急包；急救箱；化妝箱；醫藥箱；野戰餐具；修理箱；針線包；剃鬚用具套裝；救生包；成套試劑；工具包 / She keeps an emergency medical kit in her car. 她在車裡一直放著一個應急醫藥箱。/ Check the acidity of the soil with a test kit. 用成套試劑檢測一下土壤的酸度。/ In the middle of the stage was a drum kit. 舞臺中央是一套鼓。/ media（美）, press（尤美）kit 媒體資料包；宣傳資料袋 / an electronic press kit that features brief interviews with the movie's actors 一套以影片演員的簡短採訪為主要內容的電子版新聞釋出材料 / There should be a needle and thread in the sewing kit. 針線包中應該有針線。/ a well-stocked first aid kit 一套完備的急救用品 / The kit consisted of about twenty cosmetic items and a lady's shaver. 整套工具包括大約20種化妝用品和一個女士褪毛器。<span class="definition">2 [U] [英] 參加某項活動所需要的一套衣服及裝備：</span>gym, PE, sports kit 健身服；體育鍛煉套裝；運動裝 / diving, football, rugby kit 潛水裝；足球服；橄欖球服 / the official supplier of the England team kit 英格蘭隊隊服的官方供應商 / a teddy bear dressed in a replica Real Madrid kit 身著仿製版皇家馬德里隊隊服的泰迪熊 / Chelsea kit 切爾西隊的隊服 / kit bag揹包
            
<span class="vocabulary">**outfit**</span> [ˈaʊtfɪt]
<span class="definition">n. [C]（做某事所需要的）全套裝備；成套工具：</span>a bicycle repair outfit 修腳踏車的整套工具
            
<span class="vocabulary">**hardware**</span> [ˈhɑ:dweə(r); 美 ˈhɑ:rdwer]
<span class="definition">n. [U]（英式英語也用ironmongery）家庭及園藝用的工具或裝置，如五金製品、金屬器具：</span>a hardware shop 五金店

<span class="vocabulary">**probe**</span> [prəʊb; 美 proʊb]
<span class="definition">n. [C] [術語] 科研人員所用的、可以放進某事物內部以進行測試或記錄資訊的一個小裝置，如探測儀：</span>瞭解即可           

<span class="vocabulary">**aid**</span> [eɪd]  
<span class="definition">n. [C]（自己所使用的）用於幫助自己做某事的一件物品、一臺機器等：</span>teaching, training, memory aid 教具；訓練輔助器械；記憶輔助裝置 / hearing, walking（英）, navigational aid 助聽器；行走輔助器械；導航裝置 / Dictionaries are essential aids to learning. 詞典是輔助學習的必備工具。

<span class="vocabulary">**implement**</span> [ˈɪmplɪment]
<span class="definition">n. [C] 工具，器具，常指簡單的戶外用具：</span>agricultural, farm, garden implement 農具；園藝工具 / digging, writing implement 挖掘工具 / flint, metal, stone, wooden implement 燧石工具；金屬器具；石器；木質工具 / Make sure that all sharp implements, such as scythes, have covers. 確保所有的鋒利器具——例如長柄大鐮刀，都有保護套。/ kitchen, surgical implement 廚房用具；手術器具 / cooking, writing implement 廚具；書寫工具

<span class="vocabulary">**contrivance**</span> [kənˈtraɪvəns]
<span class="definition">n.  [C] [正式] 為某一目的而製作的精巧或複雜的裝置或工具：</span>瞭解即可
            
<span class="vocabulary">**facility**</span> [fəˈsɪləti]
<span class="definition">n. [C] 可提供額外服務的特色等（這些附加特色並非必需，只是起到錦上添花的作用），有時可翻譯為“工具”：</span>editing, help, mail, search facility（均為計算機用語）編輯工具；幫助工具；電郵工具；搜尋工具 / a facility for checking spelling 檢查拼寫的工具
 
周邊詞彙補充：
· weapon可指武器；
· steel可指兵器；
· prop可指戲劇或電影中用的道具。

<span class="vocabulary">**dial**</span> ['daɪəl]  
<span class="definition">n. 1 [C] 鍾或表的盤面，或機器、裝置、交通工具上的顯示時間、數量、速度、溫度等的盤面：</span>an alarm clock with a luminous dial 夜光鬧鐘 / Check the tyre pressure on the dial. 檢查一下儀表盤顯示的輪胎壓力。<span class="definition">2 [C] 收音機、爐、灶等的控制盤（例如可用於選擇某一電臺或溫度）：</span>He turned the dial on the radio. 他轉動收音機的電臺旋鈕。/ The heat dial was set at 150 degrees. 溫度調節器設定在 150 度。/ Set the dial for the number of copies required. 在控制板上設定需要的影印數目。/ I went to my locker, spun the dial, and got my books out. 我走到我的寄物櫃跟前，轉動號碼盤，取出書。
           
<span class="vocabulary">**panel**</span> [ˈpænl]
<span class="definition">n. <span class="vocabulary">**console**</span> [ˈkɒnsəʊl; 美 ˈkɑ:nsoʊl]

<span class="vocabulary">**needle**</span> ['ni:dl]

<span class="vocabulary">**pointer**</span> [ˈpɔɪntə(r)]

<span class="vocabulary">**indicator**</span> [ˈɪndɪkeɪtə(r)]

<span class="vocabulary">**knife**</span> [naɪf] 
<span class="definition">n. [C] 用於切削或用作武器的刀。是一個基礎用詞：</span>electric knife 電刀 / plastic, steel knife 塑膠刀；鋼刀 / kitchen, table knife 菜刀；餐刀 / bread, butter, steak knife 切麵包刀；塗黃油用刀；牛排餐刀 / pocket, hunting, combat（尤美）knife 小折刀；獵刀；戰刀 / utility（美）, palette knife 多用途刀；調色刀 / go under the surgeon’s knife（尤英）接受手術 / knife blade, knife handle 刀刃；刀柄 / knife attack, knife fight 持刀襲擊；持刀鬥毆 / knife cut, knife wound 刀口；刀傷 / She pointed her knife at Richard. 她把刀指向理查德。

<span class="vocabulary">**sword**</span> [sɔ:d] 
<span class="definition">n. [C] 作為武器的劍，刀：</span>ceremonial, dress sword 禮儀用劍；禮服佩劍 / a Samurai sword 日本武士刀 / long, short sword 長劍；短劍 / I thrust my sword into his chest. 我把劍刺入了他的胸膛。/ sword blade, sword tip 劍刃；劍尖 / sword stroke, sword fight 劍擊；劍戰 / the sword of justice（比喻用法）正義之劍 / a double-edged sword（比喻用法）雙刃劍（即利弊並存的事物）

<span class="vocabulary">**dagger**</span> [ˈdægə(r)]
<span class="definition">n. [C] 兩側有刃的短劍，也可稱為匕首：</span>curved dagger 短彎刀 / jewelled dagger 鑲有寶石的匕首

<span class="vocabulary">**stiletto**</span> [stɪˈletəʊ; 美 -toʊ]
<span class="definition">n. [C] 帶有尖刃的小刀，如匕首、短劍：</span>瞭解即可

<span class="vocabulary">**cutlass**</span> [ˈkʌtləs]
<span class="definition">n. [C] 舊時水手和海盜用的短彎刀、短劍：</span>瞭解即可

<span class="vocabulary">**bayonet**</span> [ˈbeɪənət]
<span class="definition">n. [C]（戰鬥中作為武器的，系在步槍前面的）刺刀：</span>troops with fixed bayonets 裝備固定刺刀的部隊 / A bayonet had been thrust through his belly. 一把刺刀刺穿了他的肚子。

<span class="vocabulary">**sabre**</span> [ˈseɪbə(r)]
<span class="definition">n. [C] 舊時騎兵用的帶有彎刃的大刀，如軍刀、馬刀：</span>瞭解即可 

<span class="vocabulary">**sickle**</span> [ˈsɪkl]
<span class="definition">n. [C] 割草、收割莊稼等所用的鐮刀（柄短，刃彎且長）：</span>瞭解即可

<span class="vocabulary">**scythe**</span> [saɪð]
<span class="definition">n. [C] 用於割高草等的長柄大鐮刀：</span>瞭解即可

相關詞義延伸：
· sabre還可指擊劍運動用的佩劍（重量較輕）。

周邊詞彙補充：
· scissors可指剪刀；
· sharpener可表示使某事物變得鋒利或尖利的一個工具或機器；
· lancet可指醫生用的柳葉刀；
· scalpel 可指解剖刀；手術刀。

☀ <span class="category">**刀具部位：**</span>
<span class="vocabulary">**edge**</span> [edӡ] 
<span class="definition">n. [C] 刀刃；刀劍的刃；刀口：</span>Be careful--it has a sharp edge. 小心點——這刀刃很鋒利。/ serrated, jagged, ragged, rough edge 鋸齒狀的刀刃；參差不齊的刀刃；凹凸不平的刀刃；粗糙的刀刃 / smooth, blunt edge 平滑的刃；鈍的刃
 
<span class="vocabulary">**blade**</span> [bleɪd]
<span class="definition">n. [C] 刀、工具（如斧頭、鋸）或機器上的用於切割的扁平部分，即刀刃、刀片：</span>knife, razor, saw, scalpel, sword blade 刀刃；剃鬚刀刀片；鋸片；手術刀刀刃；劍身 / rotary blade 旋轉刀片 / He swung the blade with all his strength. 他使出全身的力氣揮舞刀。/ Many of these tools have sharp blades, so be careful. 這些工具中很多都有鋒利的刀片，務必小心。

相關詞義延伸：
· blade還可指溜冰鞋的冰刀。
 
周邊詞彙補充：
· handle可指刀柄；
· pull可指用於拉拽的手柄或繩子等。

☀ <span class="category">**鋒利的形容詞和動詞：**</span>
<span class="vocabulary">**sharp**</span> [ʃɑːp] 
<span class="definition">adj.（邊緣或尖端）鋒利的，尤指可用於切削或扎出洞：</span>sharp knife, sharp sword, sharp teeth 鋒利的刀；利劍；鋒利的牙齒
           
<span class="vocabulary">**keen**</span> [ki:n]
<span class="definition">adj. [常用於名詞前] [文學]（刀）有鋒利的刃或銳利的尖的，即鋒利的、銳利的：</span>瞭解即可

<span class="vocabulary">**blunt**</span> [blʌnt]
<span class="definition">adj.（刃或尖端）不鋒利的，即鈍的：</span>a blunt knife 鈍刀子 / This pencil's blunt! 這支鉛筆不尖了！/ The police said he had been hit with a blunt instrument/object. 警方說他遭到了鈍器襲擊。

<span class="vocabulary">**sharpen**</span> ['ʃɑːpən] 
<span class="definition">vt.＆vi. 變得鋒利；使…變得鋒利：</span>This knife needs sharpening. 這把刀需要磨了。
           
<span class="vocabulary">**grind**</span> [graɪnd]
<span class="definition">vt.（在堅硬表面上）磨鋒利、磨快、磨光某事物：</span>a special stone for grinding knives 磨刀用的石頭 / The tip can be ground to a much sharper edge to cut smoother and faster. 可以把尖端磨得更鋒利些，這樣切割起來更順更快。

<span class="vocabulary">**blunt**</span> [blʌnt]
<span class="definition">vt. 使尖端或刃變鈍：</span>瞭解即可

相關詞義延伸：
· sharp還可指一個形狀或物體的一部分（或一端）形成一個尖或角的，即尖的。也可指在醫學領域中帶有鋒利邊緣或尖端的物品，如針、注射器等。

周邊詞彙補充：
· snag可指粗糙尖銳的物體，或物體的粗糙尖銳部分。
