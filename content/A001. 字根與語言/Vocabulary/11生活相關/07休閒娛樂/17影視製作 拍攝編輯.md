☀ <span class="category">**影視製作 拍攝編輯：**</span>
<span class="vocabulary">**studio**</span> ['stju:dɪəʊ] 
<span class="definition">n. 1 [C] 電影的攝影棚：</span>瞭解即可 <span class="definition">2 [C] 電影公司或製片廠：</span>film studio 電影製片廠 / a major Hollywood studio 好萊塢一家大型電影公司工作 / studio executive, studio head 電影公司高管

<span class="vocabulary">**material**</span> [mə'tɪərɪəl] 
<span class="definition">n. [U] 用以創作電影等所需的素材：</span>to collect material for the film 為該部影片收集素材

<span class="vocabulary">**direct**</span> [dɪ'rekt, daɪ-] 
<span class="definition">vt.＆vi. 導演電影或戲劇：</span>Who did direct the movie? 這部電影是誰導演的？

<span class="vocabulary">**direction**</span> [dɪ'rekʃn, daɪ-] 
<span class="definition">n. [U]（電影導演的）指點，指示：</span>There is some clever direction and the film is very well shot. 由於導演指導有方，影片拍得非常成功。

<span class="vocabulary">**cast**</span> [kɑːst] 
<span class="definition">vt.＆vi. cast sb (as sb)（為一部電影、戲劇等）選擇演員（以扮演不同的角色）；選擇某一演員（以扮演某一角色）：</span>The play is being cast in the US. 目前正在美國挑選這部戲的演員。/ He has cast her as an ambitious lawyer in his latest movie. 他選定她在他最近的一部影片裡扮演一名雄心勃勃的律師。
           
<span class="vocabulary">**prop**</span> [prɒp; 美 prɑ:p]
<span class="definition">n. [usually pl.] 戲劇或電影中用的道具：</span>He is responsible for all the stage props and lighting. 他負責全部舞臺道具和燈光。/ the backdrop and props for a stage show 一場舞臺劇的背景幕和道具           

<span class="vocabulary">**prompt**</span> [prɒmpt; 美 prɑ:mpt]
<span class="definition">vt.&vi. 給演員在忘詞的時候提醒詞語：</span>瞭解即可 <span class="definition">n. [C]（給演員的）提示詞：</span>瞭解即可

<span class="vocabulary">**produce**</span> [prə'dju:s] 
<span class="definition">vt. 拍攝或製作電影、戲劇等以供觀眾觀賞。整個流程通常包括了一系列的活動：</span>She produced a TV series about adopted children. 她拍了一部描寫收養兒童的電視系列片。

<span class="vocabulary">**production**</span> [prə'dʌkʃn] 
<span class="definition">n. [C, U] 電影、戲劇或廣播節目的製作、播出、上映；也可指藝術類的作品：</span>artistic production 藝術創造 / stage production 舞臺作品 / studio production 工作室作品 / TV, video production 電視作品；錄影作品 / film（尤英）, movie（尤美）production 電影作品 / theatrical production 戲劇作品

<span class="vocabulary">**make**</span> [meɪk] 
<span class="definition">vt. 出產影視作品。是一個簡單用詞：</span>She has made (= directed or acted in) several movies. 她已經出了好幾部電影。

<span class="vocabulary">**adapt**</span> [ə'dæpt] 
<span class="definition">vt. 將一本書（或一部戲劇）改編或改寫成戲劇、電影、電視節目等：</span>Her novel has been adapted for television. 她的小說已改編成了電視節目。
           
<span class="vocabulary">**transfer**</span> [trænsˈfɜ:(r)]
<span class="definition">vt.&vi. 改編：</span>The novel does not transfer well to the movies. 這部小說不太適宜改編成電影。

<span class="vocabulary">**plot**</span> [plɒt] 
<span class="definition">vt.（為小說、戲劇等）寫情節，設計故事情節，即佈局：</span>a tightly-plotted thriller 情節絲絲入扣的驚險小說

<span class="vocabulary">**set**</span> [set] 
<span class="definition">vt. 常用被動式，指把電影中的情節設定在某個背景下（如時間、地點等）：</span>The film is set in Paris in the 1920s. 這部電影以20世紀20年代的巴黎為背景。<span class="definition">adj. 指把電影中的情節設定在某個背景下（如時間、地點等）的，即以…為背景的：</span>The film is set in a small Northeastern town. 這部電影是以東北部一個小鎮為背景的。

<span class="vocabulary">**cut**</span> [kʌt] 
<span class="definition">vt. 1 電影、磁帶等的剪輯，剪接：</span>瞭解即可 <span class="definition">2 cut sth (from sth) 刪減（電影）：</span>The scene was cut from the movie. 這場戲在這部電影中被刪除了。<span class="definition">n. [C] cut (in sth) 對電影等所做的刪減，刪節：</span>瞭解即可

<span class="vocabulary">**rate**</span> [reɪt] 
<span class="definition">vt. 常用被動式，對電影或影片錄影進行分級（使其適合於特定的觀眾）：</span>瞭解即可
           
<span class="vocabulary">**rating**</span> [ˈreɪtɪŋ]
<span class="definition">n. [C] 對某部電影進行分級的數字或字母，即電影的等級：</span>The film was given a 15 rating by British censors. 英國審查員將這部電影定為15歲以上級。/ The movie carries an R rating. 這部電影帶有限制級R的標誌。
 
周邊詞彙補充：
· set可指電影的佈景。也可指電影的拍攝場地。
