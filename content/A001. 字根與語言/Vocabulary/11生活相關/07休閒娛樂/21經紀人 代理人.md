☀ <span class="category">**經紀人 代理人：**</span>
<span class="vocabulary">**agent**</span> ['eɪdӡənt] 
<span class="definition">n. [C] 演員、音樂人等的代理人或經紀人（負責與外界打交道）。是此義的一個基礎用詞：</span>theatrical agent 劇院經紀人 / talent agent 星探
           
<span class="vocabulary">**manager**</span> ['mænɪdӡə] 
<span class="definition">n. [C] 為演員、音樂人等處理商業事務的一個人，即（演員、音樂人等的）經紀人，其含義已為agent所包括：</span>瞭解即可

<span class="vocabulary">**scout**</span> [skaʊt]
<span class="definition">n. [C] 星探（即發掘明星的人），相當於talent scout：</span>瞭解即可

周邊詞彙補充：
· broker可指為別人進行買賣交易的人，即經紀人、掮客。還可指股票經紀人。
