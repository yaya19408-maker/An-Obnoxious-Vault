<span class="vocabulary">**container**</span> [kən'teɪnə] 
<span class="definition">n. [C]（用於儲存或運輸的）容器：</span>sealed, container 密封的容器 / storage container 貯藏容器 / cardboard, glass, metal, plastic container 硬紙板容器；玻璃容器；金屬容器；塑膠容器 / food, water container 盛食物的容器；盛水的容器 

<span class="vocabulary">**box**</span> [bɒks] 
<span class="definition">n. [C] 尤用於構成複合詞，指由木頭、硬紙板或金屬所做成的一個容器，側面和底部平整，常有一個蓋子，尤用於存放固體物品，可稱為盒子，匣子，箱子等：</span>cardboard, metal, plastic box 紙板箱；金屬盒；塑膠盒 / jewel box 珠寶盒 / gift box 禮品盒 / tool box 工具箱 / first-aid box 急救箱 / security box 保險箱 / collection box 募捐箱 / ballot box 投票箱 
           
<span class="vocabulary">**tub**</span> [tʌb]
<span class="definition">n. [C]（塑膠或紙質的）小型帶蓋容器（通常為圓形），用於盛裝食物等，如飯盒、食品盒：</span>a tub of margarine 一盒人造黃油 / He peeled the paper top off a little white tub and poured the cream into his coffee. 他把小白盒上的紙蓋撕去，然後把奶油倒進他的咖啡裡。

<span class="vocabulary">**bin**</span> [bɪn]
