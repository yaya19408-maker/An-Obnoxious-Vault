☀ <span class="category">**針具 釘子 夾子：**</span>
<span class="vocabulary">**pin**</span> [pɪn] 
<span class="definition">n. [C]（一端有尖的）大頭針或大頭釘：</span>safety pin 安全別針 / drawing pin（英）圖釘 / push pin（美）（通常用pushpi）按釘 / The map had a lot of little pins stuck into it. 地圖上釘了許多大頭針。

<span class="vocabulary">**nail**</span> [neɪl] 
<span class="definition">n. [C] 釘子（用於釘在牆上掛東西，或把木頭釘在一起）：</span>She hammered the nail in. 她把釘子敲了進去。/ six-inch nail 六英寸長的釘子 / iron nail 鐵釘 / finishing nail 飾面釘 / nail head 釘頭 / nail hole 釘眼 / nail bomb, nail gun 長釘炸彈；射釘槍
           
<span class="vocabulary">**screw**</span> [skru:]
<span class="definition">n. [C] 螺絲釘：</span>wood screw 木螺絲 / loose screw 鬆動的螺絲 / One of the screws is loose. 有一顆螺絲釘鬆了。/ Now tighten all the screws. 現在把全部螺絲釘都擰緊。/ Each bracket is fixed to the wall with just three screws. 每個托架只用了三顆螺釘固定在牆上。
           
<span class="vocabulary">**tack**</span> [tæk]
<span class="definition">n. 1 [C]（尤指把地毯釘在地板上的）平頭釘，大頭釘：</span>a carpet tack 地毯釘 <span class="definition">2 [C] [美] 圖釘。相當於drawing pin：</span>瞭解即可

<span class="vocabulary">**pin**</span> [pɪn] 
<span class="definition">n. [C] 某種夾子（如晾衣服的夾子）：</span>clothes pin（通常用clothespin）（美）（英式英語用clothes peg）（晾衣服的）衣夾
                
<span class="vocabulary">**clamp**</span> [klæmp]
<span class="definition">n. [C] 夾子、夾具、夾鉗：</span>瞭解即可        

<span class="vocabulary">**peg**</span> [peg]
<span class="definition">n. 1 [C]（也可用clothes peg）[均英]（美式英語用clothespin）晾衣夾子：</span>瞭解即可 <span class="definition">2 [C] 用以固定的短木頭、金屬或塑膠，如釘子，楔子：</span>He builds furniture using wooden pegs instead of nails. 他用木楔子取代釘子製作傢俱。
           
<span class="vocabulary">**clip**</span> [klɪp]
<span class="definition">n. [C]（常構成複合詞）（金屬或塑膠的）小夾子、回形針：</span>bicycle（英）, hair, nose, paper clip（騎腳踏車時用的）褲管夾；髮夾；鼻夾；回形針 / belt, pocket（美）clip 皮帶夾；口袋夾 / bull-dog（英）, spring clip 大鋼夾；彈簧夾 / alligator（尤美）, crocodile（尤英）clip 鱷魚嘴夾 / butterfly clip（美）蝶形夾 / She put a clip in her hair. 她別了一個髮夾。/ He put his bicycle clips on and set off. 他夾上腳踏車褲管夾出發了。/ Paper clips held the picture in place. 回形針把圖片固定住了。/ toe clips on a bicycle 腳踏車上的踏腳夾套

<span class="vocabulary">**wedge**</span> [wedʒ]
<span class="definition">n. [C]（木質、橡膠、金屬等材質的）楔子（用於使門開著、將兩個東西分開、將木頭或岩石劈開）：</span>He hammered the wedge into the crack in the stone. 他用錘子把楔子砸入石縫裡。/ I don't want to drive a wedge between the two of you (= to make you start disliking each other).（比喻用法）我不想在你們倆中間挑起不和。

相關詞義延伸：
· pin還可指作為首飾而起到裝飾作用的一端有尖的小型針狀物，如飾針，胸針等。也可指背後有針的一種首飾或徽章，可別在衣服上。亦可指某種夾子（如晾衣服的夾子）。或可指醫療上用於接骨的鋼釘。最後可指電器插頭上突出來的金屬之一（可插在插座裡），即管腳。

周邊詞彙補充：
· needle可指縫紉用的針、編織用的針（通常要兩個一起用）、注射器的針頭（用於注射藥物或抽血）、科學儀器上用來指示量度或方向的指標、唱機的唱針或磁針；
· hand可指鐘錶的指標。
