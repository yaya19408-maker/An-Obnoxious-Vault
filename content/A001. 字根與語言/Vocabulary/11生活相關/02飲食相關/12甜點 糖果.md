☀ <span class="category">**甜點 糖果：**</span>
<span class="vocabulary">**dessert**</span> [dɪ'zə:t] 
<span class="definition">n. [U, C]（一頓飯結束時所吃的）甜食，甜點。是此義的常規用詞，也是一個概括性用詞（各種甜食都是一種dessert）：</span>chocolate, fruit dessert 巧克力甜點；水果甜點 / dessert apple 蘋果甜點 / dessert wine 餐末甜酒 / dessert menu 甜點選單 / dessert plate 甜品盤

<span class="vocabulary">**sweet**</span> [swi:t] 
<span class="definition">n. 1 [C] [英] 一小塊甜食，通常用糖或巧克力製成，在各餐之間食用，可稱為糖果，相當於candy：</span>chewy sweet 耐嚼的糖果 / boiled sweet 硬糖 / cough, throat sweet 止咳糖；潤喉糖 / sweet shop 糖果店 <span class="definition">2 [C, U] [英]（一頓飯結束時所吃的）甜食，甜點，相當於afters，dessert或pudding：</span>Would you like some more sweet? 你想再吃點甜食嗎？

<span class="vocabulary">**candy**</span> ['kændɪ] 
<span class="definition">n. [U, C] 由糖或巧克力所製成的一種甜食，在各餐之間食用，可稱為糖果或巧克力；也可指一塊糖果或一塊巧克力。類似於sweet：</span>a box of candy 一盒糖果 / chocolate candy 巧克力糖果 / sugar-free candy 無蔗糖糖果 / candy wrapper 糖果紙 / candy bar 糖塊 / candy store 糖果店 / candy jar 糖果罐 / a piece of candy 一塊糖

<span class="vocabulary">**sugar**</span> ['ʃʊɡə] 
<span class="definition">n. 1 [U] 從各種植物汁液中提取的一種甜味物質，呈晶體狀，常為白色或棕色，用於烹飪、沏茶、泡咖啡等：</span>raw, unrefined sugar 原糖；粗糖 / processed, refined sugar 精製糖 / brown, white sugar 紅糖；白糖 / beet, cane sugar 甜菜糖；蔗糖 / spun sugar 棉花糖 / sugar beet, sugar cane 甜菜；甘蔗 / sugar industry, sugar mill 製糖業；糖廠 <span class="definition">2 [C] 一小湯匙的糖；一塊立方體的糖（用於放入茶、咖啡等之中）：</span>How many sugars do you take in coffee? 您在咖啡裡放幾塊方糖？<span class="definition">3 [C, usually pl.] [術語] 植物、水果等天然所含的任何一種甜味物質，也可稱為糖：</span>fruit sugars 果糖 / Fruit juices contain natural sugars. 果汁含有天然糖分。/ complex, simple sugar 多糖；單糖 / blood sugar 血糖
           
<span class="vocabulary">**lump**</span> [lʌmp]
<span class="definition">n. [C] [英] 小方塊形狀的糖，用於加在茶或咖啡中，相當於sugar lump：</span>a nugget of rough gold about the size of a lump of sugar 一塊方糖般大小的粗金 / 'No sugar,' I said, and Jim asked for two lumps. “不加糖，”我說道。吉姆則要了兩塊。
           
<span class="vocabulary">**gum**</span> [gʌm]
<span class="definition">n. 1 [U] 口香糖，相當於chewing gum：</span>瞭解即可 <span class="definition">2 [C]（堅硬的可咀嚼的）果味糖：</span>瞭解即可

<span class="vocabulary">**drop**</span> [drɒp] 
<span class="definition">n. [C] 小球狀糖果：</span>fruit drops 水果糖 / cough drops 止咳糖

<span class="vocabulary">**rock**</span> [rɒk] 
<span class="definition">n. [U] [英] 棒棒糖：</span>a stick of rock 一支棒棒糖

<span class="vocabulary">**chew**</span> [tʃu:] 
<span class="definition">n. [C] 口香糖：</span>瞭解即可
           
<span class="vocabulary">**mint**</span> [mɪnt]
<span class="definition">n. [C] 薄荷糖：</span>after-dinner mints 餐後薄荷糖

<span class="vocabulary">**cream**</span> [kri:m] 
<span class="definition">n. [C] 構成複合詞，內部有奶油等物質的一種糖果，如奶油夾心糖：</span>a chocolate cream 巧克力奶油夾心糖

<span class="vocabulary">**chocolate**</span> ['tʃɒkəlɪt] 
<span class="definition">n. 1 [U] 巧克力，朱古力（用可可豆製成的一種硬質的棕色甜食）：</span>milk chocolate 牛奶巧克力 / a bar of chocolate 一條巧克力 / chocolate cake 巧克力蛋糕 <span class="definition">2 [C] 由巧克力製成（或由巧克力覆蓋）的一種糖果，即巧克力糖；夾心巧克力糖：</span>soft-centred chocolate 軟心巧克力糖果

<span class="vocabulary">**ice-cream**</span> ['aɪskri:m] 
<span class="definition">n. [U, C] 冰激凌；一份冰激凌：</span>strawberry ice-cream 草莓冰激凌

<span class="vocabulary">**ice**</span> [aɪs] 
<span class="definition">n. [U] [尤美] 一種由搗碎並加了味道的冰所製成的甜食：</span>瞭解即可

<span class="vocabulary">**dumpling**</span> ['dʌmplɪŋ] 
<span class="definition">n. [C]（作為甜點食用的）布丁，尤指水果布丁：</span>apple dumplings 蘋果布丁

<span class="vocabulary">**fool**</span> [fu:l] 
<span class="definition">n. [U, C] [英] 常構成複合詞，指奶油果泥，是一種甜食，即把水果碾碎，再將其與奶油或奶油凍混合而製成：</span>rhubarb fool 奶油大黃泥

<span class="vocabulary">**split**</span> [splɪt] 
<span class="definition">n. [C] 用水果做成的一種甜食（尤指香蕉船，即將其縱向切成兩部分，然後在上面放上奶油、冰激凌等）：</span>a banana split 香蕉船
           
<span class="vocabulary">**whip**</span> [wɪp]
<span class="definition">n. [U, C] 用奶油、雞蛋、糖和水果混合而成的一種甜食：</span>瞭解即可

<span class="vocabulary">**cookie**</span> ['kʊkɪ] 
<span class="definition">n. [C] [尤美] 曲奇餅（一種小的扁平的供單人吃的甜食，常烘焙至酥脆）：</span>chocolate cookie 巧克力曲奇餅 / home-made cookie 自制曲奇餅

<span class="vocabulary">**popcorn**</span> ['pɒpkɔ:n] 
<span class="definition">n. [U]（玉米制成的）爆米花：</span>瞭解即可

☀ <span class="category">**糖衣 糖霜：**</span>
<span class="vocabulary">**icing**</span> [ˈaɪsɪŋ]
<span class="definition">n. [U]（撒在蛋糕上作為裝飾的）糖霜：</span>a birthday cake with yellow icing 掛黃色糖衣的生日蛋糕 / fondant, royal icing 軟糖衣；糖霜酥皮 / chocolate, lemon icing 巧克力味糖衣；檸檬味糖衣 / Pipe a little green icing around the strawberries. 在草莓周圍裱上點兒綠糖霜吧。/ icing sugar（英）糖粉 / icing bag 擠花袋
           
<span class="vocabulary">**frosting**</span> [ˈfrɒstɪŋ; 美 ˈfrɔ:st-]
<span class="definition">n. [U] [美]（用於裝點糕餅的）糖霜。相當於icing：</span>a huge pastry with green frosting on it 撒有綠色糖霜的一大塊油酥糕點 / buttercream, vanilla frosting 黃油乳糖霜；香草糖霜 / colored frosting 彩色糖霜 / Spread a little frosting on the cupcakes. 在紙杯蛋糕上撒點兒糖箱。
