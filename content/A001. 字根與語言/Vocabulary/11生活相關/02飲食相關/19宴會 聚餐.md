☀ <span class="category">**宴會 聚餐：**</span>
<span class="vocabulary">**dinner**</span> ['dɪnə] 
<span class="definition">n. [C] 大型正式的社交宴會：</span>formal dinner 正式宴會 / informal dinner 非正式宴會 / annual dinner 年度宴會 / wedding dinner（美）婚宴 / farewell dinner 送別宴會 / charity dinner 慈善宴會 / state dinner 國宴 / dinner party 宴會 / dinner guests 宴會賓客
           
<span class="vocabulary">**banquet**</span> [ˈbæŋkwɪt]
<span class="definition">n. 1 [C]（尤指特殊場合的）大型正式的宴會（常有人在此場合發表演講）：</span>five-course banquet 五道菜的宴會 / formal, official banquet 正式宴會；官方宴會 / royal, state banquet 王室宴會；國宴 /awards（美）, farewell, wedding banquet 頒獎宴會；告別宴；婚宴 / The winners will be announced at the annual awards banquet. 獲勝者將會在年度頒獎宴會上宣佈。/ medieval banquet 中世紀風格的宴會 / The kitchens are preparing for a lavish banquet. 廚房正在準備豐盛的宴會。/ The president attended a state banquet last night. 總統昨晚出席了一場國宴。/ a banquet for the president 為總統舉行的宴會 / Over 1 600 people attended a banquet in his honour/honor. 超過1,600人出席了為他舉行的宴會。<span class="definition">2 [C] 令人印象深刻的一頓大餐，如筵席：</span>瞭解即可

<span class="vocabulary">**barbecue**</span> ['bɑːbɪkju:] 
<span class="definition">n. [C] 戶外燒烤餐（本質上是一頓餐飯）；或戶外燒烤聚會：</span>Let’s have a barbecue! 咱們來一次戶外燒烤吧！

<span class="vocabulary">**roast**</span> [rəʊst] 
<span class="definition">n. [C] [美] 常構成複合詞，在某人的園子（或院子）裡舉行的戶外燒烤野餐聚會：</span>a hot dog roast 熱狗燒烤野餐

<span class="vocabulary">**drink**</span> [drɪŋk] 
<span class="definition">n. [pl.] [英] drinks 可指需要喝酒的一種社交場合，即酒宴，酒會：</span>Would you like to come for drinks on Sunday? 星期天來參加酒宴好嗎？/ a drinks party 酒會

<span class="vocabulary">**feast**</span> [fi:st] 
<span class="definition">n. [C] [正式]（尤指很多人參與的，慶祝某事的）大型宴會，即盛宴：</span>family, wedding feast 家庭宴會；婚宴 / Christmas feast 聖誕節大餐 / Villagers used to hold a great feast at harvest time. 村民們過去常常在收穫季節舉辦盛大的宴會。/ The salmon provide a delicious feast for the brown bear. 三文魚是棕熊的美餐。

<span class="vocabulary">**function**</span> ['fʌŋkʃn] 
<span class="definition">n. [C] 大型正式的社交宴會。不常用：</span>a private function hosted by him 由他做東的一場私人宴會 / functions such as weddings 如婚宴等的社交宴會

<span class="vocabulary">**blowout**</span> [ˈbləʊaʊt; 美 ˈbloʊ-]
<span class="definition">n. [usually sing.] [非正式] 大餐；盛宴（通常是家人和朋友為慶祝某事而舉辦的）：</span>a four-course blowout 有四道菜的大餐 / Once in a while we had a major blowout. 我們偶爾會吃頓大餐。

<span class="vocabulary">**benefit**</span> ['benɪfɪt] 
<span class="definition">n. [C] 一場活動，例如為某人或某個慈善組織而籌款所組織的一場宴會：</span>瞭解即可
           
<span class="vocabulary">**luncheon**</span> [ˈlʌntʃən]
<span class="definition">n. [C]（例如為慶祝重要事件，或進行慈善籌款而舉辦的）午宴：</span>charity luncheon 慈善午餐會 / fund-raising luncheon 籌款午宴 / buffet luncheon 自助午宴 / awards luncheon 頒獎午宴 / anniversary, business, literary luncheon（均尤英）週年紀念日午宴；商務午宴；文學交流午宴 / Earlier this month, a luncheon for former UN staff was held in Vienna. 本月早些時候在維也納設午宴招待了前聯合國工作人員。/ luncheon menu 午宴選單 / luncheon party（英）午宴聚會 / luncheon address, luncheon speech, luncheon speaker 午宴講話；午宴演講人
