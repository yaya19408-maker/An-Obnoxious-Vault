☀ <span class="category">**海峽 運河 水道：**</span>
<span class="vocabulary">**strait**</span> [streɪt] 
<span class="definition">n. [C] 聯結兩片海洋或其它大型水域的一條狹窄水道，即海峽。是此義的常規用詞。尤用於地名：</span>the Straits of Hormuz 霍爾木茲海峽 / Many vessels pass through the straits each year. 每年有很多艘船透過海峽。

<span class="vocabulary">**channel**</span> ['tʃænl] 
<span class="definition">n. 1 [C] 連線兩片水域的一條水道，尤指連線兩片海洋的水道，即海峽。不過此時遠不如strait常用：</span>瞭解即可 <span class="definition">2 [sing.] the Channel 專指英吉利海峽，也叫the English Channel：</span>the Channel Tunnel 英吉利海峽隧道

<span class="vocabulary">**sound**</span> [saʊnd] 
<span class="definition">n. [C] 常用於地名，指連線兩大片水域的一片狹長水域，即海峽，相當於strait。不常用：</span>瞭解即可

<span class="vocabulary">**course**</span> [kɔ:s] 
<span class="definition">n. [C, usually sing.] 江河的流向，河道：</span>the river’s twisting course 蜿蜒的河道 / The path follows the course of the river. 小徑沿著河道延伸。

<span class="vocabulary">**channel**</span> ['tʃænl] 
<span class="definition">n. [C] 河流或海岸附近用於行船的航道：</span>shipping channel 航道 / river channel 河道

<span class="vocabulary">**canal**</span> [kə'næl] 
<span class="definition">n. [C] 陸地上開鑿的一條狹長水道，以供行船，或把水引入某一區域，即運河：</span>the Panama Canal 巴拿馬運河 / ship canal（可行船的）運河 / canal boat 運河船隻 / canal bank 運河堤岸
