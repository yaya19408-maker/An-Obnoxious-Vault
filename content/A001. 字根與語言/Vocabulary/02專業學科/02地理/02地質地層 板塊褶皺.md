☀ <span class="category">**地質地層 板塊褶皺：**</span>
<span class="vocabulary">**shelf**</span> [ʃelf] 
<span class="definition">n. [C] 地質學中可指板子一樣的東西，尤指懸崖上（或海底）突出的一塊岩石，如陸架：</span>the continental shelf 大陸架

<span class="vocabulary">**deposit**</span> [dɪ'pɒzɪt] 
<span class="definition">n. [C] 地下自然形成的物質沉積層：</span>mineral, oil, gold, coal deposit 礦床；石油礦床；金礦；煤礦 / glacial, sedimentary deposit 冰河沉積；沉積礦床

<span class="vocabulary">**deposition**</span> [ˌdepəˈzɪʃn]
<span class="definition">n. [U, C] [術語] 沉積（如沉積到岩石或土壤上一層…），也可指此種沉積物：</span>marine, river deposition 海洋沉積物；河流沉積物 / deposition of silt along the coast 海岸上泥沙的沉積 

<span class="vocabulary">**sediment**</span> [ˈsedɪmənt]
<span class="definition">n. [U] [地理] 由水、風所帶來的，滯留在河流、湖泊、海洋等底部的沙子、石頭、泥等，即地理沉積物：</span>ocean sediments 海洋沉積物

<span class="vocabulary">**level**</span> ['levl] 
<span class="definition">n. [C] 土地的地層。並不強調沉積，也並不常用：</span>Archaeologists found pottery in the lowest level of the site. 考古學家在挖掘現場的最下層發現了陶器。

<span class="vocabulary">**fault**</span> [fɔ:lt] 
<span class="definition">n. [C]（地殼岩層的）斷層處：</span>earthquake, geological fault 地震斷層；地質斷層 / fault line 斷層線 / fault zone 斷層帶

<span class="vocabulary">**fold**</span> [fəʊld] 
<span class="definition">n. [C]（地殼岩石層的）褶皺：</span>瞭解即可

<span class="vocabulary">**plate**</span> [pleɪt] 
<span class="definition">n. [C]（構成地球表面且緩慢移動的）板塊：</span>the Pacific plate 太平洋板塊 / tectonic plates 地殼板塊
           
<span class="vocabulary">**crust**</span> [krʌst]
<span class="definition">n. [C] 地殼：</span>Earthquakes leave scars in the earth's crust. 地震會在地殼上留下裂痕。
 
相關詞義延伸：
· deposit還可指河流、洪水等底部的沉積物；亦可指（河流）使沉積。
