☀ <span class="category">**系統平臺 架構原理：**</span>
<span class="vocabulary">**system**</span> ['sɪstəm] 
<span class="definition">n. [C] 被連線在一起（或在一起執行）的一組事物，如一套裝置等，即系統：</span>computer system 計算機系統 / operating system 作業系統 / database system 資料庫系統 / an information retrieval system 資訊檢索系統

<span class="vocabulary">**platform**</span> ['plætfɔ:m] 
<span class="definition">n. [C] 計算機平臺或軟體平臺。強調為各種計算機操作提供一套基礎系統：</span>Windows platform Windows作業系統平臺 / software platform 軟體平臺

<span class="vocabulary">**environment**</span> [ɪn'vaɪrənmənt] 
<span class="definition">n. [sing.] 計算機的執行環境或工作平臺。涉及到與某種操作有關的系統中的所有因素，其含義已將platform涵蓋，可以說platform只是environment的一部分：</span>a desktop development environment 桌面開發環境

<span class="vocabulary">**architecture**</span> ['ɑːkɪtektʃə] 
<span class="definition">n. [C, U] 計算機系統的結構、層次：</span>system, hardware, information, network, processor, software architecture 系統架構；硬體架構；資訊架構；網路架構；處理器架構；軟體架構

<span class="vocabulary">**gate**</span> [ɡeɪt] 
<span class="definition">n. [C] 邏輯閘或邏輯閘，即以兩種方式之一對所輸入資料進行輸出的電子開關，也可用logic gate：</span>瞭解即可
           

<span class="vocabulary">**logic**</span> [ˈlɒdʒɪk; 美 ˈlɑ:dʒɪk]

<span class="definition">n. [U] [計算機] 邏輯系統；操作規則：</span>瞭解即可

· fuzzy可指（計算機）模糊（邏輯）的。
