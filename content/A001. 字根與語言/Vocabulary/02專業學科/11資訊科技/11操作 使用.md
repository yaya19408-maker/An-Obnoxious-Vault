☀ <span class="category">**操作 使用：**</span>
<span class="vocabulary">**surf**</span> [sɜ:f; 美 sɜ:rf]
<span class="definition">vt. ~ the Net/Internet（在網際網路上）衝浪，即瀏覽（網際網路）：</span>I was surfing the Net looking for information on Indian music. 我正上網查詢關於印度音樂的資料。/ No one knows how many people currently surf the Net. 沒人知道現在有多少人在網上衝浪。

<span class="vocabulary">**click**</span> [klɪk] 
<span class="definition">vt.＆vi. [非正式] click (on sth)（在計算機螢幕上用滑鼠）點選：</span>Click the OK button to start. 單擊確認鍵啟動。<span class="definition">n. [C]（對計算機滑鼠的）點選：</span>single click 單擊 / a double click on the filename 雙擊檔名 / mouse click 滑鼠點選 

<span class="vocabulary">**search**</span> [sə:tʃ] 
<span class="definition">n. [C] 在資料庫中或網際網路上進行的搜尋、檢索：</span>computer, database, online, Web search 計算機搜尋；資料庫搜尋；線上搜尋；網路搜尋 / I did an Internet search for free music sites. 我在網際網路上搜尋免費的音樂網站。/ advanced, keyword search 高階搜尋；關鍵詞搜尋 / search engine 搜尋引擎 / search box 搜尋欄 / search results 搜尋結果
           
<span class="vocabulary">**retrieve**</span> [rɪˈtri:v]
<span class="definition">vt. [計算機]（從計算機的儲存器或人的大腦中）檢索資料或資訊：</span>to retrieve data 檢索資料 / to retrieve information from the database 從資料庫檢索資料 / to retrieve documents from the disk 從磁碟中檢索文件 / The program allows you to retrieve items quickly by searching under a keyword. 這個程式透過關鍵詞進行搜尋，能讓你迅速獲取資料項。/ As the child gets older, so his or her strategies for storing and retrieving information improve. 隨著孩子們長大，他們儲存和檢索資訊的能力也會提高。

<span class="vocabulary">**download**</span> ['daʊn͵ləʊd] 
<span class="definition">vt. 下載：</span>to download the material to the disk 將資料下載到磁碟裡

· 表示“儲存”的詞群見[[09資料資訊 儲存歸檔]]
           
<span class="vocabulary">**install**</span> [ɪnˈstɔ:l]
<span class="definition">vt.（在計算機上）安裝程式：</span>I'll need some help installing the software. 我得找人幫忙安裝這個軟體。/ A Trojan horse, secretly installed on your computer, can change your Internet settings. 偷偷安裝到計算機上的木馬程式會改變網際網路設定。/ Linux can be installed on most PCs or Macs. 多數個人電腦和蘋果電腦都可以安裝Linux作業系統。

<span class="vocabulary">**update**</span> [ʌp'deɪt] 
<span class="definition">vt. 更新計算機程式等：</span>to updated my software 更新我的軟體 <span class="definition">n. [C] 計算機程式的更新：</span>the latest update 最近更新 / regular update 定期更新 / automatic update 自動更新 / online update 線上更新 / antivirus update 防毒軟體更新

<span class="vocabulary">**burn**</span> [bə:n] 
<span class="definition">vt. 燒錄光碟等：</span>瞭解即可

<span class="vocabulary">**stream**</span> [stri:m] 
<span class="definition">vt. 用流式技術來播放影片或音訊（即無須待整個檔案已下載到計算機上便可播放網際網路上的影片或音訊檔案）：</span>瞭解即可

<span class="vocabulary">**pause**</span> [pɔ:z] 
<span class="definition">vt.（按暫停鍵）暫停播放一個磁帶、唱片等：</span>to pause the video 暫停播放影片

<span class="vocabulary">**freeze**</span> [fri:z] 
<span class="definition">vt. 使影片、電影等的畫面定格（以檢視某一圖片）：</span>to freeze the action 將畫面定格

<span class="vocabulary">**port**</span> [pɔ:t] 
<span class="definition">vt. 將軟體（或程式）從一個系統或一部機器移植到另一個系統或機器上：</span>瞭解即可

<span class="vocabulary">**cut**</span> [kʌt] 
<span class="definition">vt.＆vi. 對文件等進行剪下：</span>You can cut and paste between different programs. 你可以在不同的程式之間剪下和貼上。
           
<span class="vocabulary">**paste**</span> [peɪst]
<span class="definition">vt.&vi. [計算機] 貼上（文字）：</span>This function allows you to cut and paste text. 本功能可使你剪下並貼上文字。/ It's quicker to cut and paste than to retype. 剪下和貼上比重新打字要快。

<span class="vocabulary">**format**</span> ['fɔ:mæt] 
<span class="definition">vt. [術語] 在螢幕上安排文字的版式（使其可以在電腦上以某種方式顯示）：</span>When text is saved from a Web page, it is often very badly formatted with many short lines. 文字從網頁上另存時版式通常會變亂，出現很多短行。
           
<span class="vocabulary">**tile**</span> [taɪl]
<span class="definition">vt. [計算機] 在計算機螢幕上平鋪顯示各視窗（使其彼此不被蓋住）：</span>瞭解即可
           
<span class="vocabulary">**portrait**</span> [ˈpɔ:treɪt; -trət; 美 ˈpɔ:rtrət]
<span class="definition">adj. [計算機]（檔案頁面）豎向的；縱向列印格式的：</span>瞭解即可
          
<span class="vocabulary">**landscape**</span> [ˈlændskeɪp]
<span class="definition">n. [U] [術語]（檔案的）橫向列印格式：</span>Select the landscape option when printing the file. 列印檔案時應選擇橫向列印格式選項。

<span class="vocabulary">**insert**</span> [ɪn'sə:t] 
<span class="definition">vt. insert sth (into sth) 將…新增到一段文字中：</span>Position the cursor where you want to insert a word. 把游標移到你想插入字詞的地方。

· 表示“錄入”、“打字”的詞群見[[22打字 錄入]]

<span class="vocabulary">**select**</span> [sɪ'lekt] 
<span class="definition">vt. 在計算機螢幕上選定；（尤指從一個選單中）選擇：</span>Select the text you want to format by holding down the left button on your mouse. 按住滑鼠左鍵選取你想要格式化的文字。/ Select ‘New Mail’ from the ‘Send’ menu. 從“傳送”選單中選擇“新郵件”。

<span class="vocabulary">**wipe**</span> [waɪp] 
<span class="definition">vt. wipe sth (off/off sth) 將資訊、聲音、影象等從計算機、磁帶或錄影機上清除，相當於erase。除了清除各種檔案，還可以清除各種痕跡，且一般是從磁碟中徹底清除：</span>You must have wiped off that programme I recorded. 你一定是把我錄製的節目給抹掉了。

<span class="vocabulary">**delete**</span> [dɪ'li:t] 
<span class="definition">vt. delete sth (from sth) 刪除（電腦中的內容）。一般僅指刪除檔案（不太用於表示清除痕跡），且被delete的檔案有時只是被挪到了回收站中：</span>to deletes files 刪除文件
           
<span class="vocabulary">**wrap**</span> [ræp]
<span class="definition">vt.&vi. [計算機] ~ (sth) (around/round) 使文字自動換行：</span>How can I wrap the text around? 我怎麼才能使文字換行呢？/ The text wraps around if it is too long to fit the screen. 如果文字太長，在顯示屏放不下的話，會自動換行。

<span class="vocabulary">**access**</span> ['ækses] 
<span class="definition">vt. 訪問或存取計算機檔案（即開啟一份計算機檔案以獲取或新增資訊）：</span>to access confidential files 存取機密檔案 <span class="definition">n. [U] access (to sth) 使用某種資訊科技的機會或權利：</span>You need a password to get access to the computer system. 使用這個計算機系統需要口令。/ broadband, Internet, wireless access 寬頻接入；網際網路接入；無線接入
           
<span class="vocabulary">**dock**</span> [dɒk; 美 dɑ:k]
<span class="definition">vt. [計算機] 將電腦連線到擴充套件塢：</span>瞭解即可

<span class="vocabulary">**open**</span> ['əʊpən] 
<span class="definition">vt.＆vi. 啟動或開啟（計算機程式或檔案）：</span>to open the file 開啟檔案

<span class="vocabulary">**refresh**</span> [rɪ'freʃ] 
<span class="definition">vt.＆vi. 計算機技術中可指獲取最新資訊（如透過在螢幕上點選按鈕而獲取一個網頁的最新資訊），可稱為重新整理，更新：</span>Click here to refresh this document. 點選此處以重新整理此檔案。/ The page refreshes automatically. 頁面會自動更新。

<span class="vocabulary">**lock**</span> [lɒk] 
<span class="definition">vt. 給計算機檔案加鎖：</span>These files are locked to protect confidentiality. 為了保密，這些檔案都加了鎖。
           
<span class="vocabulary">**debug**</span> [ˌdi:ˈbʌg]
<span class="definition">vt. [計算機] 修復錯誤程式：</span>The production lines ground to a halt for hours while technicians tried to debug software. 生產線停工達數小時之久，在此期間技術人員試圖糾除軟體錯誤。
           
<span class="vocabulary">**filter**</span> [ˈfɪltə(r)]
<span class="definition">vt. 用一個特殊的計算機程式來過濾（傳送到你計算機上的電子郵件或網站的內容）：</span>The software filters out Internet sites whose content is not suitable for children. 這個軟體可篩除含兒童不宜內容的網際網路網站。

<span class="vocabulary">**drag**</span> [dræɡ] 
<span class="definition">vt. 用滑鼠在螢幕上拖拽文字、圖示等：</span>瞭解即可

<span class="vocabulary">**exit**</span> ['eksɪt, -zɪt] 
<span class="definition">vt.＆vi. 退出計算機程式。是此義的常規用詞，最為常用：</span>to exit the database 退出資料庫 / to exit from this page 退出本頁面 / to exit the application 退出該應用程式

<span class="vocabulary">**quit**</span> [kwɪt] 
<span class="definition">vt.＆vi. 退出一個計算機程式（或退出一個應用程式）。不過不如exit常用：</span>瞭解即可

<span class="vocabulary">**network**</span> ['netwə:k] 
<span class="definition">vt. 將一些計算機或其他裝置連線成網路（以共享裝置或資訊）：</span>瞭解即可

<span class="vocabulary">**sort**</span> [sɔ:t] 
<span class="definition">n. [sing.] 計算機技術中可指將資料按照某種順序進行排列的過程，即分類、排序：</span>to do a sort 進行分類
           
<span class="vocabulary">**grab**</span> [græb]
<span class="definition">n. [C] [計算機] 抓取（或擷取、採集）的影象：</span>a screen grab from Wednesday's programme 從星期三的節目中抓取的熒屏影象
           
<span class="vocabulary">**capture**</span> [ˈkæptʃə(r)]
<span class="definition">vt. 把…輸入計算機，使其可以被計算機利用，即（計算機）採集資訊：</span>瞭解即可 <span class="definition">n. [U] 計算機的資訊採集：</span>data capture 資料採集

周邊詞彙補充：
· play可表示播放磁帶或唱片等；
· write可表示以書寫的形式來程式設計序，也可表示將資料寫入電腦儲存器（或磁碟）；
· visit可指訪問（網站）；
· escape可指電腦鍵盤上的退出鍵。
