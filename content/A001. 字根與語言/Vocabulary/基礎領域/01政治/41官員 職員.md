☀ <span class="category">**官員 職員：**</span>
<span class="vocabulary">**official**</span> [ə'fɪʃl]  
<span class="definition">n. [C] 常構成複合詞，指政府官員、政府要員：</span>high-ranking, prominent, senior, top official 高階職員；高階官員；要員 / junior, minor official 初級職員；小職員 / local, provincial official 地方官員；省級官員 / public official 公務員 / administration, departmental, federal, government, state official 行政主管；部門主管；聯邦要員；政府要員；州要員 / the official responsible for handling the case 負責處理此案的官員 / corrupt official 貪官 / council, court, customs, intelligence, military, ministry（尤英）, royal official 議會官員；法院官員；海關官員；情報官員；軍方官員；部級官員；王室官員

<span class="vocabulary">**officer**</span> ['ɒfɪsə]  
<span class="definition">n. [C] 常構成複合詞，指政府中擁有權力的官員或職員：</span>administrative, customs, financial, liaison, press, welfare（尤英）, prison officer 行政官員；海關官員；財政官員；聯絡官員；新聞官員；福利官員；典獄官員 / officers of state (= ministers in the government) （政府各部）部長 / regional officer 地方官員
           
<span class="vocabulary">**chancellor**</span> [ˈtʃɑ:nsələ(r); 美 ˈtʃæns-]
<span class="definition">n. [C]（常用於頭銜）用於英國某些高階政府官員的頭銜中：</span>the Lord Chancellor (= a senior law official) 大法官
           
<span class="vocabulary">**governor**</span> [ˈgʌvənə(r); 美 -vərn-]
<span class="definition">n. 1 [C]（也可用Governor）被別國統治的一個國家（或地區）中的首腦，如總督：</span>the former governor of the colony 該殖民地的前總督 / colonial, district, provincial, regional governor 殖民地總督；轄區長官；省長；地方長官 / She was appointed as acting governor until an election could be held. 她被任命為代省長，任期到舉行選舉為止。/ military governor 司令官 <span class="definition">2 [C]（也可用Governor）（美國州的）州長：</span>state governor 州長 / the governor of Arizona 亞利桑那州州長 / the Arizona governor 亞利桑那州州長 / Governor Tom Kean 湯姆·基恩州長
           
<span class="vocabulary">**mayor**</span> [meə(r); 美 ˈmeɪər]
<span class="definition">n. 1 [C]（英格蘭、威爾士和北愛爾蘭由議員選舉產生的）鎮長，市長，郡長：</span>the Lord Mayor of London 倫敦市長 <span class="definition">2 [C]（民選的）市長，鎮長：</span>the Mayor of New York 紐約市長 / the mayor of Moscow 莫斯科市市長 / Mayor Bob Anderson 鮑勃·安德森市長 / Democratic mayor 民主黨的市長 / right-wing mayor 右翼市長 / socialist mayor 奉行社會主義的市長 / big-city, small-town mayor（均美）大城市市長；小城市市長 / deputy mayor 副市長 / local mayor 當地市長
           
<span class="vocabulary">**commissioner**</span> [kəˈmɪʃənə(r)]
<span class="definition">n. [C]（政府部門的）首長：</span>the agriculture commissioner 農業廳長 / the health commissioner 衛生廳長 / Commissioner Rhodes was unavailable for comment. 羅茲局長無暇評論。
           
<span class="vocabulary">**collector**</span> [kəˈlektə(r)]
<span class="definition">n. [C]（南亞某些國家的）地方行政長官：</span>瞭解即可

說明：①official和officer表“政府官員”時，二者無明顯差別，很多情況下可通用，只是official更常用；②officer可專門指“軍官”、“警官”，official雖然也可指“軍官”，但並不常見。
