☀ <span class="category">**交通網：**</span>
<span class="vocabulary">**network**</span> ['netwə:k] 
<span class="definition">n. [C] 複雜的相互交叉並相互連線的一些道路、線路。強調各個交通站點之間的交通往來活動：</span>train, road network 鐵路網；道路網 / an extensive network of underground tunnels 四通八達的地下隧道網 / canal network 運河網

<span class="vocabulary">**web**</span> [web] 
<span class="definition">n. [C] 彼此緊密聯絡在一起的一些道路，即道路網。不過其一般只強調道路的複雜交錯，而不太強調各交通站點之間的交通往來活動：</span>a web of streets 縱橫交錯的街道
