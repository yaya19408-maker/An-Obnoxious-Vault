☀ <span class="category">**飛機 航班：**</span>
<span class="vocabulary">**aircraft**</span> ['eəkrɑːft] 
<span class="definition">n. [C] 指能夠飛行，同時能夠攜帶乘客或貨物的任何一種交通工具，本身可表示任何飛機，也可和其它詞連用以表示某種型別或用途的飛機。其單複數相同，也就是沒有aircrafts這種形式：</span>fixed-wing, jet, supersonic aircraft 固定翼飛機；噴氣式飛機；超音速飛機 / twin-engined aircraft 雙引擎飛機 / unmanned aircraft 無人駕駛的飛機 / cargo, civil, civilian, commercial, passenger, private aircraft 貨機；民用飛機；商用飛機；客機；私人飛機 / attack, fighter, military, reconnaissance, stealth, transport aircraft 攻擊機；戰鬥機；軍用飛機；偵察機；隱形飛機；運輸機 / enemy aircraft 敵機 / aircraft parts 飛機零件 / aircraft hangar 飛機庫 / aircraft carrier 航空母艦

<span class="vocabulary">**airplane**</span> ['eə͵pleɪn] 
<span class="definition">n. [C] [美] 飛機，與plane和aeroplane[英]同義：</span>an airplane crash 飛機墜毀 / military airplane 軍用飛機

<span class="vocabulary">**air**</span> [eə] 
<span class="definition">n. [C] 飛機。是一個簡單用詞：</span>air travel 乘飛機旅行 / air fare 飛機票價 / air crash, air disaster 飛機失事；空難 / air attack, air strike, air defence/defense 空襲；防空
           
<span class="vocabulary">**craft**</span> [krɑ:ft; 美 kræft]
<span class="definition">n. [C]（其複數形式為craft）飛機：</span>瞭解即可

<span class="vocabulary">**plane**</span> [pleɪn] 
<span class="definition">n. [C] 飛機。是此義的基礎用詞，使用比較廣泛：</span>light, small plane 輕型飛機；小型飛機 / commercial, passenger plane 商用飛機；客機 / cargo plane 貨機 / transport plane 運輸機 / fighter, spy plane 戰鬥機；偵察機 / jet plane 噴氣式飛機 / private plane 私人飛機 / plane crash 飛機失事 / plane ticket 機票 / The plane was carrying 80 people. 飛機上載有80名乘客。

<span class="vocabulary">**flight**</span> [flaɪt] 
<span class="definition">n. [C] 一起飛行的一群飛機：</span>an aircraft of the flight 一個機群中的一架
           
<span class="vocabulary">**fleet**</span> [fli:t]
<span class="definition">n. [C] ~ (of sth)（共同行動或隸屬於同一組織機構的）一群飛機，如機群：</span>瞭解即可

<span class="vocabulary">**helicopter**</span> ['helɪkɒptə] 
<span class="definition">n. [C] 直升飛機：</span>police helicopter 警用直升機 / civilian, private helicopter 民用直升機；私人直升機 / rescue helicopter 救援直升機 / ambulance helicopter（英）救護直升機 / armed helicopter 武裝直升機 / transport helicopter 運輸直升機 / cargo helicopter 貨運直升機 / to take a helicopter tour 乘坐直升機參觀 / helicopter attack, helicopter strike 直升機空襲 / helicopter rescue, helicopter search 直升機救援；直升機搜尋 / helicopter pilot 直升機駕駛員 / helicopter crash 直升機墜落 

<span class="vocabulary">**jet**</span> [dӡet] 
<span class="definition">n. [C] 噴氣式飛機，即由噴氣式發動機所驅動的飛機：</span>passenger jet 噴氣式客機 / military jet 軍用噴氣式飛機 / jet fighter 噴氣式戰鬥機 / jet engine 噴氣式發動機

<span class="vocabulary">**shuttle**</span> ['ʃʌtl] 
<span class="definition">n. 1 [C] 在兩地之間頻繁而規律地往來的飛機，即班機。強調往來穿梭的特點：</span>to take the shuttle 搭乘班機 / diplomacy shuttle（比喻用法）穿梭外交 <span class="definition">2 [C]（在地球和宇宙空間站之間穿梭從而為兩者運送物資的）太空梭：</span>space shuttle 太空梭 / shuttle landing 太空梭著陸

<span class="vocabulary">**service**</span> ['sə:vɪs] 
<span class="definition">n. [C, usually sing.] 在某一時間固定地開往某地的一架飛機，即班機，航班。強調交通的單向性：</span>US Airways also added service recently to the Dominican Republic. 美國航空公司最近也增加了飛往多明尼加共和國的航班。

<span class="vocabulary">**flight**</span> [flaɪt] 
<span class="definition">n. [C] 行駛於某個特定路線的飛機，即航班。強調航線的固定性：</span>BA flight 286 was two hours late. 英國航空286次航班晚點兩個小時。/ relief flights 用於救援的班機 / inbound, return flight 進港航班；返程航班 / connecting flight 轉接航班 / shuttle flight 往返航班 / regular, scheduled flight 定期航班 / charter flight 包機 / weekly flight 每週一次的航班 / direct flight 直達航班 / short flight 短途航班 / night flight 夜間航班 / cargo, passenger flight 貨運航班；客運航班 / cheap flight 廉價航班 / flight number 航班號 / flight schedule 航班時刻表 / flight delay 航班晚點 / flight cancellation 航班取消
           
<span class="vocabulary">**tanker**</span> [ˈtæŋkə(r)]
<span class="definition">n. [C] 運輸大量石油、天然氣等的一輛飛機：</span>The Air Force urgently needs a new tanker aircraft. 空軍迫切需要一種新型加油機。

<span class="vocabulary">**drone**</span> [drəʊn; 美 droʊn]
<span class="definition">n. [C]（地面控制的）無人駕駛飛機：</span>瞭解即可
            
<span class="vocabulary">**balloon**</span> [bə'lu:n] 
<span class="definition">n. [C] 也可用hot-air balloon，大型的熱氣球（常帶有一個籃子，可運送乘客）：</span>hot-air, weather balloon 熱氣球；氣象氣球
           
<span class="vocabulary">**parachute**</span> [ˈpærəʃu:t]
<span class="definition">n. [C]（也可用非正式的chute）（用以投放人員或物品的）降落傘：</span>a parachute regiment 空降兵團 / They fell 41,000 ft. before opening their parachutes. 他們下降了41,000英尺後才拉開傘。/ UN troops could be landed by helicopter or even by parachute. 聯合國部隊可以乘直升機著陸，甚至可以傘降。/ parachute drop, parachute jump 空投；跳傘/ He made his first parachute jump at the age of seventy. 他在七十歲時首次跳傘。/ parachute instructor（尤英）跳傘教練 / Supplies were dropped by parachute. 補給透過降落傘空投下來。/ Planes dropped supplies by parachute. 飛機用降落傘空投補給。
           
<span class="vocabulary">**chute**</span> [ʃu:t]
<span class="definition">n. [C] [非正式] 降落傘。相當於parachute：</span>You can release the chute with either hand, but it is easier to do it with the left. 你隨便哪隻手都能開啟降落傘，不過用左手容易些。
 
<span class="vocabulary">**coach**</span> [kəʊtʃ] 
<span class="definition">n. [U] [美] 飛機的經濟艙：</span>coach fares 經濟艙票價 / coach seats 經濟艙座位 / to fly coach 坐飛機的經濟艙

<span class="vocabulary">**port**</span> [pɔ:t] 
<span class="definition">n. [U] 飛機等的左側，即左舷：</span>瞭解即可

<span class="vocabulary">**forward**</span> ['fɔ:wəd] 
<span class="definition">adv. [術語] 在（或向）飛機的前部：</span>瞭解即可 <span class="definition">adj. [術語] 只用於名詞前，位於前部的（尤指位於飛機或其他交通工具的前部的）：</span>the plane’s forward door 飛機的前艙門

<span class="vocabulary">**nose**</span> [nəʊz] 
<span class="definition">n. [C] 交通工具（如飛機、太空船等）的前部：</span>the nose of the aeroplane 飛機機頭
           
<span class="vocabulary">**canopy**</span> [ˈkænəpi]
<span class="definition">n. [C] 飛機的座艙蓋：</span>瞭解即可

<span class="vocabulary">**hold**</span> [həʊld] 
<span class="definition">n. [C]（飛機上用於裝載和運輸貨物的）貨倉：</span>瞭解即可

<span class="vocabulary">**hatch**</span> [hætʃ] 
<span class="definition">n. [C]（也可用hatchway）飛機底部的一個開口或門，搬運貨物要經由此處，即貨艙門：</span>瞭解即可

<span class="vocabulary">**wing**</span> [wɪŋ] 
<span class="definition">n. [C] 飛行器的翅膀或機翼：</span>wing of a plane 飛機的機翼 / left, port wing 左機翼 / right, starboard wing 右機翼 / fixed wing 固定翼 / folding wing 可摺疊機翼 / wing tip 翼尖
           
<span class="vocabulary">**flap**</span> [flæp]
<span class="definition">n. [C] 飛行器的襟翼（是機翼的一部分，能上下移動以控制飛行器的上下運動）：</span>the sudden slowing as the flaps were lowered 襟翼放下時的突然減速 / The pilot lowered the flaps as the aircraft came into land. 飛機著陸時飛行員放下了襟翼。
              
<span class="vocabulary">**blade**</span> [bleɪd]
<span class="definition">n. [C] 直升機上的旋轉的葉片、槳葉：</span>the blades of a propeller 螺旋槳葉 / a helicopter's rotor blades 直升機的旋翼 

<span class="vocabulary">**prop**</span> [prɒp; 美 prɑ:p]
<span class="definition">n. [C] 螺旋槳（飛機或輪船的推進器），相當於propeller：</span>瞭解即可
                      
<span class="vocabulary">**screw**</span> [skru:]
<span class="definition">n. [C]（船隻、飛行器的）螺旋槳：</span>瞭解即可

<span class="vocabulary">**elevator**</span> [ˈelɪveɪtə(r)]
<span class="definition">n. [C] 飛行器尾部的升降舵：</span>瞭解即可

<span class="vocabulary">**stick**</span> [stɪk] 
<span class="definition">n. [C] [尤美，非正式]（飛機的）操縱桿，駕駛杆：</span>瞭解即可

<span class="vocabulary">**ceiling**</span> ['si:lɪŋ] 
<span class="definition">n. [C] [術語] 飛行器（如飛機）的最高飛行限度：</span>瞭解即可
