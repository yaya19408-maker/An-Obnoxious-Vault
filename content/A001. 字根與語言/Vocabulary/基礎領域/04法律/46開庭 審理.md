☀ <span class="category">**開庭 審理：**</span>
<span class="vocabulary">**sit**</span> [sɪt] 
<span class="definition">vi.（法庭）開庭：</span>瞭解即可

<span class="vocabulary">**session**</span> ['seʃn] 
<span class="definition">n. [C] 法庭開庭處理業務的一段時間。狹義上可指每日的“開庭期”或“庭審期”；但在廣義上也可指從第一次開庭到最終休庭之間的整個時間段（此期間也許包括若干天、若干周，甚至更長的時間），在此意義上與term或sitting同義，很多權威人士都將session與term作為同義詞而互換使用：</span>The court is now in session. 法庭正處於庭審期。

<span class="vocabulary">**judge**</span> [dӡʌdӡ] 
<span class="definition">vt. 在法庭上做出審判：</span>to judge a case 審理案件

<span class="vocabulary">**trial**</span> ['traɪəl] 
<span class="definition">n. [U, C] 通常意義上的“審理”、“審判”，可以包括從法院立案到做出裁決的全過程，也可指法庭審訊（即“庭審”）。是此義的常規用詞：</span>full trial 正式的審訊 / criminal trial 刑事審訊 / fraud trial 欺詐罪審訊 / court, jury trial 法庭的審訊；陪審團的審問 / civil trial 民事審判 / public trial 公審 / show trial（為宣傳目的而舉行的）擺樣子的公審 / summary trial（尤英）即決審訊 / trial court, trial judge, trial lawyer 初審法院；承審法官；庭審律師 / trial verdict 法庭的裁決 / trial procedure 審訊的程式 / She faces trial for murder. 她因涉嫌謀殺而面臨審訊。/ He did not receive a fair trial. 他沒有受到公正的審判。

<span class="vocabulary">**arbitration**</span> [ˌɑ:bɪˈtreɪʃn; 美 ˌɑ:rb-]
<span class="definition">n. [U]（第三方做出的）仲裁：</span>binding arbitration 有約束力的仲裁 / Both sides have agreed that the arbitration will be binding. 雙方都贊同仲裁具有約束力。/ independent（尤英）, international 獨立仲裁；國際仲裁 / the independent arbitration service, ACAS 獨立的仲裁部門，諮詢調解仲裁處 / salary arbitration（尤美）薪資仲裁 / Both parties agreed to independent arbitration. 雙方都同意進行獨立仲裁。/ Both sides in the dispute have agreed to go to arbitration. 爭執雙方已同意提請仲裁。/ The matter will go to arbitration. 這件事將訴諸仲裁。/ The matter will be settled by arbitration. 這件事將透過仲裁解決。/ arbitration procedure（尤英）, arbitration process, arbitration scheme（英）, arbitration service（英）, arbitration system 仲裁程式；仲裁過程；仲裁方案；仲裁服務；仲裁製度 / arbitration case 仲裁案例 / arbitration hearing 仲裁聽證會 / arbitration panel 仲裁小組 / He had won an arbitration award against the company. 仲裁裁決認定他狀告公司獲勝。

<span class="vocabulary">**hear**</span> [hɪə] 
<span class="definition">vt.（在法庭上對案件進行）聽審或審理：</span>The appeal was heard in private. 這樁上訴案不公開審理。/ Today the jury began to hear the evidence. 今天陪審團開始聽審。

<span class="vocabulary">**hearing**</span> ['hɪərɪŋ] 
<span class="definition">n. [C]（為犯罪所舉行的）聽審，審訊，審理；（為投訴、事件或問題所舉行的）聽證會。其目的在於為爭議各方（尤其是可能被剝奪權益的一方）創造陳述意見的機會。舉辦者可以是司法機關，也可以是立法機關或行政機關：</span>court hearing 庭審 / final, preliminary hearing 終審；預審 / formal hearing 正式審理 / open, public hearing 公開審理；公開的聽證會 / private, secret hearing 秘密審理 / appeal, disciplinary, pretrial hearing 上訴聽證會；紀律聆訊；審判前聽證會 / congressional hearing 國會聽證會 / to hold the hearing in public 舉行公開聽證會 / The judge dismissed the case without a hearing. 法官還未審理就駁回了此案。

<span class="vocabulary">**try**</span> [traɪ] 
<span class="definition">vi. try sb (for sth) / try sth在法庭上檢察證據並判定某人是否有罪，即審判、審理、審訊：</span>He was tried for murder. 他因謀殺罪而受審。/ The case was tried before a jury. 此案是由陪審團參與審理的。

<span class="vocabulary">**justice**</span> ['dӡʌstɪs] 
<span class="definition">n. [U] 用於懲治罪犯的法律制度，即司法、審判：</span>civil, criminal justice 民事審判；刑事審判 / restorative justice 修復性司法 / retributive justice（尤美）報應性司法 / summary justice 即決審判 / Justice must be done in every case. 每一樁案件都必須得到公正審理。/ to bring sb to justice 將某人繩之以法 / So far the robbers have escaped justice. 那些搶劫犯至今依然逍遙法外。/ to pervert the course of justice（英）妨礙司法公正 / obstruct justice（美）妨礙司法公正 / justice department 司法部門 / justice system 司法系統（或司法制度）/ miscarriage of justice 誤判 / vigilante justice 法外執法 / divine justice 神的懲罰 / poetic justice 報應

<span class="vocabulary">**circuit**</span> ['sə:kɪt] 
<span class="definition">n. [C] 法官在某一地區的各個法院所進行的審判，即巡迴審判：</span>瞭解即可

<span class="vocabulary">**examine**</span> [ɪɡ'zæmɪn] 
<span class="definition">vt.（尤指在法庭上）正式地問某人一些問題，即審問、查問：</span>瞭解即可
