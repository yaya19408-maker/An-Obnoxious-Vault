☀ <span class="category">**體檢 化驗：**</span>
<span class="vocabulary">**medical**</span> ['medɪkl] 
<span class="definition">n. [C] [英]（如入職前所做的）全面體檢：</span>full, regular medical 全面體檢；定期體檢 / to pass the medical 體檢合格

<span class="vocabulary">**examine**</span> [ɪɡ'zæmɪn] 
<span class="definition">vt. examine sth/sb (for sth) 給某人做醫學檢查，以看其是否有問題，或為查詢一個疾病的原因。是此義的常規用詞：</span>The doctor examined her but could find nothing wrong. 醫生給她做了檢查，但沒發現什麼問題。/ to examine the effectiveness of the drug 檢查該藥物的有效性

<span class="vocabulary">**inspect**</span> [ɪn'spekt] 
<span class="definition">vt. inspect sth/sb (for sth) 仔細檢查身體，尤指檢視一切是否正常，相當於examine，不過遠不如examine常用：</span>He is regularly inspected for disease. 他定期做疾病檢查。
           
<span class="vocabulary">**inspection**</span> [ɪnˈspekʃn]
<span class="definition">n. [U, C] 體檢：</span>medical inspection 醫學檢查

<span class="vocabulary">**examination**</span> [ɪɡ͵zæmɪ'neɪʃn] 
<span class="definition">n. [C] 醫學檢查：</span>clinical, forensic, medical, physical, post-mortem, psychiatric examination 臨床檢查；法醫檢驗；體檢；屍檢；精神病檢查 / a full eyesight examination 一次全面的視力檢查 / X-ray examination X光檢查 / a periodic health examination 定期健康檢查 / examination table 檢查臺 / examination room（尤美）檢查室

<span class="vocabulary">**exam**</span> [ɪɡ'zæm] 
<span class="definition">n. [C] [美] 對身體進行的健康檢查。是examination的縮寫形式：</span>medical, physical exams 體檢 / an stomach exam 胃部檢查 / thorough exam 全面的檢查 / routine exam 例行檢查 / follow-up exam 後續檢查 / abdominal exam 腹部檢查 / chest exam 胸腔檢查 / X-ray exam X光檢查 / exam table 檢查臺

<span class="vocabulary">**physical**</span> ['fɪzɪkl] 
<span class="definition">n. [C] 也可叫physical examination，即體格檢查（例如檢查其是否適合於某一工作）：</span>complete physical 全面體檢 / annual physical 年度體檢 / routine physical 例行體檢 / to pass the physical 體檢合格

<span class="vocabulary">**screen**</span> [skri:n] 
<span class="definition">vt. screen (sb) (for sth) 常用被動式，指對人們進行體檢，以檢查其是否患有某一疾病，即篩查：</span>Men over 55 should be regularly screened for prostate cancer. 55歲以上的男性應定期做前列腺癌檢查。/ All pregnant women are to be screened for the infection. 所有孕婦都要做這種傳染病篩查。
           
<span class="vocabulary">**check-up**</span>
<span class="definition">n. [C] 對某事物的檢查，尤指人體檢查（以確定是否健康）：</span>thorough check-up 全面體檢 / general check-up 全身體檢 / dental, health, medical，physical 牙科檢查；健康檢查；體格檢查 / At your age, you should have regular check-ups. 你這樣的年齡，應該定期體檢。/ The company doctor does check-ups on Wednesdays. 公司的醫生每個星期三為大家做體檢。/ The disease was detected during a routine check-up. 這個病是在做常規體檢時查出來的。/ a regular check-up at the dentist 去牙醫診所的常規牙齒檢查

<span class="vocabulary">**test**</span> [test] 
<span class="definition">n. [C] 醫療檢查或化驗：</span>medical test 醫學檢驗 / screening tests for cancer 癌症篩查 / forensic test 法醫檢驗 / hearing test 聽力檢查 / blood, DNA, urine test 驗血；DNA檢測；尿檢 / drug test 興奮劑檢測 / fitness test 體能測驗 / breath test 呼吸酒精測試 / personality, psychological test 人格測驗；心理測驗 / lie-detector test 測謊試驗 <span class="definition">vt.＆vi. test sb/sth (for sth) 檢查身體，化驗血液等：</span>to test for heart disease 檢查心臟病 / I’m having my eyes tested tomorrow. 明天我要去檢查一下眼睛。/ to test positive for HIV 艾滋病病毒化驗呈陽性 

<span class="vocabulary">**scan**</span> [skæn] 
<span class="definition">vt. 用X光、超聲波、電磁波等對某人的身體部位進行掃描以獲取影象：</span>to scan his brain 掃描他的大腦 <span class="definition">n. 1 [C] 用X光對身體所做的醫學掃描檢查：</span>to have a brain scan 做腦部掃描檢查 <span class="definition">2 [C] 用機器發出的超聲波對孕婦所做的胎兒掃描檢查：</span>to have a scan 做胎兒掃描檢查

<span class="vocabulary">**X-ray**</span> ['eks͵reɪ] 
<span class="definition">n. 1 [C] X光片：</span>a chest X-ray 胸部X光片 / to take an X-ray of my jaw 拍攝我下頜的X光片 <span class="definition">2 [C] 用X射線進行的醫學檢查：</span>to go for an X-ray 去做X光檢查 <span class="definition">vt. 給…拍攝醫學X光片：</span>I have to have my chest X-rayed. 我得做個胸部X光檢查。
           
<span class="vocabulary">**smear**</span> [smɪə(r); 美 smɪr]
<span class="definition">n. [C] [英] 子宮頸塗片檢查（一種探查早期癌變的檢查），相當於smear test：</span>cervical smear（英）宮頸塗片 / Pap smear（美）巴氏塗片 / abnormal smear 異常塗片

<span class="vocabulary">**quarantine**</span> [ˈkwɒrənti:n; 美 ˈkwɔ:r-; ˈkwɑ:r-]
<span class="definition">n. [U]（為防疾病傳染而對動物或人員進行隔離的）隔離期、檢疫：</span>She was sent home to Oxford and put in quarantine. 她被送回牛津的家中隔離起來。/ No mammals other than people may enter the country without lengthy quarantine. 除人之外，所有的哺乳動物進入這個國家都必須經過長期隔離。/ The animals are still in quarantine at the port. 那些動物仍然在港口接受檢疫隔離。/ The dog was kept in quarantine for six months. 這條狗被檢疫隔離了六個月。/ quarantine period 檢疫隔離期 / quarantine zone 隔離區 / quarantine facility, quarantine station 檢疫設施；檢疫站 / quarantine regulations, quarantine restrictions 檢疫隔離規定；檢疫隔離限制

<span class="definition">vt. 對動物或人進行隔離或檢疫：</span>Dogs have to be quarantined for six months before they'll let them in. 狗要進行6個月的隔離檢疫之後才被允許進入。

<span class="vocabulary">**post-mortem**</span> [ˌpəʊst ˈmɔ:təm; 美 ˌpoʊst ˈmɔ:rtəm]
<span class="definition">n. [C]（也可用post-mortem examination）~ (on sb) 對屍體進行醫學檢查以查詢死因，即驗屍：</span>The post-mortem on the child revealed that she had been poisoned. 驗屍證明這孩子是被人毒死的。/ The coroner says we will have to do a post-mortem. 驗屍官說我們得進行屍體解剖。/ post-mortem examination 屍體檢驗 / They're doing a post-mortem on her today. 他們今天會對她進行屍體解剖。
           
<span class="vocabulary">**autopsy**</span> [ˈɔ:tɒpsi; 美 ˈɔ:tɑ:psi]
<span class="definition">n. [C] 驗屍，屍體解剖（以探查死因）：</span>The autopsy revealed that he had been poisoned. 屍檢表明他是被毒死的。/ The diagnosis was confirmed by autopsy. 診斷結果得到了屍體剖檢的證實。/ The autopsy report gave the cause of death as poisoning. 驗屍報告上顯示死因為中毒。/ autopsy findings, autopsy results 屍檢結果 / They carried out an autopsy on the victim. 他們對遇害者進行了屍體剖檢。

<span class="vocabulary">**positive**</span> ['pɒzɪtɪv] 
<span class="definition">adj. 在科學試驗中可指某種物質或某種生理條件是存在的，即陽性的：</span>The test proved positive. 測試結果呈陽性。/ He tested positive for HIV. 他艾滋病病毒測試呈陽性。<span class="definition">n. [C]（化驗或試驗所呈現的）陽性結果或陽性反應：</span>瞭解即可
           
<span class="vocabulary">**negative**</span> [ˈnegətɪv]
<span class="definition">adj. 科學或醫學化驗的結果為陰性的（即不含有某種物質或沒顯示某種醫學跡象）：</span>He tested negative for HIV infection. 他的艾滋病病毒檢測為陰性 / The antibody test proved negative. 抗體檢測結果為陰性。/ The urine tests were negative for protein. 尿液蛋白檢測結果呈陰性。/ Her pregnancy test was negative. 她的孕檢呈陰性。/ negative test results 陰性的測試結果 / So far 57 have taken the test and all have been negative. 到目前為止，參加測試的有57人，測試結果均呈陰性。<span class="definition">n. [C] 化驗中的陰性結果（即某種物質或條件不存在）：</span>false negatives generated by the cancer test 癌症檢查結果中的假陰性

周邊詞彙補充：
· analyse（analyze）可指對…進行精神（或心理）分析（或治療）。
