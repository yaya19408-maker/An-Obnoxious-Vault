☀ <span class="category">**引擎 馬達：**</span>
<span class="vocabulary">**engine**</span> ['endӡɪn] 
<span class="definition">n. [C] 主要指交通工具的發動機或引擎：</span>four-cylinder engine 四缸發動機 / a 580-horsepower engine 一臺580馬力的發動機 / diesel engine 柴油發動機 / internal-combustion engine 內燃發動機 / aircraft, rocket engine 飛機發動機；火箭發動機 / car engine 汽車發動機 / engine power, engine speed 發動機功率；發動機轉速 / engine component, engine part 發動機零部件 / engine plant 發動機廠 

<span class="vocabulary">**motor**</span> ['məʊtə] 
<span class="definition">n. [C] 廣義上的“發動機”或“馬達”，指利用電或燃料來產生動力以驅動機器、交通工具、船隻等運轉的一種裝置：</span>to start the motor 發動引擎 / large, powerful motor 大功率發動機 / small, tiny motor 小型發動機；微型發動機 / diesel, electric motor 柴油發動機；電動機 / rocket motor 火箭發動機 <span class="definition">adj. 只作前置定語，有引擎的；利用發動機來驅動的：</span>motor vehicles 機動車輛
           
<span class="vocabulary">**turbine**</span> [ˈtɜ:baɪn; 美 ˈtɜ:rb-]
<span class="definition">n. [C] 渦輪機：</span>steam turbine 蒸汽渦輪機（汽輪機）/ water turbine 水力渦輪機
          
<span class="vocabulary">**cylinder**</span> [ˈsɪlɪndə(r)]
<span class="definition">n. [C] 發動機的氣缸：</span>a 2.5 litre, four-cylinder engine 2.5升的四缸發動機
           
<span class="vocabulary">**distributor**</span> [dɪˈstrɪbjətə(r)]
<span class="definition">n. [C]（發動機的）配電器，配電盤：</span>瞭解即可
           
<span class="vocabulary">**radiator**</span> [ˈreɪdieɪtə(r)]
<span class="definition">n. [C]（車輛或飛機發動機的）冷卻裝置，如水箱：</span>He filled his car radiator with water. 他給汽車散熱器加滿水。/ radiator cap, radiator grille, radiator hose 散熱密封蓋；散熱護柵；散熱軟管
