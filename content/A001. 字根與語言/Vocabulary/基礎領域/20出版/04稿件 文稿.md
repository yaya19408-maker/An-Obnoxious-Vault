☀ <span class="category">**稿件 文稿：**</span>
<span class="vocabulary">**copy**</span> ['kɒpɪ] 
<span class="definition">n. [U]（將被刊印在報刊、雜誌等刊物上的）文字稿件；（可用於報紙文章或廣告的）訊息，資訊，此時可稱為“新聞素材”或“廣告文案”：</span>ad(非正式), advertising, marketing, promotional copy 廣告文案；營銷文案；促銷文案 / editorial copy 社論稿 / reporters’ copy 記者的稿件 / to make copy 成為（新聞報道的）素材 / copy editor 文字編輯 / copy deadline 截稿日期 / copy desk（美）編輯部 

<span class="vocabulary">**contribution**</span> [͵kɒntrɪ'bju:ʃn] 
<span class="definition">n. [C] 為一本書或一本雜誌所撰寫的“稿件”。 強調稿件的供應：</span>All contributions for the May issue must be received by Friday. 所有要在五月這一期發表的稿件必須在星期五以前寄到。

周邊詞彙補充：
· text可表示一本書或一篇文章，即文稿，尤指與科學或學術相關的文稿。

· 表示“篇幅”、“版面”的詞群見[[13篇幅 版面]]
· 表示“字元”、“字型”的詞群見[[08字元 字母 字型]]
