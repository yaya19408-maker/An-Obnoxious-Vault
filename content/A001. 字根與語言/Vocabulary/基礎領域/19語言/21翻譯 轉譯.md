☀ <span class="category">**翻譯 轉譯：**</span>
<span class="vocabulary">**translate**</span> [træns'leɪt] 
<span class="definition">vt.＆vi. translate (sth) (from sth) (into sth) / translate sth (as sth) 用一種不同的語言來表達意思，即翻譯：</span>He translated the letter into English. 他把這封信譯成了英文。/ Most poetry does not translate well. 詩歌大多翻譯不好。<span class="definition">vi. translate (as sth)被翻譯成另一種語言：</span>The Welsh name translates as ‘Land’s End’. 這個威爾士語的地名可譯成“蘭茲角”。

<span class="vocabulary">**translation**</span> [træns'leɪʃn] 
<span class="definition">n. [U] translation (of sth) (into sth) / translation (from sth) (into sth) 將一種語言翻譯成另一種語言的過程，即翻譯：</span>an error in translation 誤譯 / direct, word-for-word translation 直譯；逐詞翻譯 / literary, poetic translation 文學翻譯；詩歌翻譯 / simultaneous translation 同聲翻譯 / A great writer survives any translation. 一位偉大的作家經得住任何翻譯。

<span class="vocabulary">**translator**</span> [træns'leɪtə] 
<span class="definition">n. [C] 以翻譯為工作的人，即翻譯員，翻譯家：</span>Japanese translator 日語譯者 / professional translator 專業譯者 / translators from Portuguese to English 把葡萄牙語翻譯成英語的譯者 / He spoke through a translator. 他透過翻譯講話。
           
<span class="vocabulary">**interpret**</span> [ɪnˈtɜ:prɪt; 美 -ˈtɜ:rp-]
<span class="definition">vi. ~ (for sb) 聽到一種語言的時候，用另一種語言翻譯出來，即口譯：</span>She couldn't speak much English so her children had to interpret for her. 她講不了幾句英語，所以她的孩子們得給她翻譯。/ Interpreters found they could not interpret half of what he said. 口譯員們發現他說的話他們連一半都翻譯不出來。

<span class="vocabulary">**interpreter**</span> [ɪn'tə:prɪtə] 
<span class="definition">n. [C]（在不同語言之間進行翻譯的）口譯工作者，即口譯員：</span>Susan acted as interpreter for us. 蘇珊擔任我們的口譯員。/ a sign language interpreter 手語譯員

<span class="vocabulary">**free**</span> [fri:] 
<span class="definition">adj. 在翻譯中不拘泥於原文的，意譯的：</span>瞭解即可
           
<span class="vocabulary">**liberal**</span> [ˈlɪbərəl]
<span class="definition">adj.（翻譯）不完全準確的、不精確的、靈活的：</span>a liberal translation of the text 不拘泥於原文的翻譯
