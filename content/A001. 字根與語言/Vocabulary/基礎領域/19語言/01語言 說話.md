☀ <span class="category">**語言 說話：**</span>
<span class="vocabulary">**language**</span> ['læŋɡwɪdӡ] 
<span class="definition">n. 1 [C] 某國或某一地區的人們在口語和書面語中所採用的一套交流體系，即語言（是一種以地域為標準而劃分出來的概念）：</span>the Chinese language 漢語 / first, native language 母語；本族語 / foreign language 外語 / official language 官方語言 / national language 國語 <span class="definition">2 [U] 人類用以交流的一套聲音和文字型系（是一種學術概念）：</span>theories about the origins of language 有關語言起源的理論 / Computers will never be able to understand natural language. 計算機永遠不能理解自然語言。/ language acquisition 語言習得 / language course 語言課程 / language skill 語言技能 / language barrier 語言障礙 <span class="definition">3 [U] 某種型別或風格的話語或書寫語言（或言語）：</span>the language of business 商業用語 / literary, legal language 文學語言；法律語言 / spoken, written language 口語；書面語 / informal language 非正式語言 / figurative language 比喻性用語 / everyday, plain language 日常語言；樸素的語言 / vague language 含糊的語言 / harsh language 狠話 / offensive language 冒犯人的語言 <span class="definition">4 [C, U]（用動作、符號和聲音來表達思想感情的）表達方式：</span>sign language 手（勢）語 / body language 身體語言 / the language of bees 蜜蜂的交流方式

<span class="vocabulary">**tongue**</span> [tʌŋ] 
<span class="definition">n. [C] [正式或文學] 一門語言：</span>mother, native tongue 母語；本族語 / common tongue 通用語；共同語 / foreign tongue 外語 / strange tongue 陌生的語言
           
<span class="vocabulary">**dialect**</span> [ˈdaɪəlekt]
<span class="definition">n. [C, U] 方言：</span>native dialect 本地方言 / northern dialect 北方方言 / Italian dialect 義大利方言 / the Yorkshire dialect 約克郡方言 / Black dialect 黑人方言 / dialect form 方言形式 / dialect word 方言詞語 / dialect speaker 講方言的人 / dialect group 方言群體 / dialect poetry 方言詩歌 / In the fifties, many Italians spoke only local dialect. 在 20 世紀 50 年代，許多義大利人只會說當地方言。/ They began to speak rapidly in dialect. 他們開始嘰裡呱啦地說起地方話來。
           
<span class="vocabulary">**vernacular**</span> [vəˈnækjələ(r); 美 vərˈn-]
<span class="definition">n. [sing.]（通常用the vernacular）一個地區或群體所說的方言、土話（尤指非官方話或非書面語）：</span>books or plays written in the vernacular 用方言寫就的書或戲劇 / To use the vernacular of the period, Peter was square. 用那時的土話講，彼得是個老古板。

<span class="vocabulary">**speech**</span> [spi:tʃ] 
<span class="definition">n. [U] 說話的能力，一般意義上的講話：</span>He temporarily lost the power of speech after the accident. 那次事故之後他暫時失去了語言能力。/ freedom of speech 言論自由 / racist hate speech 種族仇恨言論

<span class="vocabulary">**idiom**</span> ['ɪdɪəm] 
<span class="definition">n. [U, C] [正式]（特定的一些人在某段時期或某個地方所用的）語言和語法：</span>瞭解即可
           
<span class="vocabulary">**parole**</span> [pəˈrəʊl; 美 pəˈroʊl]
<span class="definition">n. [U] [語言學]（單個人所說的，而非作為作為某個群體的交流體系的）言語：</span>瞭解即可

<span class="vocabulary">**family**</span> ['fæmɪlɪ] 
<span class="definition">n. [C] 一組有關的語言，此時可稱為語族：</span>the Germanic family of languages 日耳曼語族

<span class="vocabulary">**branch**</span> [brɑːntʃ] 
<span class="definition">n. [C] 語言的分支：</span>瞭解即可
           
<span class="vocabulary">**descendant**</span> [dɪˈsendənt]
<span class="definition">n. [C] 由一門語言派生出來的一支語系：</span>Quechua, the lineal descendant of the Inca language 克丘亞語——印加語的直系派生語言

<span class="vocabulary">**variety**</span> [və'raɪətɪ] 
<span class="definition">n. [C] variety (of sth) 某種語言的一個變體：</span>different varieties of English 各類英語變體
           
<span class="vocabulary">**related**</span> [rɪˈleɪtɪd]
<span class="definition">adj. ~ (to sth/sb)（語言）同源的：</span>related languages 同系語言 / Sanskrit is related very closely to Latin, Greek, and the Germanic and Celtic languages. 梵語和拉丁語、希臘語、日耳曼和凱爾特語是聯絡密切的同源語言。

<span class="vocabulary">**grammar**</span> ['ɡræmə] 
<span class="definition">n. [U] 一個人的語言知識及語言運用：</span>His grammar is appalling. 他運用語言的能力糟透了。

<span class="vocabulary">**speak**</span> [spi:k] 
<span class="definition">n. [sing.] [非正式，常含貶義] 構成名詞，指（某一團體、組織或學科的）語言或行話：</span>computerspeak 計算機語言 / eurospeak 歐盟語言

相關詞義延伸：
· language還可表示計算機語言。

· 表示“說話”的詞群見[[08溝通交流]]
· 表示“出版”的詞群見[[00出版組目錄]]
· 表示“通訊”的詞群見[[00資訊通訊組目錄]]

<span class="vocabulary">**linguistic**</span> [lɪŋˈgwɪstɪk]
<span class="definition">adj. 語言的；語言學的：</span>linguistic skills 語言技能 / linguistic theory 語言學理論

<span class="vocabulary">**native**</span> ['neɪtɪv] 
<span class="definition">adj. 母語的，第一語言的：</span>native language 母語
                      
<span class="vocabulary">**bilingual**</span> [ˌbaɪˈlɪŋgwəl]
<span class="definition">adj. 1 會說兩種語言的：</span>She is bilingual in English and Punjabi. 她會說英語和旁遮普語。<span class="definition">2 用兩種語言的；用兩種語言書寫的：</span>bilingual education, bilingual communities 雙語教育；雙語社群 / a bilingual dictionary 雙語詞典

<span class="vocabulary">**borrow**</span> ['bɒrəʊ] 
<span class="definition">vt.＆vi. borrow (sth) (from sb/sth) 從另一種語言（或另一個人等）借用或引用字詞、話語等：</span>The author borrows heavily from Henry James. 那位作家大量引用亨利·詹姆斯的作品。/ Some musical terms are borrowed from Italian. 一些音樂術語是從義大利語引入的。

<span class="vocabulary">**speaker**</span> ['spi:kə] 
<span class="definition">n. [C] 講某種語言的人。是此義的常規用詞，最為常用：</span>Chinese speakers 講漢語的人 / a native speaker of English 以英語為母語的人 / non-native speaker 非母語使用者

<span class="vocabulary">**phone**</span> [fəʊn] 
<span class="definition">adj., adv., n. [C] –phone構成形容詞和副詞，指講某種語言的；還可指講某種語言的人（作為此義時遠不如speaker常用）：</span>anglophone 講英語的（人）/ Francophone 講法語的（人）

<span class="vocabulary">**speak**</span> [spi:k] 
<span class="definition">vt. 1 會說或會講（某種語言）：</span>to speak several languages 會講幾種語言 / to speak a little Japanese 會說一點日語 / He speaks German fluently. 他德語說得很流利。/ Do you speak English? 你會說英語嗎？<span class="definition">2 構成形容詞，講…語的：</span>French-speaking Canada 加拿大法語區 / non-English-speaking students 不會講英語的學生 <span class="definition">vt.＆vi. 用某種語言來表達自己，即用某種語言來說話：</span>What language is it they’re speaking? 他們講的是什麼語？/ Would you prefer it if we spoke in German? 我們用德語講好嗎？

<span class="vocabulary">**talk**</span> [tɔ:k] 
<span class="definition">vt.＆vi.（用某種語言）講，說；或帶某種口音：</span>The baby is just starting to talk. 這嬰兒剛開始咿呀學語。/ to talk (in) Chinese 說漢語

<span class="vocabulary">**use**</span> [ju:z] 
<span class="definition">vt. 說或寫（詞語或特定種類的語言），即使用（詞語或語言）：</span>The poem uses simple language. 這首詩用語簡單。/ That’s a word I never use. 那是我從來不用的字眼。/ You have to use the past tense. 要使用過去時。

<span class="vocabulary">**in**</span> [ɪn] 
<span class="definition">prep. 用於表明使用的語言：</span>to write a letter in English 用英語寫一封信
