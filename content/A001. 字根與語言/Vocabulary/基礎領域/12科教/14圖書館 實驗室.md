☀ <span class="category">**圖書館 實驗室：**</span>
<span class="vocabulary">**library**</span> ['laɪbrərɪ] 
<span class="definition">n. 1 [C] 圖書館，藏書閣，是一棟建築：</span>public library 公共圖書館 / private library 私人圖書館 / school, university library 學校圖書館；大學圖書館 / library catalogue 圖書館的目錄 / library card, library ticket（英）借書證 / library staff 圖書館的工作人員 <span class="definition">2 [C] 建築物內（如學校、醫院等）的圖書室，閱覽室：</span>hospital, prison library 醫院閱覽室；監獄閱覽室

<span class="vocabulary">**lab**</span> [læb] 
<span class="definition">n. [C] lab為laboratory的非正式用法，指實驗室：</span>science labs 科學實驗室 / a lab technician 實驗室技術員

<span class="vocabulary">**laboratory**</span> [lə'bɒrətərɪ] 
<span class="definition">n. 1 [C] 做科學實驗、分析與研究的一個房間或一棟建築物，即實驗室，實驗大樓：</span>industrial laboratory 工業實驗室 / underground laboratory 地下實驗室 / independent laboratory 獨立實驗室 / government laboratory 政府實驗室 / biology, chemistry laboratory 生物學實驗室；化學實驗室 / medical laboratory 醫學實驗室 / laboratory assistant 實驗室助理 / laboratory equipment 實驗室裝置 / laboratory animal 實驗室動物 / laboratory data 實驗室資料 <span class="definition">2 [C] 學校中用於教學的實驗室：</span>the chemistry laboratory of our college 我們學院的化學實驗室 / school laboratory 學校實驗室
