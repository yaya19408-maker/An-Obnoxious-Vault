☀ <span class="category">**課程 課堂：**</span>
<span class="vocabulary">**class**</span> [klɑːs] 
<span class="definition">n. 1 [C, U] 一群學生集中聽講的場景，即上課，相當於lesson：</span>I’ll have a class this afternoon. 今天下午我有一節課。/ history class 歷史課 / He’s taking classes in pottery. 他在上陶藝課。/ to disrupt class 擾亂課堂 / Class dismissed! 下課！<span class="definition">2 [C] 某個科目的一系列課程，與course近義：</span>to take economics classes 學習經濟學課程 / The college offers classes in many subjects. 學院開設有多門學科的課程。

<span class="vocabulary">**lesson**</span> ['lesn] 
<span class="definition">n. 1 [C] 在課堂上學習…的一段時間，即課：</span>individual lesson 單獨授課 / private lesson 私人授課 / driving, English, geography, swimming lesson 駕駛課；英語課；地理課；游泳課 / She gives music lessons. 她教音樂課。/ to prepare lesson 備課 / to skip lesson 逃課 / No talking was allowed during the lesson. 上課期間不許交頭接耳。<span class="definition">2 [C] 所學習的內容，即課，教學單元：</span>The textbook is divided into 26 lessons. 這本教科書分為26課 

<span class="vocabulary">**course**</span> [kɔ:s] 
<span class="definition">n. 1 [C] course (in/on sth) 某一學科的一系列課程或講座：</span>computer course 計算機課程 / crash, intensive course 速成課程；強化課程 / two-day course 兩天的課程 / advanced, beginners’, intermediate, introductory course 高階課程；初級課程；中級課程；入門課程 / required course（美）必修課程 / He enrolled in a remedial mathematics course. 他參加了一個數學補習班。/ training course 培訓課程 / correspondence, online, Web-based course 函授課程；網路課程 / to complete a course in first aid 修完一門急救課程 <span class="definition">2 [C] [尤英]（大學中要考試或取得學歷所進行的）一段時間的學習，即課程：</span>a two-year postgraduate course leading to a master’s degree 兩年制碩士研究生課程 / full-time, part-time course 全日制課程；非全日制課程 / undergraduate course 本科生課程 / degree, diploma, honours/honors course 學位課程；文憑課程；榮譽學位課程 / mathematics course 數學課程 / vocational course 職業課程 / sandwich course（英）工讀交替制課程

<span class="vocabulary">**curriculum**</span> [kə'rɪkjələm] 
<span class="definition">n. 1 [C] 學校等的全部課程：</span>The present curriculum is too narrow. 現在的課程設定學科範圍過窄。/ core curriculum 核心課程 / national, official curriculum 國家統一課程；正式課程 / standard curriculum 標準課程 / undergraduate curriculum 本科生課程 / course curriculum 課程設定 / curriculum content 課程內容 / curriculum area 課程門類 / curriculum design, curriculum planning 課程設計；課程規劃 / curriculum objectives 課程目標 <span class="definition">2 [C] 學校的某一門課程，相當於course：</span>mathematics curriculum 數學課程 

<span class="vocabulary">**period**</span> ['pɪərɪəd] 
<span class="definition">n. [C] 學校中的任何一節課；（練習、訓練或學習的）時段：</span>‘What do you have next period?’ ‘French.’ “你下一節是什麼課？”“法語。”/ free, study period 自習課 / exercise period 練習時段

<span class="vocabulary">**programme**</span> ['prəʊɡræm]（美）
<span class="definition">n. [C] 北美可指一門課程：</span>certificate program（美）認證課程 / undergraduate program（美）本科課程 / degree program（尤美）學位課程 / the two-year MA degree programme 兩年制碩士學位課程 / history program（尤美）歷史課程 / online program（尤美）線上課程

<span class="vocabulary">**extension**</span> [ɪk'stenʃn] 
<span class="definition">n. [C]（大學中為非全日制學生開設的）進修課程：</span>extension courses 大學進修課程

<span class="vocabulary">**set**</span> [set] 
<span class="definition">adj. [英] 指圖書被指定用於學習某一門課程的：</span>a set book 指定用書

<span class="vocabulary">**shop**</span> [ʃɒp] 
<span class="definition">n. [U] 北美可指工藝課，即學生用工具和機器將木頭，金屬等做成某件物品，也可叫shop class或industrial arts：</span>瞭解即可

<span class="vocabulary">**game**</span> [ɡeɪm] 
<span class="definition">n. [pl.] [舊時，英] 學校的體育課或其體育活動：</span>瞭解即可

<span class="vocabulary">**degree**</span> [dɪ'ɡri:] 
<span class="definition">n. [C] [英]（通常要攻讀三年以上的）大學學位課程：</span>to do a literature degree 攻讀文學學位課程 / degree course（尤英）, degree programme/program 學位課程

<span class="vocabulary">**major**</span> ['meɪdӡə] 
<span class="definition">adj. [美]（學校裡的課程）主修的：</span>瞭解即可 <span class="definition">n. [C] [美]（學院或大學的）主修課程；專業課：</span>Her major is French. 她的專業課是法語。<span class="definition">vi. [美]（在大學中）以…為專業：</span>He majored in finance. 他主修金融。
           
<span class="vocabulary">**minor**</span> [ˈmaɪnə(r)]
<span class="definition">n. [C] [尤美]（美國大學中的）輔修科目：</span>瞭解即可 <span class="definition">vi. [美] ~ in sth（在大學裡）輔修（某個科目）：</span>I'm minoring in computer science. 我輔修電腦科學。
           
<span class="vocabulary">**option**</span> [ˈɒpʃn; 美 ˈɑ:p-]
<span class="definition">n. [C] 選修課：</span>The course offers options in design and computing. 這一學程開了設計和計算機技術的選修科目。/ Several options are offered for the student's senior year. 為畢業班學生開設了幾門選修課。

<span class="vocabulary">**content**</span> [ˈkɒntent] 
<span class="definition">n. [U]（課程的）內容：</span>course content 課程內容
           
<span class="vocabulary">**syllabus**</span> [ˈsɪləbəs]
<span class="definition">n. [C] 學校裡某門課程的教學大綱；也可指一門課的提綱或摘要：</span>class（尤美）, course syllabus 課程大綱 / exam, examination syllabus（均尤英）考試大綱 / A level syllabus 中學高階水平考試大綱 / college, school（尤英）, university（尤英）syllabus 學院教學大綱；學校教學大綱；大學教學大綱 / geography, history, mathematics syllabus 地理教學大綱；歷史教學大綱；數學教學大綱 / online syllabus 網上教學大綱 / sample syllabuses for undergraduate courses 本科生課程大綱樣本 / The courses do not follow the syllabus of any particular examination board. 這些課程安排不遵循任何特定考試委員會的大綱。/ Several schools in Britain already teach this syllabus. 英國有幾所學校已經開始教授這個大綱規定的教學內容。/ Students follow different syllabuses according to their ability. 學生們根據自己的能力學習不同課程大綱規定的課程。/ to revise syllabus（尤英）修訂教學大綱 / It was impossible to cover the syllabus in a year. 不可能在一年內教完這一課程大綱的全部內容。/ Does the syllabus cover modern literature? 這份大綱包括現代文學的內容嗎？/ syllabus design 大綱設計 / syllabus content（英）大綱內容 / questions from last year's syllabus 根據去年教學大綱出的一些試題 / Let's include that in this year's syllabus. 咱們把那一項納入到今年的大綱裡吧。/ Is geometry on the syllabus? 這個大綱裡有幾何嗎？/ some syllabuses for basic courses in geography 地理基礎課程教學大綱 / the course syllabuses in arts subjects 文科科目的課程大綱

周邊詞彙補充：
· honour（美honor）可表示大學的榮譽學位課程。
