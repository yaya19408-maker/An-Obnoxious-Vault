☀ <span class="category">**輻射 放射：**</span>
<span class="vocabulary">**radium**</span> ['reɪdɪəm] 
<span class="definition">n. [U] 鐳元素，鐳（具有放射性的一種白色金屬，可用於治療癌症）：</span>瞭解即可
      
<span class="vocabulary">**uranium**</span> [juˈreɪniəm]
<span class="definition">n. [U]（符號為U）鈾（一種放射性化學元素）：</span>natural uranium 天然鈾 / a tonne of highly enriched uranium 一噸高純度濃縮鈾 / depleted uranium 貧化鈾 / weapons-grade uranium 武器級的鈾 / yellowcake uranium 黃餅鈾 / uranium deposits, uranium ore, uranium reserves 鈾礦；鈾礦石；鈾儲量 / uranium fuel 鈾燃料 / depleted uranium weapons 貧鈾武器 / uranium mine, uranium miner, uranium mining 鈾礦；鈾礦工人；鈾礦業 / uranium enrichment 鈾濃縮

<span class="vocabulary">**plutonium**</span> [plu:ˈtəʊniəm; 美 -ˈtoʊ-]
<span class="definition">n. [U]（符號為Pu）鈽（一種放射性化學元素）：</span>radioactive plutonium 放射性懷 / weapons-grade plutonium 武器級鈽 / one ton of plutonium extracted from spent nuclear fuel 從廢核燃料中提取的—噸鈽 / the capability to enrich uranium or reprocess plutonium 濃縮鈾或再處理鈽的能力 / plutonium fuel 鈽燃料 / plutonium production 鈽的生產 / the plane that dropped a plutonium bomb on the city of Nagasaki 在長崎市扔下一顆鈽彈的那架飛機

<span class="vocabulary">**radiation**</span> [͵reɪdɪ'eɪʃn] 
<span class="definition">n. 1 [U, C] 放射性物質所輻射出的威力強大且極其危險的射線；也可指輻射物（其中包含著放射性物質的微小粒子）：</span>the radiations emitted by radium 鐳釋放出的射線 / low-level radiation 低強度輻射 / nuclear radiation 核輻射 / radiation sickness 輻射病 <span class="definition">2 [U] 以射線的形式輻射出的熱或其他能量：</span>electromagnetic, gravitational, microwave, thermal radiation 電磁輻射；引力輻射；微波輻射；熱輻射 / gamma radiation 伽馬射線輻射 / solar radiation 太陽輻射 / radiation from computer screens 電腦螢幕的輻射 / radiation therapy 放射療法

<span class="vocabulary">**radioactive**</span> [reɪdɪəʊ'æktɪv] 
<span class="definition">adj. 放射性的；有輻射的：</span>radioactive waste 放射性廢物
                      
<span class="vocabulary">**radioactivity**</span> [ˌreɪdiəʊækˈtɪvəti; 美 -oʊæk-]
<span class="definition">n. [U] 放射性：</span>the study of radioactivity 放射性研究 / radioactivity levels 輻射強度 / a rise in the level of radioactivity 輻射強度的增加 / The soil contains 30 times the acceptable level of radioactivity. 這片土壤的輻射強度高於標準 30倍。/ the amount of radioactivity released by nuclear power plants 核電廠釋放出的輻射量

<span class="vocabulary">**radiant**</span> [ˈreɪdiənt]
<span class="definition">adj. [只用於名詞前] [術語] 從一箇中心點發射出射線的，即輻射的、放射的：</span>the radiant energy of the sun 太陽的輻射能 / The earth would be a frozen ball if it were not for the radiant heat of the sun. 如果沒有太陽輻射的熱量，地球將是一個冰封的世界。           

<span class="vocabulary">**fission**</span> [ˈfɪʃn]
<span class="definition">n. [U]（也可用nuclear fission）[物理學]（核）裂變：</span>fission reactor 核裂變反應堆
           
<span class="vocabulary">**fusion**</span> [ˈfju:ʒn]
<span class="definition">n. [U] [物理學]（也可用nuclear fusion）核聚變：</span>research into nuclear fusion 核聚變研究 / fusion reactor 核聚變反應堆
           
<span class="vocabulary">**reactor**</span> [riˈæktə(r)]
<span class="definition">n. [C] 核反應堆。相當於nuclear reactor：</span>nuclear, nuclear-power reactor 核反應堆；核動力反應堆 / breeder, fast, fast-breeder, fission, fusion, gas-cooled, light-water reactor 增殖反應堆；快速反應堆；快中子增殖反應堆；裂變反應堆；聚變反應堆；氣冷反應堆；輕水反應堆 / reactor building, reactor site 反應堆建築；反應堆場所 / reactor design 反應堆的設計 / reactor coolant, reactor fuel 反應堆冷卻劑；反應堆燃料 / reactor core 反應堆活性區 / reactor accident, reactor melt-down 反應堆事故；反應堆熔燬 / reactor safety 反應堆安全 / reactor shutdown 停堆

<span class="vocabulary">**microwave**</span> ['maɪkrəweɪv] 
<span class="definition">n. [C] [術語] 微波（一種電磁波）：</span>microwave communication 微波通訊
 
相關詞義延伸：
· radiation還可指對疾病所採用的放射性療法；
· microwave還可指微波爐。

周邊詞彙補充：
· radio可指無線電廣播。
