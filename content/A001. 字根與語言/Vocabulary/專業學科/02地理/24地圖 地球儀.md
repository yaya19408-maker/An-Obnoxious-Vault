☀ <span class="category">**地圖 地球儀：**</span>
<span class="vocabulary">**map**</span> [mæp] 
<span class="definition">n. [C] 地圖：</span>world map 世界地圖 / road, street map 公路圖；街道圖 / bus map 公交線路圖 / treasure map 藏寶圖 / geological map 地質平面圖 / weather map 氣象圖 / tourist map 旅遊地圖 / Are you good at reading maps? 你會看地圖嗎？<span class="definition">vt. 繪製某一地區的地圖，相當於chart：</span>an unexplored region that has not yet been mapped 一個尚未繪製地圖的未經勘察的地區

<span class="vocabulary">**chart**</span> [tʃɑːt] 
<span class="definition">n. [C] 海圖或星圖：</span>charts of Greek waters 希臘水域圖 / nautical, navigation chart 航海圖；導航圖 / weather chart 氣象圖 / astrological, star chart 占星圖；恆星圖 <span class="definition">vt. 繪製一片陸地、海洋或天空的圖，相當於map：</span>He charted this mountain area. 他繪製了這片山區的地圖。
           
<span class="vocabulary">**atlas**</span> [ˈætləs]
<span class="definition">n. [C] 地圖冊、地圖集：</span>a world atlas 世界地圖冊 / a road atlas of Europe 歐洲交通地圖冊

<span class="vocabulary">**key**</span> [ki:] 
<span class="definition">n. [C]（地圖中）對符號的說明：</span>瞭解即可
                      
<span class="vocabulary">**legend**</span> [ˈledʒənd]
<span class="definition">n. [C] [術語] 地圖（或書中圖表）的解釋：</span>瞭解即可

<span class="vocabulary">**grid**</span> [grɪd]
<span class="definition">n. [C]（地圖上的）座標方格：</span>grid line, grid reference（英）, grid square 座標線；格網座標；格網方格 / The grid lines on the map run north-south. 地圖上的座標線呈南北走向。/ A grid reference gives the position of a place to within 100 m. 格網座標標示某一地點的位置精確到100米以內。/ The grid reference is C8. 地圖上的座標方格數字為C8。

<span class="vocabulary">**plot**</span> [plɒt] 
<span class="definition">vt. plot sth (on sth) 在地圖上畫出或標出（例如標出飛機或船的位置或航向）：</span>to plot the course of the submarine 標出那艘潛艇的航線 / The earthquake centres had been plotted on a world map. 地震的震中位置均被標註在一張世界地圖上。
