☀ <span class="category">**語言指令 字元磅值：**</span>
<span class="vocabulary">**language**</span> ['læŋɡwɪdӡ] 
<span class="definition">n. [C, U] 計算機語言（即用於操作電腦的一套符號和規則體系）：</span>a programming language 程式語言 / the digital language of computers 計算機數字語言 / computer language 計算機語言 

<span class="vocabulary">**instruction**</span> [ɪn'strʌkʃn] 
<span class="definition">n. [C] 計算機的指令。它可以是一個編碼，也可以是一串資料序列。其作用在於定義並執行一項操作或任務。其直接發出者一般為計算機本身：</span>a set of instructions 一套指令 / CPUs are designed to process instructions in a predictable order. 中央處理器按預定順序處理各種指令。/ instruction set 指令集

<span class="vocabulary">**command**</span> [kə'mɑːnd] 
<span class="definition">n. [C] 向計算機發出的指令，命令。其直接發出者可以是人：</span>keyboard command 鍵盤命令 / command line, command prompt 命令列；命令提示符
           
<span class="vocabulary">**loop**</span> [lu:p]
<span class="definition">n. [C] [計算機] 程式中的一套重複指令（直到某一條件得到滿足）：</span>瞭解即可           

<span class="vocabulary">**code**</span> [kəʊd; 美 koʊd]
<span class="definition">n. [U] [計算機] 編碼：</span>binary code (= a system of computer programming instructions) 二進位制編碼 / executable, HTML, source code 可執行檔案程式碼；超文字標記語言程式碼；原始碼 / malicious code 惡意程式碼 / error code（計算機上的）錯誤程式碼 / code generator 編碼發生器；編碼生成程式              

<span class="vocabulary">**password**</span> [ˈpɑ:swɜ:d; 美 ˈpæswɜ:rd]
<span class="definition">n. [C] [計算機] 密碼、口令：</span>to crack passwords 破解密碼 / Enter a username and password to get into the system. 進入系統請鍵入使用者名稱稱和密碼。/ No-one could use the computer unless they had a password. 除非知道密碼，否則誰也用不了那臺計算機。/ Please supply a valid user password. 請提供有效的使用者密碼。        

<span class="vocabulary">**prompt**</span> [prɒmpt; 美 prɑ:mpt]
<span class="definition">n. [C] [計算機] 提示符（計算機螢幕上顯示的，表明某項任務已經完成，並做好了執行新任務的準備的一個符號）：</span>瞭解即可

<span class="vocabulary">**dot**</span> [dɒt] 
<span class="definition">n. [C] 計算機中的點（用以分隔域名、電子郵件地址等的各部分）：</span>瞭解即可

<span class="vocabulary">**character**</span> ['kærɪktə] 
<span class="definition">n. [C]（計算機上的）字元，字母，符號，標記：</span>a string of characters 一組字元；一串字元 / character set, character string 字元組；字串

<span class="vocabulary">**string**</span> [strɪŋ] 
<span class="definition">n. [C] 計算機技術中可指一系列字元（如字母、數字等），即字串：</span>瞭解即可

<span class="vocabulary">**point**</span> [pɒɪnt] 
<span class="definition">n. [U] 計算機螢幕等上面字型大小的單位，即點，磅值：</span>to change the text to 12 point 把文字字型大小變為12點
           
<span class="vocabulary">**underscore**</span> [ˌʌndəˈskɔ:(r); 美 -dərˈs-]
<span class="definition">n. [C] [計算機] 下劃線、底線（用於字母下劃線或計算機命令和網際網路地址中）：</span>瞭解即可
