☀ <span class="category">**故障 病毒：**</span>
<span class="vocabulary">**freeze**</span> [fri:z] 
<span class="definition">vi. 計算機螢幕上的內容因系統出問題而固定不動，即螢幕凍結：</span>瞭解即可

<span class="vocabulary">**down**</span> [daʊn] 
<span class="definition">adj. 不用於名詞前，（一部計算機或一個計算機系統）停止執行：</span>The system was down all morning. 系統整個上午都無法執行。
           
<span class="vocabulary">**hack**</span> [hæk]
<span class="definition">vt.&vi. [計算機] ~ (into) (sth) 非法侵入別人的計算機系統：</span>He hacked into the bank's computer. 他侵入了這家銀行的計算機。/ They had hacked secret data. 他們竊取了保密資料。/ The saboteurs had demanded money in return for revealing how they hacked into the systems. 那些故意破壞者索要錢財，以作為透露自己是如何非法進入該電腦系統的交換條件。

<span class="vocabulary">**clone**</span> [kləʊn] 
<span class="definition">vt. 非法複製（別人的信用卡或手機的儲存資訊，從而使卡主或機主付費）：</span>瞭解即可
           
<span class="vocabulary">**skim**</span> [skɪm]
<span class="definition">vt.&vi. 非法複製並使用信用卡上的電子資訊：</span>瞭解即可

<span class="vocabulary">**virus**</span> ['vaɪərəs] 
<span class="definition">n. [C] 計算機病毒：</span>computer, email virus 計算機病毒；電子郵件病毒 / virus protection, virus scanning 病毒防護；病毒掃描

<span class="vocabulary">**crash**</span> [kræʃ] 
<span class="definition">n. [C] 計算機技術中可指一個機器或系統（尤指一臺計算機或一個計算機系統）突然崩潰：</span>瞭解即可 <span class="definition">vt.＆vi. 計算機系統崩潰（突然停止執行）：</span>The system suddenly crashed. 系統突然崩潰了。

<span class="vocabulary">**cheat**</span> [tʃi:t] 
<span class="definition">n. [C] [尤英；美尤用cheater] 電腦遊戲中的作弊軟體，或欺騙性程式、秘技，即不需要親自玩遊戲即可立即跳到遊戲的下一步：</span>There’s a cheat you can use to get to the next level. 有種秘技，你可以用來到達下一關。

<span class="vocabulary">**worm**</span> [wə:m] 
<span class="definition">n. [C] 一種計算機病毒，即蠕蟲病毒：</span>a new computer worm 一種新型計算機蠕蟲
           
<span class="vocabulary">**bug**</span> [bʌg]
<span class="definition">n. [C]（機器，尤指計算機程式的）錯誤、漏洞：</span>computer, software bug 計算機故障；軟體故障 / There is a bug in the software. 軟體有漏洞。

<span class="vocabulary">**corrupt**</span> [kə'rʌpt] 
<span class="definition">adj.（計算機的執行過程）已癱瘓的，有缺陷的，不在最初狀態的：</span>corrupt software 崩潰的軟體 / The text seems to be corrupt. 這個文字好像崩潰了。<span class="definition">vt.＆vi. 引起計算機檔案等的錯誤，使其中的資訊不再正確：</span>The program has somehow corrupted the system files. 這程式莫名其妙地導致了系統檔案出錯。/ corrupted data 被破壞的資料
           
<span class="vocabulary">**infect**</span> [ɪnˈfekt]
<span class="definition">vt. 使計算機病毒從一臺電腦上傳播到另一臺電腦上：</span>This virus infected thousands of computers within days. 這種病毒在短短几天內感染了數千臺計算機。
           
<span class="vocabulary">**overflow**</span> [ˌəʊvəˈfləʊ; 美 ˌoʊvərˈfloʊ]
<span class="definition">n. [C, usually sing.] [計算機] 是計算機的一種故障，指數字或資料項（如計算結果的數值）太大時，計算機不能準確將其顯示出來。可稱為溢位，上溢（運算產生的數值位數或字的長度等超過儲存單元的長度）：</span>瞭解即可

周邊詞彙補充：
· scan可指為搜尋計算機病毒而對磁碟檔案進行掃描。
