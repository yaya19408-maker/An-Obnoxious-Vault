☀ <span class="category">**細胞組織 器官系統：**</span>
<span class="vocabulary">**cell**</span> [sel] 
<span class="definition">n. [C] 生物的細胞：</span>living cell 活細胞 / healthy cell 健康細胞 / blood, brain, skin cell 血細胞；腦細胞；皮膚細胞 / plant cell 植物細胞 / cell division 細胞分裂 / cell wall 細胞壁
           
<span class="vocabulary">**membrane**</span> [ˈmembreɪn]
<span class="definition">n. [C, U] 植物的細胞膜：</span>cell, cellular membrane 細胞膜

<span class="vocabulary">**fibre**</span> ['faɪbə]（美fiber）
<span class="definition">n. [C]（人、動植物身體組織及天然物質的）纖維：</span>muscle fibres 肌肉纖維 / wood fibres 木纖維

<span class="vocabulary">**tissue**</span> ['tɪʃu:] 
<span class="definition">n. [U] 有時也用複數形式tissues，指（人、動植物細胞的）組織：</span>animal, human, plant tissue 動物組織；人體組織；植物組織 / body, brain, lung tissue 身體組織；腦組織；肺組織 / connective tissue 結締組織 / soft tissue 軟組織
           
<span class="vocabulary">**valve**</span> [vælv]
<span class="definition">n. [C]（心臟或血管的）瓣（膜）（使血液只朝著一個方向流動）：</span>He also has problems with a heart valve. 他的一個心瓣也有毛病。
           
<span class="vocabulary">**chamber**</span> [ˈtʃeɪmbə(r)]
<span class="definition">n. [C]（人體、植物體內的）腔、室：</span>the chambers of the heart 心腔

<span class="vocabulary">**organ**</span> ['ɔ:ɡən] 
<span class="definition">n. [C]（人或動植物的）器官。是此義的常規用詞和專業用詞，最為常用：</span>digestive, sex organ 消化器官；性器官 / human organ 人體器官 / organ donation 器官捐贈 / organ failure 器官衰竭
           
<span class="vocabulary">**organic**</span> [ɔ:ˈgænɪk; 美 ɔ:rˈg-]
<span class="definition">adj. [常用於名詞前] [術語] 器官的；器質性的：</span>organic disease 器官疾病

<span class="vocabulary">**part**</span> [pɑːt] 
<span class="definition">n. [C]（人或動植物的）身體器官。是一個簡單用詞：</span>瞭解即可

<span class="vocabulary">**apparatus**</span> [ˌæpəˈreɪtəs; 美 -ˈrætəs]
<span class="definition">n. [C, usually sing.] [術語] 身體的系統或器官：</span>the sensory apparatus 感覺器官

<span class="vocabulary">**tube**</span> [tju:b] 
<span class="definition">n. [C]（動植物的身體）內部的管道（用於輸送氣體、液體等），常為柔軟的管道：</span>bronchial, capillary, Eustachian, Fallopian, neural tube 支氣管；毛細管；咽鼓管；輸卵管；神經管 / pollen tube 花粉管
                      
<span class="vocabulary">**cord**</span> [kɔ:d; 美 kɔ:rd]
<span class="definition">n. [U, C] 身體內的索帶：</span>nerve, spinal, umbilical, vocal cord 神經索；脊髓；臍帶；聲帶

<span class="vocabulary">**system**</span> ['sɪstəm] 
<span class="definition">n. [C] 一個人（或動物）的身體系統，是一種生物學或醫學概念：</span>digestive, immune, reproductive system 消化系統；免疫系統；生殖系統 / the central nervous system 中樞神經系統 / body system 身體系統 / The drugs have passed out of your system. 藥物已經排出你的體外。

<span class="vocabulary">**digestion**</span> [daɪˈdʒestʃən; dɪ-]
<span class="definition">n. [C] 身體的消化系統：</span>My digestion ain't so hot these days, either.（ain't為非標準或幽默用法）我的消化系統這幾天也不怎麼對勁兒。

相關詞義延伸：
· fibre還可指（食物中可助消化的）纖維素。

周邊詞彙補充：
· transplant可指移植所用的器官。
