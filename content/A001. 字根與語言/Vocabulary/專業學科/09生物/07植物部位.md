☀ <span class="category">**植物部位：**</span>
<span class="vocabulary">**root**</span> [ru:t] 
<span class="definition">n. [C] 植物的根莖，根：</span>plant, tree root 植物的根；樹根 / root vegetables 根莖蔬菜 / the twisted roots 盤根錯節的根 / root system 根系 <span class="definition">vt.＆vi. 植物生根；使植物生根：</span>The plant will root in about eight weeks. 這種植物會在八週內生根。
           
<span class="vocabulary">**stem**</span> [stem]
<span class="definition">n. [C]（植物的）莖；（花或葉的）梗，柄：</span>flowering, woody stem 花莖；木質莖 / iris, rose stem 鳶尾花莖；玫瑰花莖 / He stooped down, cut the stem for her with his knife and handed her the flower. 他彎下腰，用刀替她將花莖割斷，然後將花遞給她。/ When the bush has finished flowering, cut back all the stems. 等這從灌木開完花後，把所有的莖幹都修剪一下。/ There are several leaves on each stem. 每一根莖上都有幾片葉子。/ Cut half an inch off the base of each stem. 從每根莖的基部剪去半英寸。<span class="definition">adj. -stemmed用於構成形容詞，指有…莖（或梗）的：</span>a long-stemmed rose 一枝長莖玫瑰
            
<span class="vocabulary">**cane**</span> [keɪn]
<span class="definition">n. [C] 某些植物（例如竹子、甘蔗）的硬而空的莖：</span>cane sugar 蔗糖 / raspberry canes 樹莓藤          

<span class="vocabulary">**stalk**</span> [stɔ:k]
<span class="definition">n. [C]（支撐葉子、花或果實，並將其和植物的其它部分連線起來的）細梗，即葉柄、花梗、果實的柄；也可指植物的主要莖稈：</span>flowers on long stalks 長莖上的花 / A single pale blue flower grows up from each joint on a long stalk. 一條長長的莖稈的每個節上都會長出一朵淺藍色的花。/ corn stalks 玉米稈 / celery stalks 芹菜莖 / He ate the apple, stalk and all. 他把那個蘋果吃了個乾淨，連梗都沒剩下。
           
<span class="vocabulary">**bulb**</span> [bʌlb]
<span class="definition">n. [C] 植物（如洋蔥）長在地下的球莖、鱗莖：</span>flower bulb 花的鱗莖 / daffodil, tulip bulb 水仙的鱗莖；鬱金香鱗莖 / I'm planting some bulbs for next year. 我正在種一些鱗莖，為明年生長做準備。/ Place the bulbs close together. 把這些鱗莖密集地埋種在一起。/ She usually waters the indoor bulbs once a week. 她通常每星期給室內的鱗莖澆一次水。
           
<span class="vocabulary">**foliage**</span> [ˈfəʊliɪdʒ; 美 ˈfoʊ-]
<span class="definition">n. [U] 植物（如樹木）的葉子；枝葉（包括葉子和枝條）：</span>dense green foliage 茂密的綠葉 / shrubs with grey or silver foliage 長著灰色或銀色葉子的灌木叢 / variegated foliage 斑駁的葉子

<span class="vocabulary">**leaf**</span> [li:f] 
<span class="definition">n. [C] 植物的葉子：</span>fallen leaf 落葉 / dead leaf 枯葉 / heart-shaped leaf 心形葉 / green leaf 綠葉 / maple leaf 楓葉
           
<span class="vocabulary">**bud**</span> [bʌd]

<span class="vocabulary">**pad**</span> [pæd]
<span class="definition">n. [C]（尤指睡蓮等水生植物的）大大的扁平的葉子：</span>floating lily pads 睡蓮的浮葉
            
<span class="vocabulary">**bang**</span> [bæŋ]
<span class="definition">n. [U] 大麻的葉和花穗（用作麻醉品）。相當於bhang：</span>瞭解即可
           
<span class="vocabulary">**vein**</span> [veɪn]
<span class="definition">n. [C]（植物葉子上的）葉脈：</span>the serrated edges and veins of the feathery leaves 羽狀葉子的鋸齒邊沿和紋理

<span class="vocabulary">**head**</span> [hed]

<span class="vocabulary">**top**</span> [tɒp] 
<span class="definition">n. [C, usually pl.] 主要長在植物根上的葉子（如蘿蔔的葉子）：</span>Remove the green tops from the carrots. 去掉胡蘿蔔的綠葉。

<span class="vocabulary">**capsule**</span> ['kæpsju:l] 
<span class="definition">n. [C]（植物包含著種子的）蒴果，莢；（動物包含著卵的）囊，被膜：</span>瞭解即可

<span class="vocabulary">**hair**</span> [heə] 
<span class="definition">n. [C] 植物葉子（或莖）上的茸毛：</span>瞭解即可
           
<span class="vocabulary">**spine**</span> [spaɪn]
<span class="definition">n. [C]（植物的）刺：</span>fine spine 細刺
           
<span class="vocabulary">**sting**</span> [stɪŋ]
<span class="definition">n. [C] [美也用stinger]（植物的）刺，刺毛：</span>瞭解即可
                      
<span class="vocabulary">**pore**</span> [pɔ:(r)]
<span class="definition">n. [C]（植物的）氣孔：</span>A plant's lungs are the microscopic pores in its leaves. 植物的肺就是其葉片上微細的氣孔。

<span class="vocabulary">**stigma**</span> [ˈstɪgmə]

<span class="vocabulary">**pollen**</span> [ˈpɒlən; 美 ˈpɑ:lən]
<span class="definition">n. [U] 花粉：</span>bees gathering pollen 正在採集花粉的蜜蜂 / the few weeks a year that corn pollen is shed 一年中玉米散落花粉的幾周 / pollen grain 花粉粒 / Hay fever sufferers have a worse time when the pollen count is high. 花粉計數處在高位時，枯草熱患者的日子就更不好過了。/ pollen analysis 花粉分析 / The pollen record shows that this plant was never common on the island. 花粉記錄顯示這種植物在島上一向罕見。

<span class="vocabulary">**bud**</span> [bʌd]
<span class="definition">n. [C]（能夠長出花的）花蕾；（能夠長出葉子或莖的）芽：</span>flower, leaf, rose bud 花蕾；葉芽；玫瑰花蕾 / developing bud 發育中的花蕾 / the first buds appearing in spring 春天的初芽 / Rosanna's favourite time is early summer, just before the buds open. 羅莎娜最喜愛的季節是初夏，恰在花蕾綻放前。/ The tree is in bud already. 樹已發芽。/ The roses are in bud. 玫瑰花含苞欲放。/ The flowers are bronzy in bud and bright yellow when open. 這些花的花蕾是青銅色，開放後呈明黃色。/ almond trees that should come into bud soon 很快就會開花的杏樹

[U]（植物體內運送養分的）汁液：</span>plant, tree sap 植物汁液；樹液 / It was spring, and the sap was rising. 春天到了，植物汁液在增多。/ The leaves, bark and sap are also common ingredients of local herbal remedies. 樹葉、樹皮和樹汁也是當地草藥的常用藥材。/ Maple syrup is made from sap extracted from the sugar maple tree. 槭糖漿是用糖槭樹中提取的樹液製成的。

周邊詞彙補充：
· flesh可表示水果或蔬菜內部的柔軟可食用部分；
· twig可指樹木的嫩枝；
· runner可指從植物的長出來的一個長長的枝條，該枝條會紮下根並長出一株新的植物，即長匐莖，纖匐枝。
