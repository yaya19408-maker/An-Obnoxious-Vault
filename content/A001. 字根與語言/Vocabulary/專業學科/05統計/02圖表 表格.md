☀ <span class="category">**圖表 表格：**</span>
<span class="vocabulary">**chart**</span> [tʃɑːt] 
<span class="definition">n. [C]（意在使資訊更容易理解的）圖表。如條形圖、餅狀圖、流程圖等（diagram也可表流程圖，二者此時一般可互換使用）。另外，chart所表示的“圖表”可帶有橫豎欄整齊的一排排數字：</span>a sales chart 銷售圖表 / bar, pie chart 條形圖；餅狀圖 / flow chart 流程圖 / accompanying chart 附圖 / flip chart 用於掛紙的白板 / wall chart（通常用wallchart）掛圖 / medical chart 病歷 / organization chart 組織架構圖 / depth chart（美）（某些體育運動中場上運動員的）陣容深度表

<span class="vocabulary">**diagram**</span> ['daɪəɡræm] 
<span class="definition">n. [C]（用線條來解釋…的位置，或…如何運轉或工作等的）一種簡易圖畫，可稱為簡圖；圖解；圖表；示意圖等。總之常帶有較強的實物性，可以看作是實物的一種紙面的簡化呈現（常配有各部位的解釋性標註）：</span>block, flow, network, schematic, tree, Venn diagram 方塊圖；流程圖；網狀圖；略圖；樹形圖；維恩圖 / circuit, wiring diagram 電路圖；佈線圖 / Draw a simple diagram of the leaf structure. 畫一張樹葉結構簡圖。/ a diagram of the human nervous system 人體神經系統圖

<span class="vocabulary">**graph**</span> [ɡrɑːf] 
<span class="definition">n. [C]（表明兩組或多組數字之間關係的）圖表，常指曲線圖（但有時也可指條形圖、線形圖、座標圖）。與chart有交集。但chart在很多情況下可呈現橫豎欄的密集數字，graph一般無此用法，一般只側重於跟具體數字有關的圖形：</span>bar, line graph 條形圖；線圖 / She drew a graph showing the relationship between costs and sales. 她畫了一張圖，揭示成本與銷售之間的關係。
           
<span class="vocabulary">**curve**</span> [kɜ:v; 美 kɜ:rv]
<span class="definition">n. [C] 曲線圖：</span>steep curve 陡曲線 / bell curve鐘形曲線 / normal curve, standard curve 正態曲線；標準曲線 / growth curve 增長曲線 / grading(= of students' grades) curve（美）（學生）分數曲線 / learning curve（= the rate at which you learn sth new）學習曲線 / grade on a curve (= to adjust scores so that they fit a normal curve）（美）按成績分佈曲線來調整分數 / to plot a curve on a graph 在圖上繪出一條曲線 / the unemployment-income curve (= a line on a graph showing the relationship between the number of unemployed people and national income)（術語）失業與國民收入曲線 / Each firm will face a downward-sloping demand curve. 每一家企業都將面臨需求曲線的下滑。/ Was it just a temporary blip on an otherwise healthy growth curve? 這是否是原本良好的增長曲線中暫時的偏離？

<span class="vocabulary">**figure**</span> ['fɪɡə] 
<span class="definition">n. [C]（書籍或雜誌中的有數字編號的）圖表：</span>See Figure 8. 見表8。

<span class="vocabulary">**form**</span> [fɔ:m] 
<span class="definition">n. [C] 一種正式檔案，包含問題和寫答案的空白處，可稱為表格：</span>application, consent, entry, order, registration, reservation（美）form 申請表；同意書；參賽表格；訂貨單；登記表；（酒店房間或餐桌等）預訂單 / to fill in a form, to complete a form 填寫表格

<span class="vocabulary">**table**</span> ['teɪbl] 
<span class="definition">n. [C] 以特定順序來排列的數字（或事實）表格（通常以橫排和豎行的形式來排列）：</span>a table of contents（書的）目錄 / statistical table 統計表 / multiplication, periodic table 乘法表；週期表
           
<span class="vocabulary">**trace**</span> [treɪs]
<span class="definition">n. [C] [術語] 機器獲取資訊後，在紙上或螢幕上顯示的一條線或一個圖形：</span>The trace showed a normal heart rhythm. 描記圖表明心率正常。

<span class="vocabulary">**questionnaire**</span> [͵kwestʃə'neə] 
<span class="definition">n. [C] questionnaire (on/about sth) 書面調查問卷（列出一系列問題，由多人解答，用於從答案中收集資訊）：</span>online questionnaire 網上問卷 / a questionnaire on the environment 環境問題調查問卷
           
<span class="vocabulary">**column**</span> [ˈkɒləm; 美 ˈkɑ:ləm]
<span class="definition">n. 1 [C]（縮略形式為col.）（書、報紙印刷頁上的分立的）豎欄：</span>a column of text 一欄正文 / a dictionary with two columns per page 每頁有兩欄正文的字典 / Put a mark in the appropriate column. 在適當的欄裡標上記號。/ Their divorce filled a lot of column inches in the national papers (= got a lot of attention). 他們的離婚引起了多家全國性報紙的關注。/ In The Dictionary of Quotations, there are no fewer than one and a half columns devoted to 'kiss'. 在《引語詞典》中，“親吻”這一條目不下一欄半。<span class="definition">2 [C]（圖表等的）縱行（數字或字）：</span>to add up a column of figures 把縱行數字相加 / We had stupidly been looking at the wrong column of figures. 我們稀裡糊塗，看串了數字欄。
          
相關詞義延伸：
· column還可指報刊雜誌的關於某一主題的專欄。
