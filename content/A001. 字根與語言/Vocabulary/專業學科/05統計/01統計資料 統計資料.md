☀ <span class="category">**統計資料 統計資料：**</span>
<span class="vocabulary">**statistic**</span> [stə'tɪstɪk] 
<span class="definition">n. 1 [pl.] statistics 可表示統計數字，統計資料：</span>population, trade, unemployment statistics 人口統計資料；貿易統計資料；失業統計資料 / bureau of statistics 統計局 / annual statistics 年度統計數字 / according to official statistics 根據官方統計 <span class="definition">2 [C] 一項統計資料：</span>an important statistic 一項重要資料

<span class="vocabulary">**data**</span> ['deɪtə] 
<span class="definition">n. [U, pl.]（尤指查詢資訊或做決策時所用的）資料、資料、資訊。是一個廣義用詞：</span>raw data 原始資料 / hard data 確鑿的資料 / experimental, scientific, statistical data 實驗資料；科學資料；統計資料 / financial, historical, technical data 財務資料；歷史資料；技術資料 / a piece of data 一項資料 / to collect data 收集資料

<span class="vocabulary">**arithmetic**</span> [ə'rɪθmətɪk,͵ærɪθ'metɪk] 
<span class="definition">n. [U] 關於某個形勢的（用數字表達出來的）資料：</span>The arithmetic was discouraging. 資料不容樂觀。

相關詞義延伸：
· statistics還可表示統計學；
· data還可表示儲存在計算機中的資料資訊。
