☀ <span class="category">**文字文件：**</span>
<span class="vocabulary">**material**</span> [mə'tɪərɪəl] 
<span class="definition">n. [C, usually pl., U] 指某一活動所需的文字材料、文字資料（或其它視覺資料、聽覺資料等）。其詞義範圍最廣，涵蓋了本組其它各詞：</span>training material 培訓資料 / confidential, sensitive material 保密資料；敏感資料 / reading material 閱讀材料 / instructional, teaching, reference material 講義；參考資料 / promotional material 宣傳推廣資料 / additional material 補充資料 / historical material 歷史資料 / original material on the First World War 關於第一次世界大戰的原始資料
           
<span class="vocabulary">**handout**</span> [ˈhændaʊt]
<span class="definition">n. [C] 課堂上發給學生的學習資料（如課程要點總結、練習題等）；也可指演講中發給聽眾的講義或提綱：</span>瞭解即可

<span class="vocabulary">**work**</span> [wə:k] 
<span class="definition">n. [U] 做某項工作所需（或所使用）的材料，尤指書籍、檔案等。也是一個廣義用詞，不過隸屬於material，且遠沒有material常用：</span>to bring work home 把辦公室裡的工作材料帶回家 / His work was spread all over the floor. 他工作的材料攤了一地。

<span class="vocabulary">**text**</span> [tekst] 
<span class="definition">n. 1 [U] 書籍或雜誌等的正文，而非引言、註釋或圖片等：</span>the text and graphics on the page 頁面上的正文和圖表 / main text 正文 <span class="definition">2 [U] 任何文字材料，即文字，文件：</span>The newspaper printed the full text of the interview. 報紙刊登了採訪的全文。/ draft text 草稿 / final text 終稿 / original text 原文 / electronic, handwritten, printed, written text 電子文字；手寫文字；印刷文字；書面文字 / Save the file in plain text format. 以純文字格式儲存檔案。/ text editor 文字編輯程式 / text box 文字框 <span class="definition">3 [C] 演講稿、廣播稿、錄音稿、劇本等文稿：</span>the full text of the speech 演講的全文 <span class="definition">4 [C] 一本書或一篇文章，即文稿，尤指與科學或學術相關的文稿：</span>academic, historical, legal, medical, literary text 學術文章；歷史文稿；法律文稿；醫學文稿；文學稿件

<span class="vocabulary">**script**</span> [skrɪpt]
<span class="definition">n. [C]（劇、電影、廣播、講話等的）劇本、講稿：</span>draft script（尤英）劇本草稿 / original script 原創劇本 / final script 定稿劇本 / comedy, film（尤英）, movie（尤美）, radio, television, TV script 喜劇劇本；電影劇本；廣播稿；電視劇本 / shooting script 拍攝指令碼 / script editor 劇本編輯 / script writer（通常用scriptwriter）編劇 / a script about a farmer's life 關於一個農民生活的劇本 / a script for children's TV 兒童電視節目指令碼 / That line isn't in the original script. 原劇本中沒有那句臺詞。/ Jenny's writing a film script. 珍妮在寫一個電影劇本。
           
<span class="vocabulary">**manuscript**</span> [ˈmænjuskrɪpt]
（縮寫為MS）<span class="definition">n. 1 [C]（書籍、樂曲等的）發表前的手稿、原稿：</span>I read her poems in manuscript. 我讀過她的詩作的手稿。/ He had seen a manuscript of the book. 他見過這本書的手稿。/ an original manuscript of the song 這首歌的原稿 / autograph, handwritten manuscript 親筆稿；手寫稿 / the original autograph manuscript of the poem 這首詩的原作真跡 / unpublished manuscript 未出版的手稿 / unsolicited manuscript 主動提供的手稿 / draft manuscript 草稿 / unfinished manuscript 未完成的底稿 / completed manuscript 完成的手稿 / book manuscript 書稿 / literary, music, musical manuscript 文學手稿；音樂手稿 / I only have one copy of the manuscript. 我只有一份底稿。/ He submitted the manuscript for publication. 他把手稿交送出版了。/ He was delighted when the manuscript was accepted for publication. 他很高興自己的稿子獲採用，可以出版。/ manuscript page 手稿頁 / The text ran to dozens of pages in manuscript form. 手稿文字有幾十頁。/ manuscript submission 稿件提交 / manuscript reviewer 審稿人 / Her autobiography remained in manuscript(= was not published). 她的自傳一直沒有出版。<span class="definition">2 [C]（印刷術發明之前書籍或文獻的）手寫本，手抄本：</span>medieval illuminated manuscripts 中世紀的裝飾過的手抄本 / early printed books and rare manuscripts 早期的印刷書籍與手抄珍本 / ancient manuscript 古代的手抄本 / the surviving manuscript of 'Beowulf 儲存下來的《貝奧武夫》的手抄本 / illustrated manuscript 配插圖的手抄本 / earlier manuscripts which no longer survive 未能儲存下來的更早期的手抄本 / manuscript illumination 手抄本的彩飾 / manuscript collection 手抄本收藏
           
<span class="vocabulary">**transcript**</span> [ˈtrænskrɪpt]
<span class="definition">n. [C]（也可用transcription（根據錄音或筆記整理的）口頭髮言的文字稿件：</span>radio transcript 廣播文稿 / interview transcript 訪談筆錄 / a transcript of the interview 採訪內容的文字稿 / court, trial transcript 法庭文字記錄；審判文字記錄 / The White House posted a transcript of the speech on its website.（post尤為美式用法）白宮在其網站上釋出了講話稿。

<span class="vocabulary">**particular**</span> [pə'tɪkjələ] 
<span class="definition">n. [pl.] [正式] particulars 可指（關於財產、業務、工作崗位等的）詳細書面材料：</span>Application forms and further particulars are available from the Personnel Office. 申請表及其他詳細資料可向人事部索取。

☀ <span class="category">**檔案簡歷：**</span>
<span class="vocabulary">**file**</span> [faɪl]
<span class="definition">n. [C] file (on sb) 一個資料夾和其中所包含的資訊（例如關於某個人或某個主題的檔案）：</span>secret police files 警方秘密檔案 / official file 官方檔案 / personal file 個人檔案 / personnel file 人事檔案 / medical file 病歷 / She closed the file and put it aside. 她合上檔案放到一邊。
           
<span class="vocabulary">**résumé**</span> [ˈrezjumeɪ; 美 ˈrezəmeɪ]
<span class="definition">n. [C] [美]（縮寫為CV）（求職用的）簡歷、履歷，相當於curriculum vitae：</span>impressive résumé 令人印象深刻的簡歷 / online résumé 線上簡歷 / You can post your résumé on the website free of charge. 你可以在網站上免費上傳簡歷。/ I received an unsolicited résumé from someone asking me to hire him. 我收到—份毛遂自薦者的簡歷，請求我聘用他。/ Yes, I've read your résumé, Very impressive. 是的，我看過你的簡歷，印象很深。/ I have been building my résumé by interning at various law firms. 透過去不同的律師事務所實習，我在不斷豐富自己的履歷。/ He padded his résumé in order to appear more experienced. 為了顯得更有經驗，他把自己的簡歷寫得誇大了一些。/ There's still a glaring hole in his résumé. 他的簡歷中仍然有個明顯的漏洞。

<span class="vocabulary">**curriculum vitae**</span>
<span class="definition">n. 1 [C] [英] [美式英語用résumé]（縮寫為CV）求職用的簡歷、履歷：</span>Applications with a full curriculum vitae and two references should reach the Principal by June 12th. 申請書連同完整個人簡歷和兩份推薦信必須在6月12日以前送達校長處。/ We read all the CVs but only interviewed three people 所有人的簡歷我們都看了，不過只面試了三個人。/ There's still a glaring hole in his resume. 他的簡歷中仍有一個很顯眼的漏洞。<span class="definition">2 [C]（縮寫為CV）美國可指（大學教師求職用的）工作履歷，也可以用vita：</span>瞭解即可

☀ <span class="category">**書籍文獻：**</span>
<span class="vocabulary">**literature**</span> ['lɪtərɪtʃə] 
<span class="definition">n. [U] 某一學科的文獻、資料、著作：</span>the literature on economic policy 關於經濟政策的文獻 / historical, medical, scientific literature 歷史文獻；醫學文獻；科學文獻 / professional（尤美）, scholarly literature 專業文獻；學術文獻
           
<span class="vocabulary">**writing**</span> [ˈraɪtɪŋ]
<span class="definition">n. [pl.] writings可尤指某一作者有關某一專題的全部著作：</span>His experiences in India influenced his later writings. 他在印度的經歷影響了他後來的著作。/ the writings of Hegel 黑格爾的著作 / Althusser's writings are focused mainly on France. 阿爾都塞的著作主要是論述法國的。/ The pieces he is reading are adapted from the writings of Michael Frayn. 他目前閱讀的作品改編自邁克爾·弗雷恩的著作。
           
<span class="vocabulary">**bible**</span> [ˈbaɪbl]
<span class="definition">n. [C] 某一主題的（經常被人引用的）權威著作：</span>the stamp-collector's bible 集郵者的寶典 / the photographer's bible--Amateur Photographer 攝影師的寶典——《業餘攝影師》

<span class="vocabulary">**library**</span> ['laɪbrərɪ] 
<span class="definition">n. [C] 由同一公司出版的外觀相似的一系列圖書、磁帶等，即系列叢書，文庫：</span>a library of Chinese classics 中國文學名著系列叢書

<span class="vocabulary">**reference**</span> ['refərəns] 
<span class="definition">n. [C] 參考書：</span>reference book, reference guide, reference manual, reference material, reference source, reference text, reference work 參考書；參考指南；參考資料；參考文獻；參考作品 / reference tool 參考工具（書）/ reference library 參考書閱覽室 <span class="definition">vt. [正式] 給（書等）附參考資料：</span>Each chapter is referenced, citing literature up to 2019. 每一章都附有參考書目，引用文獻近至2019年。
           
<span class="vocabulary">**bibliography**</span> [ˌbɪbliˈɒgrəfi; 美 -ˈɑ:g-]
<span class="definition">n. [C] （參考或引證的）文獻目錄，參考書目（是將所有參考書目列出來的一種清單）：</span>瞭解即可
 
相關詞義延伸：
· text還可尤指為考試而學習的一本書或劇本等、考試（或課文）中的文章（該文章附有一些需要回答的問題）、教科書和手機簡訊；
· particular還可尤指正式寫下的一個事實或細節、詳情；
· literature還可指（旨在銷售產品或提出建議的）書面資料；
· reference還可指書籍或論文等材料中的引文（該引文出自於一本書、一首詩、一部戲劇等），亦可指（註明引文出處的）註釋。

周邊詞彙補充：
· file可指（儲存在計算機上的）檔案或資料夾，也可指一個資料夾和其中所包含的資訊（例如關於某個人或某個主題的檔案）。
