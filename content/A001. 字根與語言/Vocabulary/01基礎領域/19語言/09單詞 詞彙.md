☀ <span class="category">**單詞 詞彙：**</span>
<span class="vocabulary">**word**</span> [wə:d] 
<span class="definition">n. [C] 字，詞，單詞，是一種有一定意義的語言單位：</span>three-letter word 三個字母的詞 / big, difficult, hard, fancy, long word 大詞；難詞；華麗的辭藻；長詞 / native word 本族語詞 / borrowed word 藉詞 / foreign word 外來詞 / content, function word 實詞；虛詞 / ambiguous word 有歧義的用詞 / key word 關鍵詞 / word game 文字遊戲 / word choice 措辭 / word list 單詞表
                      
<span class="vocabulary">**writing**</span> [ˈraɪtɪŋ]
<span class="definition">n. [U]（已被書寫或印在…上的）文字：</span>There was writing all over the desk. 書桌上寫滿了字。/ 'It's from a notebook,' the sheriff said, 'And there's writing on it.'. “這是在一個筆記本里找到的，”法官道，“上面還有文字。”/ If you have a complaint about your holiday, please inform us in writing. 如果您對您的假日有什麼意見，請書面告知我們。

<span class="vocabulary">**script**</span> [skrɪpt]
<span class="definition">n.  [U] 手寫的字：</span>neat script 整潔的字跡 / cursive, flowing script 草書；行書 / She admired his neat script. 她欣賞他寫的一手好字。/ She writes in a neat, flowing script. 她寫得一手工整流暢的好字。
                      
<span class="vocabulary">**scribble**</span> [ˈskrɪbl]
<span class="definition">n. [U, sing.] 潦草的文字。相當於scrawl：</span>How do you expect me to read this scribble? 這種寫得歪歪扭扭的東西，讓我怎麼看？/ I'm sorry what I wrote was such a scribble. 很抱歉我寫得如此潦草。
     
<span class="vocabulary">**graffiti**</span> [grəˈfi:ti]
<span class="definition">n. [U, pl.]（公共場所牆上等處的）塗鴉的文字：</span>The subway was covered in graffiti. 過街地道里塗滿了亂七八糟的圖畫和文字。/ Racist graffiti was sprayed on the walls. 牆上噴塗有種族主義的標語。
           
<span class="vocabulary">**legend**</span> [ˈledʒənd]
<span class="definition">n. [C] [正式]（標誌、徽記、硬幣等物品上的）銘文：</span>瞭解即可

<span class="vocabulary">**particle**</span> [ˈpɑ:tɪkl; 美 ˈpɑ:rt-]
<span class="definition">n. [C] [語法] 小品詞（與動詞構成短語動詞的副詞或介詞）：</span>In ‘She tore up the letter ’, the word ‘up’ is a particle. 在She tore up the letter句中，up是小品詞。

<span class="vocabulary">**vocabulary**</span> [və'kæbjələrɪ] 
<span class="definition">n. 1 [C, U] 一個人所掌握或使用的全部詞彙，即詞彙量：</span>big vocabulary 龐大的詞彙量 / limited vocabulary 有限的詞彙量 / vocabulary list 詞彙表 <span class="definition">2 [C, U] 根據重要性、常用程度、語體的正式與非正式、教學目的等標準所劃分出的某種詞彙：</span>basic, key vocabulary 基礎詞彙；重點詞彙 / formal vocabulary 正式詞彙 / everyday vocabulary 日常詞彙  / specialized vocabulary 專業詞彙 / common vocabulary 通用詞彙 / active, expressive vocabulary 積極詞彙；產出性詞彙 / passive, receptive vocabulary 消極詞彙；接受性詞彙 <span class="definition">3 [C, U] 某一門語言的全部詞彙，即詞彙量：</span>a new word in the German vocabulary 德語詞彙中的一個新詞 / English has a rich vocabulary. 英語有著豐富的詞彙。<span class="definition">4 [C, U] 某一學科中人們所使用的詞彙：</span>business, musical, political, scientific vocabulary 商業詞彙；音樂詞彙；政治詞彙；科技詞彙 / the essential vocabulary for tourism 旅遊基本詞彙 / advertising vocabulary 廣告詞彙 <span class="definition">5 [C, U] 附有釋義的一張詞彙表（尤指外語學習書籍中附有釋義的詞彙表），vocab為其非正式用法：</span>瞭解即可
           
<span class="vocabulary">**glossary**</span> [ˈglɒsəri; 美 ˈglɔ:s-; ˈglɑ:s-]
<span class="definition">n. [C] 某一學科的書籍末尾的（尤指某一篇文章的）詞彙表、術語彙編：</span>瞭解即可           

<span class="vocabulary">**lexicon**</span> [ˈleksɪkən;美-kɑ:n]
<span class="definition">n. 1 [sing.] [語言學]（也可用the lexicon）（某語言或學科、某人或群體使用的）全部詞彙和短語：</span>the lexicon of finance and economics 財經詞彙 / the lexicon of management 管理學詞彙 / Chocolate equals sin in most people's lexicon. 巧克力在大多數人的字典裡等同於罪惡。<span class="definition">2 [C]（某學科或語言的）詞彙表：</span>a lexicon of technical scientific terms 科技術語詞彙表

<span class="vocabulary">**complex**</span> ['kɒmpleks] 
<span class="definition">adj.（單詞或句子）複合的（即由一個主要部分和一個或多個其他部分組成的，具體可指詞根加有詞綴或主句含有從句）：</span>瞭解即可
           
<span class="vocabulary">**compound**</span> [ˈkɒmpaʊnd; 美 ˈkɑ:m-]
<span class="definition">n. [C] [語法] 複合詞：</span>瞭解即可

<span class="vocabulary">**root**</span> [ru:t] 
<span class="definition">n. [C] 單詞的詞根：</span>瞭解即可
           
<span class="vocabulary">**stem**</span> [stem]
<span class="definition">n. [C] [語法] 詞幹：</span>‘Writ ’ is the stem of the forms ‘writes ’, ‘writing ’ and ‘written’. writ是writes、writing和written三個詞的詞幹。
           
<span class="vocabulary">**affix**</span> [əˈfɪks]
<span class="definition">n. [C] [語法] 詞綴（包括字首和字尾，例如unhappy中的un-和careless中的-less都是詞綴）：</span>瞭解即可

<span class="vocabulary">**ending**</span> ['endɪŋ] 
<span class="definition">n. [C] 附著於一個詞的主要部分的該詞最後一部分，即詞尾：</span>verb endings 動詞詞尾 / a feminine ending 陰性詞尾

<span class="vocabulary">**phrase**</span> [freɪz] 
<span class="definition">n. [C] 短語，片語：</span>to change a phrase 改個短語
           
<span class="vocabulary">**chunk**</span> [tʃʌŋk]
<span class="definition">n. [C] [語言學] 可以作為一個整體來學習的語言單位，如短語等，即語塊：</span>瞭解即可

<span class="vocabulary">**coin**</span> [kɒɪn] 
<span class="definition">vt. 創造一個新詞或短語（後來其他人也開始使用該詞或短語）：</span>The term ‘cardboard city’ was coined to describe communities of homeless people living in cardboard boxes. 人們創造了cardboard city一詞，用來指居住在紙板棚屋裡無家可歸者的居住區。
           
<span class="vocabulary">**flourish**</span> [ˈflʌrɪʃ; 美 ˈflɜ:rɪʃ]
<span class="definition">n. [C]（尤指手寫體的）花彩字：</span>瞭解即可
