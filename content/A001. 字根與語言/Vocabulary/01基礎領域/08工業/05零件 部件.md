☀ <span class="category">**零件 部件：**</span>
<span class="vocabulary">**component**</span> [kəm'pəʊnənt] 
<span class="definition">n. [C] 零件，尤指機械或交通工具的零部件或元件：</span>components of the machine 機器部件 / electrical, electronic component 電器部件；電子零件 / aircraft, engine component 飛機零件；發動機的零件 / component failure 零件故障 / component supplier 零部件供應商

<span class="vocabulary">**part**</span> [pɑːt] 
<span class="definition">n. [C] 機器的部件，零件。是一個簡單用詞。但在詞義上與component未見明顯區別：</span>car, motorcycle part 汽車零件；摩托車零件 / aircraft part 飛行器零件 / the working parts of the machinery 機器的執行部件 / replacement, spare part 備件 / part number 部件編號

<span class="vocabulary">**section**</span> ['sekʃn] 
<span class="definition">n. [C] 整體結構中的一個部件，不太常用：</span>The table comes in sections that you assemble yourself. 桌子以散件出售，需要自己組裝。

<span class="vocabulary">**piece**</span> [pi:s] 
<span class="definition">n. [C]（組成…的）零件，部件。是一個廣義用詞，其含義已經包括了機器零件。同時也是一個簡單用詞和基本用詞，日常生活中比較常用：</span>He took the clock to pieces. 他把鍾拆散了。/ The bridge was taken down piece by piece. 橋樑被一部分一部分地拆毀。/ a 500 piece jigsaw 一副500片的拼圖玩具 / standard pieces 標準部件

<span class="vocabulary">**unit**</span> ['ju:nɪt] 
<span class="definition">n. [C] 用於組成一個大機器的小機器，即裝置；機件；部件；元件。該裝置又常由若干個基礎零件組成，來完成機器內部的某項任務：</span>air-conditioning, control, power unit 空調裝置；控制裝置；電力裝置 / the central processing unit in a computer 計算機的中央處理器 / The unit plugs into any TV set. 該部件可接入任何電視機。
   
<span class="vocabulary">**kit**</span> [kɪt]
<span class="definition">n. [C] 一套將被做成…的元件：</span>a kit for a model plane 一套飛機模型元件 / Her popular pot holder is also available in do-it-yourself kits. 她那廣受歡迎的防燙布墊也可以自己購買原料套裝自己製作。/ construction kit 建構套件 / starter kit 入門套件 / I spent Sunday assembling emergency kits for the expedition. 星期天一整天我都花在了組裝探險應急套件上。/ Each kit comes with a sealable bag. 每組套件都配有一個可密封的袋子。/ The kit contains everything you need to make six candles. 這組套件包含了製作六支蠟燭所需的全部材料。/ They built the garage from a kit. 他們用一套配件建成了那個車庫。/ The doll's house comes in kit form. 這個玩偶房是用配套元件拼裝成的。

<span class="vocabulary">**module**</span> [ˈmɒdju:l; 美 ˈmɑ:dʒul]
<span class="definition">n. [C]（機器的）元件、模組：</span>瞭解即可
 
<span class="vocabulary">**machinery**</span> [məˈʃi:nəri]
<span class="definition">n. [U] 使機器運轉的一些零部件：</span>瞭解即可
           
<span class="vocabulary">**attachment**</span> [əˈtætʃmənt]
<span class="definition">n. [C] 機器的附件、附加裝置：</span>an electric drill with a range of different attachments 配有各種附件的電鑽 / Some models come with attachments for dusting. 一些模型上附帶了除塵器。
           
<span class="vocabulary">**accessory**</span> [əkˈsesəri]
<span class="definition">n. [usually pl.] 機器裝置等的附件、配件；起裝飾作用的附屬物：</span>essential, must-have accessory 必不可少的附件；必備的配件 / car accessory 汽車配件 / bicycle accessories 腳踏車附件 / a range of furnishings and accessories for the home 各種各樣的家居裝飾物及配件 / an exclusive range of hand-made bedroom and bathroom accessories 獨家經營的一系列手工臥室和浴室配件
           
<span class="vocabulary">**fitting**</span> [ˈfɪtɪŋ]
<span class="definition">n. 1 [usually pl.]（裝置或傢俱的）小配件：</span>light fittings 燈具配件 / a pine cupboard with brass fittings 黃銅鑲配的松木櫥櫃 / industrial fittings for kitchen and bathroom 工業化生產的廚衛零配件 / standard fittings 標準配件 / special fittings 特別配件 / metal, plastic fittings 金屬配件；塑膠配件 <span class="definition">2 [usually pl.] [英] 房屋內的安裝好的但可拆除的裝置（如櫥、灶、燈、擱架）：</span>original fitting 原裝配件 / interior, internal fitting 室內配件 / electrical, plumbing fitting 電器配件；管道裝置配件 / bathroom, kitchen fitting 浴室裝置；廚房裝置 / Our bathroom fittings are easy to install. 我們的浴室裝置很容易安裝。/ The room still has the original fixtures and fittings. 這個房間還留著當初的固定裝置和附加裝置。
           
<span class="vocabulary">**fixture**</span> [ˈfɪkstʃə(r)]
<span class="definition">n. [C] [尤英] 固定設施（如房屋內安裝的浴缸或抽水馬桶，搬家時不能搬走的東西）：</span>The price of the house includes fixtures and fittings. 房屋價格包括固定裝置和附加裝置。/ an exceptional example of Victorian architecture with the original fixtures intact 原有固定裝置完好無損的維多利亞時期建築的優秀典範 / The apartment still has the original light fixtures. 這所公寓仍然保留著原來安裝的燈具。/ fixtures and fittings（英）固定裝置和附加裝置
 
<span class="vocabulary">**control**</span> [kən'trəʊl] 
<span class="definition">n. [C, usually pl.]（機器或車輛的）操縱裝置（如開關、按鈕等）：</span>the controls of an aircraft 飛機的操縱裝置 / remote control 遙控器 / volume control 音量調節器 / control panel 控制（面）板 / control device 控制裝置 / a programmable control unit 可程式設計的控制單元 / control circuit, control valve 控制電路；控制閥
           
<span class="vocabulary">**controller**</span> [kənˈtrəʊlə(r); 美 -ˈtroʊ-]
<span class="definition">n. [C] [術語] 機器的控制裝置、調節裝置：</span>a temperature controller 溫控器 / game controller（美）遊戲控制器 / remote, wireless controller（均美）遙控器；無線控制器

<span class="vocabulary">**lever**</span> [ˈli:və(r); 美 ˈlevər]
<span class="definition">n. [C]（機器的）操縱桿，控制桿：</span>brake, control, gear, shift（美）leverage 制動杆；控制桿；變速桿 / safety leverage 保險桿 / To release the brake, pull the leverage. 要鬆開剎車，得拉動操縱桿。/ Push the gear leverage into first. 將變速桿推到一擋。/ The machine will stop immediately once the leverage is released.一鬆開操縱桿，機器就會馬上停下來。/ Set all three leverages to the 0 position. 將三根操縱桿全部歸零。/ The machine is operated by means of a leverage. 這臺機器透過一個操縱桿操控。/ Pull the lever towards you to adjust the speed. 把操縱桿向你身體一側拉動以調節速度。/ Push the tiny lever on the lock. 按一下鎖上的小柄。/ The taps have a lever to control the mix of hot and cold water. 水閥上有一個手柄控制冷熱水。

<span class="vocabulary">**button**</span> ['bʌtn] 
<span class="definition">n. [C] 機器的按鈕：</span>the right mouse button 滑鼠右鍵 / on, start button 開機鍵；啟動按鍵 / off, stop button 關機鍵；停止鍵 / control button 控制鍵 / fast-forward, pause, play button 快進鍵；暫停鍵；播放鍵 / alarm, panic button 報警鍵；緊急按鈕 / elevator（美）, lift（英）button 電梯按鈕 / self-destruct button（常用作比喻義）自毀按鈕 / to release button 鬆開按鈕
              
<span class="vocabulary">**knob**</span> [nɒb; 美 nɑ:b]
<span class="definition">n. [C] 機器的球形開關：</span>the knobs on the radio 收音機的旋鈕 / control knob 控制旋鈕 / volume knob 音量旋鈕 / the volume control knob 音量控制旋鈕 / Adjust the control knobs by pressing lightly. 輕輕按壓來調節控制旋鈕。/ I've tried fiddling with the knobs, but nothing seems to happen. 我試著撥弄那些旋鈕，但似乎什麼作用也沒有。

<span class="vocabulary">**starter**</span> [ˈstɑ:tə(r); 美 ˈstɑ:rt-]
<span class="definition">n. [C] 車輛發動機的啟動裝置：</span>A car's starter is basically an electric motor. 汽車的啟動裝置基本上是個電動機。
            
<span class="vocabulary">**pedal**</span> [ˈpedl]
<span class="definition">n. [C]（機器上的）踏板：</span>To stop the machine push the foot pedal. 踩腳踏板可以讓機器停止運轉。/ Her foot was working the pedal of the sewing machine. 她用腳蹬著縫紉機的踏板。/ pedal bin（英）腳踏式小垃圾桶

<span class="vocabulary">**belt**</span> [belt] 
<span class="definition">n. [C]（連續執行的，用於運送物品或帶動機器的）傳送帶或傳動帶：</span>conveyor, drive, fan, timing belt 傳送帶；驅動帶；風扇皮帶；正時皮帶
           
<span class="vocabulary">**bearing**</span> [ˈbeərɪŋ; 美 ˈber-]
<span class="definition">n. [C] [術語]（機器的）承座；（尤指）軸承：</span>An oil seal was replaced, along with both front wheel bearings. 換了一個油封和兩個前輪軸承。
           
<span class="vocabulary">**shaft**</span> [ʃɑ:ft; 美 ʃæft]
<span class="definition">n. [C]（常構成複合詞）（機器的）軸、傳動軸：</span>a drive shaft 驅動軸 / the propeller shaft 螺旋槳的傳動軸
 
<span class="vocabulary">**grill**</span> [ɡrɪl] 
<span class="definition">n. [C]（置於機器前以對其起保護作用的，由金屬桿或鐵絲做成的）金屬網罩：</span>瞭解即可

<span class="vocabulary">**work**</span> [wə:k] 
<span class="definition">n. [C] the works 複數，專指機器等的活動部件，相當於mechanism：</span>瞭解即可

<span class="vocabulary">**spare**</span> [spεə] 
<span class="definition">n. [pl.] [尤英] spares 可指（機器的）備品，備件，相當於spare parts：</span>It can be difficult to get spares for some older makes of car. 有些舊型號汽車可能不好買配件。

<span class="vocabulary">**arm**</span> [ɑːm] 
<span class="definition">n. [C]（尤指可以移動的）臂狀物體或機器部件：</span>a lever arm of the machine. 該機器的一個槓桿臂

<span class="vocabulary">**claw**</span> [klɔ:]

<span class="vocabulary">**bucket**</span> [ˈbʌkɪt]

<span class="vocabulary">**paddle**</span> ['pædl]

<span class="vocabulary">**blade**</span> [bleɪd]
<span class="definition">n. 1 [C] 發動機內部或直升機上的旋轉的葉片、槳葉：</span>the blades of a propeller 螺旋槳葉 / turbine blade 渦輪葉片 / a helicopter's rotor blades 直升機的旋翼 <span class="definition">2 [C] 船槳的槳葉：</span>瞭解即可

<span class="vocabulary">**sail**</span> [seɪl]

<span class="vocabulary">**spring**</span> [sprɪŋ]

<span class="vocabulary">**exhaust**</span> [ɪgˈzɔ:st]

<span class="vocabulary">**overflow**</span> [ˌəʊvəˈfləʊ; 美 ˌoʊvərˈfloʊ]

<span class="vocabulary">**register**</span> ['redӡɪstə]

<span class="vocabulary">**valve**</span> [vælv]

<span class="vocabulary">**intake**</span> [ˈɪnteɪk]

<span class="vocabulary">**cock**</span> [kɒk; 美 kɑ:k]

<span class="vocabulary">**reservoir**</span> [ˈrezəvwɑ:(r); 美 ˈrezərv-]

<span class="vocabulary">**chamber**</span> [ˈtʃeɪmbə(r)]

<span class="vocabulary">**jaw**</span> [dӡɔ:]

周邊詞彙補充：
· wiring可指（給機器供電的）線路；
· drive可表示交通工具中的傳動（或驅動）裝置；
· safety可指槍炮（或機器裝置）的保險栓；
· instrument可指（交通工具、機器等上面用於測量速度、高度、聲音等的）儀器，儀表；
· console可指（機器、電子裝置等的）控制檯、操縱檯、儀表板。
