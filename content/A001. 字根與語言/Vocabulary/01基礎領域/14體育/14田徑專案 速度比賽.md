☀ <span class="category">**田徑專案 速度比賽：**</span>
<span class="vocabulary">**track**</span> [træk] 
<span class="definition">n. [C] 美式英語中可指徑賽運動：</span>track event 徑賽專案 / track athlete 徑賽運動員；賽跑運動員 / track coach 徑賽教練

<span class="vocabulary">**mile**</span> [maɪl] 
<span class="definition">n. [sing.] the mile可指一英里的速度比賽：</span>He ran the mile in less than four minutes. 他用了不到四分鐘就跑完一英里。

<span class="vocabulary">**race**</span> [reɪs] 
<span class="definition">n. [C] race (between A and B) / race (against sb) 比拼速度的比賽，如賽跑、游泳、賽車等：</span>relay, road race 接力賽；公路賽 / boat race 划船比賽 / bicycle, car, motorcycle race 腳踏車賽；汽車賽；摩托車賽 / dog, horse race 賽狗；賽馬 / race meet（尤美）, race meeting（英）賽馬大會
                      
<span class="vocabulary">**racing**</span> [ˈreɪsɪŋ]
<span class="definition">n. [U] 通常構成複合詞，指任何一種速度比賽：</span>competitive racing 競爭激烈的比賽 / auto（美）, car, drag（尤美）, motor, NASCAR（美）, stock-car（英）racing 汽車賽；減重短程高速汽車賽；（美國）全國賽車聯合會汽車賽；（改裝型）賽車汽車賽 / dog, greyhound racing 賽狗；靈緹賽 / bicycle, bike（非正式）, yacht racing 腳踏車賽；快艇賽 / Grand Prix racing 國際汽車大獎賽 / road, track racing 公路賽；徑賽 / racing champion 比賽冠軍 / racing legend, racing star 比賽傳奇人物；比賽明星 / racing legend Ayrton Senna 賽車傳奇人物艾爾頓·森納 / racing cyclist（尤英）, racing driver 腳踏車賽手；賽車手 / racing team 參賽隊 / racing event 賽事 / racing programme/program, racing season 賽車賽程；賽車賽季 / racing scene 賽車圈 / racing career, racing days 賽車生涯；賽車歲月 / He made his racing debut for Ferrari three years ago. 三年前，他首次代表法拉利車隊參賽。/ racing car（英）（美式英語用race car）賽車 / racing circuit, racing track 賽道 / The bikes sped by on the racing circuit. 腳踏車在賽道上飛馳而過。/ racing enthusiast, racing fan 賽車迷；賽馬迷

<span class="vocabulary">**formula**</span> [ˈfɔ:mjələ; 美 ˈfɔ:rm-]（複數形式為formulas，科技領域常用formulae）
<span class="definition">n. [C] 在Formula One用法中，指方程式（按發動機大小等對賽車的分級）：</span>Formula One™ racing 一級方程式賽車
           
<span class="vocabulary">**scramble**</span> [ˈskræmbl]
<span class="definition">n. [C] 摩托車越野賽：</span>瞭解即可
           
<span class="vocabulary">**pursuit**</span> [pəˈsju:t; 美 pərˈsu:t]
<span class="definition">n. [sing.]（腳踏車賽和滑冰競賽中的）追逐賽：</span>Moreau took gold in the five-kilometre individual pursuit competition. 莫羅在個人五公里追逐賽中獲得金牌。

<span class="vocabulary">**start**</span> [stɑːt] 
<span class="definition">n. [C, usually pl.] 體育運動中可指某人所參加的比賽（如速度比賽）：</span>She has been beaten only once in six starts. 她參加了六次比賽，只敗過一次。

<span class="vocabulary">**run**</span> [rʌn] 
<span class="definition">vi. 1 跑步，奔跑：</span>John can run very fast. 約翰跑得很快。<span class="definition">2 跑步作為一項運動：</span>I often go running before work. 我常常在上班前跑步。<span class="definition">vt. 跑一段距離：</span>to run a mile in under four minutes 用不到四分鐘跑完一英里 <span class="definition">vt.＆vi. run (in sth) 參加跑步比賽：</span>to run the marathon 參加馬拉松比賽 / a horse running in the race 參加比賽的一匹馬 <span class="definition">n. [C] 奔跑；跑步所花的時間，跑步的距離：</span>I go for a run every morning. 我每天早晨都去跑步。/ a five-mile run 五英里跑步 / training run 跑步訓練
                      
<span class="vocabulary">**running**</span> [ˈrʌnɪŋ]
<span class="definition">n. [U] 跑，跑步（運動）：</span>We chose to do cross-country running. 我們選擇了越野賽跑。/ fell running（英）山地賽跑 / distance, long-distance, marathon, middle-distance running 長跑；馬拉松賽跑；中距離賽跑 / running event, running race 賽跑 / running gear 跑步用裝備 / running clothes（美）, running pants（美）, running shoe, running shorts, running tights（美）, running vest（英）跑步服；賽跑短褲；跑鞋；賽跑緊身褲；跑步衫 / running track 跑道

<span class="vocabulary">**sprint**</span> [sprɪnt]
<span class="definition">vt.&vi. 短距離快速奔跑或游泳，即衝刺：</span>He sprinted for the line. 他向終點線衝去。/ Three runners sprinted past. 三名運動員飛跑了過去。<span class="definition">n. 1 [C] 短距離速度比賽（如賽跑、游泳比賽）：</span>the women's 100-metres sprint 女子 100 米短跑 / the world sprint champion 短跑世界冠軍 / Rob Harmeling won the sprint in Bordeaux. 羅布·哈梅林在波爾多獲得短跑冠軍。/ Lewis will compete in both sprints in Stuttgart. 劉易斯將參加在斯圖加特舉行的兩個短跑專案的比賽。/ I knew there were other riders who could beat me in a sprint. 我知道還有其他騎手可能在短程賽馬中打敗我。/ Jamaica won the gold in the sprint relay. 牙買加隊在短跑接力中獲得金牌。/ sprint hurdles（英）短距離跨欄 / This car could comfortably outpace its rivals in a sprint. 這輛車在短距離賽中能夠輕鬆超過對手。<span class="definition">2 [usually sing.] 短距離快速奔跑或游泳等，即衝刺：</span>quick, short sprint 飛奔；短跑 / final sprint 衝刺 / The runners broke into a sprint a hundred yards from the finish. 賽跑選手們在離終點一百碼的地方開始衝刺。/ a sprint for the line 向終點線的衝刺 / She won in a sprint finish. 她在最後的衝刺中取得勝利。/ He just pipped the Kenyan runner in a sprint finish. 他在最後衝刺時才險勝肯亞選手。/ The Moroccan came out best in a frantic sprint for the line. 在向終點線的瘋狂衝刺中摩洛哥運動員勝出。/ Gilles Delion, of France, won the Tour of Lombardy in a sprint finish at Monza yesterday. 法國選手吉勒斯·德利翁昨天在蒙扎憑藉最後的衝刺獲得倫巴第巡迴賽冠軍。
           
<span class="vocabulary">**spurt**</span> [spɜ:t; 美 spɜ:rt]
<span class="definition">vi. 短暫地加速前進（以更快到達某地），如衝刺：</span>She spurted past me to get to the line first. 她衝刺超過我，率先抵達終點線。

<span class="vocabulary">**dash**</span> [dæʃ] 
<span class="definition">n. [C, usually sing.] [尤美] 短距離快速賽跑，相當於sprint：</span>the 100-meter dash 百米賽跑

· 表示“猛衝”的詞群見[[01移動 靜止]]

<span class="vocabulary">**marathon**</span> ['mærəθɒn] 
<span class="definition">n. [C] 馬拉松賽跑：</span>the Beijing Marathon 北京馬拉松賽跑 / full, half marathon 全程馬拉松；半程馬拉松 / marathon runner 馬拉松賽選手

<span class="vocabulary">**relay**</span> ['ri:leɪ] 
<span class="definition">n. [C] 接力賽，例如接力賽跑或接力游泳等，也可叫relay race：</span>freestyle, sprint relay 自由泳接力；短跑接力賽 / She ran in the 4 by 400 relay. 她參加了4×400米接力。/ relay race 接力賽 / relay team 接力隊
           
<span class="vocabulary">**chase**</span> [tʃeɪs]
<span class="definition">n. [C] 越野障礙賽，相當於steeplechase：</span>瞭解即可

<span class="vocabulary">**metre**</span> ['mi:tə]（美meter）
<span class="definition">n. [C, U] 米，公尺（用於速度比賽的名稱，縮寫為m）：</span>She came second in the 200 metres. 在200米比賽中，她取得了第二名。/ the 4×100 metre(s) relay 4×100米接力賽

<span class="vocabulary">**off**</span> [ɔ:f, ɒf] 
<span class="definition">adv. 速度比賽開始，如起跑：</span>They’re off (= the race has begun). 他們起跑了。<span class="definition">n. [sing.] the off 可指起跑：</span>They’re ready for the off. 他們準備起跑了。
