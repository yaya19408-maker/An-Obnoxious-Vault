☀ <span class="category">**複習 作業：**</span>
<span class="vocabulary">**review**</span> [rɪ'vju:] 
<span class="definition">n. [C] [美]（尤指為準備考試所上的）複習課：</span>review class, review session 複習課 / review materials 複習資料 / review question 複習題 / We need to do the review for the test tomorrow. 我們需要為明天的測驗進行復習。<span class="definition">vt.＆vi. [尤美]（尤指為準備考試而）複習，溫習：</span>review for the test（美）為考試而複習
           
<span class="vocabulary">**revise**</span> [rɪˈvaɪz]
<span class="definition">vt. [英] 為考試而複習、溫習：</span>I spent the weekend revising for my exam. 我花了整個週末複習備考。/ I can't come out tonight. I have to revise. 我今晚不能出去。我得複習。/ I'm revising Geography today. 我今天覆習地理。/ Have you revised for the test tomorrow? 明天的測驗你複習了嗎?

<span class="vocabulary">**revision**</span> [rɪ'vɪӡn] 
<span class="definition">n. [U] [英]（為準備考試所做的）複習：</span>I’ve got to do some history revision for tomorrow’s exam. 我得複習一下歷史以應對明天的考試。/ revision class, revision course, revision lesson 複習課 / revision question 複習題 
           
<span class="vocabulary">**assignment**</span> [əˈsaɪnmənt]
<span class="definition">n. [C, U] 學校佈置的作業：</span>reading, writing, written assignment 閱讀作業；寫作作業；書面作業 / homework assignment 家庭作業 / class, group assignment（均尤美）班級作業；小組作業 / The teacher gave us an assignment on pollution. 老師給我們佈置了有關汙染的作業。/ to do assignment, to write assignment 做作業；寫作業 / to complete assignment, to finish assignment 完成作業 / The students handed in their assignments. 學生們交了作業。/ an assignment on modern life-styles 有關現代生活方式的作業 / You will need to complete three written assignments per semester. 你每學期要完成三個書面作業。/ The assessment for the course involves written assignments and practical tests. 這門功課的考核包括書面作業和實際操作考試。

<span class="vocabulary">**homework**</span> ['həʊmwə:k] 
<span class="definition">n. [U]（教師給學生布置的）家庭作業：</span>English homework 英語作業 / to do a piece of homework 做一項家庭作業 / to hand in homework 交家庭作業 / to give a lot of homework 佈置很多家庭作業
