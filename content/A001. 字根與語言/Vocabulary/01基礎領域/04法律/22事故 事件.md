☀ <span class="category">**事故 事件：**</span>
<span class="vocabulary">**accident**</span> ['æksɪdənt] 
<span class="definition">n. [C] 一次令人不快的事件，尤指意外發生且造成傷害的事故，且多指交通事故：</span>hit-and-ran accident（汽車駕駛員等）肇事逃逸事故 / car, road, traffic accident 汽車事故；公路事故；交通事故 / aircraft accident 飛行事故 / boating, climbing accident 划船事故；登山事故 / accident black spot（英）事故多發地段 / accident victim 事故受害者 / accident prevention 事故的預防 / accident investigation 事故的調查 / accident report 事故的報道 / accident rate 事故發生率 / the scene of the accident 事故現場

<span class="vocabulary">**incident**</span> ['ɪnsɪdənt] 
<span class="definition">n. [C, U] 一次嚴重或暴力事件，如犯罪、事故或襲擊。其詞義已經包含了accident，可以說accident是一種incident：</span>violent incident 暴力事件 / terrorist incident 恐怖事件 / nuclear incident 核事件 / bombing, shooting, stabbing incident 爆炸事件；槍擊事件；刺傷事件 / pollution incident 汙染事件 / alcohol-related incident 涉酒事件 / incident room 事件調查辦公室 / incidents arising out of an industrial dispute 由勞資糾紛引發的事件 / It’s not an isolated incident 這不是一起孤立事件。/ The demonstration passed without incident. 遊行示威很順利，中間沒出什麼亂子。
