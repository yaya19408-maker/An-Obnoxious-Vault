☀ <span class="category">**國界 邊境：**</span>
<span class="vocabulary">**border**</span> ['bɔ:də] 
<span class="definition">n. [C] 兩國或兩個地區之間的分界線；也可指此種分界線附近的地區：</span>closed border 封閉的邊境 / national border 國界 / Nevada’s northern border 內華達州的北部邊界 / border crossing, border post 邊境過境處；邊境哨卡 / border control, border guard 邊界管控；邊防哨兵 / border clash 邊境衝突 / border region, border town 邊境地區；邊陲小鎮 / a farm on the border 交界處的一個農場 / to be stopped on the border 在邊境被攔住 / to slip across the border 偷越國境 / A river forms the border between A and B. 一條河流成為AB兩地之間的邊界。<span class="definition">vt.（國家或地區）有共同的邊界線：</span>the countries bordering the Baltic 波羅的海沿岸國家 

<span class="vocabulary">**boundary**</span> ['baʊndərɪ] 
<span class="definition">n. [C] 一條真實或虛擬的地理邊界線或分界線：</span>national, state boundary 國界；州界 / county boundaries（英）郡界 / territorial boundary 領土邊界 / district boundary 行政區界 / geographical boundary 地理分界 / northern boundary 北部邊界 / boundary fence, boundary line 邊界圍欄；邊界線 / boundary dispute 邊境爭端 / to drive across the boundary 開車穿過邊界 / The river forms the boundary. 這條河流形成了邊界。/ The fence marks the boundary between my property and hers. 那道籬笆是我和她的住宅之間的分界。/ She had never strayed beyond the city boundaries. 她從未出過這個城市。/ within the boundaries of the old city walls 在老城牆的範圍以內

<span class="vocabulary">**frontier**</span> ['frʌntɪə] 
<span class="definition">n. 1 [C] [英] frontier (between A and B) / frontier (with sth) 兩地（常是兩國）之間的邊界線；也可指此種邊界線附近的地區：</span>the frontier between India and Pakistan 印度和巴基斯坦之間的邊界 / frontier town 邊陲小鎮 / eastern frontier 東部邊界 / to cross the frontier 越過邊界 / frontier controls, frontier post 邊境檢查站；邊境哨所 / frontier guard 邊防戰士 / frontier zone 邊境地帶 / people living on the German frontier 居住在德國邊境地區的人 <span class="definition">2 [sing.] the frontier（尤指19世紀美國西部的）開發地區的邊緣地帶（人們在此居住並建設城鎮，更遠的地方則被視為荒野和未知地帶）：</span>America’s wild frontier 美國的荒野地帶 / the western frontier 西部的邊疆地帶 / frontier town 邊疆城鎮 / frontier spirit 邊疆精神

<span class="vocabulary">**division**</span> [dɪ'vɪӡn] 
<span class="definition">n. [C]（劃分…的）邊界線：</span>A hedge forms the division between their land and ours. 他們的土地與我們的土地之間以一道樹籬隔開。

<span class="vocabulary">**crossing**</span> [krɒsɪŋ] 
<span class="definition">n. [C] 從一國到另一國所經過的過境處：</span>the main crossing point on the border 邊境上的主要過境點 / They were stopped and searched at the border crossing. 他們在邊境過境處被攔下來搜查。

說明：表“國境線”時，①border常指實地的國界線，是此義的常規用詞；②boundary常指地圖上標出的國界線。雖亦可表示實地的國界線，但此時遠不如border常用；③frontier表“國境線”時，為英式用法，且常暗含一種邊遠、荒蕪、動盪、危險的色彩。在語法搭配上與border多有交集；④line一般專指地圖上標示出來的國界線，此時與boundary在詞義上幾無差別，但boundary是此義的常規用法，而line的此義是其基本詞義衍生出來的一個用法。二者也可連用，共同表示地圖上的“國界線”，如the boundary line between two countries（兩國之間的邊界線）。另需注意，在美國line不用於指國界線，而一般用來指州縣之間的界線，如the state line（州界線）；⑤limit本義是指一個區域（或一個地方）的最遠的邊緣。強調一種單向的向外擴張的極限性，而不強調border等詞的那種“劃分”的屬性。該詞在實際使用中不太用來指“國界線”，只是其詞義範圍已經涵蓋了“國界線”，所以瞭解一下即可。

周邊詞彙補充：
· bound可指形成一個區域的邊緣或界限；
· limit可指一個區域（或一個地方）的最遠的邊緣，如地界、範圍。
