☀ <span class="category">**家畜 畜類：**</span>
<span class="vocabulary">**domestic**</span> [dəˈmestɪk]
<span class="definition">adj.（動物）馴養的，非野生的：</span>domestic animal 家畜        
           
<span class="vocabulary">**domesticated**</span> [dəʊ'mestɪkeɪtɪd；美dəˈmɛstəˌketɪd]
<span class="definition">adj.（動物）被馴養的：</span>domesticated animals 馴養的動物 / These animals are only partly domesticated. 這些動物只是部分馴化了。

<span class="vocabulary">**stock**</span> [stɒk; 美 stɑ:k]
<span class="definition">n. [U]（飼養以供獲取其肉、皮毛等的）牛、羊等家畜、牲畜：</span>breeding stock 種畜

<span class="vocabulary">**cattle**</span> ['kætl] 
<span class="definition">n. [pl.] 被作為家畜來馴養的牛，包括bull（公牛）和cow（母牛）：</span>beef, dairy cattle 菜牛；奶牛 / wild cattle 野牛 / domestic cattle 家養牛 / live cattle 活牛 / a head of cattle 一頭牛 / a herd of cattle 一群牛 / cattle industry 養牛業 / cattle dealer 牛販子 / cattle farm 養牛的牧場 / cattle herd 牛群 / cattle feed, cattle food 牛飼料

<span class="vocabulary">**cow**</span> [kaʊ] 
<span class="definition">n. [C] 被飼養以產奶的大型雌牛，即奶牛；有時也把該物種的雄性和雌性動物統稱為cows：</span>dairy cows 奶牛 / beef cow 肉牛 / a herd of cows 一群牛 / milch cow（比喻用法）搖錢樹 / cow pasture 養牛的牧場

<span class="vocabulary">**ox**</span> [ɒks] 
<span class="definition">n. 1 [C] 被閹割的公牛，用於拉動農具或拉動交通工具、搬運貨物：</span>瞭解即可 <span class="definition">2 [C] 舊時可指任何被作為家畜來馴養的公牛或母牛，相當於現在的cattle：</span>瞭解即可
           
<span class="vocabulary">**steer**</span> [stɪə(r); 美 stɪr]
<span class="definition">n. [C] 飼養以供食用的被閹割的公牛：</span>瞭解即可
           
<span class="vocabulary">**calf**</span> [kɑ:f; 美 kæf]
<span class="definition">n. [C] 小牛、牛犢：</span>bull，heifer calf 小公牛；小母牛 / beef, dairy, veal calf 小菜牛；小乳牛；肉用小牛 / newborn calf 初生小牛犢 / weaned calf 斷奶的小牛 / A dairy cow needs to produce a calf each year. 奶牛每年都需要產一頭小牛。/ These calves are reared for beef. 這些小牛是作為肉牛飼養的。

<span class="vocabulary">**low**</span> [ləʊ] 
<span class="definition">vi. [文學] 牛（哞哞地）叫：</span>The cow is lowing. 那頭牛在哞哞叫。

<span class="vocabulary">**sheep**</span> [ʃi:p] 
<span class="definition">n. [C] 綿羊：</span>a flock of sheep 一群羊 / hill（英）, mountain sheep 山坡上的羊；大山裡的羊 / lost（常為比喻用法）, stray sheep 迷途的羊 / domestic, wild sheep 家養羊；野羊 / sheep farm 牧羊場 / sheep industry 綿羊養殖業 / sheep pen 羊圈 / sheep shearing 剪羊毛

<span class="vocabulary">**goat**</span> [ɡəʊt] 
<span class="definition">n. [C] 山羊：</span>mountain goat 山羊 / wild goat 野山羊 / goat’s milk 山羊奶

<span class="vocabulary">**kid**</span> [kɪd] 
<span class="definition">n. [C] 小山羊：</span>瞭解即可

<span class="vocabulary">**lamb**</span> [læm] 
<span class="definition">n. [C] 小綿羊，即羔羊：</span>a lost lamb 走失的羔羊 / sacrificial lamb（通常為比喻用法）犧牲品

<span class="vocabulary">**horse**</span> [hɔ:s] 
<span class="definition">n. [C] 馬：</span>fast horse 快馬 / wild horse 野馬 / race horse（通常用racehorse）賽馬 / horse hooves 馬蹄 / horse race 賽馬 / a horse and carriage, a horse and cart 馬車
                       
<span class="vocabulary">**pony**</span> [ˈpəʊni; 美 ˈpoʊni]
<span class="definition">n. [C] 矮種馬、小型馬：</span>wild pony 小野馬 / polo pony 馬球用馬 / pit pony（英，歷史用法）井下用馬 / Pit ponies were used in most mines at the turn of the last century. 上世紀初多數礦井都用矮種馬進行井下運輸。/ pony club 矮種馬馬術俱樂部 / pony ride騎小馬/ pony riding, pony trekking（均尤英）騎小馬消遣；騎小馬出遊 / a pony and trap（尤英）小馬拉的雙輪輕便馬車

<span class="vocabulary">**mount**</span> [maʊnt]
<span class="definition">n. [C] [正式或文學] 一匹供騎乘的馬，即坐騎：</span>the number of owners who care for older mounts 願意照料老馬的馬主的數量          

<span class="vocabulary">**hack**</span> [hæk]
<span class="definition">n. [C] 供日常騎的馬；供出租的騎用馬：</span>瞭解即可
            
<span class="vocabulary">**novice**</span> [ˈnɒvɪs; 美 ˈnɑ:v-]
<span class="definition">n. [C] 尚未贏過重要賽事的一匹馬：</span>瞭解即可

<span class="vocabulary">**hunter**</span> ['hʌntə] 
<span class="definition">n. [C] [英] 捕獵狐狸時所用的一匹快速強壯的馬：</span>瞭解即可

<span class="vocabulary">**pair**</span> [peə] 
<span class="definition">n. [C] 一起拉馬車的兩匹馬：</span>a carriage and pair 雙駕馬車

<span class="vocabulary">**bay**</span> [beɪ] 
<span class="definition">n. [C] 深棕色（或稱栗色）的馬：</span>He was riding a big bay. 他騎著一匹高大的栗色馬。
           
<span class="vocabulary">**chestnut**</span> [ˈtʃesnʌt]
<span class="definition">n. [C] 栗色馬；紅棕色的馬：</span>瞭解即可
 
<span class="vocabulary">**grey**</span> [ɡreɪ]（尤英）（美尤用gray）
<span class="definition">n. [C] 灰色的馬或白色的馬：</span>She’s riding the grey. 她騎著一匹灰馬。
           
<span class="vocabulary">**donkey**</span> [ˈdɒŋki; 美 ˈdɑ:ŋ-; ˈdɔ:ŋ-]
<span class="definition">n. [C] 驢：</span>to ride a donkey 騎驢 / A donkey is braying. 驢在叫。/ donkey cart 驢車 / donkey ride（尤英）騎驢 / donkey work（比喻用法，尤英）枯燥乏味的苦差事 / I've done all the donkey work(= the hard boring part of a job). 我已完成了所有的苦差事。
           
<span class="vocabulary">**mule**</span> [mju:l]
<span class="definition">n. [C] 騾子：</span>pack mule 馱騾 / stubborn mule 倔強的騾子 / I heard a mule braying. 我聽見一隻騾子叫。/ They crossed the mountains by mule. 他們騎騾子越過一座座山 / a man on a mule 騎著騾子的男子

<span class="vocabulary">**pig**</span> [pɪɡ] 
<span class="definition">n. [C] 豬：</span>domestic pig 家豬 / feral, wild pig 野豬 / suckling pig 乳豬 / pig farm, pig farmer 養豬場；豬倌
           
<span class="vocabulary">**draught**</span> [drɑ:ft;美dræft]
<span class="definition">adj. [只用於名詞前] 用於拖拽重物的：</span>a draught horse 役馬 / an Irish draught mare 愛爾蘭役用母馬

周邊詞彙補充：
· stable可指由同一人所擁有並訓練的一群賽馬；
· runner可指參加賽馬的馬匹；
· stray可指走失的寵物（或家畜）；無主的寵物（或家畜）。
