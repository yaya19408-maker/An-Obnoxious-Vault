☀ <span class="category">**毒品 吸毒：**</span>
<span class="vocabulary">**drug**</span> [drʌɡ] 
<span class="definition">n. [C] 毒品。是此義的常規用詞和基本用詞，其詞義範圍常涵蓋了擁有各種效果的毒品（如麻醉劑、興奮劑等）：</span>to take drugs 吸毒 / addictive drug 成癮性藥物 / class A, dangerous, hard drug A級毒品；危險毒品；硬性毒品（即毒性強的毒品）/ soft drugs 軟性毒品 / recreational drug 娛樂性毒品 / drug lord 毒梟 / drug pusher 毒販 / drug cartel 販毒集團 / drug misuse 毒品濫用
                                 
<span class="vocabulary">**opium**</span> [ˈəʊpiəm; 美 ˈoʊ-]
<span class="definition">n. [U] 鴉片：</span>opium poppy 罌粟 / opium den 鴉片煙館

<span class="vocabulary">**dope**</span> [dəʊp; 美 doʊp]
<span class="definition">n. 1 [U] [非正式] 毒品（尤指大麻，在美國尤指海洛因）：</span>A man asked them if they wanted to buy some dope. 一名男子問他們想不想買些毒品。/ He has failed a dope test for cocaine. 他未能透過可卡因藥檢。/ dope smoking 吸食毒品 / dope fiend 吸毒成癮者 / dope pedlar/peddler（尤美）毒販 <span class="definition">2 [C]（人或動物使用的）興奮劑（能提高體育運動的表現）：</span>The athlete failed a dope test (= a medical test showed that he had taken such drugs).（dope test尤為英式用法）這個運動員未能透過藥檢。
           
<span class="vocabulary">**marijuana**</span> [ˌmærəˈwɑ:nə]
<span class="definition">n. [U] 大麻（毒品）：</span>home-grown marijuana 自家種植的大麻 / medical, medicinal marijuana 藥用大麻 / a joint of marijuana 一支大麻煙卷 / marijuana cigarette, marijuana joint 大麻煙 / marijuana plant 大麻植株

<span class="vocabulary">**narcotic**</span> [nɑ:ˈkɒtɪk; 美 nɑ:rˈkɑ:-]
<span class="definition">n. [C] 泛指毒品（如海洛因、可卡因）（亦寫作narcotics，尤用於美國英語中）：</span>mild, powerful narcotic 溫和的毒品；強效毒品 / addictive narcotic 致癮毒品 / narcotics agent, narcotics detective, narcotics officer, narcotics squad 緝毒警察；緝毒偵探；緝毒警官；緝毒小隊 / He was indicted for dealing in narcotics. 他因從事毒品交易被起訴。/ He appears to be high on some sort of narcotic. 他看起來飄飄欲仙，像是吸食了某種毒品。
             
<span class="vocabulary">**heroin**</span> [ˈherəʊɪn; 美 -roʊ-]
<span class="definition">n. [U] 海洛因：</span>pure heroin 純海洛因 / cheap heroin 廉價海洛因 / street heroin 市售海洛因 / a wrap of heroin 一包海洛因 / heroin possession 海洛因私藏 / a fatal heroin overdose 致命的海洛因使用過量 / heroin dealing 海洛因交易 / heroin addict 吸海洛因上癮的人

<span class="vocabulary">**crack**</span> [kræk]
<span class="definition">n. [U]（也可叫crack cocaine）強效可卡因：</span>a crack addict 吸強效可卡因成癮的人
         
<span class="vocabulary">**cocaine**</span> [kəʊˈkeɪn; 美 koʊ-]
<span class="definition">n. [U] 可卡因：</span>crack, freebase, powder（尤美）cocaine 強效純可卡因；精煉可卡因；可卡因粉 / cocaine possession 私藏可卡因 / cocaine injection 可卡因注射 / cocaine baron 可卡因毒梟 / cocaine cartel 可卡因販毒集團        
           
<span class="vocabulary">**coke**</span> [kəʊk] 
<span class="definition">n. [U] [非正式] 可卡因，相當於cocaine：</span>a line of coke 一小份可卡因 / to sniff coke 用鼻吸可卡因

<span class="vocabulary">**line**</span> [laɪn] 
<span class="definition">n. [C] [俚] 被散放成一條細線以備吸食的可卡因：</span>瞭解即可

<span class="vocabulary">**smack**</span> [smæk]
<span class="definition">n. [U] [俚語] 海洛因：</span>smack addicts 吸白粉成癮的人
            
<span class="vocabulary">**gear**</span> [gɪə(r); 美 gɪr]
<span class="definition">n. [U] [俚語] 毒品（尤指海洛因）：</span>Are these people using gear and amphetamines at the same time? 這些人同時服用海洛因和安非他明嗎？
 
<span class="vocabulary">**fix**</span> [fɪks] 
<span class="definition">n. [sing.] [非正式] 致癮物（尤指毒品）的一次用量：</span>my daily fix of coffee 我每天喝咖啡的量 / He was like a heroin junkie needing his next fix. 他就像一個吸食海洛因成癮的人，吸了這次還想著下一次。/ caffeine fix 咖啡因用量

<span class="vocabulary">**hit**</span> [hɪt] 
<span class="definition">n. [C] [俚語] 一次使用毒品的數量，即毒品的一劑：</span>瞭解即可

<span class="vocabulary">**acid**</span> ['æsɪd] 
<span class="definition">n. [U] 毒品。非正式用法，指LSD（全稱Lysergic acid diethylamide，即“麥角酸二乙基醯胺”，簡稱“麥角二乙胺”），是一種強烈的致幻劑，或稱迷幻藥：</span>They smoke joints and drop acid. 他們抽大麻煙卷，吸食迷幻藥。

<span class="vocabulary">**glue**</span> [ɡlu:] 
<span class="definition">n. [U, C] 膠毒，是某些膠水釋放出的可以令神經興奮或產生幻覺的有毒氣體：</span>He took to sniffing glue. 他吸上了膠毒。

<span class="vocabulary">**cannabis**</span> [ˈkænəbɪs]
<span class="definition">n. [U] 大麻，大麻製品：</span>Traces of cannabis were found in the pilot's blood. 在飛行員的血液中發現了少量的大麻成分。/ cannabis consumption, cannabis dependence, cannabis use 大麻消耗；大麻癮；吸食大麻 / cannabis user 大麻吸食者 / cannabis leaf, cannabis plant, cannabis resin 大麻葉；大麻植株；大麻脂

<span class="vocabulary">**pot**</span> [pɒt] 
<span class="definition">n. [U] [非正式] 毒品中的大麻，相當於marijuana：</span>pot smoking 吸食大麻
           
<span class="vocabulary">**skunk**</span> [skʌŋk]
<span class="definition">n. [C] [俚語] 一種強效大麻，相當於skunkweed：</span>瞭解即可
           
<span class="vocabulary">**ecstasy**</span> [ˈekstəsi]
<span class="definition">n. [U] Ecstasy（縮寫為E）搖頭丸、迷幻藥：</span>ecstasy pill, ecstasy tablet（尤英）搖頭丸

<span class="vocabulary">**upper**</span> ['ʌpə] 
<span class="definition">n. [usually pl.] [非正式] 使人感到興奮且充滿能量的一種藥品，即興奮劑：</span>瞭解即可

<span class="vocabulary">**speed**</span> [spi:d] 
<span class="definition">n. [U] [非正式] 一種非法藥品（或稱毒品、禁藥，例如苯丙胺，或稱安非他命），是一種興奮劑，用以增加興奮度和精力：</span>瞭解即可
           
<span class="vocabulary">**nicotine**</span> [ˈnɪkəti:n]
<span class="definition">n. [U] 尼古丁，或稱菸鹼：</span>Nicotine produces a feeling of well-being in the smoker. 尼古丁給吸菸者帶來一種愉悅的感覺。/ Nicotine marks stained his chin and fingers. 尼古丁漬染黃了他的下巴和手指。/ The nicotine stains on his fingers told me he was under stress. 他手指上的煙痕表明他壓力很大。/ nicotine addiction, nicotine craving, nicotine dependence 抽菸上癮；煙癮；吸菸上癮 / nicotine use 抽菸 / nicotine exposure 尼古丁接觸 / nicotine addict 抽菸上癮者 / nicotine withdrawal 戒菸 / nicotine gum 尼古丁戒菸口香糖 / nicotine replacement therapy 尼古丁替代療法

<span class="vocabulary">**abuse**</span> [ə'bju:z] 
<span class="definition">n. [U, sing.] abuse (of sth)（酒精、毒品等的）濫用，相當於misuse：</span>alcohol, drug abuse 酗酒；吸毒 <span class="definition">vt. 誤用或濫用（即用得太多）而損害身體健康：</span>to abuse alcohol, to abuse drugs 酗酒；嗜毒 / He systematically abused his body with heroin and. 他因吸食海洛因一步一步地把身體搞垮了。

<span class="vocabulary">**drug**</span> [drʌɡ] 
<span class="definition">vt. 1 給人或動物用麻醉藥，或給其使用興奮劑以使其在比賽中表現更好：</span>He was drugged and bundled into the back of the car. 他被麻醉後塞入汽車後座。/ It’s illegal to drug horses before a race. 比賽前給馬服用興奮劑是違法的。<span class="definition">2 在某人的食物或飲料中放入麻醉藥，使其失去意識或睡覺：</span>Her drink must have been drugged. 她的飲料中肯定被摻了麻醉藥。
           
<span class="vocabulary">**dope**</span> [dəʊp; 美 doʊp]
<span class="definition">vt. 在體育運動中給人或動物使用興奮劑：</span>瞭解即可 <span class="definition">vt.（常指在食物或飲料中）摻入麻醉劑（使人失去意識）：</span>Thieves doped a guard dog and stole $10 000 worth of goods. 盜賊將看門狗麻醉後偷走了價值1萬元的東西。/ The wine was doped. 這酒摻有麻醉藥。/ Anyone could have got in and doped the wine. 任何人都可能會進來在酒中摻麻醉劑。/ I'd been doped with Somnolin. 我被人下了安眠藥。<span class="definition">vt. [非正式] [常用被動式] ~ sb (up) 使（某人在毒品的作用下）昏昏沉沉，不能頭腦清晰地思考或做事：</span>瞭解即可

<span class="vocabulary">**pop**</span> [pɒp] 
<span class="definition">vt. [非正式]（經常）服用大量毒品：</span>瞭解即可

<span class="vocabulary">**use**</span> [ju:z] 
<span class="definition">vt.＆vi. 吸食（或服用）毒品。是一個簡單用詞和委婉用詞，一般不帶有歧視性：</span>to used drugs 吸食毒品 / She’s been using since she was 13.（俚語）她從13歲起就吸毒。

<span class="vocabulary">**do**</span> [du:] 
<span class="definition">vt. [非正式] 吸毒：</span>He doesn’t do drugs. 他不吸毒。
           
<span class="vocabulary">**snort**</span> [snɔ:t; 美 snɔ:rt]
<span class="definition">vt. 用鼻子吸毒品：</span>He died of cardiac arrest after snorting cocaine. 他死於吸食可卡因之後的心臟停搏。<span class="definition">n. [C] 用鼻子吸入的少量毒品；用鼻子吸毒的行為：</span>to take a snort of cocaine 吸一下可卡因
           
<span class="vocabulary">**puff**</span> [pʌf]
<span class="definition">n. [C] 吸（毒品）：</span>瞭解即可

<span class="vocabulary">**dirty**</span> ['də:tɪ] 
<span class="definition">adj. [美，俚] 使用毒品的：</span>瞭解即可
 
<span class="vocabulary">**user**</span> ['ju:zə] 
<span class="definition">n. [C] [俚] 一個吸毒的人，如癮君子：</span>a casual user of cocaine 一個偶爾吸食可卡因的人 / drug, substance, heroin, tobacco user 吸毒者；海洛因吸食者；吸菸者

周邊詞彙補充：
· joint可指含有大麻的一種香菸；
· cut可表示把毒品（如海洛因等）與另一種物質混合。
