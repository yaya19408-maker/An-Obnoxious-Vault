☀ <span class="category">**車票 機票：**</span>
<span class="vocabulary">**ticket**</span> ['tɪkɪt] 
<span class="definition">n. [C] ticket (for/to sth) 交通工具的票（如車票）。是此義的常規用詞：</span>one-way（尤美）, single（英）ticket 單程票 / return（英）, round-trip（美）ticket 往返票 / season ticket 季票 / business-class ticket 商務艙客票 / plane, bus, train ticket 飛機票；公交車票；火車票 / Tickets for the train on this Friday are available. 本週五的火車票還有票。<span class="definition">vt. [術語] 為行程等生產並銷售票，即售票；還可指送票：</span>Passengers can now be ticketed electronically. 旅客現在可以電子購票。

<span class="vocabulary">**check**</span> [tʃek] 
<span class="definition">n. [C] 也可用check mark，用於美式英語，也可指車票：</span>瞭解即可

<span class="vocabulary">**single**</span> ['sɪŋɡl] 
<span class="definition">adj. [英；英美還均可用one-way] 只作前置定語，（票等）單程的：</span>the single fare 單程票價 <span class="definition">n. [C] [英] 單程票：</span>How much is a single to London? 去倫敦的單程票多少錢？

<span class="vocabulary">**return**</span> [rɪ'tə:n] 
<span class="definition">n. [C] 往返票：</span>‘Brighton, please.’ ‘Single or return?’ “買一張到布賴頓的票。”“單程的還是往返的？”/ the return fare to London 到倫敦的往返票價 / day, period return 當日往返票；限期往返票 / economy return 經濟艙往返票

· round trip可指（火車票、汽車票、飛機票等）往返的，來回的。
