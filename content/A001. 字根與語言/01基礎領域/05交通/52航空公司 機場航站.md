<span class="vocabulary">**aviation**</span> [ˌeɪviˈeɪʃn]
<span class="definition">n. [U] 航空器的設計、建造和飛行，即航空：</span>civil, military aviation 民用航空；軍用航空 / aviation business, aviation industry 航空業；航空工業 / aviation fuel 航空燃料

<span class="vocabulary">**airline**</span> ['eəlaɪn] 
<span class="definition">n. [C] 航空公司，即用航班定期運送旅客或貨物的公司：</span>international airline 國際航空公司 / commercial airline 商業航空公司 / He operates a private airline. 他經營一傢俬人航空公司。/ airline company, airline industry 航空公司；航空業 / airline passenger 飛機乘客 / airline reservation, airline ticket 飛機票預訂；飛機票

<span class="vocabulary">**airport**</span> ['eəpɔ:t] 
<span class="definition">n. [C] 飛機場：</span>international airport 國際機場 / hub airport 樞紐機場 / commercial, military airport 商用機場；軍用機場 / departure, destination airport 始發機場；到達機場 / airport terminal 機場航站樓 / airport lounge 機場候機室 / airport hotel 機場旅店 / airport security 機場安全保衛 / We fly from Heathrow Airport, 我們從希思羅機場起飛。

<span class="vocabulary">**terminal**</span> ['tə:mɪnl] 
<span class="definition">n. [C]（機場的）航空站，航站樓（為一棟或一組建築，用於接收或傳送旅客）：</span>airport terminal 機場航站樓

<span class="vocabulary">**lounge**</span> [laʊndӡ] 
<span class="definition">n. [C]（機場等地的）候機室，候車室：</span>airport lounge 機場候機室 / departure lounge 出站候機室 / VIP lounge 貴賓候機室

<span class="vocabulary">**gate**</span> [ɡeɪt] 
<span class="definition">n. [C] 乘客離開機場而登上飛機時所經過的一條通道，即登機門；登機口：</span>to board at gate 16 在16號登機口登機 / departure gate 登機口

<span class="vocabulary">**apron**</span> ['eiprən] 
<span class="definition">n. [C] [術語] 機場的停機坪，即飛機轉彎或降落等的地方：</span>瞭解即可
           
<span class="vocabulary">**pad**</span> [pæd]
<span class="definition">n. [C] 太空梭或直升機起飛或降落用的一個扁平的平臺，如發射臺、（直升機的）停機坪：</span>a little round helicopter pad 小型圓形直升機停機坪 / a landing pad on the back of the ship 船尾的停機坪
