<span class="vocabulary">**engineer**</span> [͵endӡɪ'nɪə] 
<span class="definition">n. 1 [C] 工程師：</span>senior engineer 高階工程師 / chief engineer 總工程師 / aerospace, aircraft, flight engineer 航空航天工程師；飛機工程師；（航班上的）隨機工程師 / automotive engineer（尤美）汽車工程師 / electrical, electronics engineer 電氣工程師；電子工程師 / software engineer 軟體工程師 / sound engineer 音響工程師 / mining engineer 採礦工程師 / design engineer 設計工程師 / project engineer 專案工程師 / agricultural, chemical, marine, mechanical engineer 農業工程師；化學工程師；航海工程師；機械工程師 <span class="definition">2 [C] 受過專業訓練的機械和電子裝置維修師、維修技工：</span>maintenance（尤英）, service, support engineer 維修技術人員；保養工程師；技術支援工程師 / heating, lighting, telephone（均英）engineer 供熱機修工；照明技師；電話機維修工
           
<span class="vocabulary">**electrician**</span> [ɪˌlekˈtrɪʃn]
<span class="definition">n. [C] 連線或修理電器裝置的一個人，如電工、電器技師：</span>licensed（美）, qualified, trained electrician 有資格認證的電工；合格的電工；受過培訓的電工 / apprentice electrician（尤英）電工學徒 / self-employed electrician 個體電工 / We need to call in an electrician to sort out the wiring. 我們需要叫電工來安排線路。/ You'll need a qualified electrician to rewire your house.你需要一位合格的電工來重新給你的房子佈線。

相關詞義延伸：
· engineer還可尤指在船上或飛機上負責維護並修理發動機的人，即輪機師、機械師。
