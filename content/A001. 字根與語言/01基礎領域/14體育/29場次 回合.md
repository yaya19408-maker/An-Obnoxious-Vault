☀ <span class="category">**場次 回合：**</span>
<span class="vocabulary">**round**</span> [raʊnd] 
<span class="definition">n. 1 [C] 體育比賽的階段或輪次，也可叫局或場。是此義的常規用詞：</span>opening, preliminary round 首輪；預賽 / qualifying round 預選賽 / final round 決賽 / He was knocked out in the third round. 他在第三輪被淘汰出局。<span class="definition">2 [C]（拳擊或摔跤比賽的）回合：</span>The fight only lasted five rounds. 比賽只持續了五個回合。<span class="definition">3 [C]（高爾夫球、馬術場地障礙賽等的）一輪比賽，一局：</span>We played a round of golf. 我們打了一場高爾夫球。/ the first horse to jump a clear round乾淨利落地完成整套跳躍表演的第一匹馬

<span class="vocabulary">**game**</span> [ɡeɪm] 
<span class="definition">n. [C] 體育比賽（如網球比賽）的一局，一場等：</span>to win four games to love in the second set 第二盤以4-0取勝 / two games all 各贏兩局

<span class="vocabulary">**leg**</span> [leɡ] 
<span class="definition">n. 1 [C] leg (of sth) 一段速度比賽的賽程，相當於section或stage：</span>final, last leg 最後一段賽程 / anchor leg（接力的）最後一棒 <span class="definition">2 [C] [英] 體育比賽中，可指同一回合的兩場比賽中的一場：</span>瞭解即可
           
<span class="vocabulary">**frame**</span> [freɪm]
<span class="definition">n. [C]（斯諾克、保齡球等運動中的）一輪、一回、一局：</span>瞭解即可

<span class="vocabulary">**set**</span> [set] 
<span class="definition">n. [C] 在體育運動中，可指網球或排球比賽等的局，盤：</span>in the first set 在首盤比賽中 / set point 盤點 / Nadal won in straight sets (= his opponent won none). 納達爾一盤未失，贏得比賽。

<span class="vocabulary">**half**</span> [hɑːf] 
<span class="definition">n. [C]（體育比賽的）半場或半局：</span>No goals were scored in the first half. 上半場沒有得分。

<span class="vocabulary">**quarter**</span> ['kwɔ:tə] 
<span class="definition">n. [C] 美式足球比賽四節中的一節：</span>瞭解即可
