☀ <span class="category">**房間 屋子：**</span>
<span class="vocabulary">**room**</span> [ru:m] 
<span class="definition">n. 1 [C] 建築物的（有某一用途的）房間，室（由牆壁、地板、頂棚組成）：</span>private room 私人房間 / back room 裡屋 / dining, living, reception room 餐廳；起居室；接待室 / laundry room 洗衣房 / dressing, make-up room 梳妝室；化妝間 / news, press room 新聞編輯室 / conference, meeting room 會議室 / waiting room 等候室 / reading room 閱覽室 / room key 房間鑰匙 / room number 房間號 / room service 客房服務 <span class="definition">2 [C] 構成形容詞，指有某個數量的房間的：</span>a three-roomed/three-room apartment 三室的套房 <span class="definition">3 [pl] [舊時，英] rooms可指某人所租住的一套兩室或多室的房間：</span>They lived in rooms in Kensington. 他們住在肯辛頓的出租房間。
                      
<span class="vocabulary">**chamber**</span> [ˈtʃeɪmbə(r)]
<span class="definition">n. [C]（構成複合詞）用於特定用途的一個房間：</span>For many, the dentist's surgery remains a torture chamber. 對許多人來說，牙醫的治療室一直是間受刑室。/ Divers transfer from the water to a decompression chamber. 潛水員從水裡轉入減壓艙。/ dark, darkened chamber 暗室 / private, secret chamber 密室 / sub-terranean，underground chamber 地下洞穴；地下室 / burial, tomb chamber 墓室 / a Bronze Age burial chamber 青銅器時代的墓室 / gas, torture chamber 毒氣室；刑訊室 / Millions died in the gas chambers in the war. 戰爭期間數百萬人死於毒氣室。/ echo, vacuum chamber 回聲室；真空室 / chamber door 房門
            
<span class="vocabulary">**cubicle**</span> [ˈkju:bɪkl]
<span class="definition">n. [C] 大房間分隔出的（用作淋浴間或換衣間等的）小房間或小隔間：</span>little, small, tiny cubicle 小隔間；微型隔間 / empty cubicle 空的小隔間 / I don't even know my colleague in the next cubicle. 我連隔壁小隔間的同事都不認識。/ changing（英）, shower, toilet cubicle 更衣室；淋浴小單間；衛生間小隔間 / a separate shower cubicle 獨立的淋浴小單間 / office cubicle（尤美）辦公室小隔間 / cubicle curtain, cubicle door 隔間的簾子；隔間的門 / cubicle wall 隔間的牆 / He made his way to the nearest toilet and locked himself in a cubicle. 他走到最近的衛生間，進了一個小隔間後把門反鎖上。

<span class="vocabulary">**stall**</span> [stɔ:l]
<span class="definition">n. [C] [尤美] 房間內的由玻璃或牆壁隔開的小隔間，如淋浴室、洗手間：</span>bathroom, shower, toilet stall（均尤美）浴室；淋浴室；洗手間 / stall door 隔間的門

<span class="vocabulary">**booth**</span> [bu:θ] 
<span class="definition">n. [C] 用屏風或薄牆與較大的公共區域隔開的一個小隔間，例如電話亭、投票間等：</span>ticket booth 售票廳 / phone booth 電話亭

<span class="vocabulary">**box**</span> [bɒks] 
<span class="definition">n. [C] 用於某一目的的一個小遮蔽處，如小亭，崗亭：</span>sentry, signal（英）box 崗亭；鐵路訊號所 / phone box（英）電話間
           
<span class="vocabulary">**boardroom**</span> [ˈbɔ:dru:m; -rʊm; 美 ˈbɔ:rd-]
<span class="definition">n. [C] 董事會會議室：</span>company, corporate, executive boardroom 公司董事會會議室；行政會議室 / boardroom battle, boardroom coup, boardroom shake-up（尤英）, boardroom showdown（美）董事會內部的鬥爭；董事會內部的人事突變；董事會內部的大調整；董事會會議上最後的較量 / boardroom row 董事會會議上的爭吵 / boardroom table 董事會會議桌 / boardroom pay（英）董事的薪水 / The massive boardroom pay awards were criticized by the workers. 董事們薪水過高，受到了工人的批評。/ The directors were working overtime in the boardroom. 董事們正在會議室裡加班。/ Everyone had already assembled in the boardroom for the 9:00 a.m. session. 所有人都已集中到了董事會會議室，準備參加上午9點鐘的會議。

<span class="vocabulary">**lounge**</span> [laʊndӡ] 
<span class="definition">n. [C] [英]（酒店、旅館等場所內部的）酒吧（該類酒吧通常較舒適，所售的飲料價格也比一般的酒吧貴），相當於lounge bar：</span>瞭解即可

<span class="vocabulary">**basement**</span> ['beɪsmənt] 
<span class="definition">n. [C] 建築物的地下室（部分或全部位於地面以下）：</span>flooded basement 被水淹的地下室 / basement apartment（尤美）, basement flat（英）, basement bar, basement studio 地下室公寓；地下室酒吧；地下室工作室 / basement door 地下室門 / basement car park（英）地下停車場 / The canteen is down in the basement. 食堂在地下室。
                      
<span class="vocabulary">**cellar**</span> [ˈselə(r)]
<span class="definition">n. 1 [C]（通常用於存放物品的）地下室、地窖：</span>damp, dark cellar 潮溼的地下室；黑暗的地下室 / vaulted cellar 拱狀地窖 / coal, root（美）煤窖；儲藏根用蔬菜的地窖 / storm cellar（美）防風地下室 / pub cellar（英）地下室酒館 / beer cellar（尤英）地下室啤酒館 / disused cellar 廢棄的地下室 / cellar door, cellar stairs（尤美）, cellar steps 地下室門；地下室樓梯；地下室臺階 / cellar bar（尤英）地下室酒吧 / The box of papers had been stored in a cellar at the family home. 那箱檔案一直藏在家中的地窖裡。<span class="definition">2 [C]（私人或飯店的）酒窖。相當於wine cellar：</span>the restaurant's extensive wine cellar 飯店藏酒品種繁多的酒窖 / I went down into the cellar for more wine.我到地下室去再取了些酒。

<span class="vocabulary">**cavern**</span> [ˈkævən; 美 -vərn]
<span class="definition">n. [C] 黑洞洞的大房間：</span>The kitchen now is a dark cavern, with an antiquated black stove in a corner. 廚房現在黑洞洞的，角落裡有隻破舊的黑色爐子。
           
<span class="vocabulary">**attic**</span> [ˈætɪk]
<span class="definition">n. [C]（緊靠屋頂的）閣樓，頂樓（有屋頂，常用來儲物）：</span>furniture stored in the attic 存放在閣樓的傢俱 / cluttered, cramped, little, small, tiny attic 凌亂不堪的閣樓；狹小的閣樓；丁點兒大的閣樓 / converted attic（尤英）改建後的閣樓 / Her photography studio was a converted attic. 她的攝影工作室是閣樓改建的。/ dark, dusty, musty（尤美）attic 黑暗的閣樓；佈滿灰塵的閣樓；有黴味的閣樓 / attic apartment（美）, attic bedroom, attic flat（英）, attic room 頂樓公寓；頂樓臥室；閣樓間 / attic stairs, attic steps 閣樓的樓梯 / attic window 閣樓的窗戶

相關詞義延伸：
· room還可指旅館等的客房；
· booth還可指市場上的小帳篷或臨時搭建起來的結構，可用於賣貨，獲取資訊或觀看…；
· lounge還可指（機場等地的）候車室、（旅館、俱樂部等用於等候或放鬆的）休息室、（私人住宅中用於坐或放鬆的）起居室。

周邊詞彙補充：
· cave可指山上或地下的大型洞穴；
· garage可指車庫；
· warehouse可指儲存大量商品的倉庫，貨棧，尤指在將貨物運送至商店進行銷售前進行儲存的倉庫；
· magazine可指儲存武器彈藥的一個房間或一棟建築物，即彈藥庫；軍火庫；軍械庫。
