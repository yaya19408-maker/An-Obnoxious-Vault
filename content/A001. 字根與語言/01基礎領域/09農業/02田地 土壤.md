☀ <span class="category">**田地 土壤：**</span>
<span class="vocabulary">**field**</span> [fi:ld] 
<span class="definition">n. [C] 鄉村用於種莊稼或飼養動物的土地，即田地或牧場，有的有圍欄：</span>grassy, green field 長滿草的田野；綠色田野 / muddy, snowy field 泥濘的田野；被雪覆蓋的田野 / fertile field 肥沃的田地 / arable field 可耕地 / fallow field 休耕地 / rice field 稻田 / a field of wheat 麥地 / People were working in the fields. 人們當時正在地裡幹活兒。
           
<span class="vocabulary">**farmland**</span> [ˈfɑ:mlænd; 美 ˈfɑ:rm-]
<span class="definition">n. [U, pl.] 農田、耕地：</span>250 acres of farmland 250英畝耕地 / the prosperous farmlands of Picardy 皮卡第的富饒農田
           
<span class="vocabulary">**terrace**</span> [ˈterəs]
<span class="definition">n. [C]（山間的）梯田：</span>The villagers had dug terraces in the hillside. 村民們在山間開墾出了梯田。/ massive terraces of maize and millet carved into the mountainside like giant steps 山坡上開墾出的像巨型臺階一樣的大塊大塊梯田，上面種著玉米和小米           

<span class="vocabulary">**allotment**</span> [əˈlɒtmənt; 美 əˈlɑ:t-]
<span class="definition">n. [C] [英]（城鎮居民可以租來種菜的）小塊土地：</span>瞭解即可

<span class="vocabulary">**garden**</span> ['ɡɑːdn] 
<span class="definition">n. 1 [C] [英；美用yard] 房子周圍種了花草樹木、蔬菜水果的一片地：</span>flower, herb, kitchen, rose, vegetable garden 花園；藥草園；自用菜園；玫瑰園；菜園 / rock garden 假山庭園 / cottage garden 鄉居花園 / private, public garden 私人花園；公共花園 / botanic, botanical garden 植物園 / beer garden 啤酒園 / garden flower, garden plant 園藝花卉；園藝植物 / garden tools 園藝工具 / garden path 庭園小徑 <span class="definition">2 [C] [美] 院子裡面種花、植物、蔬菜的一塊區域：</span>瞭解即可

<span class="vocabulary">**bed**</span> [bed] 
<span class="definition">n. [C] 園子或院子中用於種植花、蔬菜等的一塊土地，如花壇，苗圃，菜園等：</span>flower, rose, strawberry, etc. bed 花壇、玫瑰花壇、草莓苗床等 / garden, planting（均美）bed 花壇；苗圃
           
<span class="vocabulary">**frame**</span> [freɪm]
<span class="definition">n. [C] 花園中的冷床，陽畦（保護育種或幼苗抗寒的床框）。相當於cold frame：</span>瞭解即可

<span class="vocabulary">**yard**</span> [jɑːd] 
<span class="definition">n. [C] [美]（住宅旁或周圍的）園圃，花園，果園，菜園等（常帶有一片草坪），相當於英式英語的garden：</span>back yard（通常用backyard）, front yard, side yard 後花園；前花園；側花園 / to water the yard 給園子澆水

<span class="vocabulary">**border**</span> ['bɔ:də] 
<span class="definition">n. [C]（草坪邊緣的）狹長花壇：</span>a lawn flanked by wide herbaceous borders 兩側栽種大片草本花的草坪 / border plants 花帶裡的植物

<span class="vocabulary">**nursery**</span> ['nə:sərɪ] 
<span class="definition">n. [C] 苗圃，即種植樹木或其他植物以供售賣或移栽的一片地方：</span>瞭解即可

<span class="vocabulary">**bank**</span> [bæŋk] 
<span class="definition">n. [C] 像“堤”一樣的土地形態，如壟、埂：</span>low banks of earth between the rice fields 稻田之間的低矮田埂 / The children ran down the grassy bank. 孩子們沿著長滿草的斜坡跑下。
           
<span class="vocabulary">**furrow**</span> [ˈfʌrəʊ; 美 ˈfɜ:roʊ]
<span class="definition">n. [C] 地上的犁溝：</span>瞭解即可
           
<span class="vocabulary">**cultivated**</span> [ˈkʌltɪveɪtɪd]
<span class="definition">adj.（土地）用於耕種的：</span>cultivated fields 耕地
