☀ <span class="category">**莊稼 穀物：**</span>
<span class="vocabulary">**crop**</span> [krɒp] 
<span class="definition">n. [C] 莊稼、作物：</span>agricultural crop 農作物 / cash crop 經濟作物 / forage, food crop 飼料作物；糧食作物 / staple crop 主要農作物 / growing crop 生長中的莊稼 / fruit, root, seed, vegetable crop 水果類作物；根用作物；種子作物；蔬菜類作物 / genetically engineered crop 轉基因作物 / crop rotation 農作物輪作 / crop failure 莊稼絕收 / crop production, crop yield 農作物產量 / crop plant 作物植株

<span class="vocabulary">**grain**</span> [ɡreɪn] 
<span class="definition">n. [U, C] 農作物的小而硬的種子顆粒，如小麥、稻米等，即穀物、穀粒；一粒種子顆粒：</span>a few grains of wheat 幾粒小麥 / rice grain 米粒 / pollen grain 花粉粒 / grain yield 糧食產量 / grain crop 糧食作物 / grain exports 穀物出口
           
<span class="vocabulary">**cereal**</span> [ˈsɪəriəl; 美 ˈsɪr-]
<span class="definition">n. 1 [C] 穀類植物：</span>the rich cereal-growing districts of the Paris Basin 巴黎盆地盛產穀物的地區 / cereal crop, cereal foods, cereal grain, cereal products 穀類作物；穀類食物；穀類糧食；穀類產品 / cereal production 穀物生產 <span class="definition">2 [U] 用穀類植物所生產出來的顆粒，即穀物（顆粒）：</span>瞭解即可 <span class="definition">3 [C, U]（常加牛奶作早餐用的）穀類食物、麥片：</span>breakfast cereals 穀類早餐食物 / bran, high-fibre/high-fiber, rice, wholegrain cereal 含麩皮穀類食物；高纖維穀類食物；大米粉；整粒穀物食物 / fortified breakfast cereals 強化穀類早餐 / a wholewheat cereal fortified with vitamins 維生素強化全麥食品 / sugary cereal 甜味麥片粥 / I have cereal for breakfast. 我早飯吃麥片粥。/ I have a bowl of cereal every morning. 我每天早上喝一碗麥片。/ cereal bowl 麥片粥碗

<span class="vocabulary">**bean**</span> [bi:n] 
<span class="definition">n. 1 [C] 豆子的總稱，豆科植物，常是一種藤蔓植物；也可指豆莢：</span>kidney bean 芸豆 / red bean 紅豆 / bean curd 豆腐 / bean sprout 豆芽 <span class="definition">2 [C] 常構成複合詞，指咖啡樹或可可樹等植物的豆形種子，可用於生產咖啡、可可粉或巧克力：</span>cocoa, coffee, vanilla bean 可可豆；咖啡豆；香草豆

<span class="vocabulary">**pulse**</span> [pʌls] 
<span class="definition">n. [pl.] 一些可被食用的種子或果實，如豌豆或其他豆子等。其詞義已為bean所涵蓋，且不常用：</span>瞭解即可

<span class="vocabulary">**corn**</span> [kɔ:n] 
<span class="definition">n. 1 [U] [英] 任何一種穀類作物，例如小麥；還可指這種穀類作物的籽粒，即穀物顆粒：</span>a field of corn 一片穀類作物 / The corn is still green. 穀物還未成熟。/ corn mill 麵粉機 <span class="definition">2 [U] [美] 相當於maize，即通常意義上的玉米，也可指玉米的顆粒：</span>transgenic corn 轉基因玉米 / seed corn 玉米種子 / yellow corn 黃玉米 / popped, roasted corn 爆玉米花；烤玉米 / an ear of corn 一穗玉米 / corn field 玉米地 / corn yield 玉米產量 / corn seed 玉米種子 / corn flour（通常用cornflour）玉米粉 / corn on the cob（煮食的）玉米棒 <span class="definition">3 [U] [美] 相當於sweetcorn，指可以作為蔬菜來食用的一種甜玉米：</span>瞭解即可

<span class="vocabulary">**potato**</span> [pə'teɪtəʊ] 
<span class="definition">n. [C, U] 土豆，馬鈴薯：</span>sweet potato 紅薯 / baked potato 烤土豆 / potato skins 土豆皮

<span class="vocabulary">**rice**</span> [raɪs] 
<span class="definition">n. [U] 大米（指大米籽粒）；水稻（指出產大米的一種作物）：</span>a grain of rice 一粒大米 / long-grain rice 長粒大米 / brown, white rice 糙米；白米 / sticky rice 糯米 / rice production 稻米產量 / rice farmer 稻農 / rice field, rice paddy 稻田

<span class="vocabulary">**wheat**</span> [wi:t, hw-] 
<span class="definition">n. [U] 小麥作物；小麥籽粒：</span>wheat flour 小麥制的麵粉 / spring, winter wheat 春小麥；冬小麥 / an ear of wheat 一穗麥子 / wheat harvest 小麥收成 / wheat field 麥田 / wheat bread（尤美）小麥麵包 / wheat production 小麥生產
           
<span class="vocabulary">**barley**</span> [ˈbɑ:li; 美 ˈbɑ:rli]
<span class="definition">n. [U] 大麥；大麥粒：</span>fields of ripening wheat and barley 一片片正在成熟的小麥和大麥 / malted, malting barley 大麥芽 / roasted barley 烘烤過的大麥 / pearl barley 珍珠大麥 / wild barley 野生大麥 / Feed barley prices are set to rise. 飼料用大麥的價格要上漲。/ spring, winter barley 春大麥；冬大麥 / an ear of barley 一穗大麥 / barley harvest大麥收割 / barley field大麥田 / barley grain大麥粒 / barley soup（尤美）大麥湯
           
<span class="vocabulary">**oats**</span> [əʊts; 美 oʊts]
<span class="definition">n. [pl.] 燕麥：</span>whole oats 全燕麥 / rolled oats（蒸軟後碾平的）燕麥片 / porridge oats（尤英）（做粥用的）燕麥片 / field of oats 燕麥田 / Oats provide good, nutritious food for horses. 燕麥是營養豐富的優質馬飼料。/ oat bran 燕麥麩
