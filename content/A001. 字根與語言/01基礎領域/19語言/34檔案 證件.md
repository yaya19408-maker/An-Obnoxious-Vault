☀ <span class="category">**檔案 證件：**</span>
<span class="vocabulary">**paperwork**</span> ['peɪpəwə:k] 
<span class="definition">n. [U] 為某事所需的全部檔案，例如訴訟案件或購置房產等所需的全部檔案：</span>Have you done all the necessary paperwork for the sale? 你已經辦妥出售所需的全部檔案了嗎？

<span class="vocabulary">**paper**</span> ['peɪpə] 
<span class="definition">n. 1 [pl.] papers 可指帶有文字的紙張，即檔案，資料（如信件、工作檔案、私人檔案等，人們可能將其儲存在家中安全的地方）：</span>The papers are in the drawer. 那些檔案在抽屜裡。/ official papers 正式檔案 / personal, private papers 私人檔案 / court, legal papers 法院檔案；法律檔案 / adoption papers 收養檔案 <span class="definition">2 [pl.] papers 可指一種官方檔案，該檔案能夠證明身份、給予許可等，即證明，證件（表許可類的證件時，相當於permission或permit，不過papers此時是一種簡單的生活化用詞）：</span>identity papers 身份證件 / divorce papers 離婚證件 / ballot papers 選票 <span class="definition">adj. 指協議、資質、利益等以正式檔案的形式存在的，即書面的（雖然可能並非真正有效或有用）：</span>paper promises, paper losses 書面承諾；賬面虧損

<span class="vocabulary">**packet**</span> ['pækɪt] 
<span class="definition">n. [C] [美]（為某一用途）作為一個整體來提供的一套檔案：</span>a training packet 一套培訓材料

<span class="vocabulary">**document**</span> ['dɒkjumənt] 
<span class="definition">n. [C] 官方或正式的檔案（可提供…的資訊，或可用作證據或證明），即檔案；公文；文獻；證件等。是一個基礎用詞，含義極廣：</span>court, government, legal, official document 法庭檔案；政府檔案；法律檔案；官方檔案 / policy document 政策檔案 / travel document 旅行證件 / contract document 合同檔案 / guidance document 指導性檔案 / historical document 歷史文獻 / internal document 內部檔案 / original document 原始檔案 / attached document 作為附件的檔案 / confidential, sensitive document 機密檔案；敏感檔案
           
<span class="vocabulary">**documentary**</span> [ˌdɒkjuˈmentri; 美 ˌdɑ:k-]
<span class="definition">adj. [只用於名詞前] 檔案的、文獻的、由檔案（或文獻）組成的：</span>documentary sources, documentary material 檔案來源；文獻資料 / We have documentary evidence that they were planning military action. 我們有書面證據證明他們正在策劃軍事行動。
           
<span class="vocabulary">**report**</span> [rɪ'pɔ:t] 
<span class="definition">n. [C] report (on sth)（由研究了某一情況或問題的一群人所寫出的一份）正式的檔案，如調查報告：</span>commission, committee report 委員會報告 / company report 公司報告 / credit, economic, financial, market report 信用報告；經濟報告；財務報告；市場報告 / environmental report 環境報告 / scientific, technical report 科學報告；技術報告 / laboratory report 實驗室報告 / survey report 調查報告

<span class="vocabulary">**brief**</span> [bri:f] 
<span class="definition">n. [C] [英] 某人所得到的一份闡釋其工作或職責的指示或說明，如任務簡介：</span>It wasn’t part of his brief to speak to the press. 交付他的任務不包括向新聞界發言。/ I was given the brief of reorganizing the department. 我被分派去改組這個部門。/ news brief（美）簡要新聞 / planning brief（英）規劃綱要 / intelligence, mission（美）brief 情報簡報；任務指令 / presidential brief（美）總統指示 / to stick to brief（英）僅做分內的事

<span class="vocabulary">**note**</span> [nəʊt] 
<span class="definition">n. [C] 證明。是一個簡單用詞：</span>a sick note from your doctor 醫生開據的病假證明

<span class="vocabulary">**certificate**</span> [sə'tɪfɪkət, sə'tɪfɪkeɪt] 
<span class="definition">n. [C] 一種正式檔案，可被用於證明該檔案所陳述的事實是真的，即證書：</span>birth, death, marriage certificate 出生證；死亡證明書；結婚證 / medical certificate 醫學證明 / school certificate 學校證書 / gift certificate（美）購物禮券 / digital certificate 數字證書 / a certificate of authenticity 真品證明書
           
<span class="vocabulary">**confirmation**</span> [ˌkɒnfəˈmeɪʃn; 美 ˌkɑ:nfərˈm-]
<span class="definition">n. [U, C] 確認書、證明書：</span>I'm still waiting for confirmation of the test results. 我仍在等待考試結果的通知書。/ Practically every new piece of documentary evidence provides additional confirmation that the charges are true. 實際上，每一份新的證明檔案都進一步證實這些指控屬實。
           
<span class="vocabulary">**credentials**</span> [krəˈdenʃlz]
<span class="definition">n. [pl.] 資格證書；證件：</span>Britain's new ambassador to Lebanon has presented his credentials to the President. 英國駐黎巴嫩的新任大使已將國書遞交給總統。/ media, press（美）credentials 媒體資格證書；記者資格證書 / Nearly 200 reporters applied for press credentials to cover the case. 近200名記者申請報道此案的資格證。

<span class="vocabulary">**consent**</span> [kənˈsent]
<span class="definition">n. [C] 正式的批准檔案，批文：</span>瞭解即可
           
<span class="vocabulary">**conveyance**</span> [kənˈveɪəns]
<span class="definition">n. [C] [法律] 產權轉讓證書：</span>瞭解即可
           
<span class="vocabulary">**attachment**</span> [əˈtætʃmənt]

<span class="definition">n. [C] 附屬於其它檔案的一份檔案，即檔案的附件：</span>Justice Fitzgerald included a 120-page discussion paper as an attachment to the annual report. 在年度報告上，法官菲茨傑拉德附上了一篇長達120頁的討論稿。
           
<span class="vocabulary">**supplement**</span> [ˈsʌplɪmənt]
<span class="definition">n. [C] ~ (to sth) 補充性的檔案：</span>This document is a supplement to the main report. 這是主要報告的補充檔案。

相關詞義延伸：
· document還可指含有文字的計算機文件；
· report還可指（關於學生在學校的表現的）成績報告單；
· brief還可指向律師提供的一份案情說明（以使其能夠在法庭上進行辯護），亦可指在一個案件中，為支援一方所寫的一份事實總結，可稱為辯護狀（其將被提交給法庭）；
· attachment還可指（用電子郵件傳送的）附件。

周邊詞彙補充：
· submission可指（向權威部門提交或呈遞的）檔案；
· badge可指（某人攜帶或佩戴的，用於證明其身份的）證章（如警徽）；
· source可指資訊來源（尤指用於研究）；原始資料；
· authorization可指官方的批准書、授權書；
· title可表示能夠昭示所有權（尤指對土地或財產的所有權）的憑證；
· passport可指護照；
· stamp可指印花（是一種付款證明，粘在一份檔案上以表明某個數額的款項已支付完畢）；
· identification可指證明身份的正式檔案；
· instrument可指正式法律檔案；
· application可指申請書，申請表；
· reference可指推薦信。
