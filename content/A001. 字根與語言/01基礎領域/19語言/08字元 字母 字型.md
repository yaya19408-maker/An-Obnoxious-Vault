☀ <span class="category">**字元 字母 字型：**</span>
<span class="vocabulary">**character**</span> ['kærɪktə] 
<span class="definition">n. [C]（書寫、印刷或計算機上的）字元，字母，符號，標記。是“字元”的基礎用詞：</span>Chinese characters 漢字 / a line 30 characters long 長達30個字元的一行 / number character 數字 / character set, character string 字元組；字串

<span class="vocabulary">**print**</span> [prɪnt] 
<span class="definition">n. [U] 在紙上印出來的字母、文字或數字等，即印刷字型。偶爾也可指在螢幕上顯示的字型（不過此時不如type常用）：</span>clear print 清晰的印刷字型 / bold print 黑體字 / large print 大字型 / fine, small print（法律檔案）小號字印刷的附加條款 / columns of tiny print 一欄欄極小的印刷字 / the tiny print on the screen 螢幕上的小字

<span class="vocabulary">**type**</span> [taɪp] 
<span class="definition">n. [U]（印刷或列印的）字母，文字，字型，字型；也可指印刷文字所用的小塊金屬（如鉛字）：</span>bold type 黑體 / italic type 斜體 / roman type 羅馬字型 / The type was too small for me to read. 字型太小，我看不清。
           
<span class="vocabulary">**bold**</span> [bəʊld; 美 boʊld]
<span class="definition">adj. [術語]（印刷的文字或字元）粗體的；黑體的：</span>Highlight the important words in bold type. 用黑體突出重要詞語。/ bold lettering 黑字型 <span class="definition">n. [U] [術語]（印刷的文字或字元的）粗體；黑體：</span>Headwords are printed in bold. 首詞用黑體印刷。

<span class="vocabulary">**point**</span> [pɒɪnt] 
<span class="definition">n. [U] 印刷物字型大小的單位，即點，磅值：</span>to change the text to 12 point 把文字字型大小變為12點

<span class="vocabulary">**dot**</span> [dɒt] 
<span class="definition">n. [C] 小圓點（尤指列印或印刷出來的點）：</span>There are dots above the letters i and j. 字母i和j上面都有一個點。/ Text are printed at 300 dots per inch. 文字按每英寸300個點的密度列印。/ a system of painting using small dots of colour 一套運用小色點作畫的方法 / The helicopters appeared as two black dots on the horizon. 直升機像兩個小黑點出現在地平線上。

<span class="vocabulary">**point**</span> [pɒɪnt] 
<span class="definition">n. [C] 書寫時所用的點（相當於dot，此時可以叫做“標點”）。尤指數學中的“小數點”，用於將整數和分數分開：</span>to broadcast on ninety-six point four (96.4) FM 以調頻96.4播音 / exclamation point（美）驚歎號 / bullet point（列舉的各項前的）方形符號

<span class="vocabulary">**letter**</span> ['letə] 
<span class="definition">n. [C]“字母”的常規用詞：</span>big, large letter 大的字母；大字母 / block, capital, upper-case letter 大寫字母 / lower-case, small letter 小寫字母 / initial letter 首字母 / silent letter 不發音的字母
                      
<span class="vocabulary">**vowel**</span> [ˈvaʊəl]
<span class="definition">n. [C] 母音字母：</span>瞭解即可

<span class="vocabulary">**consonant**</span> [ˈkɒnsənənt; 美 ˈkɑ:n-]
<span class="definition">n. [C] 子音字母：</span>瞭解即可

<span class="vocabulary">**capital**</span> ['kæpɪtl] 
<span class="definition">n. [C] 大寫字母，也可叫capital letter：</span>Please write in capitals(=in capital letters). 請用大寫字母書寫。<span class="definition">adj. 只作前置定語，（字母表中的字母）大寫的：</span>English is written with a capital ‘E’. English一詞中字母E大寫。

<span class="vocabulary">**small**</span> [smɔ:l] 
<span class="definition">adj. 常作前置定語，指（書寫或列印、印刷出來的字母）小寫的：</span>Should I write ‘god’ with a small ‘g’ or a capital? god裡面的字母g，我該寫成小寫還是大寫？/ She’s a socialist with a small ‘s’. 她信仰社會主義，但不是社會黨人。

<span class="vocabulary">**short**</span> [ʃɔ:t] 
<span class="definition">adj. short for sth（名稱或單詞）簡略的；縮寫的：</span>Call me Jo--it’s short for Joanna. 叫我喬好了——這是喬安娜的簡稱。/ file transfer protocol or FTP for short“file transfer protocol（檔案傳輸協議）”或簡稱“FTP” 

<span class="vocabulary">**I**</span> [aɪ] 
<span class="definition">n. [C, U] 也可用i，英語字母表的第九個字母：</span>‘Island’ begins with (an) ‘I’. Island一詞以字母I開頭。
           
<span class="vocabulary">**delta**</span> [ˈdeltə]
<span class="definition">n. [C] 希臘字母表的第四個字母：</span>瞭解即可

<span class="vocabulary">**initial**</span> [ɪ'nɪʃl] 
<span class="definition">n. 1 [C] 一個人的名字（first name）的第一個字母（即首字母）：</span>‘What initial is it, Mrs Owen?’ ‘It’s J, J for Jane.’ “首字母是什麼，歐文太太？”“是J，Jane的J。” <span class="definition">2 [pl.] initials 一個人的全名的各個部分的首字母之和：</span>John Fitzgerald Kennedy was often known by his initials JFK. 人們常以姓名的首字母JFK稱約翰·菲茨傑拉德·肯尼迪。/ Just write your initials. 寫下你的姓名首字母即可。
           
<span class="vocabulary">**cipher**</span> [ˈsaɪfə(r)]
<span class="definition">n. [C] [英] 用在一個設計圖案中的某人名字的各部分的首字母（以代表某種含義）：</span>瞭解即可

<span class="vocabulary">**alphabet**</span> ['ælfəbɪt] 
<span class="definition">n. [C] 一種語言中用於書寫的一套字母，可叫做字母表（在該表中，字母的順序常是固定的）：</span>Arabic, Latin alphabet 阿拉伯字母表；拉丁字母表 / the letters of the alphabet 字母表中的字母
           
<span class="vocabulary">**script**</span> [skrɪpt]
<span class="definition">n. [U, C] 一種語言的字母系統、字母表。相當於alphabet：</span>phonetic script 標音書寫體系；音標 / Arabic, Cyrillic, Hebrew, Japanese script 阿拉伯文字；西里爾文字；希伯來文字；日本文字 / a document in Cyrillic script 一份用西裡爾字母書寫的檔案 / a text in the Malay language but written in Arabic script 一篇使用阿拉伯字母書寫的馬來語演講稿

周邊詞彙補充：
· string可指計算機技術中的一系列字元（如字母、數字等），即字串；
· dot可指計算機中的點（用以分隔域名、電子郵件地址等的各部分）。
