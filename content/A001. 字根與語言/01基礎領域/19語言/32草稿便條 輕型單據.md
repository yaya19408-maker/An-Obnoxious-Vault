☀ <span class="category">**草稿便條 輕型單據：**</span>
<span class="vocabulary">**draft**</span> [drɑːft] 
<span class="definition">n. [C] 起草出來的粗略的檔案版本，即草稿；草案；草圖：</span>initial, original draft 初稿；原稿 / revised draft 修訂稿 / final draft 定稿 / draft agreement, draft budget 協議草案；預算草案 / law draft 法律草案 / draft plan, draft proposal 計劃草案；提議草案

<span class="vocabulary">**note**</span> [nəʊt] 
<span class="definition">n. 1 [C] 一則簡短的、非正式的書信，即便條：</span>brief note 便條 / scribbled note 草草寫下的留言 / credit, delivery（英）, love, ransom, sick（英）, suicide, thank-you note 欠條；送貨單；情書；贖金通知；病假條；絕命書；致謝便函 / sticky note（尤美）便利貼 / crumpled note 皺巴巴的便條 / to leave a note 留便條 / to pass a note 傳紙條 <span class="definition">2 [C] notes，複數形式可表示某人聽講、讀書、或閱讀其它材料時所寫下的資訊：</span>brief notes 簡短的筆記 / lecture notes 講稿 / field notes 野外記錄 / case notes 病歷 / I’ve made a note of the book’s title. 我記下了書名。/ The booklet has full notes on each artist. 這本小冊子裡有每一位藝術家的詳盡資料。/ She made a mental note to call them in the morning. 她心裡記下要在早晨給他們打電話。

<span class="vocabulary">**slip**</span> [slɪp] 
<span class="definition">n. [C] 一個小片紙，尤指用於寫字或印有內容的小紙條：</span>I wrote it down on a slip of paper. 我把它記在一張紙條上。/ rejection slip（尤英）退稿單 / betting, voting（均英）slip 投注單；選票單 / permission slip（尤美）許可單 / deposit slip 存款單據 / wage slip（英）工資單 / pink slip（美）解僱通知單 / Please detach and return the slip below. 請撕下下面的憑條並交回。

<span class="vocabulary">**excuse**</span> [ɪk'skju:z] 
<span class="definition">n. [C] [美] 家長或醫生寫的為何一個學生不能去上課或某人為何不能去工作的紙條，即請假條：</span>瞭解即可

相關詞義延伸：
· draft還可指給銀行開具的向某人付款的單據，如匯票；
· note還可指協助人們記起某事（即起提醒作用）的一小段文字；
· sticky可指（可以寫資訊的）小貼紙。
