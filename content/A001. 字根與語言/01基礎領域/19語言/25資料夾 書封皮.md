☀ <span class="category">**資料夾 書封皮：**</span>
<span class="vocabulary">**envelope**</span> ['envɪləʊp] 
<span class="definition">n. [C] 用於儲存紙張檔案的扁平塑膠封皮：</span>envelope of certificates 證書的封套

<span class="vocabulary">**wallet**</span> ['wɒlɪt] 
<span class="definition">n. [C] 由皮革、塑膠或硬紙板製成的用於攜帶檔案的扁平式資料夾：</span>a document wallet 資料夾

<span class="vocabulary">**file**</span> [faɪl] 
<span class="definition">n. 1 [C] 裝檔案用的容器（如檔案櫃或摺疊式的資料夾等）。是此義的常規用詞：</span>a file of insurance papers 一個保險單資料夾 / paper file 紙質資料夾 / six box files bulging with notes 六個裝滿筆記鼓鼓囊囊的盒式卷宗 
           
<span class="vocabulary">**folder**</span> [ˈfəʊldə(r); 美 ˈfoʊld-]
<span class="definition">n. [C]（紙板或塑膠材質的）資料夾：</span>瞭解即可
           
<span class="vocabulary">**portfolio**</span> [pɔ:tˈfəʊliəʊ; 美 pɔ:rtˈfoʊlioʊ]
<span class="definition">n. [C] 扁平的公事包（用於攜帶檔案、圖畫等）：</span>瞭解即可
           
<span class="vocabulary">**briefcase**</span> [ˈbri:fkeɪs]
<span class="definition">n. [C] 扁平的公文包（用於攜帶檔案）：</span>battered briefcase（尤英）破舊不堪的公文包 / He wore a torn suit and carried a battered briefcase. 他穿一身破西服，提著一個破舊不堪的公文包。/ bulging briefcase 鼓脹的公文包 / leather, metal, plastic briefcase 皮製公文包；金屬公文包；塑膠公文包           

<span class="vocabulary">**pocketbook**</span> [ˈpɒkɪtbʊk; 美 ˈpɑ:k-]
<span class="definition">n. [C] 一種小的扁平的袋子，常用皮革或塑膠製成，用於攜帶檔案：</span>瞭解即可

<span class="vocabulary">**note**</span> [nəʊt] 
<span class="definition">n. [C] 常用複數，指印在一部特殊的書籍封面、CD或唱片封面等上面的資訊，該資訊介紹了一次演出、一個演員的職業履歷、或一段音樂等：</span>introductory notes 介紹性說明 / biographical notes 生平簡介 / booklet, sleeve（英）notes 說明性小冊子；（唱片）封套內容簡介

<span class="vocabulary">**jacket**</span> ['dӡækɪt] 
<span class="definition">n. [C] 也可用dust jacket，指書籍的鬆散的護封或書套（常帶有圖案或畫面）：</span>瞭解即可

<span class="vocabulary">**cover**</span> ['kʌvə] 
<span class="definition">n. [C] 圖書或雜誌的封皮。是此義的常規用詞：</span>a small book with a green cover 一本綠色封面的小冊子 / He always reads the paper from cover to cover. 他總是把報紙從頭到尾全看一遍。/ back, front cover 封底；封面 / book, magazine cover 圖書封面；雜誌封面 / cover image, cover story, cover character 封面圖片；封面故事；封面人物 
           
<span class="vocabulary">**binding**</span> [ˈbaɪndɪŋ]
<span class="definition">n. [C, U] 書的封皮：</span>Its books are noted for the quality of their paper and bindings. 其圖書以紙質優良、封面精美而聞名。

相關詞義延伸：
· file還可指（儲存在計算機上的）檔案或資料夾；
· folder還可指（計算機系統中的）資料夾。
