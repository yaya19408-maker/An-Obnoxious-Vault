☀ <span class="category">**學習 攻讀：**</span>
<span class="vocabulary">**learn**</span> [lə:n] 
<span class="definition">vt.＆vi. 1 learn (sth) (from sb/sth) 透過學習或訓練來獲得知識或技能，即學習：</span>to learn English 學習英語 / to learn to play the piano 學習彈鋼琴 / I learned a lot from my father. 我從父親那兒學了很多東西。/ Children learn very quickly. 孩子們學得很快。/ Children learn through play. 孩子透過玩耍來學習。<span class="definition">2 learn (from sth) 從…中吸取教訓：</span>to learn from mistakes 從錯誤中吸取教訓 <span class="definition">vt. 1 學習或重複…以使自己記住該內容，即背誦，學會：</span>I learned the poem last night. 我昨晚背誦了這首詩。<span class="definition">2（透過改變態度來）學會以某種方式來行事：</span>You have to learn to face your problem. 你得學會面對自己的問題。
           
<span class="vocabulary">**learning**</span> [ˈlɜ:nɪŋ; 美 ˈlɜ:rnɪŋ]
<span class="definition">n. [U] 學習：</span>a model for effective learning 有效的學習模式 / distance learning（= by correspondence course）遠端學習 / early learning（= of very young children）早期教育 / higher learning 高等教育 / institutions of higher learning 高等院校 / lifelong learning 終身學習 / active, passive learning 主動學習；被動學習 / online learning 線上學習 / computer-assisted learning 計算機輔助學習 / language learning 語言學習 / learning curve 學習曲線 / The whole team has been on a steep learning curve since the project began. 在這個專案開始後整個組都在克難攻堅。/ The school aims to provide a supportive learning environment. 學校致力於提供一種有助於學習的環境。/ learning programme/program 學習計劃 / The dance department's multimedia learning centre/center is equipped with eight laptops. 舞蹈系的多媒體學習中心配備了八臺膝上型電腦。/ learning differences（尤美）, learning difficulty, learning disability學習障礙 / a child with severe learning difficulties 有嚴重學習障礙的孩子 / a seat of learning 學習場所 / Last season was a learning experience for me. 上個季度對我來說是一次學習。/ The brochure described the library as the focal point of learning on the campus. 這本小冊子把圖書館形容為校園學習的中心。

<span class="vocabulary">**study**</span> ['stʌdɪ] 
<span class="definition">n. 1 [U] 對知識的學習：</span>language study 語言學習 / the study of geography 學習地理 / academic study 理論學習 / course of study 學習科目 / full-time study 全日制學習 / study group 學習小組 <span class="definition">2 [pl.] [正式] studies 可指某人的學習活動，例如在大學的學習活動，即學業，課業，功課：</span>to continue my studies 繼續我的學業 / Many undertake further studies after college. 很多人在大學畢業後繼續深造。/ to completed studies 完成學業 <span class="definition">vt.＆vi. study (sth) (at ...) / study (for sth) 學習某一科目或某些科目：</span>to study geography 學習地理 / She is studying hard for her exams. 她正為考試而努力學習。/ He studied under Professor Tom. 他曾在湯姆教授的指導下學習。
            
<span class="vocabulary">**audit**</span> [ˈɔ:dɪt]
<span class="definition">vt. [美] 旁聽大學課程：</span>瞭解即可          

<span class="vocabulary">**imbibe**</span> [ɪmˈbaɪb]
<span class="definition">vt. [正式] 吸收，尤指吸收資訊、思想或觀點：</span>As a clergyman's son he'd imbibed a set of mystical beliefs from the cradle. 作為牧師的兒子,他從尚在襁褓時起就接受了一套神秘主義信仰。

<span class="vocabulary">**at**</span> [æt] 
<span class="definition">prep. 在學習的地點：</span>She’s at Yale (= Yale University). 她在耶魯大學。

<span class="vocabulary">**task**</span> [tɑːsk] 
<span class="definition">n. [C]（尤指在語言教學中）旨在幫助學習者達到某一學習目標的一項活動：</span>task-based learning 任務型學習 / one of the first tasks in language learning 語言學習的首要任務之一

<span class="vocabulary">**do**</span> [du:] 
<span class="definition">vt. 學習某事物。是一個簡單用詞，不太常用：</span>I’m doing physics. 我在學物理。

<span class="vocabulary">**shadow**</span> ['ʃædəʊ] 
<span class="definition">vt. 與正在做某項工作的人在一起進行學習，即實地學習：</span>It is often helpful for teachers to shadow managers in industry. 教師跟隨業界的管理人員實地學習，常常會很有收穫。

<span class="vocabulary">**commit**</span> [kə'mɪt] 
<span class="definition">vt. commit sth to memory 把…學得足夠好，好到可以精準記住：</span>to commit the knowledge to memory 把知識記得很牢

<span class="vocabulary">**take**</span> [teɪk] 
<span class="definition">vt. 學習（課程）。也是一個簡單用詞，不太常用，不過比do常用：</span>to take a computer course 修一門計算機課程 / How many subjects are you taking this year? 你今年修多少門課？

<span class="vocabulary">**acquire**</span> [ə'kwaɪə] 
<span class="definition">vt. [正式]（透過自己的努力、能力或行為表現）學到：</span>She has acquired a good knowledge of English. 她英語已經學得很好了。

<span class="vocabulary">**acquisition**</span> [͵ækwɪ'zɪʃn] 
<span class="definition">n. [U] 獲得某事物（尤指知識、一種技能等）的行為：</span>Language acquisition begins in the first months of a baby’s life. 語言習得在嬰兒出生後的頭幾個月就開始了。

<span class="vocabulary">**read**</span> [ri:d] 
<span class="definition">vt.＆vi. [英，過時] read (for sth)（尤指在大學）學習某一科目，即攻讀：</span>I read for a literature degree at this university. 我曾在這所大學攻讀文學學位。

說明：表“學習”時，①learn和study只有在各自作為動詞的第一義項時，才存在明顯交集，此時二者有時可互換；②learn此時無名詞形式（但有動名詞形式learning）；③learn一般只是對學習活動進行客觀描述，不暗含某些色彩，而study則有時暗含出學習者為了學習而付出了一定的時間和努力，有一點“刻苦”的意味。

周邊詞彙補充：
· probation可指（為表現不理想的學生而設的）試讀期、試讀。
