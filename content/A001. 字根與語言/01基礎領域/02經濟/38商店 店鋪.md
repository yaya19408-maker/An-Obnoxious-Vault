☀ <span class="category">**商店 店鋪：**</span>
<span class="vocabulary">**shop**</span> [ʃɒp] 
<span class="definition">n. [C] 可以購買貨物或服務的商店，店鋪，商鋪，尤用於英式英語：</span>corner（英）, village shop 街頭小店；鄉村商店 / high-street shop（英）商業街上的店鋪 / butcher’s shop 肉鋪 / shoe shop 鞋店 / electrical shop 電器商店 / specialist, specialty（美）shop 專賣店 / duty-free shop 免稅商店 / antique, junk shop 古董商店；舊貨店 / retail shop 零售商店 / one-stop shop 一站式商店 / coffee shop 咖啡店 / a row of shops 一排商店 / shop assistant, shop manager, shop owner 店員；商店經理；店主 / shop shelves 商店貨架

<span class="vocabulary">**store**</span> [stɔ:] 
<span class="definition">n. 1 [C] 大型百貨商店：</span>a big department store 大型百貨商店 <span class="definition">2 [C] 美式英語中可指大大小小的商店，有些用法可用於英式英語：</span>big, large store 大商店 / small store 小商店 / retail store 零售商店 / high-street（英）, village（尤英）store 商業街上的商店；鄉村商店 / chain store 連鎖店 / online store 網上商店 / discount store 折扣店 / convenience, corner（尤英）, general store 便利店；街頭小店；雜貨鋪 / specialty store（美）專賣店 / outlet store 工廠直營店 / toy store 玩具店 / flagship store 旗艦店 / store shelf, （尤美）商店貨架 / store clerk（美）店員 / store owner 店主

<span class="vocabulary">**supermarket**</span> ['sju:pə͵mɑːkɪt] 
<span class="definition">n. [C] 超級市場，超市，是一種大型食品及居家用品商店，美式英語也用grocery store：</span>high-street（英）, neighborhood（美）supermarket 商業街的超市；居民區的超市 / out-of-town supermarket（英）市郊的超市 / supermarket chain, supermarket giant 連鎖超市；大型超市 / supermarket checkout, supermarket shelf 超市付款臺；超市貨架 / supermarket brand 超市品牌 / supermarket manager 超市經理
           
<span class="vocabulary">**mall**</span> [mɔ:l; British English also mæl]
<span class="definition">n. [C] [尤美] 封閉式的大型購物區，即商場。相當於shopping mall：</span>Let's go to the mall. 我們去商場吧。/ shopping mall 購物中心 / strip mall（美）（位於商業街的）小型購物中心 / outlet mall 品牌特價購物中心 / suburban mall 郊區商城 / local, regional mall 當地購物中心；地區購物中心 / indoor, outdoor mall 室內商城；露天商城 / pedestrian mall（美）商業步行街 / The website is a virtual shopping mall. 這個網站是一家虛擬商城。/ They hit the mall for a wild shopping spree. 他們去商城瘋狂購物。/ teenagers hanging out at the mall 一在商城閒逛的青少年 / mall store 購物中心的商店 / mall parking lot（美）商城停車場 / mall food court 商城美食區 / mall rat (= a young person who spends a lot of time in malls with their friends）（美，非正式）愛逛商城的年輕人 / mall security, mall security guard 商城安保部門；商城保安 / They reported him to mall security. 他們向商城安保部門舉報了他。/ They spend a lot of time at the mall. 他們花很多時間逛商場。

<span class="vocabulary">**chain**</span> [tʃeɪn] 
<span class="definition">n. [C] 由同一家公司所擁有的一些商店或旅館等，即連鎖店：</span>fast-food, hotel, supermarket, etc. chain 連鎖快餐店；連鎖酒店；連鎖超市等 / chain store 連鎖商店
           
<span class="vocabulary">**outlet**</span> [ˈaʊtlet]
<span class="definition">n. 1 [C] [商業] ~ (for sth) 專賣某公司的產品（或某類產品）的專營店：</span>The business has 34 retail outlets in this state alone. 那家商號僅在本州就有34個零售店。/ specialist outlet（英）專賣店 / high-street outlet（英）商業街上的經銷店 / commercial, distribution, fast-food, media, news, retail, sales outlet 商貿經銷店；分銷網點；快餐店；傳媒公司；新聞公司；零售商店；銷售點 / a retail outlet for exotic plants 外來植物零售店 / the largest retail outlet in the city 該市最大的零售商店 <span class="definition">2 [C] [尤美] ~ (for sth) 以低價出售某類產品的商店，即折扣品經銷店：</span>the factory outlet store in Belmont 位于貝爾蒙特的廠家直銷型折扣店 / the Nike outlet in the outlet mall 折扣品商場裡的耐克折扣店
           
<span class="vocabulary">**franchise**</span> [ˈfræntʃaɪz]
<span class="definition">n. [C] 獲特許權的商業（或服務）機構：</span>They operate franchises in London and Paris. 他們在倫敦和巴黎經營專賣店。/ a burger franchise 漢堡包特許經銷店 / fast-food franchises 快餐特許經營店 / He runs a local pizza franchise. 他在當地開著一家比薩餅專賣店。/ the franchise to build and operate the tunnel 建造和經營隧道的特許機構           

<span class="vocabulary">**boutique**</span> [bu:ˈti:k]
<span class="definition">n. [C] 出售時裝或昂貴禮品的一種小商店：</span>chic, elegant, exclusive, smart（尤英）, trendy boutique 雅緻的精品店；高檔的精品店；時髦的精品店 / fashion boutique 時裝精品店 / designer boutique 設計師品牌店 / expensive boutique 昂貴的精品店

<span class="vocabulary">**butcher**</span> ['bʊtʃə] 
<span class="definition">n. 1 [C] 商店中的肉販，或屠夫：</span>He was a butcher ten years ago. 他十年前是個屠夫。/ halal butcher 售賣伊斯蘭教合法畜肉的肉商 / pork butcher（尤英）殺豬匠 <span class="definition">2 [C] butcher’s（複數為butchers）指肉店或肉鋪：</span>He owns the butcher’s in the main street. 他在大街上開了一家肉鋪。
           
<span class="vocabulary">**forge**</span> [fɔ:dʒ; 美 fɔ:rdʒ]
<span class="definition">n. [C] 鐵匠鋪：</span>the blacksmith's forge 鐵匠鋪 / Woodbury Blacksmith & Forge Co. 伍德伯裡鍛造公司

<span class="vocabulary">**barbershop**</span> ['bɑːbəʃɒp] 
<span class="definition">n. [C] [尤美] [英常用barber’s] 理髮師工作的一個地方，即理髮店：</span>瞭解即可

<span class="vocabulary">**barber**</span> ['bɑːbə] 
<span class="definition">n. [C] [英]（也可用barber’s[英]）男子理髮店：</span>a barber’s shop in south London 倫敦南部的一家理髮店 / My Mum took me to the barber’s. 我媽帶我理髮去了。
     
<span class="vocabulary">**salon**</span> [ˈsælɒn; 美 səˈlɑ:n]
<span class="definition">n. [C] 美髮店；美容店；（售賣昂貴衣服的）高階服裝店：</span>beauty, hair, hairdressing, nail, tanning salon 美容院；美髮廳；美甲店；美黑沙龍

<span class="vocabulary">**optician**</span> [ɒpˈtɪʃn; 美 ɑ:p-]
<span class="definition">n. [C] optician's（複數為opticians）可指眼鏡商店：</span>to go to the optician's 去配眼鏡 / Some may need specialist treatment at the optician's. 有些人可能需要到眼鏡店進行專業驗光。
 
<span class="vocabulary">**café**</span> ['kæfeɪ, kə'feɪ] 
<span class="definition">n. [C] [南非] 便利商店（出售糖果、報紙、食物等，通常比其他商店晚一些關門）：</span>瞭解即可

<span class="vocabulary">**bar**</span> [bɑː] 
<span class="definition">n. [C] 尤用於構成複合詞，指主要（或專門）出售某類食品或飲料的地方：</span>breakfast, burger, coffee, juice, snack, sushi bar 早餐店；漢堡店；咖啡店；果汁店；小吃店；壽司店

<span class="vocabulary">**greengrocer**</span> ['ɡri:nɡrəʊsə]（尤英）
<span class="definition">n. [C] greengrocer’s（複數形式為greengrocers）指蔬菜水果店：</span>瞭解即可

<span class="vocabulary">**grocer**</span> ['ɡrəʊsə] 
<span class="definition">n. [C] grocer’s（複數為grocers）指食品雜貨店：</span>瞭解即可

<span class="vocabulary">**grocery**</span> ['ɡrəʊsərɪ] 
<span class="definition">n. [C] 賣食品或其它家庭用品的商店，尤用於英式英語。美式英語常用grocery store，且其常用以指supermarket：</span>local grocery 當地的雜貨店 / the corner grocery（尤英）街角處的雜貨店 / grocery checkout 雜貨店的收銀臺 / grocery chain 連鎖雜貨店

<span class="vocabulary">**parade**</span> [pəˈreɪd]
<span class="definition">n. [C]（常指遠離大街的）一小排店鋪：</span>瞭解即可	
           
<span class="vocabulary">**precinct**</span> [ˈpri:sɪŋkt]
<span class="definition">n. [C] [英]（不能停車的）步行商業區：</span>shopping precinct 步行購物區 / town-centre precinct 城慎中心的步行商業區 / The Centre was a pedestrian precinct with a bandstand in the middle. 該中心是步行商業區，正中央有一個室外音樂演奏臺。

<span class="vocabulary">**retailer**</span> [ˈri:teɪlə(r)]
<span class="definition">n. [C] 零售商；零售店（可以是一個人，也可用是一家公司）：</span>the country's biggest food retailer 該國最大的食品零售商 / niche（尤英）, specialist（英）, specialty（美）retailer 針對特定顧客的零售商；專業零售商；專營零售商 / high-end（美）, luxury（尤美）, upscale（美）高階零售店；奢侈品零售店；高階零售店 / discount retailer 折扣零售商 / brick-and-mortar（美）, bricks-and-mortar（美）, traditional retailer 實體零售店；傳統零售店 / Internet, online, Web 網上零售店 / apparel（美）, clothing, electrical（英）, electronics, fashion, food, furniture, grocery, music, toy retailer 服飾零售商；服裝零售商；電器零售店；電子產品零售店；時裝零售店；食品零售店；傢俱零售店；雜貨零售店；音樂零售店；玩具零售商 / Furniture and carpet retailers are among those reporting the sharpest annual decline in sales. 據稱年銷售額下滑幅度最大的零售商中包括傢俱和地毯零售商。

說明：cleaner和laundry表“洗衣店”時，①二者詞義幾無區別，只是laundry更為常用；②dry-cleaner’s（乾洗店）強調洗滌方式為“乾洗”，而laundry不強調是乾洗還是其他洗滌方式。
  
周邊詞彙補充：    
· franchised store可指特許經營店、專賣店；
· cleaner可指乾洗店，是一個簡單用詞；
· laundry可指洗衣店，是一個常規用詞；
· gallery可指（可供觀看或買賣字畫的）私人字畫店；
· chemist可指（通常也出售肥皂、化妝品等的）藥店，藥房。
