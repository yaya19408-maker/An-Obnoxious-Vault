<span class="vocabulary">**money**</span> ['mʌnɪ]

<span class="vocabulary">**currency**</span> ['kʌrənsɪ] 
<span class="definition">n. [C, U] 一國在交易流透過程中發揮交換之媒介作用的物品，即“通貨”，因絕大部分情況下都是錢幣充當此媒介，故也常稱為“貨幣”。currency是本組其它表示“錢幣”之詞的基礎，是一個高度概括性的學術概念，強調錢幣作為一個系統而具有的流通性和媒介屬性：</span>national currency 本國貨幣 / foreign currency 外幣 / Japanese currency 日本貨幣 / weak currency 疲軟貨幣 / hard currency 硬通貨 / paper currency 紙幣 / currency exchange 貨幣兌換 / currency markets 貨幣市場 / currency devaluation 貨幣貶值 / currency system 貨幣體系（或貨幣制度）/ currency reform 貨幣改革

說明：coin和change表“硬幣”時，①二者在作不可數名詞時，都可表示“硬幣”；②coin是此義的常規用詞，是硬幣的統稱；而change的“硬幣”之義是由“零錢”之義擴充套件而來的（“零錢”在特定環境中表現為“硬幣”），此時遠不如coin常用。
	
exchange和convert表“本外幣兌換”時，①exchange是表示此義的最常用詞，相對正式。可用於各種跟貨幣兌換有關的交易行為或機構的名稱中；②convert是一種正式用法，但同時也是一種學術術語。除了此義，convert還可指不同種類的股票之間的兌換或不同單位之間的數量換算。如果說在經濟金融領域的日常各類書面和口頭表述中最常使用“exchange”，那麼“convert”則傾向於一種理論上的學術表達。
	
pence、pennies表“便士的複數”之義時，①在英國、美國和加拿大，兩者均為penny的複數。但pence為便士的款額，是純粹的貨幣額度，如six pence（六便士）。此時可縮寫為“p”，如4p（四便士）。還可用“數字+penny”的組合形式來表示此義，如a six-penny stamp（一枚六便士的郵票）；②而pennies則為硬幣的個數，如six pennies（六枚一便士的硬幣）。
