☀ <span class="category">**進口 出口：**</span>
<span class="vocabulary">**import**</span> [ɪm'pɔ:t] 
<span class="definition">n. [U, C] 可數名詞時為複數，產品或服務的進口：</span>imports and exports 進出口 / the import of hazardous waste 危險廢棄物的進口 / annual import 年進口量 / total import 進口總額 / illegal import 非法進口 / coal import 煤炭進口 / import duty, import tariff 進口關稅 / import quota 進口配額 / a ban on imports, a restriction on imports 進口禁令；進口限制 <span class="definition">vt.＆vi. import sth (from ...) / import sth (into ...) 將產品、服務從一國引入至另一國，即進口：</span>to import raw materials 進口原材料 / goods imported from Japan into the US 從日本輸入到美國的貨品

<span class="vocabulary">**export**</span> ['ekspɔ:t] 
<span class="definition">n. [U, C] 可數名詞時為複數， 商品或服務的出口貿易：</span>illegal export 非法出口 / capital, commodity export 資本輸出；商品出口 / beef export 牛肉出口 / live export 活動物出口 / volume of exports 出口量 / export crop, export goods 出口作物；出口商品 / export controls, export quota, export restrictions, export subsidy, export tax 出口管控；出口配額；出口限制；出口補貼；出口關稅 / export order 出口訂單 <span class="definition">vt.＆vi. export (sth) (to sb) 將商品或服務出口到另一國：</span>The island export fruit. 該島出口水果。/ 90% of the engines are exported to Europe. 90%的發動機都出口到歐洲。/ to be widely exported 被出口到很多地方 / illegally exported works of art 非法出口的藝術作品 / Last year 2 000 birds were exported from the island. 去年有2 000只鳥從這個海島輸出。

<span class="vocabulary">**exporter**</span> [ekˈspɔ:tə(r); 美 ekˈspɔ:rt-]
<span class="definition">n. [C] 從事貨物出口業務的一個人、公司或國家，即出口商；出口公司；出口國：</span>the world's largest exporter of cars 世界上最大的汽車出口國 / the biggest oil exporter in the world 世界上最大的石油輸出國 / China is the top exporter of telecommunications equipment. 中國是通訊裝置的第一大出口國。/ The country is now a net exporter of fuel (= it exports more than it imports). 目前這個國家是燃料淨輸出國。/ France is the world's second-biggest exporter of agricultural products. 法國是世界第二大農產品出口國。/ food, grain, oil exporter 食品出口國；穀物出口國；原油輸出國
