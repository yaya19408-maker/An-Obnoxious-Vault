☀ <span class="category">**政府的 國有的：**</span>
<span class="vocabulary">**national**</span> ['næʃnəl]  
<span class="definition">adj.（常用於名詞前）由政府所有、所控制或所支付的，即國有的、國立的、國營的：</span>national airline, national museum, national theatre 國營航空公司；國立博物館；國家劇院

<span class="vocabulary">**state**</span> [steɪt]  
<span class="definition">adj.（也可用State，只用於名詞前）由一國政府所提供、控制、資助或組織的：</span>state education（英）公辦教育 / state school（可指英國的）公立學校 / state funding, state subsidy, state support 政府出資；政府補貼；政府扶持 / state benefit（尤英）政府救濟金 / state spending 政府開支 / state employee 政府僱員 

<span class="vocabulary">**public**</span> ['pʌblɪk]  
<span class="definition">adj. 1 只用於名詞前。提供給公眾使用的（尤指由政府提供），如公共的、公立的，與private相反：</span>public education system 公共教育體系 / public library 公共圖書館 / public transportation 公共交通 / public areas of international airports 國際機場裡的公共區域 <span class="definition">2 只用於名詞前。政府的、與政府提供的服務有關的，與private相反：</span>the public purse（英）國庫 / to be in public ownership 歸國有 / public money, public expenditure, public funding 公款；公共開支；政府撥款 / to work in public office 從事政府工作
