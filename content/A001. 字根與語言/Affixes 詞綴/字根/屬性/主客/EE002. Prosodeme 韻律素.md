### Linguistics Units and Terms
(by $\mbox{G}{\ddot{o}\mbox{ran Hammarstr}{\ddot{o}}m}$) 
韻律（prosody）指的是講話的過程中非獨立音段（元音和輔音）的成分，即音節或更大單位的性質。這些性質形成語調、聲調、重讀和等時性等語言功能。

suprasegmental 超音段
intonation 語調, intonation pattern 語調模式
contour (旋律)輪廓, pitch contour 音高輪廓
sonogram 聲波圖
- **prosody** 韻律 (通常語境)
	- **prosode** 韻律 (此處語境)
		- **alloprosode** 韻律變體
			- **prosodeme** 韻律素
				- 同為發聲單位，但不屬phone一類
				- 大致能分成
					- 音節韻律素 (*syllab prosodeme*)
						- 單位定義為syllabeme prosodeme
						- 一個含建構功能的音素 (phoneme)組合 (即語音變體variphone)的韻律素，從而組成音節 (syllable)
						- 分四要素 (以/mi: n/這個音為例):
							- <u>course of pitch 音高過程</u>
								- 中間的syllabeme (即母音i)的音高最高，接著下降
							- <u>course of loudness 音量過程</u>
								- i (中間的母音)最大聲
							- <u>course of quality 音質過程</u>
								- m音還蠻清楚
								- i音最清楚
								- n音不太清楚
							- <u>course of length 音長過程</u>
								- m音長
								- i音最長
								- n音短
						- syllab (音節) 屬於兼性 (可自由變換)，並不拘於固定形式，其中分布於幾個等級 (levels):
							- contoureme 輪廓素
							- idiolectal 個體方言: idioprosodeme 個體方言韻律素
								- = a set of idioprosodes 個體方言韻律實體
									- 衍生
										- <u>idiosyllabeme</u>
										- <u>idioallosyllab</u>
										- <u>idiosyllab</u>
										- <u>idiosyllabememe</u>
										- 依規律往下類推
											- idio-, socio-, dia-, glotto-
											- morph, lex, syntagm
							- sociolectal 社會方言: socioprosodeme 社會方言韻律素
							- dialectal 方言: diaprosodeme 方言韻律素
								- <u>allosyllabeme prosodeme</u> 音節位素變體韻律素 = a set of allosyllab
								- <u>allosyllab prosodeme</u> 音節變體韻律素
									- 要是一組的音節韻律素 (syllab prosodeme)是可以聽得出在音節、字形、詞位和句法上有區別，則該組可稱一個allosyllab prosodeme
									- 一個allosyllab prosodeme可稱為一個音節位素韻律素 (syllabememe prosodeme)的方言韻律變體 (dialloprosode)，也可稱為一個音節韻律素 (syllabeme prosodeme)的韻律變體 (alloprosode)
									- <u>allosyllab</u> 音節變體 = a set of syllabeme
								- <u>varisyllab</u> 變式音節 = a set of syllab
									- 一個varisyllab會有一個變式音節韻律素 (varisyllab prosodeme)且可拆分成多個音節韻律 (syllab prosodemes)
								- <u>pseudosyllab</u> 假音節
									- 聽到和實際的音節數量不對等 (多/少了)
								- <u>metasyllab</u> 後設音節
									- 分析音節本質的單位
					- 形態韻律素 (*morph prosodeme*)
						- 通常morpheme不會有prosodeme (除非真的要探討)
						- <u>morpheme prosodeme</u> 詞素韻律素
							- a set of morph prosode 形態韻律素
							- or a set of varimorph prosodeme 變式形態韻律素
						- <u>varimorph</u> 變式詞素
						- <u>allomorph</u> 同位異形
						- <u>allomorpheme</u> 義基素變體
						- <u>morphememe</u> 義基素
					- 詞位韻律素 (*lex prosodeme*)
						- <u>lexeme</u> = 某單詞的所有實現加總起來
						- <u>lex</u> = 上述其中的任何一種實現
						- <u>varilex</u> 變式詞位
						- <u>varilexeme</u> 變式詞位素
						- <u>lexememe</u> 詞位基
					- 句法韻律素 (*syntagm prosodeme*)
						- <u>syntageme</u> 句法素
							- 一系列出現在句法中的形式 (patterns)
						- <u>sytagm</u> 句法
							- syntax的具體實現
						- <u>period</u> 週期
							- 一段句子的實體 (instance)，相似的週期會合併成一集合
							- periodeme 週期素
								- 一項週期的集合 (a set of periods)
							- alloperiod 同位異期
								- 拆散periodeme的各個單位
								- dialloperiod, trialloperiod... (subsets)
							- varisytagm 變式句法
								- 因應詞位結構和句法位置不同導致創造多種可能性而衍生的單位
							- sytagmeme 句法基
				- <u>pr-glottoprosodeme</u>: 方言型韻律素在韻律基礎上 (pr = prosodemically)相符
				- <u>pr-glottofuncteme</u>: 方言功能 (all functions in dialects)完全相符
				- <u>f-glottofuncteme</u>: 方言功能(f = functemically)完全相符，韻律卻不相符
				- <u>f-glottoprosodeme</u>: 方言功能和韻律皆相符
				- 其中如果以韻律 (prosody)本身探討，可以分為這些基本要素:
					- intonation 語調: voice height
					- accentuation 重音: voice intensity
					- rhythm 節奏: voice length over time
					- prominence 突顯: the emphasis in context, 包括intonation/停頓/其他
					- tempo 語速: voice speed
			- **contoureme** 輪廓素 (尚無定義，姑且稱輪廓contour基本單位)
