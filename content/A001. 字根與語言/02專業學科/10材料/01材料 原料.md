☀ <span class="category">**材料 原料：**</span>
<span class="vocabulary">**material**</span> [mə'tɪərɪəl] 
<span class="definition">n. [C, U]（用於製作…的）材料，即原料：</span>higher raw material costs 更高的原材料成本 / building, construction material 建築材料 / natural, organic material 天然材料；有機材料 / recyclable material 可回收材料 / synthetic material 合成材料 / biological material 生物材料 / industrial material 工業材料 / drawing, writing material 繪畫材料；書寫用品

<span class="vocabulary">**fibre**</span> ['faɪbə]（美fiber）
<span class="definition">n. [C] 纖維（是一種天然或人造的細絲，尤用於製造織物或繩子）：</span>strong fibre 結實的纖維 / natural fibre 天然纖維 / synthetic fibre 合成纖維 / wool fibre 羊毛纖維 / carbon fibre 碳纖維 / glass fibre 玻璃纖維 / fibre optics 纖維光學

<span class="vocabulary">**glass**</span> [ɡlɑːs] 
<span class="definition">n. [U] 玻璃：</span>a sheet of glass 一片玻璃 / clear glass 透明玻璃 / broken glass 碎玻璃 / glass bottle 玻璃瓶 / glass window 玻璃窗
           
<span class="vocabulary">**crystal**</span> [ˈkrɪstl]
<span class="definition">n. [U] 高質量的玻璃，如水晶玻璃：</span>crystal chandelier, crystal vase 水晶玻璃枝形吊燈；水晶玻璃花瓶 / Some of the finest drinking glasses are made from lead crystal. 一些最高檔的酒杯是用鉛晶質玻璃製成的。
           
<span class="vocabulary">**paste**</span> [peɪst]
<span class="definition">n. [U] 用於製作人造寶石（如鑽石）的鉛質玻璃：</span>paste emeralds 鉛玻璃翡翠
           
<span class="vocabulary">**pane**</span> [peɪn]
<span class="definition">n. [C]（窗戶或門上的）一片玻璃：</span>glass pane 窗玻璃 / broken pane 打破的窗玻璃 / window pane 窗戶上的玻璃 / Her face was pressed against the pane. 她的臉貼在窗玻璃上。/ The rain ran down the window pane. 雨水沿著窗玻璃往下流。/ There was dirt on the pane. 那塊玻璃上有灰塵。/ The burglars got in by breaking a pane of glass in a door.竊賊是打碎了門上的一塊玻璃進了屋的。
           
<span class="vocabulary">**porcelain**</span> [ˈpɔ:səlɪn; 美 ˈpɔ:rs-]
<span class="definition">n. [U] 瓷：</span>porcelain bowl, porcelain doll, porcelain figure, porcelain vase瓷碗；瓷娃娃；瓷像；瓷花瓶 / There were lilies everywhere in tall white porcelain vases. 高高的白色瓷瓶上繪滿了百合花。

<span class="vocabulary">**metal**</span> ['metl] 
<span class="definition">n. [C, U] 金屬：</span>a piece of metal 一塊金屬 / pure metal 純金屬 / heavy metal 重金屬 / metal door 金屬門

· 表示“金屬”的詞群見[[04酸鹼鹽 金屬礦物]]

<span class="vocabulary">**plastic**</span> ['plæstɪk, plɑːstɪk] 
<span class="definition">n. [U, C, usually pl.] 塑膠：</span>chairs made from plastic 塑膠椅子 / a sheet of clear plastic 一張透明的塑膠 / the plastic industry 塑膠工業 <span class="definition">adj. 由塑膠製成的：</span>a plastic bag 塑膠袋
           
<span class="vocabulary">**membrane**</span> [ˈmembreɪn]
<span class="definition">n. [C, U]（可起防水、防風等作用的）膜材料：</span>a waterproof membrane 防水薄膜

<span class="vocabulary">**rubber**</span> ['rʌbə] 
<span class="definition">n. [U] 橡膠：</span>a ball made of rubber 皮球 / a rubber tree 橡膠樹

<span class="vocabulary">**wood**</span> [wʊd] 
<span class="definition">n. [U, C] 木頭、（建築等用的）木材、（做燃料的）木柴：</span>dead, rotten wood 枯木；朽木 / pine wood 松木 / wood grain 木材的紋理 / wood floor 木地板
           
<span class="vocabulary">**log**</span> [lɒg; 美 lɔ:g; lɑ:g]
<span class="definition">n. [C] 從樹上切下（或從樹上落下）的一大塊木頭，可稱為原木：</span>The road was blocked by fallen logs. 倒下的原木擋住了路。/ pine logs 松樹原木 / floating logs 漂浮的原木 / a pile of sawn logs 一堆鋸好的木頭 / log cabin 原木小屋 / logs for the fire 燒火用的木材 / log fire 柴火 / burning logs 燃燒的原木 / I added another log to the fire. 我往火裡又加了一塊木柴。/ logs cracking in the fireplace 壁爐裡燒得噼裡啪啦的木頭
           
<span class="vocabulary">**lumber**</span> [ˈlʌmbə(r)]
<span class="definition">n. [U] [尤美；英式尤用timber] 建築等用的木材、木料（是已經制備好，等待使用的木頭）：</span>softwood lumber 軟木木材 / treated lumber 處理過的木材 / scrap lumber 廢木料 / lumber mill 木材廠 / lumber yard 木料場 / lumber camp 伐木場 / lumber company 木材公司 / lumber industry 木材行業
           
<span class="vocabulary">**timber**</span> [ˈtɪmbə(r)]
<span class="definition">n. 1 [U]（用於建築或製作傢俱等物品的）樹木、林木：</span>timber industry 林木業 / timber company 木材公司 / timber production 木材生產 / timber trade 木材貿易 / timber harvest 木材採伐 / timber sale 木材銷售 / timber supply 木材供應 <span class="definition">2 [U] [尤英]（美式英語通常用lumber）建築等用的木材、木料：</span>houses built of timber 木屋 / a single-story timber building 木質平房 / These Severn Valley woods have been exploited for timber since Saxon times. 從撒克遜時代起就開始採伐塞文山谷的這些樹林獲取木料。/ solid timber 實木 / timber structure 木結構 / timber beam 木樑 / timber building 木房屋 / timber floor 木地板 <span class="definition">3 [C, usually pl.]（建造房屋或船隻用的）又大又重的木料：</span>roof timbers 房檁 / a bird nestling in the timbers of the roof 棲息於屋頂橫樑上的鳥 / ship's timber 造船用的木材

<span class="vocabulary">**maple**</span> ['meɪpl] 
<span class="definition">n. [U] 楓木：</span>a maple table 一張楓木桌子

<span class="vocabulary">**pine**</span> [paɪn] 
<span class="definition">n. [U] 松木：</span>a pine table 一張松木桌子

<span class="vocabulary">**ash**</span> [æʃ] 
<span class="definition">n. [U] 梣木：</span>The rafters are made from ash. 這些椽子是用梣木做的。

<span class="vocabulary">**walnut**</span> ['wɔ:lnʌt] 
<span class="definition">n. [U] 核桃樹（或胡桃樹）的木頭，即核桃木，胡桃木：</span>瞭解即可
           
<span class="vocabulary">**cherry**</span> [ˈtʃeri]
<span class="definition">n. [U]（也可用cherrywood）櫻桃木：</span>cherry wood 櫻桃木

<span class="vocabulary">**box**</span> [bɒks] 
<span class="definition">n. [U] 也可用boxwood，指黃楊木：</span>瞭解即可
           
<span class="vocabulary">**cane**</span> [keɪn]
<span class="definition">n. [U] 用於製作傢俱等的竹竿、藤條：</span>cane chair 藤椅 / cane furniture 竹藤傢俱 / cane basket 藤籃

<span class="vocabulary">**bone**</span> [bəʊn] 
<span class="definition">adj.（工具或裝飾品）由骨頭製成的，或骨質的：</span>a bone tool 骨制的工具

<span class="vocabulary">**stone**</span> [stəʊn] 
<span class="definition">n. 1 [U] 常用於名詞前或構成複合詞，指石頭；石料（常用於建築領域）；岩石：</span>stone arch, stone bridge, stone house 石拱；石橋；石屋 / stone tablet 石碑 <span class="definition">2 [C] 通常構成複合詞，指（為某種用途而加工成某種形狀的）石塊（常指石碑）：</span>These words are carved on the stone beside his grave. 在他的墓碑上刻著這樣的話。/ memorial stone 紀念碑 / foundation stone 基石 / stepping stone 墊腳石 / sharpening stone 磨刀石
                      
<span class="vocabulary">**repellent**</span> [rɪˈpelənt]
<span class="definition">n. [U, C]（用於織物、石料等的）防水劑：</span>(a) water repellent 防水劑

<span class="vocabulary">**in**</span> [ɪn] 
<span class="definition">prep. 用於表明使用的材料等：</span>瞭解即可

相關詞義延伸：
· material還可指用以創作書籍、電影、戲劇等所需的素材；
· plastic還可指塑膠學，也可指（某種材料或物質）可以輕易塑造成不同形狀的（即有可塑性的）。

周邊詞彙補充：
· plate可指鍍金（或鍍銀）的常規金屬；
· wrap可指用於包裹某事物的紙張、塑膠等材料；
· cardboard可指硬紙板（常用於製造盒子）。
