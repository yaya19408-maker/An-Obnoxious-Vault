☀ <span class="category">**昆蟲 害蟲：**</span>
<span class="vocabulary">**insect**</span> ['ɪnsekt] 
<span class="definition">n. [C] 昆蟲：</span>flying insect 飛蟲 / aquatic insect 水生昆蟲 / beneficial insect 益蟲 / harmful, poisonous insect 害蟲；有毒昆蟲 / bloodsucking insect 吸血的昆蟲 / insect attack 蟲害 / insect bite 昆蟲咬傷 / insect eggs 昆蟲卵 / insect spray 噴霧殺蟲劑
           
<span class="vocabulary">**bug**</span> [bʌg]
<span class="definition">n. [C] [尤美] 小昆蟲：</span>We noticed tiny bugs that were all over the walls. 我們注意到牆上爬滿了小蟲子。/ a bloodsucking bug which infests poor housing 貧民區滋生的一種吸血的蟲子

<span class="vocabulary">**queen**</span> [kwi:n] 
<span class="definition">n. [C] 為整個群體產卵的一個雌性昆蟲，即王、後：</span>a queen bee 蜂王

<span class="vocabulary">**worm**</span> [wə:m] 
<span class="definition">n. 1 [C] 蠕蟲：</span>The worm was wriggling on the hook. 蠕蟲在鉤子上扭動。<span class="definition">2 [C] 昆蟲的幼蟲（此時看上去像一條短的蠕蟲）：</span>The wood is full of worms. 這塊木頭長滿了蟲子。

<span class="vocabulary">**pest**</span> [pest] 
<span class="definition">n. [C] 對植物、莊稼、水果等有害的昆蟲或動物：</span>agricultural, plant pest 農業害蟲；植物害蟲 / insect pest 害蟲 / pest control 害蟲防治 / pest population 害蟲種群 / pest infestation 蟲害
  
<span class="vocabulary">**vermin**</span> [ˈvɜ:mɪn; 美 ˈvɜ:rmɪn]
<span class="definition">n. [pl.] 破壞植物或莊稼（或襲擊家畜或家禽）的野生動物或鳥類（如攜帶致病菌的老鼠），即害獸、害鳥：</span>On farms the fox is considered vermin and treated as such. 在農場裡狐狸被當成有害動物來對待。/ vermin control 害蟲控制

<span class="vocabulary">**locust**</span> [ˈləʊkəst; 美 ˈloʊ-]
<span class="definition">n. [C] 蝗蟲：</span>A swarm/plague of locusts descended on the countryside. 一大群蝗蟲襲擊了鄉村地區。/ The locusts have destroyed a lot of the crops. 蝗蟲毀壞了大批莊稼。/ locust swarm 蝗蟲群

<span class="vocabulary">**flea**</span> [fli:]
<span class="definition">n. [C] 跳蚤：</span>The dog has fleas. 這條狗有跳蚤。/ This drug is given to dogs to control fleas. 這種藥是用來滅狗身上的跳蚤的。/ flea bite 跳蚤咬的紅斑 / flea collar, flea powder 滅蚤項圈；去蚤粉

<span class="vocabulary">**tick**</span> [tɪk] 
<span class="definition">n. [C] 一種小型的吸血昆蟲（可叮咬並吸食人類和動物的血），該昆蟲有若干種，有些種類可傳播疾病。如蜱：</span>a tick bite 蜱叮咬之處

<span class="vocabulary">**ant**</span> [ænt] 
<span class="definition">n. [C] 螞蟻：</span>ants’ nest 螞蟻窩
          
<span class="vocabulary">**cricket**</span> [ˈkrɪkɪt]
<span class="definition">n. [C] 蟋蟀、蛐蛐兒：</span>the chirping of crickets 蟋蟀的唧唧叫聲 / The only sound was a cricket chirping. 只有蟋蟀唧唧的叫聲。/ A cricket is singing. 一隻蟋蟀在鳴叫。

<span class="vocabulary">**fly**</span> [flaɪ] 
<span class="definition">n. [C] 昆蟲中的蠅、蒼蠅：</span>A fly was buzzing against the window. 一隻蒼蠅貼在窗戶上嗡嗡地叫著。/ fly spray 滅蠅噴霧劑

<span class="vocabulary">**mosquito**</span> [mə'ski:təʊ] 
<span class="definition">n. [C] 蚊子：</span>mosquito bite 蚊子叮咬處 / mosquito net 蚊帳
           
<span class="vocabulary">**jumper**</span> [ˈdʒʌmpə(r)]
<span class="definition">n. [C] 跳躍的一個昆蟲：</span>瞭解即可

相關詞義延伸：
· worm還可表示寄生蟲；
· fly還可指釣魚時被用作誘餌的蒼蠅；
· vermin還可指（動物或人體的）體外寄生蟲（或害蟲）。

周邊詞彙補充：
· vector可指傳播疾病的媒介昆蟲、帶菌生物。

· 表示“寄生蟲”的詞群見[[38寄生蟲 病原蟲]]
