☀ <span class="category">**網路 通訊：**</span>
<span class="vocabulary">**network**</span> ['netwə:k] 
<span class="definition">n. [C] 連線在一起的若干臺計算機或其他裝置（以共享裝置和資訊），即網際網路絡。僅表示一種網路結構（而不表示其它屬性），是一個純技術用語。是本組一個最基礎、最廣義的用詞，包含了各種型別的網路（如區域網、廣域網、全球資訊網、國際網際網路等）：</span>computer network 計算機網路 / communications, mobile network 通訊網路；行動電話網路 / Wi-Fi, wireless network 無線網路 / network engineer 網路工程師 / network address 網址

<span class="vocabulary">**web**</span> [web] 
<span class="definition">n. [sing.] the Web可指計算機的網際網路，也稱全球資訊網（World Wide Web）。在詞義上與Internet沒有明顯區別，不過Internet使用範圍更廣，也更為常用：</span>Web page 網頁 / Web address 網址 / Web browser 網路瀏覽器 / Web network 網路 / I found the information on the Web. 我在網上找到了這條訊息。

<span class="vocabulary">**Internet**</span> ['ɪntənet] 
<span class="definition">n. [sing.] 國際計算機互聯所形成的一個網路，即網際網路（通常寫作the Internet，非正式用法為the Net）。最為常用：</span>I looked it up on the Internet. 我在網際網路上查過此事。/ All the rooms have access to the Internet. 所有的房間均可接入網際網路。/ Internet company, Internet firm 網際網路公司 / Internet industry 網際網路產業 / Internet security 網際網路安全

<span class="vocabulary">**net**</span> [net] 
<span class="definition">n. [sing.] [非正式] the Net 可表示網際網路，是Internet的非正式用法：</span>瞭解即可
           
<span class="vocabulary">**grid**</span> [grɪd]
<span class="definition">n. [sing.] [計算機]（常用the Grid）聯入網路的一些計算機（彼此之間傳遞資料等）：</span>瞭解即可
 
<span class="vocabulary">**website**</span> ['websaɪt] 
<span class="definition">n. [C] 網站：</span>I found this information on their website. 我在他們的網站上發現了這一資訊。/ official website 官方網站 / government, news, travel website 政府網站；新聞網站；旅遊網站
                      
<span class="vocabulary">**site**</span> [saɪt]
<span class="definition">n. [C] [計算機] 網站、站點。相當於website：</span>online dating sites 相親網站 / official site 官方網站 / personal site 個人網站 / home site 主站點 / major news sites 主要新聞網站 / portal site 入口網站 / search site 搜尋網站 / a music download site 音樂下載站點 / business, commercial, corporate, e-commerce site 企業網站；商業網站；公司網站；電子商務網站 / government site 政府網站 / auction, gambling, gaming, media, music, news, newspaper, review, shopping, travel site 拍賣網站；賭博網站；遊戲網站；媒體網站；音樂網站；新聞網站；報紙網站；評論網站；購物網站；旅遊網站 / site design 網站設計 / site content 網站內容 / site link 網站連結 / site administrator 網站管理員 / site visitor 網站訪問者 / At this site you'll find all the latest news and gossip. 在這個網站你會找到所有最新的新聞和八卦訊息。/ The sofeware sell on the site for $30. 這個軟體在網上售價30美元。/ sites about Japan 關於日本的網站 / the most popular sites for children 最受兒童歡迎的網站 / I'm searching for sites on aromatherapy. 我正在搜尋關於芳香療法的網站。

<span class="vocabulary">**protocol**</span> [ˈprəʊtəkɒl; 美 ˈproʊtəkɔ:l; -kɑ:l]
<span class="definition">n. [C] [計算機]（計算機間傳遞資料的）協議：</span>瞭解即可           

<span class="vocabulary">**domain**</span> [dəˈmeɪn; dəʊ-; 美 doʊ-]
<span class="definition">n. [C] [計算機] 網際網路上的域（是一套網址，可以表明一個網址所屬的類別或地理範圍，以相同字母結尾的網址屬於同一個域，如.com、.org等）：</span>Internet domain 網際網路域 / Register a domain name if you want people to find your website. 想要別人找到你的網址，就註冊一個域名。/ domain registration 域名註冊
           
<span class="vocabulary">**alias**</span> [ˈeɪliəs]
<span class="definition">n. [C] [計算機術語]（檔案、網際網路地址等用的）別名，假名字：</span>email, online alias 電子郵件中的化名；網名

<span class="vocabulary">**page**</span> [peɪdӡ] 
<span class="definition">n. [C] 顯示在計算機螢幕上的（資料或資訊的）頁面或版面；網頁：</span>web page 網頁 / home, main page 主頁

<span class="vocabulary">**connect**</span> [kə'nekt] 
<span class="definition">vt.＆vi. connect (sb) (to sth) 將一臺計算機連線到網際網路（或一個計算機網路）中：</span>Click ‘Continue’ to connect to the Internet. 點選 Continue 連線到網際網路。

<span class="vocabulary">**connection**</span> [kə'nekʃən] 
<span class="definition">n. 1 [U, C] connection (to sth)（行為意義上的）連線；（狀態意義上的）處於被連線的狀態：</span>I’m having problems with my Internet connection. 我的網際網路連線有問題。<span class="definition">2 [C] 將兩個部分連線起來的一個點，即連線點，尤指電力系統中的接頭：</span>phone connection 電話接頭 / mobile connection 行動通訊連線 / broadband, Internet, wireless connection 寬頻連線；網際網路連線；無線連線

<span class="vocabulary">**link**</span> [lɪŋk] 
<span class="definition">n. [C] 電子文件中的連結（透過該連結可進入另一個電子文件或連結所在的文件的另一部分）：</span>Internet, text, Web, website link 網際網路連結；文字連結；網路連結；網站連結 / broken link 斷開的連結 <span class="definition">vt. 1 link A to/with B / link A and B (together) 常用被動式，在兩個機器之間建立物理聯絡或電子聯絡。此時相當於connect，二者未見明顯區別，常用度也相近：</span>The video cameras are linked to a powerful computer. 這些攝像機是與一臺效能強大的計算機相連線的。/ The computers are linked into a network. 這些計算機是聯網的。<span class="definition">2 建立電子文件的連結：</span>to use hypertext to link Internet documents 用超文字來連結網際網路文件

<span class="vocabulary">**terminal**</span> ['tə:mɪnl] 
<span class="definition">n. [C]（計算機技術中的）終端或終端機（該終端與一箇中央計算機系統相連）：</span>computer, Internet terminal 計算機終端；網際網路終端 / data terminal 資料終端 / mobile, wireless terminal 移動終端；無線終端 / terminal screen 終端顯示器
           
<span class="vocabulary">**client**</span> [ˈklaɪənt]
<span class="definition">n. [C] [計算機]（連線在伺服器上的）客戶機：</span>瞭解即可

<span class="vocabulary">**host**</span> [həʊst] 
<span class="definition">n. [C] 計算機網路的主機，伺服器（可控制該網路中的其他計算機或向這些計算機傳送訊息）：</span>transferring files from the host to your local computer 從主機向你的本地計算機傳送檔案
           
<span class="vocabulary">**browser**</span> [ˈbraʊzə(r)]
<span class="definition">n. [C] [計算機]（尤指用來在網際網路上搜尋資訊的）瀏覽器：</span>Internet, Web browser 網際網路瀏覽器；網路瀏覽器 / What do you use as your default browser? 你的預設瀏覽器是什麼?/ browser toolbar, browser window 瀏覽器工具欄；瀏覽器視窗 / Click the 'back' button on your browser. 在瀏覽器上點選back（返回）按鈕。
           
<span class="vocabulary">**navigate**</span> [ˈnævɪgeɪt]
<span class="definition">vt.&vi. [計算機]（在網際網路或某個網站上）導航：</span>Customers will be able to navigate easily throughout the site. 顧客將能夠輕鬆查閱網站資訊。/ Browsers are used to navigate through the Web. 瀏覽器是用於網上瀏覽的。

<span class="vocabulary">**navigation**</span> [ˌnævɪˈgeɪʃn]
<span class="definition">n. [U] 網站的導航：</span>a well laid out design and easy navigation 佈局很好的設計和容易操作的網站導航 / Web navigation 網路導航 / Make site navigation easy and intuitive. 使網站導航既簡單又直觀。/ navigation bar, navigation button, navigation menu 網站導航條；網站定位按鈕；網站導航選單 / You can use the navigation bar at the bottom of the pictures to zoom in or out. 你可用圖片下方的導航條來放大或縮小圖片。/ Keep the navigation of your site really simple. 把你的站點導航弄得真正簡單些。/ navigation through complex documents 在各種複雜的檔案中查閱 / A mouse or joystick makes navigation simpler. 滑鼠或遊戲杆使查閱變得更簡單。

<span class="vocabulary">**visit**</span> ['vɪzɪt] 
<span class="definition">vt. 訪問（網際網路上的網站）：</span>For more information, visit our website. 欲知詳情，請訪問我們的網站。<span class="definition">n. [C] 對網際網路網站的訪問：</span>Visits to our website have doubled in a year. 我們網站的訪問人次一年內翻了一番。

<span class="vocabulary">**remote**</span> [rɪ'məʊt] 
<span class="definition">adj.（計算機或系統）遠端的（即可以在較遠的地方透過電子連結來聯接起來的）：</span>remote terminal, remote database 遠端終端；遠端資料庫

<span class="vocabulary">**email**</span> ['i:meɪl] 
<span class="definition">n. 1 [U]（正式寫法為electronic mail）透過計算機網路給別人傳送訊息的一種通訊方式，即電子郵件：</span>junk e-mail 垃圾電子郵件 / e-mail address 電郵地址 / e-mail account 電郵賬戶 <span class="definition">2 [C, U] 用電子郵件所傳送的訊息：</span>瞭解即可
           
<span class="vocabulary">**attachment**</span> [əˈtætʃmənt]
<span class="definition">n. [C] [計算機]（用電子郵件傳送的）附件：</span>When you send an e-mail you can also send a sound or graphic file as an attachment. 在傳送電子郵件時，你也可以附上一份聲音或圖片檔案。/ email attachment 電子郵件附件 / unsolicited attachment 未經請求而傳送的附件 / infected attachment 感染了病毒的附件 / No attachment was included. 沒有附件。
 
<span class="vocabulary">**flame**</span> [fleɪm] 
<span class="definition">n. [C] [非正式] 火藥味的電子郵件（或網際網路資訊）：</span>The best way to respond to a flame is to ignore it. 回應攻擊性電子郵件的最好方式就是置之不理。

<span class="vocabulary">**mailbox**</span> ['meɪlbɒks] 
<span class="definition">n. [C] 電腦上儲存電子郵件的地方，即電子郵箱：</span>瞭解即可

<span class="vocabulary">**account**</span> [ə'kaʊnt] 
<span class="definition">n. [C] 網際網路、電子郵件等的賬戶、賬號：</span>Internet, email account 網際網路賬號；電子郵件賬號

<span class="vocabulary">**address**</span> [ə'dres] 
<span class="definition">n. [C] 用計算機查詢資訊時所使用的一個地址（是一串文字和符號，如網際網路地址）：</span>email address 電子郵件地址 / Internet, Web, website address 網際網路網址；網址；網站地址

<span class="vocabulary">**at**</span> [æt] 
<span class="definition">prep. 可用於電子郵件地址中的符號@：</span>His email address is xxx@xxx.com. 他的電子郵件地址是xxx@xxx.com。

<span class="vocabulary">**live**</span> [laɪv] 
<span class="definition">adj. 常作前置定語，（網際網路的電子連結）有效的，功能正常的，能連線到網際網路上的其他檔案或頁面的：</span>some live links to relevant resources 相關資源的一些有效連結 / The new website is expected to go live in October. 新網站預計十月上線。

<span class="vocabulary">**chat**</span> [tʃæt] 
<span class="definition">vi. 在網際網路上（尤指在網上聊天室中）與某人交談或聊天：</span>He’s been on the computer all morning, chatting with his friends. 他整個上午都在網上和朋友聊天。<span class="definition">n. [C] 網際網路聊天：</span>Internet, live, online chat 網聊；直播聊天；線上聊天

<span class="vocabulary">**publish**</span> ['pʌblɪʃ] 
<span class="definition">vt.（在網際網路上向公眾）發表，公佈：</span>The report will be published on the Internet. 報告將在網際網路上公佈。/ The study was published online. 這項研究發表在網上。

<span class="vocabulary">**post**</span> [pəʊst] 
<span class="definition">n. [C] 網上發的帖子：</span>To respond to your comments, please see my previous post. 對您的評論的答覆，請參閱我以前的帖子。<span class="definition">vt. 常用被動式，指在網際網路上發帖子：</span>Relevant information will be posted on the Internet. 相關資訊將在網際網路上公佈。

<span class="vocabulary">**posting**</span> [ˈpəʊstɪŋ; 美 ˈpoʊ-]
<span class="definition">n. [C] 發到網際網路討論組的一則訊息或資訊，即帖子：</span>Postings on the Internet can be accessed from anywhere in the world. 網際網路上的帖子在世界任何一個地方都能讀到。/ Internet, online posting 網際網路上的帖子；線上帖子 / blog posting 部落格帖子 / online job postings on the website 網站上的線上招聘帖子 / I've made a couple of postings about identity theft. 我發了幾個關於身份盜竊的帖子。/ Have you read my posting? 你讀了我的帖子嗎?/ a posting about cats 關於貓的帖子 / postings on our message board 我們留言板上的帖子

<span class="vocabulary">**thread**</span> [θred] 
<span class="definition">n. [C] 網際網路留言板上由不同的人所發出的一系列相關資訊：</span>瞭解即可
       
<span class="vocabulary">**blog**</span> [blɒg; 美 blɑ:g]
<span class="definition">n. [C] 個人寫的網路日誌，也可叫部落格：</span>When Tom started his blog, his aspirations were small; he simply hoped to communicate with a few people. 湯姆開博時並沒抱多大的期望；他只是希望和一些人聊聊天。/ academic, corporate, political blog 學術部落格；公司部落格；政治部落格 / baseball, food, news blog 棒球部落格；美食部落格；新聞部落格 / I'm making my students keep blogs this semester. 這學期我讓我的學生寫網路日誌。/ blog entry, blog post 部落格目錄；部落格文章 / blog reader 部落格讀者 / blog site 部落格地址 / blog posting 部落格發帖 / a blog about music關於音樂的部落格 / You can read about my trip in my blog. 你可以在我的部落格裡讀到有關我的旅行的帖子。<span class="definition">vi. 寫部落格：</span>瞭解即可

<span class="vocabulary">**email**</span> ['i:meɪl] 
<span class="definition">vt.＆vi. email sth (to sb) 給…發電子郵件：</span>I’ll email the documents to her. 我將用電子郵件把這些檔案發給她。

<span class="vocabulary">**flame**</span> [fleɪm]
