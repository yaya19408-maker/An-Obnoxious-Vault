<span class="vocabulary">**number**</span> ['nʌmbə] 
<span class="definition">n. [C] 代表了一個數量的文字或符號，即數字，相當於figure。是此義的基礎用詞，使用範圍最廣：</span>Think of a number and multiply it by two. 想出一個數，然後乘以二。/ high, low number 高位數；低位數 / three-digit number 三位數 / cardinal, ordinal number 基數；序數 / even, odd number 偶數；奇數 / real, rational number 實數；有理數 / whole, decimal number 整數；小數 / negative, positive number 負數；正數 / random number 隨機數字 / lucky number 幸運數字

<span class="vocabulary">**figure**</span> ['fɪɡə] 
<span class="definition">n. 1 [C, often pl.] 數字，尤指官方資料中的統計數字：</span>official figure 官方數字 / latest figure 最新數字 / inflated figure 誇大的數字 / target figure 目標數字 / attendance, sales, trade figure 出席人數；銷售數字；貿易數字 / the inflation figure 通貨膨脹的數字 / unemployment figure 失業數字 / audience figure（尤英）觀眾數字 / real figure 真實的數字 <span class="definition">2 [C] 0到9（包含0到9）之間的十個個位數字；還可指位數：</span>The figures on the screen read 8:45 a.m. 螢幕上的數字顯示上午8點45分。/ the figure ‘3’ 數字“3” / a six-figure salary 六位數的薪水 / single figures 一位數 / double figures 兩位數 / high figure 高數值
          
<span class="vocabulary">**digit**</span> [ˈdɪdʒɪt]
<span class="definition">n. [C]（從 0 到 9 的任何一個）數字，數位：</span>binary, decimal digit 二進位制數字；十進位制數字 / the decimal digits 0 to 9 從0到9的十進位制數字 / double, single digit 兩位數；個位數 / double-digit inflation 兩位數的通貨膨脹 / a four-digit number 四位數 / The number 57 306 contains five digits. 數字57 306是個五位數。/ a sequence of binary digits 一組二進位制數字 / three digits long 三位數 / The number can be up to eight digits long. 這個數字可達八位數。/ Her telephone number differs from mine by one digit. 她的電話號碼和我的只差一個數字。

<span class="vocabulary">**unit**</span> ['ju:nɪt] 
<span class="definition">n. [C] 個位數（即0-9中的任意一個）：</span>a column for the tens and a column for the units 十位數欄和個位數欄
           
<span class="vocabulary">**cardinal**</span> [ˈkɑ:dɪnl; 美 ˈkɑ:rd-]
<span class="definition">n. [C]（也可用cardinal number）基數、純數（而非序數）：</span>瞭解即可
           
<span class="vocabulary">**odd**</span> [ɒd; 美 ɑ:d]
<span class="definition">adj.（無比較級和最高階）（數字）奇數的、單數的（即不能被2整除的）：</span>1, 3, 5 and 7 are odd numbers. 1、3、5和7是奇數。/ The odd numbers are on the left as you walk up the street. 沿這條街走，在左邊的是單號。/ There's an odd number of candidates. 候選人總數為單數。

<span class="vocabulary">**even**</span> ['i:vn] 
<span class="definition">adj.（數目）可以被2整除的，即偶數的：</span>4, 6, 8, 10 are all even numbers. 4、6、8、10都是偶數。
            
<span class="vocabulary">**mode**</span> [məʊd; 美 moʊd]
<span class="definition">n. [sing.] [數學] 眾數（即一組數字中出現次數最多的數）：</span>瞭解即可          

<span class="vocabulary">**index**</span> [ˈɪndeks]
<span class="definition">n. 1 [C]（物價、工資等的）指數：</span>a general index calculated from death and population information 根據死亡人數和人口資訊計算出來的一個綜合指數 / cost-of-living, market, price index 生活費用指數；市場指數；價格指數 / the consumer price index（尤美）消費者價格指數 / the retail price index（英）零售價格指數 / commodity, futures, share, stock, stock-market index 商品指數；期貨指數；股票指數；股票市場指數 / Dow Jones, FT-SE 100 index 道瓊斯指數；富時100指數 / dollar, sterling index 美元指數；英鎊指數 / Those who lived in the inner cities had a high index of deprivation. 生活在市中心的居民貧困指數很高。/ The retail price index is published monthly. 零售價格指數每月釋出一次。/ A wave of frenzied buying pushed the index up 136.2 points. 一股搶購風潮將指數推高了136.2點。/ Most commodity funds track a specific commodity index. 多數商品基金都會跟蹤某種具體的商品價格指數。/ The NYSE Financial index gained 20%. 紐約證券交易所金融指數上漲了20%。/ The commodities index fell 3.1%. 商品價格指數下降了3.1%。/ an index designed to measure monthly changes in the volume of industrial production 用於衡量工業產量變化的指數 / The Morgan Stanley Cyclical index posted a small advance. 摩根士丹利迴圈指數出現小幅上升。/ an index based on incidents causing a loss of production 以導致產量下降的事件為基礎的指數 / The increase in our standard rates will be linked to the retail price index. 我們的標準價格的漲幅將與零售價格指數掛鉤。/ index-linked pensions（英）與物價指數掛鉤的養老金 / an index covering some 1,700 companies 涵蓋約1,700家公司的指數 / The hundred shares index closed down 15 points. 百種股票綜合指數收盤時下跌了15點。/ dividends on shares in the index 指數成分股的股息 / The company is listed on the Nasdaq technology stocks index. 該公司被列入納斯達克科技股指數成分股。/ a 28.2 point drop in the FT-SE 100 index 富時100指數下跌28.2點 / Inflation, as measured by the retail price index, is expected to drop. 正如零售價格指數所顯示的，通貨膨脹率預期下降。/ body mass index（縮寫為BMI）體重指數 / glycaemic/glycemic index（縮寫為GI）血糖指數 / refractive index 屈光率指數 <span class="definition">2 [usually pl.] [數學] 指數（在等式42 = 16中，2是指數）：</span>瞭解即可

<span class="vocabulary">**factor**</span> [ˈfæktə(r)]
<span class="definition">n. [C] 係數：</span>a suntan lotion with a protection factor of 10 防護係數為10的防曬油 / The wind-chill factor will make it seem colder. 風寒係數大，會使人覺得比實際溫度更冷一些。<span class="definition">vt. [常用被動式] ~ sth (to sth) 將（工資等）與（物價水平等）掛鉤；使…指數化：</span>Minimum pensions and wages are to be indexed to inflation. 最低養老金和最低工資要與通貨膨脹掛鉤。
           
<span class="vocabulary">**infinity**</span> [ɪnˈfɪnəti]
<span class="definition">n. [U, C] [數學]（符號為∞）一個無窮大的數字：</span>As x approaches infinity, y approaches zero. 當x值趨向無窮大時，y值趨近於0。/ These permutations multiply towards infinity. 這些排列可以增加到無窮大。/ There is always an infinity of numbers between any two numbers. 任何兩個數字之間都存在無窮多的數字。/ The usual convention is to choose the reference point at infinity. 通常的做法是在無窮大處取一個參照點。
