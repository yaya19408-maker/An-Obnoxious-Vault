☀ <span class="category">**池塘 水池：**</span>
<span class="vocabulary">**pond**</span> [pɒnd] 
<span class="definition">n. [C] 池塘（比湖要小，且尤指人工池塘）：</span>muddy pond 渾濁的池塘 / backyard pond（美）後院的池塘 / fish, lily pond 魚塘；睡蓮池塘 / pond water 池塘裡的水 / There were some ducks swimming on the pond. 池塘裡有幾隻鴨子在遊動。

<span class="vocabulary">**pool**</span> [pu:l] 
<span class="definition">n. [C] 水池，水塘，水坑等一小片靜止的水域（尤指自然形成的水塘）：</span>muddy pool 泥水坑 / freshwater, saltwater pool 淡水池塘；鹹水池塘 / rock（英）, tide（美）pool 岩石區潮水潭；滿潮湖 / ornamental pool 裝飾性水池 / pools of water 水坑
           
<span class="vocabulary">**reservoir**</span> [ˈrezəvwɑ:(r); 美 ˈrezərv-]
<span class="definition">n. [C]（天然或人工的）水庫、蓄水池：</span>natural reservoir 天然蓄水池 / artificial, man-made reservoir 人工蓄水池；人造蓄水池 / underground reservoir 地下蓄水池 / water reservoir 蓄水池 / large reservoir 大型水庫

<span class="vocabulary">**tank**</span> [tæŋk] 
<span class="definition">n. [C] 在印度英語中可指一個人工挖掘或開鑿的水池、湖泊或水庫、蓄水池：</span>瞭解即可
