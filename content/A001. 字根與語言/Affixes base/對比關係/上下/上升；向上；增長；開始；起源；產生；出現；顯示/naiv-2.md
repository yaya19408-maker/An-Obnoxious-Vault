<font color="#f79646">含義：初始</font>
<font color="#f79646">變體：無</font>
<font color="#f79646">來源：法語單詞naïf（意為natural或ingenuous，即自然的或天然的、天生的）</font>
<font color="#f79646">用途：構成初始相關詞彙</font>

naive T cell 初始T細胞，幼稚T細胞（一種尚未接觸到特定病原體或抗原的T細胞，因此能夠發生初級免疫反應） 
naive immunity 初始免疫力（存在於尚未暴露於特定病原體或抗原的個體中的自然免疫力）
naive virus 初始病毒（一種尚未感染宿主細胞的病毒，因此能夠感染易感宿主）
naive host 初始宿主（尚未接觸過某種病原體或抗原的個體，因此對該病原體或抗原缺乏免疫力）
naive state 初始狀態（對某一特定病原體或抗原沒有事先接觸或免疫的狀態）
naive patient 尚未接觸過某種病原體或抗原的病人（因此對該病原體或抗原缺乏免疫力）
naive infection 發生在尚未接觸過特定病原體或抗原的人身上的感染
naive vaccinee 尚未接種某種特定疫苗的人，因此對該疫苗所針對的病原體或抗原缺乏免疫力
naive herd 尚未接觸過特定病原體或抗原的群體，因此對該病原體或抗原缺乏免疫力
naive antigen 一個個體或群體的免疫系統還沒有遇到的抗原
naive memory cell 一種記憶細胞，尚未透過接觸特定的病原體或抗原而被啟用，因此缺乏進行二次免疫反應的能力
naive lymphocyte pool 存在於尚未暴露於特定病原體或抗原的個體中的淋巴細胞（包括T細胞和B細胞）的總群體
naive specificity 尚未透過接觸特定的病原體或抗原來確定的免疫反應的特異性
